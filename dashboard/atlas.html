Atlas requires the Tauri shell

The Atlas code-structure visualiser runs inside the Tauri desktop shell where IPC access to the Intelligence KB is available. Open this page from within the gsd-skill-creator desktop app to enable all four panes (System Map, Symbol Graph, Mission Archeology, Code View) and the Cmd-K search palette.

Browser-only mode will show this banner. Symbol indexing, snapshot diffing, and live graph updates all require Tauri IPC.
