  3. Execute Wave 0 first (Haiku agent for schemas and source index) before launching Wave 1 parallel tracks.
  4. Run Wave 1 in 3 parallel tracks: CRAFT-PHYS (acoustic + signal processing), CRAFT-ELEC (EM + magnetic), CRAFT-PNW (PNW species + GPU/ML).
  5. Gate at each wave boundary via CAPCOM (HITL interface). Safety-critical tests SC-01 through SC-04 must pass before release.
