# PNW Avian Taxonomy -- Module 8: Compiled Bibliography

> **Module 8 deliverable for the PNW Avian Taxonomy mission ("Wings of the Pacific Northwest")**
>
> This document compiles, deduplicates, and normalizes all sources cited across Modules 00 through 07 of the PNW Avian Taxonomy compendium. Sources are organized by category (Government/Agency, Peer-Reviewed, Professional Organizations, Ethnographic/Cultural) with full bibliographic entries, module cross-references, and source quality verification.
>
> **Source quality standard:** All sources are traceable to government agencies, peer-reviewed journals, professional ornithological organizations, university press publications, or museum/tribal cultural center public materials. No Wikipedia, personal blogs, social media, entertainment media, or anonymous web content.
>
> **Taxonomic authority:** AOS Check-list of North and Middle American Birds, 7th edition through 62nd Supplement.

## How to Use This Bibliography

Source codes are used throughout all modules for inline citation. The format is `[CODE]` (e.g., `[G1]`, `[P13]`, `[O1]`, `[C9]`). Where a source was introduced by a specific module, the "First cited" column indicates origin. Sources from `00-source-index.md` are marked "Base" for first citation.

**Note on code collisions:** Some modules independently assigned the same P-codes to different sources (e.g., Module 1c uses P21 for Koenig & Mumme 1987, while Module 2a uses P21 for Buchanan 2006; Module 4 uses P21 for Benkman 1993/2003). These collisions reflect parallel module development. In this compiled bibliography, each entry is listed under its module of origin with the code as used in that module. The "Modules citing" column clarifies scope.
