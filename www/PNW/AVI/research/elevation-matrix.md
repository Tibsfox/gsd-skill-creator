## Matrix Structure

### Columns: 11 Canonical Ecoregion Bands

| ID | Zone | Elevation/Depth |
|----|------|----------------|
| ELEV-ALPINE | Alpine/Glacial | 6,500+ ft |
| ELEV-SUBALPINE | Subalpine Forest | 4,500–6,500 ft |
| ELEV-MONTANE | Montane Forest | 2,000–4,500 ft |
| ELEV-LOWLAND | Western Foothills/Valley | 500–2,000 ft |
| ELEV-RIPARIAN | Riparian Corridors | Cross-cutting |
| ELEV-COASTAL | Coastal Influence | Sea level–500 ft |
| ELEV-SHRUB-STEPPE | East Side/Rain Shadow | Varies |
| ELEV-SUBTERRANEAN | Fossorial/Cave | Cross-cutting |
| ELEV-INTERTIDAL | Tidal Zone | Sea level |
| ELEV-SHALLOW-MARINE | Nearshore | 0–200m depth |
| ELEV-DEEP-MARINE | Pelagic/Offshore | 200m+ depth |

### Rows: Species (by Order)

Organized by taxonomic order. Each species gets one row.

### Value Vocabulary

| Code | Meaning |
|------|---------|
| **P** | Primary habitat — breeds and/or resides year-round |
| **S** | Secondary/seasonal — uses during migration, winter, or foraging |
| **V** | Vagrant/occasional — documented but not regular |
| **X** | Absent — not documented in this zone |

## Temporal Layer (Wave 2 Extension)

The temporal layer adds 12 monthly presence vectors to each cell. Each cell becomes:

```
value: P|S|V|X
temporal: [Jan, Feb, Mar, Apr, May, Jun, Jul, Aug, Sep, Oct, Nov, Dec]
  where each month = 0 (absent), 0.5 (possible), 1 (present)
```

Example — Rufous Hummingbird at ELEV-LOWLAND:
```
value: S
temporal: [0, 0, 0.5, 1, 1, 1, 1, 1, 0.5, 0, 0, 0]
```

The temporal layer is produced in Wave 2 by MATRIX-TEMPORAL agent, after migratory species profiles are complete.
