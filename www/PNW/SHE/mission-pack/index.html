Usage

  1. Download the Mission Package PDF — this is the complete, self-contained document.
  2. To execute via GSD skill-creator, feed the PDF or .tex source to the orchestrator with: gsd mission execute smart-home-diy-electronics-mission.pdf
  3. The mission will activate a Squadron profile with 12 agents across 4 waves.
  4. Wave 1 runs two parallel tracks (Hardware Foundations + Software & Connectivity) to maximize throughput.
  5. Wave 2 synthesizes 36 projects across 6 tiers, with Opus handling capstone projects and safety analysis.
  6. Human checkpoints (CAPCOM) occur at each wave boundary.

Recompilation

  1. Download the .tex source file
  2. Ensure DejaVu fonts are installed: sudo apt install fonts-dejavu
  3. Compile with XeLaTeX (three passes for cross-references):
    xelatex mission.tex && xelatex mission.tex && xelatex mission.tex
