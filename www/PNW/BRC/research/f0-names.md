# F0-NAMES — PNW Naming Glossary

**Wave:** 0 (Foundation)
**Status:** COMPLETE
**Consumed by:** All Wave 1A, 1B, and Wave 2 components (highest-reuse artifact — cache first)

---

All Virtual Black Rock City component names are drawn from Pacific Northwest bioregion taxonomy. This glossary is the single authoritative source for name→species→source mappings. No component may use a PNW name not listed here.

## Naming Rules

1. All names sourced from **ecological taxonomy** — never Indigenous cultural names (Safety: cultural appropriation boundary)
2. Names reference species, landforms, or ecological processes documented in tibsfox.com/PNW research
3. Each name includes its **source taxonomy** (COL, CAS, ECO, GDN, AVI, MAM, or GEO)
4. Compound names use title case with possessive or descriptive form (e.g., "Raven's Welcome", "Tide Pool Medical")
