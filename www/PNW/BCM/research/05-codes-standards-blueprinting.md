> **Abstract:** This module maps the complete building code landscape governing construction in Oregon and Washington, with detailed coverage of code adoption mechanisms, state amendment processes, blueprint reading conventions, permit workflows, and inspection sequences. The Pacific Northwest operates under two distinct state code systems — Oregon's Building Codes Division (BCD) administering specialty codes per ORS 455.148 [GOV-01], and Washington's State Building Code Council (SBCC) adopting codes via WAC 51-xx per RCW 19.27 [GOV-02]. Both states adopt International Code Council (ICC) model codes with state-specific amendments, but differ in adoption cycles, amendment authority, and administrative structure. Portland and Seattle add local amendments that exceed state minimums in seismic and fire safety [GOV-03, GOV-04]. This document serves all five audience levels with progressive depth, from homeowner permit awareness through engineering-level code analysis and ABET curriculum alignment.

## Part 1: The PNW Code System — How It Works

### L1 View — What Homeowners Need to Know

Building codes are the rules that govern how buildings are constructed, modified, and maintained. Think of them as the recipe book for safe construction — they exist because decades of experience (and failures) have taught us what works and what does not. Every requirement in the code traces back to a real problem that hurt real people [CODE-01].

In the Pacific Northwest, building codes are not optional. They are state law. When you build, remodel, or repair your home, the work must meet the code in effect at the time the permit is issued. The code protects you, your family, and your neighbors.

> **PNW Regional Note:**
> The Pacific Northwest has unique hazards that make building codes especially important here. The Cascadia Subduction Zone (CSZ) is capable of producing magnitude 9.0+ earthquakes — the last one struck in 1700 [CODE-04, GOV-05]. PNW codes include seismic provisions that go beyond what many other regions require. Our marine climate brings 37-60+ inches of rain annually in western Oregon and Washington, making moisture management critical. And our volcanic soils and steep terrain add landslide and erosion risks that codes must address.
>
> *Applies to: Both OR and WA*

**How codes reach your project:**

1. **Model codes** are written by national organizations (ICC, NFPA, IAPMO) [CODE-01, CODE-02, CODE-03]
2. **State agencies** adopt those model codes with state-specific changes (amendments)
3. **Local jurisdictions** (your city or county) may add further local requirements
4. **Your building department** enforces the codes through permits and inspections

You do not need to read the codes yourself. Your contractor, architect, or engineer handles compliance. But understanding that codes exist — and that permits are how compliance is verified — helps you make informed decisions about your home.

### L3 View — How Code Adoption Works

The code adoption pipeline in the PNW follows a structured path from model code development to local enforcement. Understanding this pipeline is essential for trade students because it explains why the same building in Portland and Seattle may be built to different standards, even though both states start from the same ICC model codes [CODE-01].

**Model Code Development Cycle:**

The ICC publishes new editions of the IBC, IRC, IMC, IFC, and related codes on a 3-year cycle [CODE-01]. NFPA publishes the NEC on a 3-year cycle [CODE-02]. IAPMO publishes the UPC on a 3-year cycle [CODE-03]. These cycles are staggered — a new IBC might arrive in 2024, while the NEC 2023 arrived the year before.

**State Adoption Lag:**

Neither Oregon nor Washington adopts model codes immediately upon publication. Each state has a review process that typically adds 1-3 years between model code publication and state adoption. During this gap, the state operates under the previous edition.

| Step | Oregon (BCD) | Washington (SBCC) |
|------|-------------|-------------------|
| Model code published | ICC publishes (e.g., IBC 2024) | ICC publishes (e.g., IBC 2024) |
| State review begins | BCD convenes advisory boards per ORS 455.148 | SBCC opens WAC rule-making per RCW 19.27 |
| Public comment period | Required by Oregon APA | Required by Washington APA |
| State amendments drafted | Oregon-specific amendments integrated | WAC amendments drafted |
| Adoption vote/filing | BCD Director files with Secretary of State | SBCC votes to adopt |
| Effective date | Published date (e.g., Oct 1, 2025) | Published date (e.g., Mar 15, 2024) |
| Mandatory date | 6-month phase-in for technical provisions | Same as effective date |
| Typical lag from model code | 1-2 years | 2-3 years |

**Source:** [GOV-01] for Oregon process, [GOV-02] for Washington process.

> **Note:**
> Code adoption timelines are not fixed. Washington's 2024 code cycle has been delayed significantly — final adoption is now scheduled for August 21, 2026, with an effective date of May 3, 2027 [GOV-02]. This means Washington will operate under 2021 IBC/IRC editions until mid-2027. Always verify the currently enforced code edition with the local building department before starting any project.

## Part 2: Oregon Code Structure

Oregon administers building codes through the Building Codes Division (BCD) of the Department of Consumer and Business Services, per ORS 455.148 [GOV-01]. Oregon uses "specialty code" nomenclature — each trade discipline has its own named code that incorporates the relevant ICC or NFPA model code with Oregon amendments.

| Oregon Residential Specialty Code | ORSC | IRC 2021 | 2023 | Oct 1, 2023 | Apr 2024 | STD-02 |
| Oregon Electrical Specialty Code | OESC | NEC 2023 (NFPA 70) | 2023 | Oct 1, 2023 | Oct 1, 2023 | STD-03 |

| ORSC R327 (Wildfire-Urban Interface) | R327 | ORSC amendment | 2025 | Aug 5, 2025 | Aug 5, 2025 | STD-08 |

### Oregon Key Amendments and Characteristics

**OSSC 2025 (Structural) [STD-01]:**
The 2025 OSSC adopts IBC 2024 as its base, a significant jump from the previous OSSC which used IBC 2021. The six-month phase-in for technical provisions means that between October 2025 and April 2026, projects permitted under the old edition may continue under those requirements. After April 2026, the IBC 2024 technical provisions are mandatory. Oregon integrates the IEBC 2024 and IFC 2024 into the OSSC package, giving Oregon a unified structural/existing building/fire code document.

Key provisions for PNW practice:
- Seismic design per ASCE 7-22, referenced by IBC 2024 Chapter 16 [CODE-04]
- Cascadia Subduction Zone design parameters: Seismic Design Category D for most of western Oregon
- ASCE 7-22 risk-targeted maximum considered earthquake (MCER) ground motion maps
- See [M1-ST:Seismic Design] for structural system requirements

**ORSC 2023 (Residential) [STD-02]:**
Based on IRC 2021, the ORSC governs one- and two-family dwellings and townhouses not more than three stories in height. The IRC/ORSC provides a prescriptive path for residential construction that does not require engineering analysis for most standard configurations.

Key provisions:
- Foundation requirements: Section R403 (footings), R404 (foundation walls)
- Wall bracing: Section R602.10 — prescriptive bracing methods for lateral (seismic and wind) resistance
- Roof framing: Section R802 — conventional framing provisions
- See [M1-ST:Residential Framing] for structural details
- See [M4-BE:Moisture Management] for envelope requirements per R703

**OESC 2023 (Electrical) [STD-03]:**
Full adoption of NEC 2023 with Oregon amendments. Key 2023 cycle changes now in effect in Oregon:
- Section 210.8(A)(6): Expanded GFCI protection — all kitchen receptacles in dwelling units [CODE-02]
- Section 230.67: Surge protection device (SPD) requirements for dwelling units, minimum 10kA nominal discharge [CODE-02]
- Section 230.85: Emergency disconnect requirements for one- and two-family dwellings [CODE-02]
- Article 625: Updated EV charging infrastructure provisions [CODE-02]
- Article 690: Updated solar PV system requirements [CODE-02]
- Article 706: Energy storage system requirements [CODE-02]
- See [M2-EL:Service Calculations] for detailed electrical code analysis

**OPSC 2023 (Plumbing) [STD-04]:**
Based on UPC 2021 with Oregon amendments. Oregon maintains its own plumbing specialty code rather than adopting the UPC directly through a different administrative path.
- See [M3-PM:Plumbing Systems] for detailed plumbing code analysis

**OMSC 2022 (Mechanical) [STD-05]:**
Based on IMC with Oregon amendments. Governs HVAC, refrigeration, and fuel gas installations.
- See [M3-PM:Mechanical Systems] for detailed mechanical code analysis

**OEESC 2025 (Energy) [STD-07]:**
Based on ASHRAE 90.1-2022. Oregon's energy code applies to both commercial and residential buildings, though residential compliance can also be achieved through the IRC energy provisions incorporated into the ORSC.
- Climate zones in Oregon: 4C (western/marine) and 5B (eastern/dry)
- Wall insulation: R-20 to R-21+5ci depending on climate zone and framing type
- Roof/ceiling insulation: R-38 to R-60
- See [M4-BE:Energy Code Compliance] for insulation and air sealing requirements

**ORSC R327 (Wildfire-Urban Interface) [STD-08]:**
Effective August 5, 2025. Applies to residential construction in designated WUI zones.
- Class A roof coverings required
- Ember-resistant venting required
- Ignition-resistant exterior materials
- Defensible space provisions
- See [M4-BE:WUI Provisions] for exterior material requirements

## Part 3: Washington Code Structure

Washington administers building codes through the State Building Code Council (SBCC), established by RCW 19.27 [GOV-02]. Unlike Oregon's specialty code system, Washington adopts ICC model codes through the Washington Administrative Code (WAC) chapter system, with state amendments integrated into each WAC chapter.

### Washington Code Inventory

| WAC Chapter | Code | Base Model Code | Edition | Effective Date | Source ID |
|------------|------|-----------------|---------|---------------|-----------|
| WAC 51-50 | Washington State Building Code | IBC 2021 | 2021 | Mar 15, 2024 | STD-09 |
| WAC 51-51 | Washington State Residential Code | IRC 2021 | 2021 | Mar 15, 2024 | STD-10 |
| WAC 51-52 | Washington State Mechanical Code | IMC 2021 | 2021 | Mar 15, 2024 | STD-11 |
| WAC 51-54A | Washington State Fire Code | IFC 2021 | 2021 | Mar 15, 2024 | STD-12 |
| WAC 51-55 | Wildland-Urban Interface Code | IWUIC | 2021 | Mar 15, 2024 | STD-13 |
| WAC 51-56 | Washington State Plumbing Code | UPC 2021 | 2021 | Mar 15, 2024 | STD-14 |
| — | Washington State Energy Code — Residential | WSEC-R 2021 | 2021 | Mar 15, 2024 | STD-15 |
| — | Washington Electrical Code | NEC 2023 | 2023 | State-specific | GOV-08 |

**Source:** Washington SBCC [GOV-02], Washington L&I [GOV-08].

> **Note:**
> Washington's electrical code is administered by the Department of Labor & Industries (L&I) [GOV-08], not by the SBCC. This is a significant administrative difference from Oregon, where BCD administers all specialty codes including electrical. In Washington, electrical permits, inspections, and code interpretations go through L&I, while all other building code matters go through the local building department operating under SBCC-adopted codes.

### Washington Code Precedence Rule

Washington establishes a formal precedence order for conflicts between codes [GOV-02]:

**IBC > IRC > IMC > IFC > WUI > UPC**

When two codes contain conflicting provisions, the code listed earlier in this sequence prevails. This precedence rule is codified and applies statewide.

### Washington Local Amendment Rules

Washington imposes restrictions on local amendments that Oregon does not [GOV-02, RCW 19.27.074]:

1. **Floor rule:** Local amendments may not reduce minimum performance standards established by state code
2. **Residential amendment review:** Amendments affecting single- or multi-family residential buildings must be reviewed and approved by the SBCC before taking effect
3. **Application:** Local jurisdictions may adopt more restrictive requirements (exceeding state minimums) but cannot weaken state code provisions

### Washington Key Amendments and Characteristics

**WAC 51-50 (IBC) [STD-09]:**
Washington's adoption of the 2021 IBC took effect March 15, 2024. The state integrates amendments directly into the WAC text. Key characteristics:
- Seismic design per ASCE 7-22 as referenced by IBC 2021 Chapter 16 [CODE-04]
- Western Washington: primarily Seismic Design Category D
- Eastern Washington: primarily Seismic Design Category B/C
- See [M1-ST:Seismic Design] for structural requirements

**WAC 51-51 (IRC) [STD-10]:**
Washington's adoption of the 2021 IRC. Same base edition as Oregon's ORSC 2023, but with different state amendments.
- See [M1-ST:Residential Framing] for structural provisions

**WAC 51-56 (UPC) [STD-14]:**
Washington adopts UPC 2021 with significant state amendments:
- Chapters 12 and 14: Not adopted [CODE-03]
- Chapter 5 (appliance venting/combustion air): Not adopted [CODE-03]
- Building sewer provisions: Handled separately from the UPC
- Residential lavatory faucets: Maximum 1.2 GPM at 60 PSI (state amendment)
- Public-use faucets: Maximum 0.5 GPM (state amendment)
- See [M3-PM:Plumbing Systems] for detailed plumbing analysis

**WSEC-R 2021 (Energy — Residential) [STD-15]:**
Washington's residential energy code. Climate zones match Oregon: 4C (western/marine) and 5B (eastern/dry).
- See [M4-BE:Energy Code Compliance] for insulation and air sealing requirements

### Washington 2024 Code Cycle Delay

> **Note:**
> The 2024 Washington code cycle has been significantly delayed [GOV-02]:
>
> - **Original timeline:** Adoption expected 2025
> - **Revised timeline:** Final adoption August 21, 2026; effective date May 3, 2027
> - **Reason:** Extended SBCC review process per RCW 19.27.074
> - **Impact:** Washington continues operating under 2021 IBC/IRC/IMC/IFC editions until May 2027
> - **Planning consideration:** Projects permitted before May 2027 will be built to 2021 edition codes. Projects permitted after May 2027 will need to comply with the new editions. Designers and contractors should monitor the SBCC adoption schedule for the transition date.

## Part 4: Code Relationship Map

Understanding how building codes relate to each other is critical for every audience level. The codes are not independent documents — they form an interconnected regulatory system where one code frequently references provisions in another.

### IBC and IRC Threshold

The boundary between the International Building Code (IBC) and the International Residential Code (IRC) is the most fundamental code relationship [CODE-01]:

| Characteristic | IRC Applies | IBC Applies |
|---------------|------------|------------|
| Dwelling units | 1-2 family dwellings, townhouses | 3+ dwelling units (apartments, condos) |
| Stories | 3 or fewer stories above grade | 4+ stories, or any height beyond IRC scope |
| Occupancy | Residential (R-3, R-4) | All other occupancy types, and R-2 |
| Design approach | Prescriptive (no engineer required for standard configs) | Engineered design (PE/SE typically required) |

**Source:** [CODE-01] IBC Section 101.2 and IRC Section R101.2.

> **PNW Regional Note:**
> In seismically active areas of the PNW, even IRC-compliant residential construction benefits from engineering review. The prescriptive bracing provisions of IRC Section R602.10 were developed for general nationwide application. In Seismic Design Category D (most of western Oregon and Washington), the prescriptive minimums represent the floor, not the optimum. Many PNW jurisdictions encourage or require additional engineering review for homes on hillsides, in liquefaction zones, or on soft soils [GOV-05, CODE-04].
>
> *Applies to: Both OR and WA*

### Code Cross-Reference Map

| Code | Primary Domain | Cross-References To | Relationship |
|------|---------------|--------------------|----|
| IBC [CODE-01] | Commercial/multi-family structural, fire, means of egress | ASCE 7 (loads), NEC (electrical), IMC (mechanical), IFC (fire), UPC (plumbing) | Master code for non-residential and large residential |
| IRC [CODE-01] | Residential structural, mechanical, plumbing, electrical | IBC (for conditions exceeding IRC scope), NEC (electrical details), energy code | Self-contained residential code with trade chapters |
| NEC [CODE-02] | Electrical installations | IBC/IRC (occupancy, construction type), NFPA 72 (fire alarm), NFPA 13 (sprinklers) | Standalone electrical; separate adoption path in WA |
| UPC [CODE-03] | Plumbing installations | IBC/IRC (occupancy, fixture counts), IMC (mechanical interactions) | Standalone plumbing; OR has OPSC, WA has WAC 51-56 |
| IMC [CODE-01] | Mechanical/HVAC | IBC (fire dampers, shaft requirements), energy code (efficiency), IFC (fuel gas) | Mechanical systems including ventilation |
| IEBC [CODE-01] | Existing building alterations | IBC (current code baseline), ASCE 41 (seismic evaluation) | Alteration levels, change of occupancy triggers |
| IFC [CODE-01] | Fire safety operations | IBC (construction type, fire ratings), NEC (fire alarm circuits), sprinkler standards | Fire prevention and operational requirements |
| ASCE 7 [CODE-04] | Structural loads | IBC (adoption by reference), ASCE 41 (existing buildings) | Seismic, wind, snow, rain, flood load definitions |
| ASCE 41 [CODE-04] | Existing building seismic evaluation | ASCE 7 (ground motion), IEBC (alteration triggers), Portland Ch. 24.85 | Tier 1/2/3 seismic evaluation methodology |

### NEC Adoption Path Difference

A key administrative distinction between Oregon and Washington that trade students and contractors must understand [GOV-01, GOV-08]:

| Aspect | Oregon | Washington |
|--------|--------|-----------|
| Electrical code | OESC (adopted by BCD) | NEC (adopted by L&I) |
| Adopting authority | BCD per ORS 455.148 | Dept of L&I (not SBCC) |
| Permit authority | Local building department | L&I or local jurisdiction with L&I delegation |
| Inspector authority | BCD-certified inspectors | L&I electrical inspectors |
| Amendment process | BCD advisory board | L&I rule-making |
| Current edition | NEC 2023 [STD-03] | NEC 2023 [GOV-08] |

**Practical impact:** In Washington, electrical permits and inspections are obtained from a different agency than building permits. A residential project in Seattle requires a building permit from SDCI [GOV-04] and an electrical permit from L&I [GOV-08]. In Oregon, all permits come through the same local building department operating under BCD authority [GOV-01].

## Part 5: Local Amendments — Portland and Seattle

Both Portland and Seattle maintain local amendments that exceed state minimum codes. These local provisions are significant for PNW practice because they apply in the two largest metro areas in the region.

### Portland Local Amendments

**Portland Bureau of Development Services (BDS) [GOV-03]:**

**Title 24 Building Regulations [STD-17]:**
Portland's local building code provisions, including adoption of IBC 2021 Chapters 32 and 33 to close regulatory gaps not addressed by the state code.

**Chapter 24.85 — Seismic Design Standards for Existing Buildings [STD-16]:**
Portland's most significant local amendment. This chapter mandates seismic evaluation and retrofit for existing buildings undergoing change of use, occupancy changes, or significant alterations.

| Trigger | Requirement | Standard | Source |
|---------|-------------|----------|--------|
| Change of use or occupancy | ASCE 41 seismic evaluation required | Tier 1 and Tier 2 deficiency-based evaluation | [STD-16, CODE-04] |
| Significant structural alteration | ASCE 41 evaluation and possible retrofit | Basic Performance Objective for Existing Buildings (BPOE) | [STD-16, CODE-04] |
| Voluntary seismic retrofit | Prescriptive retrofit program available for residential | Portland prescriptive methods | [GOV-03] |

**Portland Prescriptive Seismic Retrofit Program [GOV-03, GOV-05]:**

For residential buildings, Portland offers a prescriptive retrofit path that does not require engineering analysis:

| Component | Specification |
|-----------|--------------|
| Foundation anchoring | Mudsill bolting to foundation |
| Cripple wall bracing material | 1/2-inch CDX five-ply plywood |
| Plywood coverage — 1-story home | 40% of cripple wall area |
| Plywood coverage — 2-story home | 50% of cripple wall area |
| Plywood coverage — 3-story home | 80% of cripple wall area |

**Historical context:** Pre-1974 Oregon homes were not required to be anchored to their foundations. Pre-1993 homes were built before seismic code requirements were added to the Oregon building code. These homes are the primary targets for seismic retrofit [GOV-05, STD-16].

> **BLOCK — Licensed Professional Required**
>
> Seismic evaluation under ASCE 41 requires a licensed structural engineer (SE). The evaluation methodology involves complex analysis of building response to earthquake ground motions. Prescriptive residential retrofits (foundation bolting, cripple wall bracing) may be performed by a licensed contractor without engineering, but only within the scope of the Portland prescriptive program.
>
> **Required professional:** Licensed Structural Engineer (SE) for ASCE 41 evaluations; Licensed General Contractor for prescriptive residential retrofits
> **Why:** Incorrect seismic evaluation or retrofit can create a false sense of security or cause structural failure during an earthquake
> **Code basis:** Portland City Code Ch. 24.85 [STD-16], ASCE 41-17 [CODE-04]

### Seattle Local Amendments

**Seattle Department of Construction & Inspections (SDCI) [GOV-04]:**

Seattle maintains its own amendments to the Washington State Building Code. Key provisions:

**Seattle Prescriptive Earthquake Retrofit Program [GOV-04]:**
Seattle's residential seismic strengthening program follows the ABC method:

| Step | Action | Description |
|------|--------|-------------|
| **A** | **Anchor** to foundation | Bolt the mudsill to the concrete foundation |
| **B** | **Brace** cripple walls | Sheathe short stud walls between foundation and first floor with structural plywood |
| **C** | **Connect** to first floor framing | Ensure positive connection between cripple wall top plate and first floor framing |

**Source:** [GOV-04] SDCI prescriptive seismic retrofit guidelines.

## Part 6: Blueprint Reading

### L1 View — Understanding Your Home's Plans

When you get a permit or hire a contractor, your project will involve drawings (plans). You do not need to be able to create plans, but understanding what you are looking at helps you communicate with your contractor and verify that the work matches what was approved.

**Types of drawings you may encounter:**

| Drawing Type | What It Shows | Analogy |
|-------------|---------------|---------|
| Floor plan | Your home from above, as if the roof were removed | Looking down into a dollhouse |
| Elevation | Your home from the outside, one side at a time | A photograph of each side of your house |
| Section | Your home sliced vertically, showing the interior layers | Cutting a layer cake and looking at the layers |
| Detail | A zoomed-in view of one specific assembly | Using a magnifying glass on one small part |
| Site plan | Your property from above, showing the building, driveway, setbacks, and lot lines | An aerial photo of your yard with measurements |

### L2 View — Reading Plans for DIY Projects

For skilled DIYers working on permitted projects, plan reading is a practical skill. You need to verify dimensions, understand material callouts, and match what is on paper to what you see on site.

**Essential plan reading skills:**

**Scale:** Every drawing has a scale notation (e.g., 1/4" = 1'-0"). This means one quarter inch on paper equals one foot in real life. A standard architectural scale ruler is the essential tool for verifying dimensions on printed plans [PRO-02].

**Line weight hierarchy:**

| Line Weight | Meaning | What It Represents |
|------------|---------|-------------------|
| Heavy (thick) | Cut lines | Where the imaginary cutting plane slices through the building |
| Medium | Visible edges | Edges and surfaces you can see from the viewing direction |
| Light (thin) | Hidden/projection lines | Edges hidden behind other surfaces, or projection/dimension lines |
| Dashed | Hidden elements | Items below the cut plane or behind visible surfaces |
| Center line (dash-dot) | Centerlines | Axes of symmetry, center of columns, center of openings |

**Dimensioning conventions:**

| Project Type | Dimensioning System | Example |
|-------------|--------------------|---------|
| Residential | Feet and inches | 12'-6" |
| Commercial/structural | Feet and inches or decimal feet | 12.50' or 12'-6" |
| Civil/site work | Decimal feet | 125.50' |
| Engineering calculations | Decimal feet or inches | 150.0" |

> **Note:**
> Dimensions on plans are nominal (the number on the drawing), not actual measured on-site dimensions. Construction tolerances mean the as-built condition will vary slightly from the plan dimension. Acceptable tolerances depend on the work type — concrete placement has different tolerances than finish carpentry. When in doubt, the dimension on the approved plan governs.

**Title block information:**
Every sheet of construction drawings includes a title block (usually in the lower right corner) containing:
- Project name and address
- Drawing title and sheet number
- Scale
- Date of drawing and any revision dates
- Name and license number of the design professional (architect, engineer)
- Professional stamp/seal (when required — see Part 9)

### L3 View — Drawing Standards and Symbol Conventions

Trade students must be fluent in the drawing standards that govern how construction documents are prepared. These standards derive from architectural graphic standards developed by professional organizations [PRO-02] and are consistent across most of the United States.

**Drawing sheet organization:**

| Sheet Prefix | Discipline | Examples |
|-------------|-----------|---------|
| A | Architectural | A1.1 Floor Plan, A2.1 Elevations, A3.1 Sections |
| S | Structural | S1.1 Foundation Plan, S2.1 Framing Plan |
| E | Electrical | E1.1 Lighting Plan, E2.1 Power Plan |
| P | Plumbing | P1.1 Plumbing Plan, P2.1 Riser Diagram |
| M | Mechanical | M1.1 HVAC Plan, M2.1 Duct Layout |
| C | Civil | C1.1 Site Plan, C2.1 Grading Plan |
| L | Landscape | L1.1 Landscape Plan |
| G | General | G0.1 Cover Sheet, G0.2 Code Summary, G0.3 Abbreviations |

**Standard architectural symbols (common to PNW practice):**

| Symbol | Meaning | Found On |
|--------|---------|----------|
| Circle with number | Section cut reference (number = detail, number below = sheet) | Floor plans |
| Triangle with number | Elevation reference | Floor plans |
| Rectangle with diagonal | Window tag (type/size reference to schedule) | Floor plans |
| Circle with cross | Column grid intersection | Structural plans |
| Half-filled circle | Electrical panel | Electrical plans |
| Arc at door opening | Door swing direction | Floor plans |
| Dashed rectangle | Equipment or item above cut plane | Floor plans |

**Drawing coordination:**
On any project involving multiple trades, the architectural plans serve as the base for all other disciplines. Structural, electrical, plumbing, and mechanical plans are drawn on top of (or referencing) the architectural floor plan. When discrepancies exist between disciplines, the following hierarchy generally applies:

1. Structural plans govern structure
2. Architectural plans govern dimensions and layout
3. MEP (Mechanical/Electrical/Plumbing) plans govern system routing
4. Specifications govern material and installation requirements
5. Details govern assembly methods

When conflicts arise between plans and specifications, the specification typically governs (because it is more specific). When conflicts arise between general notes and details, the detail typically governs. These precedence rules should be documented in the project's General Requirements (Division 01 of the specifications).

### L5 View — Professional Drawing Standards

**Drawing standards per professional licensing requirements:**

Licensed architects and engineers in Oregon and Washington are required to prepare construction documents that meet professional standards of care [PRO-02]. Key requirements:

| Requirement | Oregon | Washington | Source |
|-------------|--------|-----------|--------|
| Professional stamp on documents | Required for work within scope of license | Required for work within scope of license | State licensing boards |
| Digital signatures | Accepted with secure certificate | Accepted with secure certificate | State licensing boards |

## Part 7: Permit Processes

### L1 View — Do I Need a Permit?

> **GATE — Permit Verification (SAFETY-CRITICAL)**
>
> Before starting any construction, remodeling, or repair project, confirm:
> - [ ] You have determined whether a permit is required for your project
> - [ ] If a permit is required, you have obtained it BEFORE starting work
> - [ ] The permit is posted at the job site in a visible location
>
> **If any item cannot be confirmed:** Contact your local building department. Working without a required permit can result in fines, required removal of work, inability to sell your home, and insurance complications. Never assume a permit is not needed — always verify.

**General rule:** Most structural, electrical, plumbing, and mechanical work requires a permit. Cosmetic work (painting, flooring, cabinetry) generally does not.

**Common projects and permit requirements:**

| Project | Permit Required? | Notes |
|---------|-----------------|-------|
| Kitchen or bath remodel (cosmetic only) | No | Unless moving/adding plumbing, electrical, or walls |
| Replace a light fixture (like for like) | No | Same circuit, same location |
| Add a new electrical circuit or outlet | Yes | Electrical permit |
| Replace a water heater | Yes | Plumbing and possibly mechanical permit |
| Build a deck | Usually yes | Depends on height and attachment to house |
| Replace a roof (same materials) | Usually no | Check local requirements; some jurisdictions require permits |
| Remove or modify a wall | Yes if load-bearing | Structural permit; may require engineering |
| Add or remodel a bathroom | Yes | Plumbing, electrical, possibly structural permits |
| Install/replace a furnace or AC | Yes | Mechanical permit |
| Build a fence | Varies | Height-dependent; check local zoning |
| Install solar panels | Yes | Electrical and building permits |
| Seismic retrofit (foundation bolting) | Yes | Building permit; prescriptive path available in Portland/Seattle |

**Source:** [GOV-01] Oregon, [GOV-02] Washington — general permit requirements. Always verify with your local building department, as requirements vary by jurisdiction.

### L2 View — The Permit Process Step by Step

**Oregon Permit Process [GOV-01]:**

1. **Determine scope and permits needed.** Contact your local building department (city or county). Describe the project. They will tell you which permits are required.
2. **Prepare application materials.** Depending on scope:
   - Simple projects: Application form, description of work, site plan
   - Complex projects: Engineered construction documents, structural calculations, energy compliance documentation
3. **Submit application and fees.** Fees vary by jurisdiction and project valuation. Typical residential permits: $200-$2,000+ depending on scope.
4. **Plan review.** The building department reviews your plans for code compliance. Timeline varies:
   - Simple residential: 2-4 weeks typical
   - Complex residential: 4-8 weeks
   - Commercial: 6-12 weeks or more
5. **Permit issuance.** Once plans are approved, the permit is issued. Post it at the job site.
6. **Construction with inspections.** Work proceeds with required inspections at key stages (see Part 8).
7. **Final inspection and certificate of occupancy.** When all work is complete and passes final inspection, the building department issues a certificate of occupancy (for new construction) or closes the permit (for remodels).

**Washington Permit Process [GOV-02]:**

The process is substantially similar to Oregon. Key differences:
- Electrical permits are obtained from L&I [GOV-08], not the local building department (though some jurisdictions have L&I delegation agreements)
- Washington local building departments operate under SBCC-adopted codes
- Some Washington jurisdictions have expedited review programs for simple residential projects

## Part 8: Inspection Sequences

### Residential New Construction — Typical Inspection Sequence

The following sequence represents a typical single-family residential new construction project. The exact inspection requirements vary by jurisdiction, but this sequence is representative of both Oregon and Washington practice [GOV-01, GOV-02].

| # | Inspection | When | What the Inspector Checks | Common Failure Points |
|---|-----------|------|--------------------------|----------------------|
| 1 | **Erosion control / site prep** | Before any ground disturbance | Silt fencing, erosion control measures per grading permit | Missing or improperly installed erosion controls |
| 2 | **Footing / foundation excavation** | After excavation, before concrete pour | Soil bearing capacity (visual), excavation depth and width per plans, reinforcement placement | Wrong depth, inadequate bearing soil, water in excavation |
| 3 | **Foundation forms and rebar** | After forms set and rebar placed, before pour | Rebar size, spacing, cover per plans/code; form dimensions; anchor bolt placement | Insufficient cover, missing anchor bolts, wrong rebar spacing |
| 4 | **Foundation / slab** | After pour, forms stripped | Concrete placement quality, anchor bolt location and projection, foundation dimensions | Misaligned anchor bolts, honeycombing, dimensional errors |
| 5 | **Underground plumbing** | Before backfill | Pipe material, size, slope per code; cleanout locations; water test | Wrong slope, incorrect material, failed pressure test |
| 6 | **Underground electrical** | Before backfill | Conduit depth, material, separation from other utilities | Insufficient cover depth |
| 7 | **Framing (structural)** | After framing complete, before insulation/drywall | Lumber species/grade, fastener schedule, shear wall nailing, hold-down hardware, header sizes, joist hangers, blocking, fireblocking | Missing hold-downs, wrong nailing pattern, missing fireblocking |
| 8 | **Rough electrical** | After wiring, before insulation/drywall | Wire gauge, circuit counts, box fill, GFCI/AFCI locations, grounding, panel wiring | Overcrowded boxes, missing GFCI protection, improper grounding |
| 9 | **Rough plumbing** | After piping, before insulation/drywall | Pipe size, venting, fixture unit calculations, water pressure test | Failed pressure test, improper venting, wrong pipe size |
| 10 | **Rough mechanical** | After ductwork/equipment, before insulation/drywall | Duct sizing, equipment clearances, combustion air, gas piping test | Inadequate combustion air, failed gas test, undersized ducts |
| 11 | **Insulation / vapor barrier** | After rough-in approvals, before drywall | Insulation R-values per energy code, vapor barrier placement, air sealing | Wrong R-value, gaps in insulation, improper vapor barrier placement |
| 12 | **Energy code compliance** | May combine with insulation inspection | Blower door test results, duct leakage test, window U-factors, HERS rating if required | Failed blower door test, excessive duct leakage |
| 13 | **Drywall (nailing)** | After drywall hung, before taping | Fastener spacing, edge distance, correct screw/nail type for fire-rated assemblies | Wrong fastener spacing on fire-rated assemblies |
| 14 | **Final building** | After all work complete | All systems operational, smoke/CO detectors, egress windows, guardrails, stairs, accessibility | Missing smoke detectors, non-compliant guardrails |
| 15 | **Final electrical** | After all work complete | Cover plates, labeling, GFCI/AFCI function test, panel schedule, emergency disconnect | Missing labels, non-functional GFCI, incomplete panel schedule |
| 16 | **Final plumbing** | After all work complete | Fixture function, hot/cold correct, backflow prevention, water heater installation | Cross-connections, missing backflow prevention |
| 17 | **Final mechanical** | After all work complete | Equipment operation, thermostat function, gas appliance venting, CO detection | Improper venting, missing CO detectors |

**Source:** [GOV-01, GOV-02, CODE-01, CODE-02, CODE-03].

### Remodel Inspection Sequence

Remodel projects require a subset of the inspections above, depending on the scope of work:

| Scope of Remodel | Typical Inspections Required |
|------------------|------------------------------|
| Kitchen remodel (layout change) | Rough plumbing, rough electrical, rough mechanical (if HVAC affected), insulation (if exterior walls opened), framing (if walls removed/added), finals |
| Bathroom addition | Rough plumbing, rough electrical, framing, insulation, finals |
| Electrical panel upgrade | Rough electrical (if new circuits), final electrical |
| Window replacement | May require permit; energy code compliance for U-factor; final building |
| Seismic retrofit | Foundation/structural (anchor bolts, cripple wall bracing), final building |
| Roof replacement | Varies by jurisdiction; may require sheathing inspection if decking replaced |

### L4 View — Inspection Preparation Best Practices

**What to have ready for every inspection:**

- [ ] Approved plans (stamped/approved set) on site and accessible
- [ ] Permit card posted and visible from the street or building entry
- [ ] Work area accessible — clear path for inspector, adequate lighting
- [ ] Relevant work exposed — do not cover work before inspection approval
- [ ] Test results ready (pressure tests, blower door, duct leakage) if applicable
- [ ] Material certifications available (engineered wood product stamps, concrete mix designs, insulation R-value labels)
- [ ] Previous inspection approvals documented

**Common rejection reasons and recovery:**

| Rejection Reason | Code Basis | Typical Fix | Schedule Impact |
|-----------------|-----------|------------|-----------------|
| Missing hold-down hardware at shear walls | IRC R602.10 / IBC 2304 [CODE-01] | Install per approved plans; request re-inspection | 1-3 days |
| GFCI protection missing at required locations | NEC 210.8 [CODE-02] | Install GFCI receptacles or breakers | 1 day |
| Insulation gaps or compression | OEESC / WSEC-R [STD-07, STD-15] | Reinstall insulation to fill cavity without compression | 1-2 days |
| Failed plumbing pressure test | UPC Chapter 7 [CODE-03] | Find and repair leak; retest | 1-2 days |
| Fireblocking missing at floor/wall intersections | IRC R302.11 [CODE-01] | Install fireblocking material; request re-inspection | 1 day |
| Wrong nailing pattern on structural sheathing | IRC R602.3 / engineering schedule [CODE-01] | Add nails to correct spacing; request re-inspection | 1-2 days |

> **From the field:** Schedule inspections for morning when possible. Many inspectors work a route — morning inspections are less likely to be delayed. Have your phone number posted on the permit card so the inspector can reach you if they have questions. A good relationship with your inspector is a professional asset.

> **BLOCK — Licensed Professional Required**
>
> Any of the following conditions require a licensed design professional (PE, SE, or registered architect):

> - Multi-family residential (3+ units)

> **Required professional:** Licensed PE (discipline-specific), Licensed SE (structural), or Registered Architect (building design)

### Practice Questions (L3)

**Question 1:** In Washington, which state agency administers the electrical code?

- a) State Building Code Council (SBCC)
- b) Department of Labor & Industries (L&I)
- c) Department of Commerce
- d) Office of the Secretary of State

**Answer:** b) L&I — Washington electrical code is administered by the Department of Labor & Industries [GOV-08], not the SBCC which administers all other building codes. This is a key administrative difference from Oregon, where BCD administers all specialty codes including electrical [GOV-01].

---

**Question 2:** A contractor is building a 4-unit apartment building in Portland. Which primary building code applies — the IRC or the IBC?

- a) IRC, because it is residential
- b) IBC, because it has 3 or more dwelling units
- c) Either code, at the contractor's choice
- d) Portland City Code only

**Answer:** b) IBC — The IRC applies to one- and two-family dwellings and townhouses. Buildings with 3 or more dwelling units are classified as R-2 occupancy under the IBC [CODE-01]. In Oregon, this means the OSSC 2025 (IBC 2024 base) applies [STD-01].

---

**Question 3:** When codes conflict in Washington, which code takes precedence?

- a) The most recently adopted code
- b) The code with the most restrictive requirement
- c) The code listed first in the precedence order (IBC > IRC > IMC > IFC > WUI > UPC)
- d) The local amendment

**Answer:** c) Washington establishes a formal code precedence order [GOV-02]: IBC > IRC > IMC > IFC > WUI > UPC. When two codes contain conflicting provisions, the code listed earlier in this sequence prevails.

---

**Question 4:** A homeowner in Portland wants to convert their single-family home's basement into a rental apartment. What code requirement is triggered by this change of occupancy?

- a) No additional requirements — it is still a residential building
- b) ASCE 41 seismic evaluation per Portland City Code Chapter 24.85
- c) Full demolition and rebuild to current code
- d) Only an electrical upgrade is required

**Answer:** b) Portland Ch. 24.85 [STD-16] mandates ASCE 41 seismic evaluation for existing buildings undergoing change of occupancy [CODE-04]. Converting a single-family home to include a rental unit changes the occupancy classification and triggers evaluation requirements.

---

**Question 5:** What is the effective date of Oregon's OEESC 2025, and when does it become mandatory?

- a) Effective October 2025, mandatory April 2026
- b) Effective January 1, 2025, mandatory July 1, 2025
- c) Effective July 2025, mandatory January 2026
- d) Effective March 2025, mandatory September 2025

**Answer:** b) The OEESC 2025 (based on ASHRAE 90.1-2022) has an effective date of January 1, 2025 and becomes mandatory July 1, 2025 [STD-07, GOV-01].
