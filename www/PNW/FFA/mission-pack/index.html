  1. Open mission.pdf and read the Stage 1 Vision Guide to orient yourself to the mission scope and 6-module architecture.
  2. Hand the full PDF to your GSD Orchestrator (Claude Code with Skill-Creator active). Say: "Execute this research mission."
  3. The Orchestrator will activate Fleet profile, run Wave 0 (schemas), then dispatch parallel Wave 1 agents across three tracks.
  4. Wave 2 (INTEG/Opus) synthesizes cross-domain connections. Wave 3 assembles and verifies the final document.
  5. Approve the human review gate (CAPCOM) before final delivery.
  6. The output is a comprehensive cross-referenced research document spanning biology, rendering, craft, fabrication, and animation arts.
