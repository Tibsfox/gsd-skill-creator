# Shared Schema and Foundation Reference

Foundation layer for the Animal Speak, Sacred Landscapes, and the Living World research study. Establishes naming conventions, the tribal name glossary, the source citation index, and document structure standards used across all five modules.

---

## Document Structure Standards

All research modules in this study follow a consistent structure:

| Element | Convention |
|---------|-----------|
| Module identifier | Three-letter code: BIO, BIB, TRB, CRT, SYN |
| Section headings | `## Major Section` / `### Subsection` / `#### Detail` |
| Citations | In-text: `[Author Year]` with full entry in source bibliography |
| Tables | Markdown pipe tables with header row |
| Sensitivity markers | Inline `[SENSITIVITY: LEVEL]` where content requires protocol |
| Cross-references | `[see Module-Section]` format, e.g., `[see TRB-2.4.2]` |
| Dates | Full format: Month Day, Year (e.g., October 24, 2009) |
| Nation names | Always specific: Coast Salish, not "Native American" |

## Tribal Name Glossary

Canonical names and common alternatives for all Pacific Northwest nations referenced in this study. **Rule: Always use the name in the "Canonical Name" column.** Update this glossary whenever a module references a nation.

| Canonical Name | Alternative Names | Language Family | Territory Summary |
|---------------|-------------------|-----------------|-------------------|
| Coast Salish | Salish, Central Coast Salish, Halkomelem speakers, Lushootseed speakers, Straits Salish | Salishan | SW British Columbia; W Washington (Puget Sound region); NW Oregon coast. Includes Squamish, Musqueam, Tsleil-Waututh, Snohomish, Duwamish, Puyallup, Nisqually, Lummi, and many other distinct communities |
| Lower Chinook | Chinook, Chinookan peoples, Shoalwater Chinook | Chinookan (Penutian) | Columbia River mouth; coastal Washington/Oregon. Historically the great trade language people; Chinook Jargon was the lingua franca of the Pacific Northwest fur trade |
| Tlingit | Lingit, Tlingit Nation | Na-Dene (Tlingit branch) | SE Alaska coastal panhandle; northern coastal British Columbia. Organized into moieties (Raven and Eagle/Wolf) with matrilineal clan structure |
| Haida | Haida Nation, Xaayda, X?aayda, Old Massett, Skidegate | Haida (language isolate) | Haida Gwaii (Queen Charlotte Islands), British Columbia; Prince of Wales Island, Alaska. Master carvers; monumental totem pole tradition |
| Tsimshian | Ts'msyen, Tsimshian Nation, Coast Tsimshian, Southern Tsimshian | Tsimshianic (Penutian) | NW British Columbia coast (Prince Rupert area); Annette Island, Alaska (Metlakatla). Four matrilineal clans: Raven, Eagle, Killer Whale, Wolf |

### Terms Requiring Sensitivity

| Term | Context | Protocol |
|------|---------|----------|
| Power animal | Originally Coast Salish "guardian spirit in animal form" (sxwayxwey tradition) | Always attribute to Coast Salish origin; note migration through Harner to Andrews |
| Spirit quest | Vision quest, Spirit Quest, guardian spirit quest | Use nation-specific terminology when available; note that practices vary significantly by nation |
| Shaman | Medicine person, healer, spiritual specialist | Note that "shaman" is an external academic term (from Evenki/Tungusic via Eliade); PNW nations have their own terms for spiritual specialists |
| Totem | Totem animal, totem pole, crest | Distinguish between: (1) clan crests/heraldic poles (Haida, Tlingit, Tsimshian), (2) guardian spirits (Coast Salish), (3) Andrews' personal spiritual ally concept. These are not the same thing |
| Medicine wheel | Sacred hoop | Not a PNW Coast tradition; originates with Plains nations. Andrews uses it; PNW nations generally do not |
| Smudging | Cleansing ceremony | Specific practices vary by nation; do not present a generic "how to smudge" |

    sonnet: [BIO, BIB, TRB]
