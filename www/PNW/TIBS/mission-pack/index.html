How to Use This Package

  1. Feed to GSD Skill Creator: Open Claude Code / CLI and load this PDF as the mission document for gsd-skill-creator to orchestrate.
  2. Reconcile baseline: Check your current GSD release against v1.49.24. If your environment is ahead, review any breaking changes before execution.
  3. Wave 0 first: Run HAIKU to produce the shared schema, tribal name glossary, and source index before any survey work begins.
  4. Parallel Wave 1: Andrews Track (BIO + BIB) and Tribal Track (TRB + CRT) can run simultaneously — assign to separate EXEC agents.
  5. Cultural sensitivity gate: All six SC-IND and SC-CER safety tests are mandatory-pass BLOCK conditions. Do not merge any module output that fails these tests.
  6. CAPCOM approval: Human-in-the-loop approval required at wave boundaries before proceeding to Wave 2 (Synthesis) and Wave 3 (Publication).
