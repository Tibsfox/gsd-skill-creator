How to Use This Package

  1. Open sop_mission.pdf and review Stage 1 (Vision) with Tibsfox before execution.
  2. Hand Stage 3 (Mission Package) to the GSD orchestrator in Claude Code on the dev or nasa branch.
  3. Orchestrator loads Stage 2 (Research Reference) as the primary context cache for all Wave 1 agents — no re-searching needed.
  4. Execute Wave 0 (Foundation schema generation), then trigger Wave 1 parallel tracks simultaneously.
  5. CAPCOM gate after Wave 1: Tibsfox reviews research outputs before Wave 2 synthesis begins.
  6. Execute Wave 2 (Opus-led M5 synthesis + cross-module patterns).
  7. CAPCOM gate after Wave 2: Tibsfox reviews synthesis before publication.
  8. Execute Wave 3: self-assessment tool, MV SOP template, verification report, final delivery.
