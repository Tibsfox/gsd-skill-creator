🚀 How to Use This Package

  1. Download fertilizer_mission.pdf — review the Vision Guide (Stage 1) and confirm framing matches your research goals.
  2. Open a new Claude Code session and hand it fertilizer_mission.pdf with the instruction: "Execute this GSD research mission starting at Wave 0."
  3. The orchestrator will request CAPCOM approval at three gates: after Wave 0 (foundation), after Wave 2 (modules 1–6 complete), and after Wave 3 (synthesis).
  4. Final output: a structured 6-module research document + 8-milestone transition roadmap + verification report.
  5. Optionally seed the .planning/staging/inbox/ via planning-bridge MCP for persistent mission tracking.
