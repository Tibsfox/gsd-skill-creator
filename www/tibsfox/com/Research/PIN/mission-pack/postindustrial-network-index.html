  1. Point GSD's new-project at the vision section of the mission package
  2. Wave 0 produces shared artist schemas, source index, and module skeletons
  3. Wave 1 executes three parallel tracks (A: TG/COUM, B: NWW+List, C: C93+DiJ)
  4. Wave 2 synthesizes network topology, Coil nexus, NIN outlier, and ideology review
  5. Wave 3 assembles, verifies, and delivers the final document package
