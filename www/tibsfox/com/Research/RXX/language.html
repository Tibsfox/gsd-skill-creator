NUMERIC DIGITS 1000
Everything is a string. Numbers are strings that happen to look like numbers. This is not duck typing. This is no typing at all.

1. The Typeless String Model

x = 42
SAY LENGTH(x)          /* 2 -- it's a two-character string */
SAY x + 1              /* 43 -- converted to number, added, converted back */
SAY x || 'kg'          /* 42kg -- string concatenation */

y = '  007  '
SAY y + 0              /* 7 -- leading/trailing spaces and zeros stripped */
SAY LENGTH(y)          /* 7 -- the original string has 7 characters */

flag = (3 > 2)
SAY flag               /* 1 -- booleans are "0" and "1" strings */
SAY DATATYPE(flag)     /* NUM */

NUMERIC DIGITS — Arbitrary Precision

NUMERIC DIGITS 9       /* default: 9 significant digits */
SAY 1/3                /* 0.333333333 */

NUMERIC DIGITS 50
SAY 1/3                /* 0.33333333333333333333333333333333333333333333333333 */

NUMERIC DIGITS 100
SAY 0.1 + 0.2          /* 0.3 -- no IEEE 754 nonsense */

/* Fibonacci at 500 digits */
NUMERIC DIGITS 500
a = 0; b = 1
DO i = 1 TO 2000
  c = a + b; a = b; b = c
END
SAY "F(2000) =" b      /* a 418-digit number, computed exactly */

2. Variables & Stems

/* Simple variables — no declaration needed */
name = "Alice"
age = 30

/* Compound variables (stems) — REXX's associative arrays */
student.1.name = "Alice"
student.1.grade = "A"
student.2.name = "Bob"
student.2.grade = "B"
student.0 = 2           /* convention: stem.0 = count */

/* Iterate over stems */
DO i = 1 TO student.0
  SAY student.i.name "got" student.i.grade
END

/* Default value: the uppercase stem name */
SAY unknown.key          /* prints "UNKNOWN.KEY" (not an error!) */

/* DROP releases a variable */
DROP name
SAY name                 /* prints "NAME" (reverts to default) */

Porting note: Stems are unique to REXX. student.i.name resolves the value of i first, then constructs the compound key "STUDENT.1.NAME". This means stem tails can be ANY expression that evaluates to a string. Implementing this requires a hash table keyed by the fully-resolved string.

3. Control Structures

/* DO — the universal loop */
DO i = 1 TO 10               /* counted */
DO WHILE condition            /* while */
DO UNTIL condition            /* until */
DO FOREVER                    /* infinite (use LEAVE to exit) */
DO 5                          /* repeat 5 times */
DO i = 1 TO 100 BY 3         /* stepped */
DO i = start TO end FOR max   /* bounded */

/* IF/THEN/ELSE */
IF age > 18 THEN
  SAY "Adult"
ELSE
  SAY "Minor"

IF x > 0 THEN DO             /* multi-statement body needs DO/END */
  SAY "Positive"
  SAY "x =" x
END

/* SELECT (switch/case) */
SELECT
  WHEN color = "red"   THEN SAY "Stop"
  WHEN color = "green" THEN SAY "Go"
  WHEN color = "amber" THEN SAY "Caution"
  OTHERWISE SAY "Unknown color"
END

/* LEAVE and ITERATE */
DO i = 1 TO 100
  IF i // 7 = 0 THEN ITERATE    /* skip multiples of 7 */
  IF i > 50 THEN LEAVE           /* exit loop */
  SAY i
END

6. Subroutines & Functions

/* Internal function */
SAY factorial(10)
EXIT

factorial: PROCEDURE        /* PROCEDURE isolates variables */
  ARG n
  IF n <= 1 THEN RETURN 1
  RETURN n * factorial(n - 1)

/* PROCEDURE EXPOSE — selective variable sharing */
counter: PROCEDURE EXPOSE count
  count = count + 1         /* modifies the caller's 'count' */
  RETURN count

7. The INTERPRET Instruction

code = 'SAY "Hello from INTERPRET"'
INTERPRET code              /* prints: Hello from INTERPRET */

/* Dynamic variable access */
varname = "total"
INTERPRET varname '= 42'
SAY total                   /* 42 */

/* Building code at runtime */
op = '+'
INTERPRET 'result =' a op b

Porting challenge: INTERPRET means every port must carry a full parser and evaluator at runtime. Pure AOT compilation is impossible. The CUDA port must handle this via dispatch table lookup — pre-compiled kernel targets indexed by INTERPRET arguments.

8. Error Handling

SIGNAL ON SYNTAX NAME handleSyntax
SIGNAL ON NOVALUE NAME handleNovalue

/* ... program code ... */

handleSyntax:
  SAY "Syntax error" RC "at line" SIGL ":" ERRORTEXT(RC)
  EXIT 1

handleNovalue:
  SAY "Uninitialized variable at line" SIGL
  SAY "Variable:" CONDITION('D')
  EXIT 2
ConditionTriggered When
SYNTAXRuntime syntax/arithmetic error
NOVALUEUninitialized variable used
ERRORHost command returns non-zero RC
FAILUREHost command not found
HALTExternal interrupt (Ctrl-C)
NOTREADYI/O not ready

9. TRACE — Built-in Debugging

TRACE Results          /* show expression results */
TRACE Intermediates    /* show intermediate evaluations */
TRACE All              /* show every clause before execution */
TRACE Off              /* disable tracing */
TRACE ?Results         /* interactive: pause after each result */

TRACE is part of the language specification. Some programs dynamically toggle TRACE for diagnostics. Any port must implement at least TRACE OFF, TRACE RESULTS, and TRACE ALL.
