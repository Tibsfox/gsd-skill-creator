1. Circuits & Ohm’s Law

Everything in electronics starts with three quantities and one law:

V = I × R

Voltage (V) is pressure — the force pushing electrons through a circuit. Current (I) is flow — how many electrons pass a point per second (amperes). Resistance (R) is opposition — how much the circuit resists the flow (ohms). Know any two, calculate the third.

The Three Passive Components

Resistor: Converts electrical energy to heat. Controls current flow. Color bands encode the value. A 10KΩ resistor with 5V across it passes 0.5 mA (V/R = 5/10000).

Capacitor: Stores energy in an electric field between two plates. Charges and discharges over time. The RC time constant (τ = R × C) governs how fast. Used for filtering, timing, and coupling signals. A 100μF capacitor with a 10KΩ resistor has τ = 1 second.

Inductor: Stores energy in a magnetic field around a coiled wire. Resists changes in current. Inductors and capacitors together create resonant circuits — the basis of radio tuning, power supplies, and the oscillators inside every clock signal on every CPU.

Power

P = V × I (watts). A 5V USB port delivering 2A supplies 10 watts. A 12V automotive system at 10A is 120 watts. The Orion spacecraft’s solar arrays generate about 11,100 watts at Earth distance — enough to power a small house. At lunar distance (same solar flux), the arrays produce the same power because the Sun-Earth distance doesn’t change significantly over a lunar transit.

Try This

  1. Buy a breadboard, a bag of resistors, LEDs, and a 9V battery. Build a circuit: battery → resistor → LED → battery. The resistor limits current so the LED doesn’t burn out. Calculate the resistor value: (9V − 2V LED drop) / 20mA = 350Ω. Use a 390Ω (nearest standard value).
  2. Add a capacitor in parallel with the LED. Watch it glow briefly after you disconnect the battery — the capacitor discharges through the LED.
  3. Get a multimeter. Measure voltage across the LED. Measure current through the circuit. Verify Ohm’s law: the numbers should match your calculations.

2. The 555 Timer

The NE555 is the most popular integrated circuit ever manufactured. Billions have been produced since its introduction in 1972 by Signetics (Hans Camenzind’s design). It contains 23 transistors, 2 diodes, and 15 resistors — and it teaches more about analog electronics than any textbook.

ModeWhat It DoesApplication
AstableFree-running oscillator — output toggles between high and low continuouslyClock signals, LED blinkers, tone generators
MonostableOne-shot pulse — trigger input produces a single timed output pulseDebouncing switches, timed delays, pulse shaping
BistableFlip-flop — output toggles on each trigger, stays until next triggerToggle switches, memory elements, divide-by-2

Astable Mode: The Oscillator

Two resistors (R1, R2) and one capacitor (C) set the frequency:

f = 1.44 / ((R1 + 2*R2) * C)

Example: R1 = 1K, R2 = 10K, C = 10uF
f = 1.44 / ((1000 + 20000) * 0.00001)
f = 1.44 / 0.21
f = 6.86 Hz  (blinks ~7 times per second)

Change the capacitor to 100nF and the frequency jumps to 686 Hz — an audible tone. Connect a speaker and you have a sound generator. The same circuit at MHz frequencies generates the clock signal that drives a CPU. Every digital system has an oscillator at its heart.

// Arduino — read temperature sensor, control LED brightness
#include <Wire.h>

void setup() {
  Wire.begin();           // I2C bus
  Serial.begin(115200);   // UART to computer
  pinMode(9, OUTPUT);     // PWM-capable pin for LED
}

void loop() {
  // Read temperature from I2C sensor at address 0x48
  Wire.requestFrom(0x48, 2);
  int raw = (Wire.read() << 8) | Wire.read();
  float tempC = raw / 256.0;

  // Map temperature to LED brightness (0-30C → 0-255 PWM)
  int brightness = constrain(map(tempC * 10, 0, 300, 0, 255), 0, 255);
  analogWrite(9, brightness);  // PWM output

  Serial.print("Temp: "); Serial.print(tempC);
  Serial.print("C  LED: "); Serial.println(brightness);
  delay(1000);
}

Try This

  1. Get an Arduino Uno or ESP32. Install the Arduino IDE. Upload the Blink example (File → Examples → Basics → Blink). The onboard LED blinks. You just programmed hardware.
  2. Connect a temperature sensor (DS18B20 or BME280) via I2C. Read the temperature and print it to the serial monitor. Compare with KPAE METAR — how close is your sensor to the airport?
  3. Build a weather station: temperature + humidity + pressure sensor, ESP32 with WiFi, POST data to a server every 5 minutes. This is how real weather stations work.

4. Signal Processing

Every sensor produces a signal. Signal processing extracts the information from that signal and removes the noise.

Analog Filtering

An RC low-pass filter is a resistor and capacitor in series. High frequencies are attenuated; low frequencies pass through. The cutoff frequency: fc = 1 / (2πRC). A 10KΩ resistor with a 100nF capacitor gives fc ≈ 159 Hz — it passes speech frequencies and blocks high-frequency noise.

Digital Filtering & the FFT

The Fast Fourier Transform converts a time-domain signal into its frequency components. A weather station sampling temperature every second can use FFT to identify: the 24-hour diurnal cycle, the ~12-hour tidal component (for coastal stations), and high-frequency noise from sensor jitter.

# Python FFT — analyze temperature data
import numpy as np

# 96 hours of hourly temperature data (KPAE, 4 days)
temps = [4.5, 4.0, 3.5, ...]  # 96 values
fft = np.fft.rfft(temps)
freqs = np.fft.rfftfreq(len(temps), d=1.0)  # 1 sample/hour

# Dominant frequency should be ~0.042 cycles/hour (24-hour period)
dominant = freqs[np.argmax(np.abs(fft[1:])) + 1]
print(f"Dominant period: {1/dominant:.1f} hours")  # should be ~24

The Nyquist Theorem

To accurately capture a signal, you must sample at at least twice its highest frequency. Our hourly sweep cadence (1 sample/hour) can accurately capture signals with periods of 2 hours or longer. The 24-hour diurnal temperature cycle is well within this. Sub-hourly fluctuations (sea breeze onset, convective showers) are aliased — which is why our sub-version marks exist: they increase the effective sample rate during interesting events.

Our math co-processor includes FFT tools (fourier_fft, fourier_spectrum) running on GPU via CUDA. The same signal processing that analyzes weather data also processes the ALICE TPC signals — thousands of ionization electron drift-time measurements per collision, each a signal to be filtered and reconstructed.

The Hardware-Software Boundary

Assembly maps to logic gates. C’s memory model maps to physical address lines. PWM is a for-loop in hardware. The 555 timer generates the clock signal that drives every CPU. The abstraction layers between software and hardware are thinner than most programmers realize.

Understanding electronics doesn’t make you a better programmer in the way that learning algorithms does. It makes you a better engineer. When you know that an I2C bus can only be so long before capacitance causes signal degradation, you understand why distributed systems have latency. When you know that a capacitor smooths voltage ripple, you understand why exponential moving averages smooth noisy data. The math is the same. The substrate changes.
