How to Execute This Mission

Open vulkan_desktop_mission.pdf. Review Stage 1 Vision and confirm scope aligns with your current gsd-skill-creator branch state (v1.49+ / DACP milestone required).

Hand Stage 3 Wave 0 specification to a Claude Code session. Paste the chipset YAML and have the Haiku-tier agent produce the shared schemas: sample-schema.yaml, dacp-bundle-template.json, version-matrix.yaml.

Run Wave 1 in three parallel Claude Code sessions (tracks A, B, C). Each track uses the Wave 0 schemas as its sole shared input. Track A: API + Framework. Track B: Extension Corpus. Track C: Performance Patterns.

HITL CAPCOM Gate 1: review the 99 skill stubs (28 API + 47 extension + 24 performance) for accuracy. Verify the 5 safety-critical items (S-01 through S-05) before proceeding.

Run Wave 2 Upstream Synthesis. Feed all Wave 1 outputs to the Opus-tier orchestrator. Output: upstream-intelligence-vulkan.yaml and the finalized version-matrix.yaml.

Run Wave 3 Wiring. Generate all 80+ DACP bundles, the version-gate skill, ZFC stamp candidate SQL, and execute the headless CI test run (10 samples via Swiftshader). Run all 31 tests. CAPCOM Gate 3 = merge decision.
