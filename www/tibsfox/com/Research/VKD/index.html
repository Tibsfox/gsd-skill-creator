The Vulkan Samples repository from KhronosGroup is not just a tutorial collection — it is the canonical living reference for best-practice GPU programming across the entire modern hardware ecosystem. Each sample is a distillation of accumulated wisdom from ARM, NVIDIA, AMD, Valve, Google, and dozens of contributors. The spaces between the samples — the connections between concepts — are where the real architectural knowledge lives.

Research Modules

M1: API Foundations

API Track A · Core samples

14 core API samples + 14 Vulkan-Hpp C++ variants. Hello Triangle through Compute N-body. HDR, instancing, terrain tessellation, dynamic rendering, timeline semaphores. Each sample mapped as a DACP-compatible skill stub with version requirements.

M2: Extension Corpus

Extensions Track B · 47 samples

Complete catalogue of extension samples: ray tracing (KHR), mesh shaders, dynamic rendering, descriptor indexing, fragment shading rate, buffer device address. Extension family maps showing promotion paths from vendor to KHR. Version requirement matrix.

M3: Performance Patterns

Performance Track C · 24 patterns

24 performance optimization samples with profiling annotations formalized as machine-readable preconditions. Multi-draw indirect, pipeline caching, render pass optimization, subpass dependencies, command buffer management, descriptor set strategies.

M4: Framework & Build

Framework Track A · Architecture

Vulkan Samples framework architecture: Application, Platform, RenderContext abstractions. CMake build system with Gradle Android integration. CI harness patterns for GPU validation. Git LFS asset management. Cross-platform compilation targets.

M5: Upstream Intelligence

Upstream Synthesis · Version tracking

Vulkan 1.4 spec changes, SDK 1.4.341 updates, Roadmap 2026 extensions. Extension promotion tracking (vendor to EXT to KHR). Breaking changes in descriptor heap architecture. Tensor processing extensions (VK_ARM_tensors). Float8 shader support frontier.

M6: skill-creator Wiring

Wiring Integration · DACP

80+ DACP bundles mapping each sample to a skill stub. Version gate skill for extension availability checking. ZFC stamp candidate for federation compliance. The artifact that makes the entire Vulkan corpus queryable by skill-creator agents.
