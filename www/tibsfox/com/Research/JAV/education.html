Java has 30 years of accumulated wisdom, most of which is still relevant and a surprising amount of which is free online. The good news: the canonical book corpus is small and well-curated, and the best references (Bloch, Goetz, Gosling) cover 90% of what you need.

A Recommended Progression

1 Start: Core Java or Head First Java if you’re new to programming; Effective Java (Bloch, 3rd ed. 2018) if you already know C++/C#/Kotlin and want to learn Java idioms quickly.
2 Reference: The Java Language Specification (Gosling, Joy, Steele, Bracha, Buckley) — free at docs.oracle.com. The authoritative source.
3 Concurrency: Java Concurrency in Practice (Goetz, Peierls, Bloch, Bowbeer, Holmes, Lea, 2006) — the definitive book on java.util.concurrent and the JMM. Still the reference 20 years later.
4 Modern style: Modern Java in Action (Urma, Fusco, Mycroft, 2018) — lambdas, streams, Optional, reactive. Covers Java 8–9 thoroughly.
5 JVM internals: Java Performance: The Definitive Guide (Scott Oaks, 2nd ed. 2020) or Optimizing Java (Evans/Gough/Newland, 2018).
6 dev.java — Oracle’s free tutorials, updated for Java 21. The closest thing to an official learning site.
7 Baeldung.com — the largest independent Java tutorial site. Search “baeldung X” for almost any topic and you’ll find a working example.
8 A real project: Spring Boot + Maven or Gradle + JUnit 5 + AssertJ + Testcontainers. Read other people’s Spring Boot code on GitHub. Read Brian Goetz’s JEPs.

By Joshua Bloch (the idiomatic canon)

  • Effective Java — 1st ed. 2001, 2nd ed. 2008 (Java 5/6), 3rd ed. 2018 (Java 7/8/9). 90 items. Every Java programmer should read this.
  • Java Puzzlers (with Neal Gafter, 2005) — 95 tricky puzzles that teach edge cases. Entertaining and instructive.

By Brian Goetz & team

  • Java Concurrency in Practice (2006) — with Peierls, Bloch, Bowbeer, Holmes, Lea. The book on Java concurrency. Still the authoritative text for threads and the JMM, even in the Loom era.

A Reading Order That Works

  1. Head First Java or Core Java — 2–4 weeks, hands-on
  2. dev.java tutorials — a week, work through the examples
  3. Effective Java (Bloch, 3rd ed.) — 3–4 weeks, one item per sitting
  4. Build something: a Spring Boot REST service with a database, tests with JUnit 5 + AssertJ + Testcontainers
  5. Java Concurrency in Practice — before you write anything concurrent in anger
  6. Baeldung as you go for “how do I do X” questions
  7. Modern Java in Action — streams, Optional, functional style
  8. JEP index for each LTS release, read the features that interest you
  9. Once you’re confident: Java Performance or Optimizing Java

This progression takes several months of serious study and produces a programmer who can write modern Java with confidence and read any Java codebase.
