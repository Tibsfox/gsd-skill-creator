Usage Instructions

  1. Point GSD's new-project at the vision section of the TeX source
  2. Wave 0 produces shared schema, level taxonomy, and source index
  3. Wave 1 executes three parallel tracks (A: Matter/Signal/Logic, B: Compute/Software/Network, C: Intelligence/Quantum/Collider/Fidelity)
  4. Wave 2 synthesizes layer bridges and the unified calibration chain
  5. Wave 3 verifies, compiles, and delivers the final 100-level curriculum PDF
