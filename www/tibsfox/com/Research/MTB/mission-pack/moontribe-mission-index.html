Usage Instructions

  1. Point GSD's new-project at the vision section of the document
  2. Wave 0 produces shared schema and source index
  3. Wave 1A executes History Track (M1 + M2 + M3 in parallel)
  4. Wave 1B executes Ecology Track (M5, parallel with 1A)
  5. Wave 2 synthesizes cultural legacy (M4), sensitivity review, cross-module integration
  6. Wave 3 assembles, verifies, and delivers the final document
