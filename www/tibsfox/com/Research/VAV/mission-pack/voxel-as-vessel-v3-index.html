Research Modules — Third Pass (M14–M21)

M14 — Temporal Imaging

Nyquist/Shannon in imaging. Long exposure vs fast frame rate. Temporal aliasing, MTF, shutter angle. TSR (6× beyond Nyquist). CUP (trillion fps). Field separation, deinterlacing.

M15 — Color Fidelity

ICC vs DNG profiles. ColorChecker workflow (7 steps). RAW processing pipeline. Studio lighting: D65/flash vs LED vs tungsten. sRGB/AdobeRGB/ProPhoto gamuts.

M16 — Audio Fidelity

Mic calibration, polar patterns, proximity effect. Semaphores & mutexes in DAW signal chain. Gating, JACK RT scheduling. FFT restoration pipeline. IRENE optical digitization. iZotope spectral recovery.

M17 — Serialization + HPC

FlatBuffers 81 ns/op vs JSON 7045 ns/op. Protobuf, MessagePack, Cap'n Proto, Avro. Reed-Solomon, LDPC, Hamming, ECC RAM. InfiniBand HDR 200 Gbps, RDMA, SHARP in-network computing. HIPPI history.

M18 — Transport Taxonomy

Acoustic modem → POTS/V.90 → ADSL/VDSL → DOCSIS 3.1 (OFDM/4K FFT) → GPON/DWDM. HIPPI. InfiniBand. RFC 1149 IPoAC (Bergen 2001). Sneakernet bandwidth. Sneakernet mesh / DTN Bundle Protocol.

M19 — Backup + Federation

3-2-1-1-0 rule. RBD snapshot/mirror, BorgBackup, CDP. Arrow Flight, Trino, Apache Drill federation. Cross-domain: data diode, CASB, break-and-inspect, format sanitization.

M20 — Zero Trust + Firewall

NIST SP 800-207 PDP/PEP architecture. DoD 7 pillars. BeyondCorp/ZTNA. Proxy types (forward/reverse/IAP/CASB/WAF). 4-zone firewall model mapped to Ceph/OpenStack. CephX ZT gap analysis. CAEP session re-eval.

M21 — Synthesis

Frequency-domain unification of all modules. Extended Ceph/Minecraft fidelity isomorphism. Transport Pareto frontier. Sneakernet mesh protocol spec outline. Through-line: "the channel does not know what it carries."

Every layer of the stack is a decision about which frequencies to trust.
The camera's shutter is a temporal low-pass filter. The Fourier transform is the language all these decisions share.
The pigeon carrying a USB drive does not know its effective bandwidth exceeds ADSL by a factor of ten thousand.

The seed is still there. The function is still there. Only the changes need the channel.
