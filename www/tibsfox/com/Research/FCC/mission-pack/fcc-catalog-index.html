  1. Point GSD's new-project at the vision section of the PDF
  2. Wave 0 produces shared Part inventory schema and source index
  3. Wave 1 executes two parallel tracks (A: Structure+Spectrum, B: Carrier+Broadcast)
  4. Wave 2 executes two parallel deep-dives (C: Private Radio+Compliance, D: Flux+NTIA)
  5. Wave 3 synthesizes cross-reference index, master catalog, and verification report
  6. Wave 4 assembles final publication package
