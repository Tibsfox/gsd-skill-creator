Recompilation

  1. Download the .tex source file
  2. Ensure DejaVu fonts are installed (standard on Ubuntu; available via fonts-dejavu)
  3. Run three XeLaTeX passes:
    xelatex -interaction=nonstopmode coil-mission.tex
    xelatex -interaction=nonstopmode coil-mission.tex
    xelatex -interaction=nonstopmode coil-mission.tex
