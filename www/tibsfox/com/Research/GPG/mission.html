The Through-Line

The GPU is the modern Denise chip. It owns the framebuffer. It executes shader programs without asking the CPU for permission. The observability stack is Paula — listening to every sensor, reporting I/O health, knowing when a channel is saturated before the system notices. The skill-creator is Agnus — orchestrating DMA, pre-staging the next wave before the current one lands. This mission documents the complete wiring harness connecting these layers: from the sub-pixel OpenGL fragment shader to the Prometheus scrape interval, from the Shadertoy uniform to the Grafana dashboard alert that fires when GPU temperature exceeds the ambient thermal envelope. The spaces between shader invocations are where the rendering happens.
