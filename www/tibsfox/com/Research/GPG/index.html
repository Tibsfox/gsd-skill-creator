The Amiga 500 shipped with a custom chipset that embarrassed workstations costing ten times its price. It did not win by having more transistors. It won because Agnus, Denise, and Paula each owned their slice of the machine and communicated through DMA channels that kept the CPU free to think. The GPU is the modern Denise — a dedicated display coprocessor executing a billion shader invocations per second, stateless, parallel, each computing its color from nothing but coordinates and uniforms. The CPU submits work orders. The GPU fills pixels.

Research Modules

M1: GPU Graphics Pipeline

Pipeline Track A · Vertex to fragment

The modern programmable pipeline: vertex processing, rasterization, fragment shading, tessellation, geometry shaders, compute shaders. GLSL and SPIR-V shader languages. GPU memory architecture. The Hello Triangle from NeHe to Vulkan.

M2: Vulkan & Modern APIs

API Track A · Explicit GPU control

Vulkan 1.3 object model, command buffers, pipeline state objects, descriptor sets, synchronization primitives. The implicit-to-explicit transition from OpenGL. Memory management and validation layers.

M3: Ray Tracing Architecture

Ray Tracing Track B · Hardware RT

BVH acceleration structures, RT Core architecture across three NVIDIA generations. Vulkan ray tracing pipeline. Hybrid rendering. Path tracing, ReSTIR, and DLSS/FSR reconstruction.

M4: XScreenSaver & Shadertoy

GLSL Track B · Visual programming

XScreenSaver 6.14 framework, GLX virtual root window, Shadertoy API wrapper, GLSL portability matrix. WebGPU and WGSL. GPGPU history. 50-hack curated catalog for GSD-OS idle render.

M5: Shader Programming

Shader Track C · GLSL techniques

SIMD execution model. Fragment shader techniques: SDFs, ray marching, noise, post-processing. Vertex skinning, instanced particles. SPIR-V toolchain. Shader debugging. The GSD-OS CRT shader engine.

M6: GPU Observability

Observability Track C · Monitoring

NVML and DCGM Exporter. OpenTelemetry for GPU workloads. Prometheus 3.0 resource attributes. Grafana Alloy pipeline. Raspberry Pi SoC monitoring. Five GSD-OS dashboard specifications. Alert engineering.
