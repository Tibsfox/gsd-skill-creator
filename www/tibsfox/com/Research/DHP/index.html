DHCP Protocol

Wire Format · Options Catalog · Lease Lifecycle · Security · DHCPv6
Overview Wire Format Options Lease Lifecycle Security DHCPv6 Mission Pack
Every network-connected device begins its participation through a brief, elegant conversation with a server it cannot yet fully address. Before it has an identity — before it knows its own IP address — it broadcasts into the void: Who am I? Where am I? What are my parameters? This is DHCP, and those 236 bytes of fixed header followed by a magic cookie and a variable options field configure every networked device on earth. Four message types in the basic flow. A lease lifecycle with precisely defined timers. Small building blocks. Massive operational consequence.

Research Modules

M1: Wire Format and Message Types

Wire Track A · Packet structure

Full byte-by-byte layout of the DHCP packet: 15 named fields across 236 bytes. All 8 message types (DISCOVER through INFORM). DORA exchange flow. Transport rules: UDP ports 67/68, broadcast vs unicast, relay agent processing. Magic cookie (99.130.83.99).

M2: Options Catalog

Options Track A · TLV encoding

Complete DHCP option registry: TLV encoding architecture, RFC 1497 vendor extensions (codes 1-18), DHCP-specific extensions (codes 50-61), 20+ post-RFC 2132 options. Option categories, encoding examples, overloading, IANA registry structure.

M3: Lease Lifecycle & State Machine

State Track B · Client FSM

Three allocation mechanisms. 8-state client FSM: INIT through BOUND through RENEWING through REBINDING. Timer semantics: T1 at 50%, T2 at 87.5%, lease expiry. Retransmission algorithms: exponential and linear backoff. ARP conflict detection.

M4: Security Architecture

Security Track B · Attack/defense

Three attack classes: rogue server, DHCP starvation, replay/spoofing. The starvation-to-MITM attack chain. Five defense mechanisms: DHCP snooping, DAI, IP Source Guard, port security, VACLs. RFC 3118 authentication and its failure analysis.

M5: DHCPv6 & Modern Extensions

IPv6 Synthesis · RFC 8415

Complete DHCPv6 architecture: SARR exchange, UDP ports 546/547, multicast addressing. DUID types (LLT, EN, LL, UUID). Identity Associations: IA_NA, IA_TA, IA_PD. SLAAC coexistence. Prefix delegation. 11-dimension DHCPv4 vs DHCPv6 comparison matrix.
