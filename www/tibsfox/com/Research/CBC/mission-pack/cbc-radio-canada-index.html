Usage Instructions

  1. Point GSD's new-project at the vision section of the PDF
  2. Wave 0 produces shared schema, source index, and directory structure
  3. Wave 1 executes two parallel tracks (A: History+Funding, B: Services+Indigenous)
  4. Wave 2 synthesizes Module 5 + OCAP review + political balance review + cross-module synthesis
  5. Wave 3 verifies sources, assembles final pack, and runs full test plan
