How to Use with GSD Skill Creator

  1. Open your terminal and navigate to the gsd-skill-creator repository (dev branch).
  2. Place the personal_care_mission.pdf in your .planning/staging/inbox/ directory, or deposit via the planning-bridge MCP endpoint.
  3. Launch the GSD orchestrator and reference the mission file as the execution spec.
  4. Confirm Wave 0 outputs (schemas, source index) before releasing Wave 1A/1B agents.
  5. Review Wave 1 module outputs at the CAPCOM gate before authorizing Wave 2.
  6. Review the Opus synthesis (Wave 3) before releasing the publication wave.
  7. Final deliverable: complete research document + practical protocol guide, ready for tibsfox.com/Missions/.
