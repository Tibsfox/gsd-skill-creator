Analysis

The mid-range server represents the practical sweet spot of the March 2008 sprint. Four AMD Opteron 8347 quad-cores on a Tyan quad-socket board deliver 16 cores at $291 per core — 2x the core count of the Budget Workstation at only 2x the cost per core.

The Opteron 8347 at 1.9GHz trades clock speed for socket density. Where the Intel builds were limited to dual-socket, AMD's HyperTransport interconnect enabled four sockets on commodity boards. This is the architecture that would scale to 32 cores in the next build.

Computing Context: March 2008

AMD's Barcelona (Opteron 8300 series) launched in late 2007 as AMD's first native quad-core server chip. The TYAN S4980G2NR was a quad-socket board using AMD's Direct Connect Architecture — each CPU had its own memory controller rather than sharing a front-side bus, a significant advantage for memory-intensive workloads like fractal rendering.

Where This Fits

The mid-range server is the bridge between the dual-socket Intel builds ($1,121 and $5,169) and the octo-socket AMD builds ($8,761 and $43,361). At $291/core it offers the best value proposition for core density in the 2008 lineup, trading per-thread performance for parallelism — exactly the trade-off that fractal rendering rewards.
