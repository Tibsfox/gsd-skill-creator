Minecraft Real-Time Weather

NOAA · Open-Meteo · GOES-19 SDR · Fabric Modding · PNW Microclimates · Voxel Atmosphere
Overview Data Sources Satellite MC Weather API Microclimate Mesh Integration Mission Pack
The sky above any Minecraft world is a lie. It rains when a random number generator says so, clears when a timer expires, and the thunderstorm over your base has no relationship whatsoever to the thunderstorm rolling across the actual mountains outside your window. For a platform that has taught generations of players about geology, agriculture, ecology, and architecture, this is a staggering missed opportunity. This mission closes that gap — replacing Minecraft's stochastic weather engine with a living atmosphere derived from the real world.

Research Modules

M1: Global Weather Data Sources

Data Track A · API survey

Open-Meteo, NOAA HRRR, NASA GEOS-5, and GFS. API endpoints, rate limits, response schemas, and normalization to WeatherState. WMO code standard. PNW-specific data considerations including convergence zones and atmospheric rivers.

M2: DIY Satellite Reception Pipeline

Satellite Track A · SDR hardware

GOES-19 HRIT direct reception via RTL-SDR. Hardware BOM ($95-$200), RF link budget, SatDump decode pipeline, ABI channel products, cloud cover extraction, fog detection via channel differencing, and continuous automation.

M3: Minecraft Weather API & Mod Architecture

Minecraft Track B · Game internals

Vanilla weather state model, biome temperature/precipitation mechanics, existing mod survey (6 mods analyzed), Fabric/Forge API surfaces, per-player weather packets, RCON bridge, Paper plugin API, and shader integration for enhanced effects.

M4: Microclimate Simulation & Variable Mapping

Simulation Track B · Translation layer

WMO code to Minecraft state translation (complete mapping table), snow level determination, cloud cover thresholds, precipitation intensity scaling, wind escalation, fog/visibility mapping, coordinate transforms, weather front propagation, and hysteresis.

M5: GSD Mesh Integration & Deployment

Mesh Synthesis · Sidecar service

Python sidecar architecture (<200 MB), data broker pipeline, cache design, RCON and Fabric mod bridges, YAML configuration schema, systemd deployment, Foxy's Playground reference config (4 PNW zones), monitoring, and security hardening.
