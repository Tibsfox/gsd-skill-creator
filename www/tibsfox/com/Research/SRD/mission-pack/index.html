How to Use This Package

  1. Read the Vision (Stage 1) to understand scope, module decomposition, and success criteria before execution.
  2. Hand to gsd-skill-creator via Claude Code: open the PDF as context and say "Execute this research mission — start with Wave 0."
  3. Confirm CAPCOM gates at the end of each wave. The package specifies 4 human approval points before proceeding.
  4. Parallel tracks in Wave 1 can run simultaneously. Assign Track A (SSH) and Track B (RDP/VNC) to separate agent instances.
  5. Opus handles synthesis in Wave 2 — security analysis and hardening guidance require judgment-level reasoning.
  6. Deliverable: 9 output files including compiled PDF, .tex source, and bibliography.
