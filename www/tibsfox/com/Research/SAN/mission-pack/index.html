Contents

  1. Stage 1 — Vision Document: Problem statement (5 problems), ecosystem architecture diagram, 5 research modules, chipset configuration, scope boundaries, 12 success criteria, through-line
  2. Stage 2 — Research Reference: Organizational history & milestones, training curriculum (6 practice areas, 85+ courses), GIAC certification pipeline (tiers, exam format, pricing), 2026 conference & summit catalog (8+ events), community programs (Holiday Hack, NetWars, ISC), ISC2 2025 Workforce Study data, salary benchmarks, AI impact analysis, bibliography
  3. Stage 3 — Mission Package: Squadron activation (12 roles), 4-wave execution plan with 2 parallel tracks, component breakdown with model assignments, 28 test cases (safety-critical + core + integration + edge), verification matrix, token budget (~66K across 18 windows), dependency graph

Usage

  1. Review the PDF for the complete research mission package
  2. Hand to gsd-skill-creator via Claude Code for orchestrated execution
  3. The mission activates at Squadron profile (12 crew roles, 2 parallel tracks)
  4. Wave 0 (Foundation) runs first, then Wave 1 parallelizes across Track A (Org + Training) and Track B (Conferences + Community)
  5. Wave 2 synthesizes workforce context across all modules
  6. Wave 3 assembles, verifies, and publishes the final integrated reference

Recompilation

  1. Ensure DejaVu fonts are installed (apt install fonts-dejavu)
  2. Run three passes: xelatex -interaction=nonstopmode sans-mission.tex (×3)
  3. Three passes resolve TOC, cross-references, and page counts
