Module Overview

Architecture

01. Network Architecture & Design

Campus, data center, WAN, and cloud topologies. Spine-leaf, CLOS, fat-tree fabrics. Network tiers and hierarchy. Capacity planning. Bill of materials. The architecture decision framework for greenfield and brownfield deployments.
Routing

02. Routing & Switching Operations

BGP operations (peering, route filtering, communities, looking glasses). OSPF/IS-IS area design. VLAN architecture. VRF segmentation. MPLS and EVPN-VXLAN. SD-WAN overlay engineering. The control plane vs data plane operational split.
Automation

03. Network Automation & NetDevOps

Ansible/NAPALM/Nornir for network config. YANG models, NETCONF, gNMI streaming. Network CI/CD pipelines. Config drift detection. Batfish verification. Intent-based networking. The CLI-to-API transformation.
Traffic

04. Load Balancing & Traffic Engineering

L4/L7 load balancing. Global server load balancing (GSLB). DNS-based load balancing. Anycast engineering. Traffic shaping and QoS. ECMP. Segment routing traffic engineering. Content delivery network integration.
Security

05. Network Security Engineering

Firewall architecture (zones, policies, microsegmentation). IDS/IPS deployment. DDoS mitigation (scrubbing, Cloudflare/Akamai). Zero-trust network architecture. mTLS everywhere. Network forensics and packet analysis.
Cloud

06. Cloud Networking

VPC design patterns. Transit gateways and hub-spoke. PrivateLink/Private Service Connect. Hybrid connectivity (Direct Connect, ExpressRoute, Cloud Interconnect). Cloud firewalls and security groups. Service mesh networking (Istio, Cilium).
Observe

07. Observability & Troubleshooting

SNMP and streaming telemetry (gNMI/OpenConfig). Flow analysis (NetFlow, sFlow, IPFIX). Packet capture methodology. Network performance monitoring. Cable testing and physical layer. Systematic troubleshooting methodology.
Wireless

08. Wireless & Edge Networking

Wi-Fi 6E/7 engineering. Site surveys and RF planning. Controller architectures (on-prem, cloud-managed). IoT networking protocols (Zigbee, LoRaWAN, Matter). 5G private networks. Edge compute networking patterns.
DNS

09. DNS Engineering at Scale

Authoritative vs recursive architecture. DNSSEC operations (key rotation, algorithm migration). DNS-based service discovery (Consul, CoreDNS). GeoDNS and latency-based routing. DNS as control plane. DNS security (DoH, DoT, RPZ).
Reliability

10. Network Reliability & DR

Redundancy design (dual-homed, diverse path routing). Convergence time engineering. BFD for fast failure detection. Graceful restart and NSF. Network DR testing. Submarine cable resilience. The 5-nines network availability target.
