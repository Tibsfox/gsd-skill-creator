Learning the bash ShellNewham & Rosenblatt2005O’Reilly classic. Systematic walk through bash features. 3rd edition. Good for intermediate users.
From Bash to Z ShellKiddle, Peek & Stephenson2004Cross-shell comparison. Covers bash, zsh, tcsh. Useful for understanding shell family differences.

Learning Path — From Zero to Pipeline Virtuoso

  1. Level 1: Navigation — cd, ls, pwd, mkdir, cp, mv, rm, cat, less. Learn to move around. Read The Linux Command Line Part 1.
  2. Level 2: Pipes & Filters — grep, sort, uniq, wc, head, tail, cut. The pipe operator. The Unix Programming Environment chapters 1-3.
  3. Level 3: Text Processing — sed, awk, regular expressions, tr, paste, join. Build real pipelines. sed & awk book.
  4. Level 4: Shell Scripting — Variables, loops, conditionals, functions, set -euo pipefail. Write scripts that run in production. ABS Guide + ShellCheck.
  5. Level 5: Process & Job Control — &, fg, bg, jobs, nohup, signals, trap, flock. Manage long-running processes.
  6. Level 6: System Administration — cron, systemd, ssh, rsync, find -exec, xargs. Automate everything. Unix Power Tools.
  7. Level 7: Terminal Mastery — tmux, custom prompts, dotfiles, readline bindings, fzf, modern tools (ripgrep, fd, bat). The Art of Unix Programming.
  8. Level 8: Pipeline Virtuoso — You think in pipes. You compose tools instinctively. You know when to reach for Perl or Python. You write tools others compose with. You are the sysadmin.

The secret: There is no shortcut. You learn the CLI by using it every day, for real tasks, making real mistakes. Read the man page. Try the flags. Break things in a VM. Fix them. Repeat for 20 years. That’s the curriculum.
