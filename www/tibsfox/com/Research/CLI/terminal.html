ANSI Escape Codes

Every modern terminal emulator speaks ANSI. The CSI (Control Sequence Introducer) is ESC [ (0x1B 0x5B):

Cursor Control

ESC[H        cursor home (1,1)
ESC[10;20H   cursor to row 10, column 20
ESC[A        cursor up     ESC[B   cursor down
ESC[C        cursor right  ESC[D   cursor left
ESC[s        save position ESC[u   restore position
ESC[2J       clear screen  ESC[K   clear to end of line

SGR (Select Graphic Rendition)

CodeEffectCodeEffect
ESC[0mReset allESC[1mBold
ESC[2mDimESC[3mItalic
ESC[4mUnderlineESC[7mReverse video
ESC[9mStrikethroughESC[22mNormal weight

Color Codes

# 8/16 colors (original ANSI)
ESC[31m      red foreground         ESC[41m      red background
ESC[32m      green foreground       ESC[42m      green background
ESC[91m      bright red (16-color)  ESC[101m     bright red bg

# 256 colors (xterm extension)
ESC[38;5;196m   foreground color 196 (bright red)
ESC[48;5;21m    background color 21 (blue)

# 24-bit true color (16.7 million)
ESC[38;2;255;100;50m   foreground RGB(255,100,50)
ESC[48;2;0;128;255m    background RGB(0,128,255)

Color test: Run printf '\033[38;2;255;100;0mTrue color!\033[0m\n' — if it shows orange text, your terminal supports 24-bit color. Most modern emulators do. The GNOME VTE-based terminals (GNOME Terminal, Tilix) added it in 2014. Alacritty, Kitty, WezTerm, and Windows Terminal have always supported it.

Pseudo-Terminals (PTY)

Modern “terminals” are not terminals at all — they are applications that talk to the kernel’s pseudo-terminal subsystem.

$ tty
/dev/pts/3       # pseudo-terminal slave #3

$ ls /dev/pts/
0  1  2  3  4  ptmx

The PTY architecture has two sides:

SideDeviceUsed By
Master/dev/ptmxTerminal emulator (xterm, Kitty, etc.)
Slave/dev/pts/NShell and programs running inside

The kernel’s TTY layer sits between them, providing line discipline (echo, canonical mode, special characters like Ctrl-C → SIGINT). This is how SSH works: the SSH server allocates a PTY on the remote machine, and the SSH client forwards your keystrokes to the master side across the network.

Terminal Multiplexing: tmux & screen

A multiplexer creates virtual terminals inside a single terminal connection. Sessions persist after disconnection — essential for remote servers.

# tmux architecture
tmux server (background daemon)
  ├─ Session "work"
    ├─ Window 0: "editor" — vim
    ├─ Window 1: "build" — make && ./run
    └─ Window 2: "logs" — tail -f app.log
  └─ Session "monitoring"
    └─ Window 0: "htop" + "iotop" (split pane)
# Essential tmux commands
tmux new -s work           # new named session
tmux ls                    # list sessions
tmux attach -t work        # reattach
# Ctrl-b d                 # detach
# Ctrl-b c                 # new window
# Ctrl-b %                 # vertical split
# Ctrl-b "                 # horizontal split
# Ctrl-b [                 # scroll mode (q to exit)

Why Multiplexers Matter

  • Persistence: SSH disconnects don’t kill your work. Reattach and everything is exactly where you left it.
  • Panes: Editor in one pane, build output in another, logs in a third — all visible simultaneously.
  • Pair programming: Two users attach to the same tmux session and see each other’s keystrokes in real time.
  • Scripting: tmux send-keys lets you automate terminal interactions from outside.

Modern Terminal Features

GPU-Accelerated Rendering

Alacritty (2017) pioneered GPU-rendered terminal text. The entire character grid is rendered as textured quads on the GPU, achieving frame rates that CPU-based terminals cannot match. Kitty, WezTerm, and Ghostty followed. The performance difference is visible when scrolling large outputs (find / -type f) or running TUI applications with rapid updates.

Graphics Protocols

ProtocolTerminalCapability
Kitty GraphicsKittyPNG, animated GIF, GPU-rendered content inline
iTerm2 Inline ImagesiTerm2, WezTermPNG/JPEG inline, file transfer
SixelMany (xterm, mlterm)Bitmap graphics, legacy DEC format

These protocols let tools like matplotlib render plots directly in the terminal, viu display images, and tty-clock render analog clock faces.

Ligatures and Font Features

Kitty, WezTerm, and Windows Terminal support programming ligatures — != renders as ≠, -> as →, == as a connected glyph. Combined with Nerd Fonts (patched fonts with 3,000+ icons), the modern terminal is visually richer than any terminal in history while remaining a character-cell grid at its core.

The pragmatic answer: Any GPU-accelerated terminal from the last 5 years is excellent. Pick one that matches your OS and workflow, install a Nerd Font, set TERM=xterm-256color, and focus on what you type, not where you type it.
