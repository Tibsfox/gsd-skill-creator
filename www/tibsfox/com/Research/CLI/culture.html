$ There is a world that exists entirely in monospaced type. It has no gradients, no rounded corners, no drop shadows. Its palette, for decades, was seven colors and a background. Its resolution is measured not in pixels but in rows and columns. And yet this world has produced some of the most enduring, elegant, and influential culture in all of computing.

ANSI Art & the BBS Scene

Before the web, there were Bulletin Board Systems — dial-up servers you connected to at 2400-14400 baud with a terminal program. The BBS scene (1980s-1990s) produced a rich visual culture built entirely from character cells and ANSI escape codes.

The Art Groups

GroupFoundedSpecialty

Blocktronics2010Modern revival. Active today, releasing art packs.

Artists used TheDraw (1986) and later PabloDraw to create images from the 256 characters in the CP437 character set — box-drawing characters (┌─┐│└─┘), block elements (░▒▓█), and the 16 ANSI colors. The constraint was the canvas: 80 columns, 16 colors, 256 glyphs. The results were extraordinary — portraits, landscapes, logos, and abstract compositions that pushed the medium to its absolute limits.

The medium: ANSI art is to the terminal what pixel art is to the sprite. A constrained medium that demands precision and creativity. The best ANSI artists achieved photorealistic shading using nothing but half-blocks and color gradients. Archives at sixteencolors.net and artscene.textmod.es preserve thousands of works.

From CP437 to Unicode

The original character set for ANSI art was Code Page 437, the IBM PC’s default character encoding. It included:

RangeCharactersUse
Box-drawing┌─┐│└┘├┤┴┬┼Borders, frames, tables
Block elements█▓▒░▾▴Shading, fills, pseudo-pixels
Braille patterns&braille; (Unicode 2800-28FF)2×4 dot matrix per cell = pseudo-graphics
Geometric shapes★♥♦♣♠♪Decoration, games

Unicode expanded the palette enormously. Braille patterns (U+2800-U+28FF) provide a 2×4 dot matrix per character cell, enabling pseudo-graphical displays at 2x horizontal and 4x vertical resolution. Tools like gnuplot and plotext use Braille patterns to render charts directly in the terminal.

Nerd Fonts: Icons in the Terminal

Nerd Fonts patches popular programming fonts with 3,000+ additional glyphs from icon libraries: Powerline symbols, Font Awesome, Devicons (language logos), Octicons (GitHub), Material Design Icons, Seti UI (file-type icons).

Tools that leverage Nerd Fonts: Starship prompt, eza/lsd (file-type icons in ls), yazi (file manager), lazygit, nvim-tree. As of 2026, Nerd Fonts is effectively mandatory for the modern terminal aesthetic — 50+ pre-patched programming fonts available at nerdfonts.com.

TUI Frameworks & Notable TUIs

The Foundation: ncurses

ncurses (new curses, Thomas Dickey, 1993) is the library that made terminal UIs possible. Originally written to support the game rogue (1980). Provides windows, panels, forms, menus, color pairs, and mouse support. Programs built on it: Midnight Commander, vim, htop, mutt, dialog/whiptail (Linux installer UIs).

Dotfiles Culture

A dotfiles repository is a version-controlled collection of configuration files: .bashrc, .zshrc, .vimrc, .tmux.conf, .gitconfig, and more. The dotfiles movement — sharing configuration publicly on GitHub — created a culture of learning by example.

Prompt Engineering (The Original Kind)

Before LLMs, “prompt engineering” meant crafting your shell prompt. The PS1 variable in bash, PROMPT in zsh, or a dedicated tool like Starship or Powerlevel10k can show: git branch, exit code, execution time, language version, AWS profile, Kubernetes context, battery level, weather, and more — all in a single line.

# A basic but informative PS1
export PS1='\[\033[32m\]\u@\h\[\033[0m\]:\[\033[34m\]\w\[\033[0m\]$ '

# Starship (cross-shell, Rust, fast)
# Install: curl -sS https://starship.rs/install.sh | sh
# Add to .bashrc: eval "$(starship init bash)"

The Rice: r/unixporn & Desktop Culture

Ricing (from car culture: adding cosmetic modifications) is the art of customizing your Unix desktop for maximum visual coherence. The community gathers at r/unixporn (900,000+ members) to share screenshots of their setups.

A typical “rice” combines:

  • Tiling window manager: i3, sway, Hyprland, bspwm, dwm
  • Bar: polybar, waybar, eww
  • Launcher: rofi, wofi, dmenu
  • Terminal: Alacritty, Kitty, st (suckless)
  • Color scheme: Catppuccin, Dracula, Gruvbox, Nord, Tokyo Night
  • Font: Nerd Font variant (JetBrains Mono Nerd, Fira Code Nerd)
  • Shell: Zsh + Powerlevel10k or Fish + Starship
  • Fetch tool: neofetch, fastfetch (system info with ASCII art logo)

The rice is simultaneously a creative expression, a declaration of taste, and a deep technical exercise. Configuring a tiling WM from scratch requires understanding X11/Wayland, IPC, keybindings, shell scripting, and CSS-like styling — all integrated through dotfiles that can be version-controlled and shared.

The CLI as Philosophy

Text as Universal Interface

The Unix pipeline model survived because text is the lowest common denominator. A program that reads lines and writes lines can interoperate with any other program that reads and writes lines. No API versioning, no schema negotiation, no serialization. A tool written in 1975 composes with a tool written in 2025. The abstraction is lines of text, and lines of text never break backward compatibility.

CLI as Literacy

There is a reason sysadmins think in pipes: the CLI is a language, and fluency in that language is a form of literacy. A fluent CLI user doesn’t think “I need to find files” — they think find . -name "*.log" -mtime -1 | xargs grep "ERROR". The command is the thought. The pipe is the composition operator. The one-liner is the program.

The CLI Renaissance (2024-2026)

The CLI is having a moment. Several forces converge:

  • DevOps and containers. Docker, Kubernetes, and cloud CLIs made the terminal the primary deployment interface.
  • SSH. Remote servers don’t have GUIs. The terminal is the only interface.
  • Git. The most important developer tool is CLI-first.
  • The Rust rewrite wave. Modern tools (ripgrep, fd, bat) made the CLI more pleasant.
  • TUI renaissance. Textual, Bubbletea, Ratatui made building terminal apps as easy as web apps.
  • AI assistants. Claude Code, GitHub Copilot CLI — AI meets you in the terminal. The CLI is the most natural interface for AI pair programming because it is already a text-in, text-out REPL.

Terminal Nostalgia

cool-retro-term renders your terminal as a CRT monitor: green phosphor glow, scan lines, screen curvature, bloom effects. It is useless and beautiful. The appeal is not just aesthetic — it connects the modern user to the physical reality that shaped the interface. The blinking cursor, the monospaced font, the 80-column line — these are not arbitrary. They are fossils of teletypes and phosphor screens, preserved because the systems built on them were too useful to abandon.
