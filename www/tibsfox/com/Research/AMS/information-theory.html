Information theory provides the formal language for understanding why single agents can outperform multi-agent chains, how token budgets constrain communication bandwidth, and where context compression necessarily loses signal. The Data Processing Inequality is the foundational result.

Module Overview

This module formalizes five information-theoretic concepts that emerged from agentic technology research during the Artemis II mission: the Data Processing Inequality as proof of single-agent advantage, Shannon channel capacity as a model for token budget limits, mutual information as the measure of agent communication quality, token budgets as information bandwidth, and context compression as quantifiable information loss.

Key Mathematical Concepts

Data Processing Inequality (DPI)

I(Y; C) ≥ I(Y; M)

For any Markov chain X → Y → Z:
I(X; Y) ≥ I(X; Z)

Processing cannot create information — only destroy or preserve it.

The DPI proves that when a query Y passes through a coordinator C to reach the model M, mutual information can only decrease. A single agent with direct access to context preserves more information than a multi-agent relay chain. This result from Cover & Thomas underpins the finding in arXiv:2604.02460 that single-agent LLMs outperform multi-agent systems on multi-hop reasoning.

Shannon Channel Capacity

C = maxp(x) I(X; Y)

For a noisy channel with input X and output Y,
capacity C is the supremum of mutual information
over all possible input distributions p(x).

Token budgets in LLM systems are analogous to channel capacity. The context window defines a hard bandwidth limit. Every token consumed by system prompts, tool outputs, or conversation history reduces the remaining capacity for task-relevant information. Shannon’s 1948 theorem gives the theoretical ceiling.

Mutual Information in Agent Communication

I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

= ∑x,y p(x,y) log [ p(x,y) / (p(x)p(y)) ]

When two agents exchange messages, mutual information quantifies how much one agent’s state tells us about the other’s. Low mutual information between coordinator and worker indicates a lossy handoff — the worker is operating on incomplete context. High mutual information means the communication channel is efficient.

Context Compression and Information Loss

Compressing a 200K-token context into a 4K-token summary necessarily discards information. The rate-distortion function R(D) gives the minimum bit rate needed to represent a source within distortion D. In agentic systems, aggressive context compression trades fidelity for bandwidth, and the DPI guarantees the compressed version can never contain more task-relevant information than the original.
