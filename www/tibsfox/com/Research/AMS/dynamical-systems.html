Dynamical systems theory describes how complex systems evolve over time — from synchronized oscillators modeling multi-agent coordination to entropy increase governing context decay. The key question: does the system converge to a stable state or diverge into chaos?

Module Overview

This module covers five dynamical systems concepts from the agentic research: self-organizing intelligence inspired by particle physics, Kuramoto oscillator synchronization as a model for agent coordination, gradient stability and dynamical isometry in deep networks, context rot as entropy increase over time, and convergence behavior in the multi-pass research pipeline.

Key Mathematical Concepts

Self-Organizing Intelligence

dx/dt = F(x) + η(t)

Emergent order: limt→∞ S(x(t)) < S(x(0))
where S = entropy, F = interaction dynamics, η = noise

Self-organizing systems reduce local entropy through interaction dynamics despite global entropy increase. Particle physics inspires the model: just as particles in a cooling gas spontaneously form crystals, agents in a well-designed orchestration system spontaneously develop coordination patterns. The Gastown convoy model exhibits this — agents self-organize into efficient work distribution without central scheduling.

Kuramoto Oscillator Synchronization

dθi/dt = ωi + (K/N) ∑j=1N sin(θj - θi)

ωi = natural frequency of oscillator i
K = coupling strength
K > Kc ⇒ synchronization emerges

The Kuramoto model (1984) describes how coupled oscillators synchronize when coupling strength K exceeds a critical threshold Kc. Applied to multi-agent systems: each agent has a natural “frequency” (processing speed, update rate). When inter-agent communication coupling K is strong enough, the agents synchronize their work cycles. Below Kc, agents drift apart and coordination fails.

Gradient Stability and Dynamical Isometry

||∇L(x + ε)|| ≈ ||∇L(x)|| for small ε

Dynamical isometry: singular values of Jk ≈ 1
where Jk = Jacobian of layer k

Deep networks suffer from vanishing or exploding gradients when the Jacobian singular values deviate from 1.0. Dynamical isometry — all singular values near unity — ensures gradients propagate stably through many layers. This is a formal stability condition for the dynamical system defined by backpropagation. The analogy to agent systems: information should propagate through hierarchical agent layers without amplification or attenuation.

Context Rot as Entropy Increase

S(t) = S(0) + ∫0t σ(τ) dτ

where σ(τ) = entropy production rate
Context relevance: R(t) = R(0) · e-λt

Context rot is the second law of thermodynamics applied to information systems. As time passes, cached context becomes stale — the world changes, models update, dependencies shift. The relevance R(t) decays exponentially with rate λ. High-entropy environments (rapidly changing codebases) have large λ; stable environments have small λ. The enterprise AI drift problem is context rot at organizational scale.

Multi-Pass Convergence

xn+1 = f(xn)

Fixed point: x* = f(x*)
Convergence: |f′(x*)| < 1 (contraction mapping)

The Research Co-Processor runs multiple passes, each refining the previous state. Convergence requires that each pass is a contraction mapping — it brings the state closer to the fixed point (complete research). If |f′(x*)| ≥ 1, the system oscillates or diverges. The 8-pass pipeline observed in the published.html outputs demonstrates convergence: each pass adds decreasing amounts of new information.
