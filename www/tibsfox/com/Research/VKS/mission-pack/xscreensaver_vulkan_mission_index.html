Open the PDF — Read the Vision section to confirm scope alignment with your current gsd-skill-creator dev branch state (v1.49+, DACP milestone).

Hand to GSD Orchestrator — Feed the PDF to Claude Code as a mission spec. The Stage 3 wave plan is self-contained: each wave task has model assignment, dependencies, and expected outputs. No additional context required.

Wave 0 first — Start Claude Code with: "Execute Wave 0 of the Vulkan Screensaver Engine mission. Produce catalog schema YAML, plugin API header, shared types header, root CMakeLists.txt, and CI workflow skeleton."

CAPCOM gates — Review and approve at each gate (after Wave 1B, after Wave 1C, after Wave 2 synthesis). The CAPCOM agent will surface issues; your approval is required before proceeding to the next wave.
5
skill-creator upstream PR — Wave 3 produces the upstream intelligence YAML diff ready for a PR to the gsd-skill-creator dev branch. Review before merging.
