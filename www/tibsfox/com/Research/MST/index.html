Global Telescope Mesh Network

Network Architecture · Fleet Mapping · Alert Systems · Yuri's Night · Citizen Science · GSD Deployment
Overview Network Fleet Alerts Yuri's Night Citizen Science GSD Deploy Mission Pack
On April 12, 1961, Yuri Gagarin completed one orbit in 108 minutes. Sixty-five years later, millions of people point telescopes at the sky in isolation. The network exists physically — distributed across six continents, pointed at the same sky — but it has never been switched on. This mission proposes to switch it on. The spaces between telescopes are where the science lives.

Research Modules

M1: Network Architecture

Network Track A · Infrastructure layer

Protocol comparison of ASCOM, INDI, and the Las Cumbres OCS. Bridge layer design connecting professional and amateur telescope networks. NASA meshNetwork assessment. FITS data format standardization across heterogeneous instruments.

M2: Telescope Fleet Mapping

Fleet Track A · Node census

Complete census of connected telescope networks: Unistellar (15,000+), Las Cumbres (25 robotic), ZTF, AAVSO, Globe at Night. Geographic coverage map with gap analysis. Protocol-per-network inventory. Equipment tier classification from visual-only to research-grade CCD.

M3: Alert & Event Systems

Alerts Track B · Event pipeline

Rubin Observatory LSST alert pipeline: 7 million transient alerts per night, 2-minute latency. All 9 community alert brokers documented (ALeRCE, ANTARES, Fink, Lasair, Pitt-Google, AMPEL, Babamul, SCiMMA, TOM Toolkit). End-to-end routing from professional detection to backyard follow-up.

M4: Yuri's Night Integration

Event Track B · Convergence event

Yuri's Night event ecosystem mapping: 567 events in 75 countries at 50th anniversary. Global Shutter protocol: 00:00 UTC April 12 simultaneous exposure across all connected telescopes. Participation tiers from casual observer to campaign-assigned target tracking. PNW anchor nodes at Museum of Flight and Burke Museum.

M5: Citizen Science Protocols

Science Track C · Research methods

Scientific methods for distributed amateur networks: occultation timing, exoplanet transit photometry, nova monitoring, comet tracking, satellite brightness surveys. Unistellar documented output: 136 asteroids, 20 exoplanet orbits refined, DART debris in Nature. Data quality gates for GPS-timestamped FITS ingestion.

M6: GSD Deployment Layer

Deploy Synthesis · Orchestration

DACP bundle structure for telescope observation requests. Skill-creator integration: observe-detect-suggest loop mapped to campaign assignment. HITL CAPCOM gates at wave boundaries. DoltHub federation for distributed observation data. Alert-to-backyard pathway specification.
