Mission Package

The module file is self-contained because portability matters. The effect commands are hex codes because the machine is the notation system. The pattern grid scrolls vertically because time is explicit, not implied.

The Through-Line

The tracker paradigm is the original proof of the Amiga Principle: four hardware channels, one Finnish programmer, a generation of electronic musicians. OctaMED did not succeed by having more resources — it succeeded by being architecturally smarter about the resources it had. Software mixing doubled polyphony at the cost of CPU time that musicians were glad to spend because OctaMED was for composition, not games.

The module file is the original self-contained bundle: samples and score in a single distributable object. Small, portable, trivially shareable over BBS networks. The DACP bundle inherits this design principle — everything the receiver needs, nothing they don't, in a format that travels without dependencies.

“The hardware is fixed; the software is leverage.”
