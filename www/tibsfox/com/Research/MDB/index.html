How to Use This Package
1
Download the PDF

Read Stage 1 (Vision) to understand scope and philosophy. Read Stage 2 (Research Reference) as your primary evidential baseline before executing any research tasks.

2
Hand to GSD Orchestrator

Feed the PDF to Claude Code via gsd-skill-creator. The Stage 3 Mission Package is a complete flight plan: wave structure, crew manifest, token budget, and test plan are ready to execute.

3
Execute Waves with CAPCOM Gates

Run Wave 0 (Foundation) first, then parallel Waves 1A/1B/1C. Pause at each CAPCOM gate for human review — particularly the LSD/Leary philosophical content calibration before Wave 2.

4
Target: College of Knowledge

Final deliverable slots into the GSD College of Knowledge music history curriculum under "Progressive Rock Origins." GSD ecosystem through-line (Amiga Principle / Spaces Between) is pre-written in the synthesis passages.
