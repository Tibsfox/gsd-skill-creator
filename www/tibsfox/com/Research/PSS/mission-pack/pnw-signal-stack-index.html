Usage Instructions

  1. Point GSD's new-project at the vision section of the TeX source
  2. Wave 0 produces shared schemas and source index
  3. Wave 1 executes four parallel tracks (Broadcast, Spectrum, DIY RF, Aerial)
  4. Wave 2 synthesizes three parallel tracks (Aerospace, Puget Sound, Pacific Rim)
  5. Wave 3 integrates M8 Global Deployment architecture
  6. Wave 4 assembles, verifies, and publishes the final document
