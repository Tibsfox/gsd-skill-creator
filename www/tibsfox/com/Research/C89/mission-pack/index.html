Usage

  1. Open c895-mission.pdf for the complete mission package ready for GSD orchestrator handoff
  2. Feed the document to gsd-skill-creator via Claude Code for execution
  3. Wave 0 (Foundation) runs first, producing shared schemas and source index
  4. Wave 1 executes in two parallel tracks: History+Technical and Education+Cultural
  5. Wave 2 synthesizes with Opus handling cross-module integration and the funding module
  6. Wave 3 assembles the final publication and runs the 28-test verification suite
