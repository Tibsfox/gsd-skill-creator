The Frog Protocol — Quick Reference

  1. SILENCE — Distributed pause, no acknowledgment needed
  2. ASSESS — Scout agent characterizes the anomaly passively
  3. PROBE — Minimal test action by scout; others remain paused
  4. CLASSIFY — Threat / Neutral / Opportunity determination
  5. RESUME — Graduated re-engagement, scout first
