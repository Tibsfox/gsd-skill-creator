Distributions & Architecture

Module 1

K8s v1.35 "Timbernetes", K3s (single binary, 512MB), MicroK8s, KubeEdge. Distribution comparison across 10+ dimensions. Amiga Principle node mapping: 68000/Agnus/Denise/Paula to K3s roles.

Networking & Traffic Management

Module 2

Gateway API as the post-Ingress era. Ingress-NGINX retirement (March 2026). CNI landscape: Cilium (eBPF), Calico, Flannel. Service mesh: Istio vs. Linkerd. The nftables migration.

Resource Management

Storage & Persistence

Module 3

CSI driver ecosystem. Longhorn for small clusters, Rook/Ceph for enterprise scale, OpenEBS Mayastor for NVMe. Decision framework for bare-metal 3–10 node deployments with mixed SSD/HDD.

GPU Scheduling & AI/ML

Module 4

NVIDIA GPU Operator and Dynamic Resource Allocation (DRA). Fractional GPU: MIG, MPS, time-slicing. Topology-aware scheduling. Kueue, Volcano, YuniKorn. Dual RTX 5090 configuration.

Security & Zero Trust

Module 5

Five defense layers: network policies, RBAC, mTLS with pod certificates (KEP-4317), admission control (OPA/Kyverno), runtime detection (Falco). Phased zero-trust implementation for the GSD mesh.

Operations & Observability

Module 6

GitOps with ArgoCD and Flux. Prometheus + Grafana + OpenTelemetry monitoring. Velero backup. FinOps and right-sizing. Self-healing patterns. Cluster lifecycle management.
