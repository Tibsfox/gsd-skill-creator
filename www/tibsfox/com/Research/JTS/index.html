Why JavaScript & TypeScript together? TypeScript is not a separate language — it is JavaScript with a structural type system that erases at compile time. Every valid JavaScript program is a valid TypeScript program. TypeScript’s rise since 2016 has been so thorough that most new projects, frameworks, and libraries are TypeScript-first. You cannot understand the modern JS ecosystem without understanding TS, and you cannot understand TS without understanding JS.

The Two Halves

JavaScript (JS)

The runtime language. Brendan Eich, Netscape, 1995. Dynamic, prototype-based, event-driven, single-threaded with async I/O. Runs in browsers, servers (Node.js/Deno/Bun), edge workers, mobile (React Native), and desktop (Electron). Standardized as ECMAScript by TC39.

TypeScript (TS)

The type layer. Anders Hejlsberg, Microsoft, 2012. Structural typing, generics, utility types, conditional types. Compiles to JavaScript — types are erased at runtime. The de facto standard for large codebases, frameworks, and libraries.

The Runtimes

V8 (Chrome, Node.js, Deno, Workers), SpiderMonkey (Firefox), JavaScriptCore (Safari, Bun), Hermes (React Native). Four competing engines, each with their own JIT pipeline, GC, and optimization strategy.

The Ecosystem

npm (2M+ packages), build tools (webpack → Vite → esbuild/SWC), frameworks (React, Vue, Svelte, Angular, Solid), meta-frameworks (Next.js, Remix, Nuxt, SvelteKit, Astro). The largest developer ecosystem in the world.

The Timeline

May 1995 Brendan Eich writes Mocha in 10 days at Netscape.
Dec 1995 Renamed JavaScript (Sun trademark deal). Ships in Netscape Navigator 2.0 beta.
Jun 1997 ECMA-262 1st edition. Guy Steele as editor.
Dec 1999 ECMAScript 3. The stable base for a decade.
2005 AJAX coined (Jesse James Garrett). Google Maps/Gmail demonstrate the pattern.
Aug 2006 jQuery 1.0 (John Resig). DOM normalization across browsers.
2008 V8 engine ships in Chrome. Douglas Crockford’s JavaScript: The Good Parts.
Nov 2009 Node.js (Ryan Dahl, JSConf EU). JavaScript breaks out of the browser.
Dec 2009 ECMAScript 5. Strict mode, JSON, Array methods.
Oct 2012 TypeScript 0.8 (Anders Hejlsberg, Microsoft).
Jun 2015 ES2015 (ES6). The biggest upgrade: let/const, arrows, classes, promises, modules.
2016 Angular 2 rewritten in TypeScript. Pivotal moment for TS adoption.
2017 ES2017: async/await. Next.js 1.0 (Vercel).
2018 Deno announced (Ryan Dahl). Rust + V8, TypeScript-first.
Feb 2019 React 16.8: Hooks release. The React world shifts.
2020 Vite 1.0 (Evan You), esbuild (Evan Wallace), SWC (Donny Wang). The Rust/Go build tool revolution.
2022 Bun 0.1 (Jarred Sumner). Zig + JavaScriptCore, extremely fast.
2024 React 19: Server Components standard. Svelte 5: runes and signals.
