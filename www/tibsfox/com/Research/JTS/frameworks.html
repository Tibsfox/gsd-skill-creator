The JavaScript framework ecosystem is the most active, most contested, and most rapidly evolving frontend technology market in computing. No other language has this many competing approaches to the same problem, and no other ecosystem generates this much opinion per line of code.

The Framework Timeline

2006 jQuery (John Resig) — DOM normalization across browsers. The first dominant JS library.
2010 Backbone (Jeremy Ashkenas) — MV* with routing. AngularJS (Google, Miško Hevery) — two-way data binding.
2013 React (Facebook, Jordan Walke) — one-way data flow, virtual DOM, JSX. Changed everything.
2014 Vue.js (Evan You) — progressive framework, single-file components.
2016 Angular 2 (Google, rewrite in TypeScript). Svelte (Rich Harris) — compiled framework, no runtime. Next.js (Vercel).
2018 Solid.js (Ryan Carniato) — React-like syntax, fine-grained reactivity.
2021 Remix (Ryan Florence, Michael Jackson) — web standards-first, acquired by Shopify 2022.
2022 Astro — islands architecture, framework-agnostic.
2024 React 19 — Server Components standard. Svelte 5 — runes and signals.

React — The Dominant UI Library

Jordan Walke at Facebook (2013). React introduced three ideas that changed frontend development: components as the unit of UI, JSX as a compile-time syntax for expressing DOM trees in JavaScript, and a one-way data flow that replaced the two-way binding of AngularJS/Knockout.

// React with TypeScript and hooks
import { useState } from "react";

interface Props { initialCount?: number; }

export function Counter({ initialCount = 0 }: Props) {
  const [count, setCount] = useState(initialCount);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(c => c + 1)}>Increment</button>
    </div>
  );
}

React 16.8 (February 2019): Hooks (useState, useEffect, useRef, useMemo, useCallback) replaced class components. React 18 (March 2022): concurrent rendering, useTransition, automatic batching. React 19 (2024): Server Components as the standard rendering model, Actions, use hook.

Vue.js

Evan You, ex-Google, 2014. Progressive framework: use as little or as much as you need. Vue 3 (2020) introduced the Composition API (setup function + reactive/ref) and Proxy-based reactivity, rewritten in TypeScript.

<!-- Vue 3 Single File Component -->
<script setup lang="ts">
import { ref, computed } from "vue";

const count = ref(0);
const doubled = computed(() => count.value * 2);
</script>

<template>
  <p>Count: {{ count }} (doubled: {{ doubled }})</p>
  <button @click="count++">Increment</button>
</template>

Svelte

Rich Harris (2016). Svelte is a compiler, not a runtime framework. It compiles .svelte files to minimal vanilla JS at build time. No virtual DOM, no runtime diff. Svelte 5 (2024) introduced runes ($state, $derived, $effect) as the new reactivity primitive.

Angular

Google (Miško Hevery). AngularJS (2010) was rewritten as Angular 2 (2016) in TypeScript — a clean break. Full framework: dependency injection, router, forms, HTTP client, CLI. TypeScript-first from day one. Angular 17+ (2023) introduced signals for reactivity and new control flow syntax.

Solid.js

Ryan Carniato (2018). React-like syntax (JSX, components) but with fine-grained reactivity and no virtual DOM. Signals are the primitive. Excellent benchmark performance.

Meta-Frameworks — The 2020s Dominant Form

A meta-framework wraps a UI framework with routing, SSR/SSG, data loading, and deployment integration. They are now the standard way to start a production web application.
