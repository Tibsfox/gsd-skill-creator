If you list the most important software running in the world today — kernels, databases, language runtimes, web servers, codecs, browsers, embedded firmware — almost all of it is written in C. Not as an ideological choice, but because when you need absolute control of memory, direct hardware access, predictable performance, and a stable ABI, C is still the answer five decades after it was invented. This is the catalog of what C actually runs.

Operating System Kernels

Linux Kernel

Linus Torvalds, 1991. Announced on comp.os.minix August 25, 1991: “I’m doing a (free) operating system (just a hobby, won’t be big and professional like gnu) for 386(486) AT clones.” It became the most successful open source project in history.

MetricValue
Total lines of code~30 million LOC
Primary languageC (with small amounts of assembly)
RustAccepted into tree in kernel 6.1 (Dec 2022). Rust for Linux project led by Miguel Ojeda. Several drivers now in Rust
Contributors15,000+ developers have contributed
Release cadence~9 weeks between major releases
Current6.13+ (2026)
Code style“Kernel Coding Style” (Documentation/process/coding-style.rst) — 8-space tabs, 80 columns, snake_case, Torvalds-approved

The Linux kernel uses heavy GCC extensions: __attribute__, statement expressions (({ ... })), typeof, anonymous structs, and more. It barely compiles with anything other than GCC or Clang. CONFIG_CC_IS_CLANG has been a supported option since ~2019 and is now mainstream.

XNU — macOS / iOS Kernel

X is Not Unix. Apple’s kernel, based on the Mach microkernel (CMU, 1985) + BSD (mostly FreeBSD-derived) + IOKit (C++ device driver framework) + Apple extensions. The Mach part is C, the BSD part is C, the IOKit part is C++. Open-sourced as part of Darwin. Every macOS, iOS, iPadOS, watchOS, and tvOS device runs XNU.

Windows NT Kernel

Dave Cutler’s team at Microsoft, early 1990s. Originally targeted MIPS (!), then added i386, Alpha, PowerPC, Itanium, x86-64, and ARM64. Written mostly in C with small amounts of C++ and assembly. Proprietary. Mark Russinovich’s Windows Internals books are the canonical external documentation.

Databases Written in C

SQLite — The Most-Deployed Database in the World

D. Richard Hipp, 2000. Public domain (not just open source, actually PD). Designed originally for Tcl scripts and embedded systems. Now runs in every iOS app, every Android app, every macOS app, every major browser (Chrome, Firefox), every mobile operating system, airplane black boxes, Bloomberg terminals, and countless other places.

SQLite’s test suite has 92 million lines of test code testing ~150 thousand lines of implementation. It is one of the most thoroughly tested pieces of software in existence. The coverage metric is 100% MC/DC (modified condition / decision coverage), the standard used in avionics certification.

Language Runtimes Written in C

Every time you run a Python script, the CPython interpreter — written in C — is what actually executes. Every time Perl runs, it’s perl the C program. Every PHP request is handled by Zend Engine, written in C. This is C’s secret monopoly: the language you think you’re running is itself just a program running on top of C.

Embedded Systems & Firmware

C is the only language that runs everywhere embedded. Every 8-bit AVR in an Arduino, every ARM Cortex-M in an STM32, every ESP32 in your smart plug, every RISC-V chip in a research board, every microcontroller shipping in an appliance — all of them run C. MicroPython and CircuitPython run on top of C interpreters. Rust embedded is growing but still a rounding error compared to C.

Embedded C Characteristics

FeatureWhy It Matters

Direct register accessMemory-mapped I/O via volatile pointers
No recursion (often)Stack size is fixed and small — overflow is a reboot
Interrupt handlersMust be reentrant, short, non-blocking

Aerospace & Mission-Critical

When lives are on the line, the language is almost always C (or Ada for some European projects). Modern C with MISRA-C restrictions, formal static analysis, and exhaustive testing. JPL’s Mars rovers. The ISS. Automotive brake controllers. Pacemakers. Airliner flight computers. Every time you board a plane, you are trusting C code.
