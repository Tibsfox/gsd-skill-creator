3
Launch Wave 0 first: haiku agent builds the source index, schema, and sensitivity checklist from Stage 2.
4
Execute Wave 1 tracks A and B in parallel. Track C (Events) follows after A completes. Each agent receives the Stage 2 Research Reference as context.
