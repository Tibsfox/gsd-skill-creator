Handing to GSD / skill-creator

  1. Download both fec_mission.pdf and fec_mission.tex from above.
  2. Place the PDF in your GSD project's .planning/staging/inbox/ directory (or use the deposit_mission MCP tool if the planning-bridge server is active).
  3. Open a Claude Code session and provide the mission package path. The Stage 3 component specs are self-contained — each agent can execute from its spec alone.
  4. Begin with Wave 0: run the schema generation pass (Paula/Haiku) to produce schema.json, sources.bib, and the shared glossary.
  5. Launch Wave 1A in parallel: three agents (CRAFT-THEORY, CRAFT-CODE, CRAFT-DECODE) running simultaneously, each consuming Wave 0 outputs.
  6. After CAPCOM Gate 1 review, launch Wave 1B (CRAFT-APP + CRAFT-FRONTIER), then Wave 2 synthesis, then Wave 3 publication.
