History & Evolution

From Punch Cards to RISC-V — 80 Years of Machine-Level Programming
Overview x86-64 ARM64 Microarch Tools Modern Rosetta Learn History
Before assemblers, programmers wrote raw binary on punch cards. The invention of symbolic assembly — giving names to opcodes and addresses — was one of the most important abstractions in computing history.

Timeline

1945 von Neumann architecture — stored-program concept. Instructions and data in the same memory.
1947 Kathleen Booth (Birkbeck) writes the first assembly language concepts.
1949 EDSAC (Cambridge) — Maurice Wilkes and David Wheeler use initial orders (a primitive assembler). Wheeler’s work earns him the first CS PhD.
1954 Nathaniel Rochester writes the first symbolic assembler for the IBM 701.
1964 IBM System/360 — BAL (Basic Assembly Language). 16 GPRs, byte addressing. The architecture that dominated mainframes for decades.
1970 PDP-11 (DEC) — MACRO-11 assembler. Clean, orthogonal design. Inspired C.
1974 Intel 8080 — the first widely-used microprocessor assembly language.
1975 MOS 6502 — Apple II, Commodore 64, NES. Simple, beloved.
1976 Zilog Z80 — CP/M, ZX Spectrum, TI calculators.
1978 Intel 8086 — the start of the x86 lineage. 16-bit, segmented memory.
1979 Motorola 68000 — Amiga, Atari ST, Mac, Sega Genesis. Clean 32-bit-ish design.
1980 RISC concept — John Hennessy (MIPS, Stanford) and David Patterson (RISC-I, Berkeley).
1985 ARM — Acorn RISC Machine, designed by Sophie Wilson and Steve Furber.
1985 Intel 80386 — 32-bit x86. Protected mode. Virtual memory. The foundation of modern x86.
1993 Pentium — superscalar x86 (two pipelines).
1995 Pentium Pro — out-of-order execution (P6 microarchitecture). Changed everything.
1997 MMX — first x86 SIMD. 64-bit packed integer operations.
1999 SSE — 128-bit SIMD, packed floating-point.
2003 AMD64 (x86-64) — AMD extends x86 to 64-bit. Intel adopts it as EM64T/Intel 64.
2007 iPhone — ARM processors become the dominant CPU family by volume.
2010 RISC-V — open-source ISA, UC Berkeley. Krste Asanovic, Andrew Waterman.
2011 AVX — 256-bit SIMD on x86.
2012 Compiler Explorer launched by Matt Godbolt. Changed how people learn assembly.
2016 AVX-512 — 512-bit SIMD.
2018 Spectre/Meltdown disclosed. Changed CPU design philosophy.
2020 Apple M1 — ARM-based desktop CPU. Proved ARM can match x86 performance.

CISC vs RISC

CISC

Complex Instruction Set. Variable-length instructions, many addressing modes, memory-to-memory operations. x86, VAX, S/360. “Do more per instruction.” The decoder is complex; the programmer has rich tools.

RISC

Reduced Instruction Set. Fixed-length instructions, load/store only, large register files. ARM, MIPS, RISC-V, SPARC. “Do less per instruction, but do it faster.” Simpler decoder, more registers, easier to pipeline.

The Blur

Modern x86 is externally CISC but internally RISC: the decoder translates x86 instructions into micro-ops that execute on a RISC-like core. The distinction matters less than it did in 1985.

The Winners

x86-64 won desktops/servers via backward compatibility. ARM won mobile/embedded via power efficiency. RISC-V is the open challenger. MIPS, SPARC, PowerPC, Itanium lost.
