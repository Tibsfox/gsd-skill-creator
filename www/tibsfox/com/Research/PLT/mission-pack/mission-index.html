Usage

Feeding to GSD / Skill-Creator

  1. Download the Mission Package PDF — this is the complete flight plan.
  2. Place it in the .planning/staging/inbox/ directory for MCP bridge pickup, or provide it directly to Claude Code.
  3. The mission is self-contained: Stage 1 (Vision) frames the scope, Stage 2 (Research) provides the evidence base, Stage 3 (Mission) decomposes into executable wave plans.
  4. The Planting Guide PDF is a standalone deliverable — it's what the mission produces. Use it directly for this week's planting decisions.

Recompiling LaTeX Sources

Both .tex files require XeLaTeX with DejaVu fonts. Three passes for cross-references:

# Mission package
xelatex -interaction=nonstopmode pnw-planting-mission.tex
xelatex -interaction=nonstopmode pnw-planting-mission.tex
xelatex -interaction=nonstopmode pnw-planting-mission.tex

# Standalone guide
xelatex -interaction=nonstopmode pnw-planting-guide-march-26-2026.tex
xelatex -interaction=nonstopmode pnw-planting-guide-march-26-2026.tex
xelatex -interaction=nonstopmode pnw-planting-guide-march-26-2026.tex

Ubuntu/Debian: sudo apt install texlive-xetex fonts-dejavu
