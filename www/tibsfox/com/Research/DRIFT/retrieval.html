Overview

Retrieval drift occurs on the output side of RAG systems: even when retrieval appears to succeed, the generation layer can semantically drift from the retrieved context toward parametric priors, effectively ignoring the retrieved documents. Module C covers four primary phenomena: Response Semantic Drift in diffusion-LM pipelines, temporal grounding via the Chronos event-evolution graph, the Semantic Grounding Index as the primary faithfulness metric, and context-entropy dilution in iterative BEE-RAG systems.

0.92–1.28
Cohen’s d: SGI grounded vs. ungrounded generation — The Semantic Grounding Index (SGI, 2025) reports Cohen’s d of 0.92–1.28 between pairs of grounded and ungrounded outputs, representing a large-to-very-large effect size. This means grounding faithfulness is reliably measurable in production, not just in controlled settings (SGI 2025, arXiv:2512.13771).

1. Response Semantic Drift in Diffusion-LM RAG

RSD (2026, arXiv:2601.11342) characterized a drift mechanism specific to diffusion-based language model RAG pipelines: the denoising process interpolates between retrieved passages and parametric priors, and in high-prior-density topic regions, the output can semantically drift toward what the model “already knows” rather than what was retrieved. This is distinct from retrieval failure (the right document was found) and from output hallucination (the model is not confabulating); it is a generation-layer faithfulness failure where the denoising trajectory is pulled toward a parametric attractor.

RSD introduces this as a separate drift surface requiring its own monitoring: standard RAG evaluation (retrieval precision, answer exact-match) does not detect it.

2. Chronos Event-Evolution Graph for Temporal Retrieval

Liu et al. (2026) developed Chronos, a system that builds an event-evolution graph over time-stamped documents and uses it to provide temporally-aware retrieval (arXiv:2604.05096). The core insight: many RAG drift failures are temporal — the retrieved document was accurate at time T but the world has moved to state T+k. Chronos models document facts as nodes in a directed temporal graph, with edges representing supersession, confirmation, or contradiction events. Retrieval is conditioned on the query timestamp, returning the most temporally proximate valid facts rather than the most semantically similar static documents.

3. Semantic Grounding Index

The Semantic Grounding Index (SGI, 2025, arXiv:2512.13771) measures how faithfully an LLM’s generated text reflects the content of retrieved context documents rather than leaking parametric knowledge. SGI computes sentence-level entailment scores between output claims and retrieved passages. The reported effect size of Cohen’s d 0.92–1.28 between grounded and ungrounded generation pairs establishes SGI as a reliable production metric: the grounded vs. ungrounded distinction is large enough to detect at the individual-response level, not just at the population level (SGI 2025).

SGI is the retrieval surface’s primary quality metric, distinguishing “semantic laziness” (parametric leakage) from genuine source-faithful responses. The term “semantic laziness” denotes the failure mode where a model produces a plausible-sounding answer from parametric memory rather than grounding in the provided context, even when the context is directly relevant.

4. BEE-RAG Context-Entropy Dilution

BEE-RAG (2025, arXiv:2508.05100) studied context-entropy dilution in iterative RAG loops: as multiple retrieval rounds accumulate retrieved passages, the attention entropy over the context window increases, diluting the model’s effective focus on any single retrieved passage. This is a different failure mode from RSD (which is about generation-layer pull) and from standard context overflow (which is about length limits) — it occurs within the capacity budget and arises from the model’s attention mechanism distributing focus across increasingly many retrieved passages. BEE-RAG’s entropy-aware retrieval strategy mitigates this by selecting passages that minimize redundancy in the attention distribution.
