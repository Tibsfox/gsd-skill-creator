Contents

Stage 1 — Vision Document

  • Vision narrative: accounting as the grammar of economic life
  • 5 core problems: licensure complexity, scope-of-practice, firm licensing, record retention, enforcement
  • Layered protocol stack model (federal standards → state statute → board rules → professional standards)
  • 5 research modules: CPA Licensure, Practice Standards, Firm Licensing, Business Compliance, Enforcement
  • 12 testable success criteria

Stage 2 — Research Reference

  • M1: Three CPA licensure pathways (Nov 2025 changes), exam structure, ethics, CPE (120 hrs/3 yrs)
  • M2: GAAP hierarchy, board-rule-prevails principle (WAC 4-30-048), independence, integrity
  • M3: Firm licensing triggers, peer review (2024 compilation exemption), firm mobility
  • M4: Record retention master table (DOR 5yr, L&I 3yr, IRS 3-7yr), scope-of-practice boundary table
  • M5: Disciplinary authority ($30K fines), prohibited acts catalog, consumer complaint process

Stage 3 — Mission Package

  • Squadron activation profile (12 roles)
  • 4-wave execution plan with 2 parallel tracks
  • 9 deliverables with acceptance criteria
  • 32 tests (5 safety-critical, 16 core, 6 integration, 5 edge case)
  • ~71,500 token budget across ~18 context windows

Usage Instructions

  1. Review the Vision Document (Stage 1) to confirm scope and the 5-module decomposition.
  2. Check the Research Reference (Stage 2) — all citations trace to RCW sections, WAC rules, or agency publications.
  3. Hand the complete package to gsd-skill-creator via Claude Code for execution.
  4. CAPCOM checkpoints occur at wave boundaries for human approval.
  5. Note the November 2025 licensure pathway changes and the Pathway 3 expiration date (December 31, 2035).

Recompiling the LaTeX

  1. Requires XeLaTeX with DejaVu fonts installed.
  2. Run three passes:
    xelatex -interaction=nonstopmode accounting-mission.tex (×3)
