This project covers both the person and the language. Blaise Pascal was a French mathematician, physicist, and philosopher who built the first mechanical calculator at age nineteen, co-founded probability theory with Fermat, proved the existence of vacuum against the Aristotelian orthodoxy, and produced two of the most important works of French prose — the Lettres provinciales and the Pensées. He died at thirty-nine. When Niklaus Wirth designed a new programming language for teaching structured programming in 1970, he named it after Pascal because of Pascal’s contributions to mechanical calculation — specifically the Pascaline (1642), a brass gear-driven adding machine that could carry tens automatically. The language inherited the name, and the name carried the weight of four centuries of intellectual ambition.

Hello World, Pascal Style

program HelloWorld;
begin
  writeln('Hello, world!');
end.

The program keyword declares the program name. begin and end. delimit the main block — note the period after end, which marks the end of the entire program (not a semicolon). writeln is a built-in procedure, not a library function — a design choice Wirth made to keep the language self-contained for teaching. Single quotes delimit string literals. The entire program fits in four lines. This is the simplicity that won a generation of computer science departments.

A Record Type and Pointer

type
  PStudent = ^Student;
  Student = record
    name: string[40];
    grade: integer;
    next: PStudent;
  end;

var
  head: PStudent;

begin
  new(head);
  head^.name := 'Pascal';
  head^.grade := 100;
  head^.next := nil;
  writeln(head^.name, ': ', head^.grade);
  dispose(head);
end.

Forward pointer declarations (PStudent = ^Student before Student is defined) allow recursive data structures in a single-pass compiler. The ^ operator dereferences a pointer. new allocates heap memory; dispose frees it. This is the foundation of every data structure in Pascal — linked lists, trees, and graphs are all built from records and pointers.
