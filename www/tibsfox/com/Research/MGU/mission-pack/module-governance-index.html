Usage Instructions

  1. Point GSD's new-project at the vision section of the PDF
  2. Wave 0 produces manifest, shared schemas, and content inventory
  3. Wave 1A promotes bridge spec + creates pointer (Track A)
  4. Wave 1B implements upstream monitor + governance contracts (Track B)
  5. Wave 2 cross-validates all deliverables and runs monitor test harness
  6. Wave 3 commits to dev branch and updates document index
