How to Use This Package

  1. Open mission.pdf and read Stage 1 (Vision) to orient yourself to the mission scope and the eight modules.
  2. Hand mission.pdf to a GSD Orchestrator Claude Code session as the primary spec document. The Stage 3 wave plan is self-contained — no additional context required.
  3. Begin with Wave 0: have Haiku generate the tag schema (docs/TAG-SCHEMA.md), file inventory, and Navigator Index scaffold before launching parallel Wave 1 tracks.
  4. Run Wave 1A, 1B, 1C in parallel across three sessions. Cache the file inventory and milestone map across all three agents.
  5. Complete the CAPCOM Safety Gate — human review of all safety annotations — before allowing Wave 2 to begin. This is a hard architectural requirement.
  6. After Wave 3 verification passes (typedoc clean, AJV validates, links resolve), merge to dev branch and update the docs site.
