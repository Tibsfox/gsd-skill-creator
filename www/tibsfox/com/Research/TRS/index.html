Dual focus. This research project covers trace scheduling from two angles. First, trace scheduling as a compiler optimization technique for VLIW architectures — the original 1981 insight by Joseph Fisher that if basic blocks average only 4–7 instructions, a compiler must look across branch boundaries to find enough instruction-level parallelism to fill a wide instruction word. Trace selection picks the most-likely execution path; trace compaction packs operations from that path into parallel slots; bookkeeping repairs all the other paths.

Second, the deep structural analogy between trace scheduling and AI agent orchestration. Context windows are VLIW slots — fixed-width resources that must be filled efficiently. Model tiers are functional units — each with different capabilities and latencies. Token budgets are instruction count limits — exceeding them is a hard fault. The formal mapping table in the AI Crossover module makes this correspondence precise.

Timeline

YearEvent
1976FPS AP-120B — early attached array processor with long instruction word, precursor to VLIW thinking
1981Joseph Fisher publishes “Trace Scheduling: A Technique for Global Compaction” — the founding paper
1983ELI-512 — Fisher’s experimental VLIW machine at Yale, 512-bit instruction word
1984Multiflow Computer founded by Fisher, Rau (briefly), and others to commercialize VLIW
1985John Ellis publishes the Bulldog compiler thesis — first complete trace scheduling compiler
1987Multiflow TRACE series ships; Cydrome Cydra-5 ships (software-pipelined VLIW)
1990Multiflow bankrupt — VLIW ahead of its time for general-purpose computing
1992Hwu, Mahlke, Chen publish superblock and hyperblock techniques at UIUC (IMPACT compiler)
1994HP and Intel announce IA-64 partnership — EPIC (Explicitly Parallel Instruction Computing)
2001Itanium launched — IA-64 reaches production, 128-bit bundles, predication, speculation
2012AMD transitions GPU architecture from VLIW4/5 to GCN (scalar + vector), ending GPU VLIW era
2021Intel discontinues Itanium (Kittson) — end of IA-64, but the ideas live on
2024VLIW thriving in DSP and mobile — Qualcomm Hexagon, TI C6x, billions of units shipped
