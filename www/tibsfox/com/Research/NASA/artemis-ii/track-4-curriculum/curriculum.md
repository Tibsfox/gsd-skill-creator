# College of Knowledge — Artemis II Curriculum

*Department mappings populated post-mission.*

## Primary Departments

1. **Aerospace Engineering** — SLS propulsion, staging, trajectory design
2. **Optical Communications** — O2O laser link, photonics, signal processing
3. **Materials Science** — AVCOAT ablative heat shield, thermal protection

## Secondary Departments

4. **Aerospace Medicine** — AVATAR organ-on-chip, ARCHAR biomarkers, radiation dosimetry
5. **Orbital Mechanics** — Free-return trajectories, patched conics, sphere of influence
6. **Mathematics** — Sophie Germain, elasticity theory, Fermat's Last Theorem
7. **History of Science** — First woman beyond LEO, international cooperation (ESA, CSA)

## TRY Session (planned)

Build a simple optical communication link using an LED, photodiode, and Arduino. Transmit data across a room using light pulses. Compare data rate to radio (speed of light is the same — the bandwidth is in the modulation).

## DIY Project (planned)

Free-return trajectory simulator using Python + poliastro. Input: TLI burn parameters. Output: does the spacecraft come home? Visualize the trajectory in 2D. Use real Artemis II mass and delta-v values.
