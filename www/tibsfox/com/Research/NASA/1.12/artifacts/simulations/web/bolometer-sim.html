How it works:
Black hemisphere: absorbs ALL radiation (solar + reflected + thermal).
White hemisphere: reflects visible light (~90%), absorbs only thermal IR.

Difference = solar + reflected component.
This separation is Suomi's key insight for measuring the radiation budget from orbit.
Try this:
1. Set solar to 0 (eclipse). Both sensors see only thermal — temperatures equalize.
2. Increase solar. Black heats up fast, white barely changes — the difference IS the solar measurement.
3. Increase albedo. More reflected light → black gets even more input.
