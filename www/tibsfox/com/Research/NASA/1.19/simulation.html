A. Simulations — What to Build Locally

A1. Python: Suborbital Trajectory Comparator (Shepard vs Ham vs Gagarin)

Python numpy + matplotlib Beginner-Intermediate 2–3 hours

Computes and visualizes the complete suborbital trajectories of Freedom 7 (Shepard), MR-2 (Ham), and Vostok 1 (Gagarin) on the same diagram. User adjusts launch parameters (burnout velocity, flight path angle, burnout altitude) and watches how the trajectory changes.

  • Plot 1: Earth-centered trajectory view — trajectories are TINY compared to Earth; inset zooms the two suborbital arcs
  • Plot 2: Altitude vs time — parabolic arcs vs Gagarin’s constant-altitude orbit
  • Plot 3: Velocity vs time — “Orbital velocity (7800 m/s) is 3.4x Freedom 7’s max, but requires 11.6x more energy (v² scaling)”
  • Plot 4: G-force vs time — Shepard’s clean symmetric profile vs Ham’s rougher abort-induced curve
The humbling scale: both suborbital trajectories are barely visible arcs against Earth’s curvature, while Gagarin’s orbit wraps around the entire planet.

A2. Python: Manual Control Bandwidth Calculator

Python numpy + matplotlib + scipy Intermediate 2–3 hours

Models the human-in-the-loop control bandwidth for Mercury manual attitude control. Inputs: human reaction time, thruster characteristics, capsule inertia. Outputs: control loop frequency, Shannon information rate, fuel consumption rate. Side-by-side comparison of Shepard’s proportional control vs the ASCS bang-bang control.

  • Plot 1: Attitude vs time — human (smooth) vs ASCS (sawtooth)
  • Plot 2: Cumulative fuel consumption — human uses 20–40% less fuel
  • Plot 3: Thruster firing pattern — human looks purposeful; ASCS looks like noise
  • Plot 4: Information rate bar chart — “Higher bandwidth does not always mean better control”
The ASCS has higher bandwidth (more bits/s) but lower efficiency (more fuel per degree). The human has lower bandwidth but higher semantic content per bit.

A3. Web: “Can You Fly Freedom 7?” — Manual Attitude Control Game

HTML5 Canvas + JS Self-contained All levels

Browser-based manual attitude control simulator. Fire H2O2 thrusters with keyboard (W/S/A/D/Q/E) to maintain Mercury capsule attitude during the five-minute weightless phase. No damping — the capsule rotates until you fire the opposing thruster. Four phases: orientation, hold, maneuver sequence (Shepard’s actual test), and reentry attitude. Performance compared to Shepard’s actual data.

  • 3D attitude ball with horizon, pitch ladder, roll indicator, target attitude ghost
  • Simulated periscope view of Earth rotating with capsule attitude
  • Telemetry panel: pitch/yaw/roll, angular rates, fuel remaining
  • Scoring: “Shepard” rank = <2° avg deviation, >60% fuel remaining
The first time the player fires a thruster and the capsule keeps rotating after they release the key. In space, there is NO damping. You have to stop it yourself.

A4. Web: Peregrine Falcon Stoop Physics Simulator

HTML5 Canvas + JS Self-contained All levels

Interactive stoop from 1000 meters altitude onto a pigeon at 200 meters. Control wing tuck angle (mouse Y-axis) and bank angle (mouse X-axis). Real-time velocity, g-force, and intercept geometry. Standard atmosphere drag model. Goal: maximum speed at intercept while keeping pullout g below 25g.

  • Side view with altitude/distance axes, falcon silhouette, trajectory trail color-coded by speed
  • Physics panel: gravity, drag, net force, terminal velocity at current tuck
  • G-force meter (0–30g): green/yellow/red/black zones
  • Compare mode: falcon stoop alongside Freedom 7 trajectory at same time scale
Maximum stoop speed (~390 km/h) produces 24g at pullout. The falcon does this every time it hunts. Shepard experienced 11.6g once and became a hero.

A5. SPICE: AGC Circuit for Mercury Rate Gyro Signal Conditioning

ngspice Intermediate

Op-amp AGC with JFET-controlled gain, compressing 74 dB gyro dynamic range to ~20 dB for the ASCS relay-logic bang-bang switching. Uses 741 op-amp model, 2N5457 JFET, peak detector for control voltage. The same AGC principle used in AM/shortwave receivers (v1.9, v1.16) applied to spacecraft instrumentation.

A6. SPICE: Audio CW Filter — Progressive Radio Circuit #19

ngspice Beginner-Intermediate

Active audio bandpass filter using twin-T notch topology in op-amp negative feedback. Center frequency ~700 Hz (CW tone), bandwidth ~200–400 Hz. Rejects adjacent signals, SSB bleed, and power line hum. 9.5 dB SNR improvement from narrowing 2700 Hz to 300 Hz bandwidth. Design equations: f0 = 1 / (2πRC), R = 22k, C = 10 nF.

Mercury connection: Ground stations used similar narrowband filters to separate biomedical subcarrier tones. Shepard’s heart rate was telemetered on a 1.3 kHz subcarrier.

B. Machine Learning — What to Train

B1. Manual Control Style Classifier

PyTorch / TensorFlow Intermediate

Binary classifier (1D CNN or LSTM) distinguishing “proportional” (Shepard-like) from “bang-bang” (ASCS-like) control strategies. Input: 30-second thruster firing time series at 10 Hz (300 samples, 7 features per sample). Expected accuracy >95% for clean patterns, ~85% for fatigued-human vs ASCS. Most discriminative feature: pulse duration variability.

C. Computer Science — The Human-in-the-Loop Problem

C1. Supervisory Control Architecture

Architecture Design Pattern

The Mercury control hierarchy as a software architecture pattern:

  • Level 0: Physics / Infrastructure — the environment, not controlled
  • Level 1: Actuators / Service mesh — execute commands, do not decide
  • Level 2: ASCS / Automated agents — closed-loop automatic, ~2–5 Hz, bang-bang
  • Level 3: Manual (Shepard) / Human operator — closed-loop human, ~0.77 Hz, proportional
  • Level 4: Mission Control / Management — strategic, ~0.01 Hz

The Shepard lesson: Level 3 should always be able to override Level 2. The override capability costs little and provides enormous value when the automatic system encounters an unplanned situation.

D. Creative Arts — What to Compose, Write, and Render

D1. GLSL Shader: “Freedom”

GLSL 1920x1080

View from Freedom 7’s periscope at apogee — 187.5 km above the Atlantic. Circular periscope viewport with crosshair overlay and degree markings. Through the lens: the blue Atlantic, white cloud formations, the Bahamas as turquoise patches, Cape Canaveral at the edge of the field, the curve of the Earth at the horizon, and the impossibly thin blue line of atmosphere transitioning to black space. Slow drift simulates attitude motion; periodic corrections simulate thruster firings.

D2. Short Story: “Fifteen Minutes”

HTML + LaTeX 3,000–4,000 words

Creative nonfiction following Shepard’s sensory experience from T-0 to helicopter recovery. Third-person limited, present tense during flight. 13 sections keyed to mission elapsed time: ignition, tower clear, max-Q, engine cutoff, tower jettison, manual control, the view, retro-sequence, retro jettison, reentry (11.6g), drogue chute, main chute, splashdown. The prose matches the flight’s pace: rapid during launch, contemplative during weightlessness, violent during reentry.

D3. FAUST Synth: MR-3 Launch Sequence

Faust DSP 48 kHz stereo, 150 s

Physically motivated synthesis: broadband engine noise (80–150 Hz resonant bandpass), structural vibration (20–40 Hz, peak at max-Q), atmospheric wind (proportional to dynamic pressure), Doppler shift as the rocket recedes (~1 octave drop by cutoff), and the dramatic silence of engine cutoff — from 78,000 lbs thrust to zero in 100 ms.

D4. FAUST Synth: Peregrine Falcon Stoop

Faust DSP 48 kHz stereo, 12 s

The stoop from the falcon’s perspective: wind over folded feathers (pink noise, rising pitch with speed), feather flutter at high velocity (narrow-band 200–800 Hz), the sub-bass thump of wing deployment at 24g pullout, the sharp transient of the strike, and the sudden quiet after — mirroring Freedom 7’s post-cutoff silence.

D5. Circuit: LED G-Force Indicator

Hardware LM3914 Beginner, ~$8

10-LED bar display driven by LM3914 dot/bar display driver IC. Potentiometer simulates the capsule’s accelerometer. Color-coded: green (1–3g), yellow (4–6g), orange (7–9g), red (10g+). Rotate the pot to trace Freedom 7’s g-profile: 1g on pad, ramp to 6.3g at burnout, drop to 0g in weightlessness, spike to 11.6g during reentry, settle to 1g under parachute.

D6. Circuit: Analog Attitude Controller

Hardware LM358 + SG90 Servo Beginner-Intermediate, ~$15

Single-axis attitude controller: potentiometer (hand controller) drives an op-amp integrator (capsule dynamics), which drives a servo motor (attitude display). Push the pot → servo rotates at ~1°/s. Release → servo holds position (no damping in space). Push the other way to stop. The student learns exactly why Shepard’s manual control test was not trivial.

E. Problem-Solving — The Qualification Test

E1. Qualification vs Survival

Methodology

Survival: The system encounters conditions beyond its design envelope and does not fail. Ham’s MR-2 survived 14.7g (design limit: 12g). This demonstrates robustness.

Qualification: The system encounters conditions within its design envelope and performs as predicted. Shepard’s MR-3 matched every prediction within engineering tolerance. This demonstrates control.

NASA required qualification, not survival, before flying Glenn. A stress test demonstrates robustness. A qualification test demonstrates control. Both are valuable. Neither substitutes for the other. Ship after qualification, not after survival.

The Mercury Ladder: MR-1 (failure→learning) → MR-1A (success) → MR-2 (survival) → MR-BD (fix) → MR-3 (qualification) → MR-4 (repeat qualification) → MA-4 (success) → MA-5 (qualification) → MA-6 (operational).

F. GSD Improvements

F1. Human-in-the-Loop Override Chipset

Chipset Design

Four chips: override-switch (state machine: AUTOMATIC → HANDOFF → MANUAL → RELEASE → AUTOMATIC), proportional-input (nuanced “do this much” inputs vs binary do/don’t-do), efficiency-monitor (flag if human intervention is using more resources than automatic), and context-capture (snapshot full agent state for seamless resume). The human does not replace the automatic system. The human complements it.

F2. Qualification Test Pattern

Chipset Design

Four chips: spec-oracle (verify predicted performance, not just non-failure), tolerance-band (define acceptable ranges — “survived” is not “qualified”), regression-guard (detect drift toward survival mode), and qualification-ladder (unit → integration → system → qualification → operational). Only advance when the current level qualifies.

Three organisms, three time scales: 15 milliseconds, 10 seconds, 928 seconds. Three leaps: the lunge, the stoop, the launch. Three commitments that cannot be unmade once they begin. Kierkegaard, born this day 148 years before the launch, knew that the hardest thing about a leap is not the landing but the leaving — the instant when both feet are off the ground and the only thing holding you up is the trajectory you chose and the physics you trusted. Shepard jumped. Fifteen minutes later, America was in space, and the question was no longer “can we?” but “how far?”
