Template Reference — McNeese & Hoag (1959)

Donald G. McNeese and Albert L. Hoag's Engineering and Technical Handbook (Prentice-Hall, 1959. 376 pages. Library of Congress Card No. 57-6690. Third printing September 1959.) organized engineering knowledge with enough care that it was reprinted seven times. Its three-element discipline — Tables / Formulas / Examples, every page earning its weight — is adopted as the structural template for TSPB. The McNeese-Hoag Cross-Reference Index (Appendix C of TSPB) maps every significant table and formula in the 1959 handbook to its TSPB equivalent, noting what has been updated with modern values and what has been extended with mathematics not available in 1959 (Euler's formula, orbital mechanics, information theory, L-systems).

How to Run Part G

  1. Wait until a complete batch of missions has run Parts A–F (e.g., all Apollo-era missions, or a version milestone boundary).
  2. Collect all RETRO output files from the batch into .planning/retros/{BATCH_ID}/.
  3. Run Part G: gsd run skill-creator --template nasa_partg_template.pdf --input {BATCH_ID}
  4. Wave 0 produces findings-registry.json — review the P0 items before Wave 1 proceeds (CAPCOM gate).
  5. Waves 1–4 run sequentially, each pass improving the previous. Approve CAPCOM gate after Pass 5 QA metrics are met.
  6. Wave 5 (Pass 8) authors new TSPB chapters and runs tools/validate-formulas.py — all tests must pass before compilation.
  7. Wave 6 compiles TSPB.pdf with bash tools/compile.sh (three-pass XeLaTeX).
  8. Approve final CAPCOM gate. Release tagged v1.50.{BATCH_ID}.INT.

Recompile TSPB

From the docs/TSPB/ directory:

cd docs/TSPB
