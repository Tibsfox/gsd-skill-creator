How to Hand Off to GSD Orchestrator

  1. Place the nasa_mission_history.pdf in your GSD project's .planning/reference/ directory.
  2. Open Claude Code and invoke the skill-creator: gsd run skill-creator --mission nasa_mission_history.pdf
  3. The orchestrator reads the Stage 3 wave plan and activates the Fleet profile. Wave 0 runs first (schema + source index).
  4. Approve each CAPCOM gate as waves complete. Wave 3B is the final human review before document delivery.
  5. The full reference document is deposited to .planning/outputs/nasa-mission-history/ as PDF + .tex source.

Recompile the LaTeX Source

To update active mission data or add new sections, edit nasa_mission_history.tex and recompile with XeLaTeX (three passes required for TOC and page numbers):

xelatex -interaction=nonstopmode nasa_mission_history.tex
xelatex -interaction=nonstopmode nasa_mission_history.tex
xelatex -interaction=nonstopmode nasa_mission_history.tex

Requires: texlive-xetex, fonts-dejavu. On Ubuntu: sudo apt install texlive-xetex texlive-fonts-recommended texlive-latex-extra fonts-dejavu
