How to Execute with GSD / skill-creator

  1. Open this package in Claude Code or gsd-skill-creator environment.
  2. Hand Stage 2 (Research Reference) to the orchestrator as the shared system-prompt substrate for all Wave 1 agents — compute once, cache for all five module executors.
  3. Launch Wave 0 (Haiku): generate schema.yaml, source_index.json, and five module templates.
  4. Launch Waves 1A and 1B concurrently: Track A (Modules 01 + 05, Sonnet) and Track B (Modules 02 + 03, Sonnet) run in parallel. Opus cultural sensitivity review runs alongside Track A.
  5. CAPCOM gate: Review Modules 01–03 and 05 before launching Wave 1C (Module 04, Opus).
  6. Launch Wave 1C: Opus produces Module 04 (Cultural Impact); Sonnet verifies all producer-DAW source attributions.
  7. CAPCOM gate: Full document review before Wave 2 synthesis.
  8. Wave 2 (Opus + Sonnet): Cross-module integration, through-line composition, final assembly.
  9. Wave 3 (Sonnet + Haiku): PDF rendering, source validation sweep, verification matrix population.
