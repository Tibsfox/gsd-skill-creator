Perl was born in 1987 from a linguist’s frustration. Larry Wall, trained in linguistics at UC Berkeley and SIL International, built Perl because neither sed, awk, nor shell scripting could handle the complex text-processing reports he needed. That origin — a linguist building a programming language — is why Perl dominated text processing for two decades, and why Perl’s regex engine became the template every modern language copied.

Regular Expressions — The Engine That Changed Everything

Perl did not invent regex. Ken Thompson implemented them in ed (1968). But Larry Wall took them and made them a first-class citizen of a programming language. By Perl 5 (1994), the regex engine had features no other language matched:

PCRE (Perl-Compatible Regular Expressions): Philip Hazel studied Perl’s regex in 1997 and created PCRE as a C library. It became the regex engine behind PHP (preg_*), Nginx, Apache, R, Julia, and influenced Python, Ruby, Java, and JavaScript. The name says it all: Perl-Compatible.

Named Captures — Self-Documenting Patterns

my $line = "2026-04-08 ERROR: disk full on /dev/sda1";

if ($line =~ /(?<date>\d{4}-\d{2}-\d{2})\s+(?<level>\w+):\s+(?<msg>.+)/) {
    say "Date:    $+{date}";    # 2026-04-08
    say "Level:   $+{level}";   # ERROR
    say "Message: $+{msg}";     # disk full on /dev/sda1
}

The /x Modifier — Regex as Prose

my $email_re = qr{
    (?<user>                     # local part
        [a-zA-Z0-9._%+-]+       #   alphanumeric plus specials
    )
    @                            # literal @
    (?<domain>                   # domain part
        [a-zA-Z0-9.-]+          #   domain name
        \.                      #   dot separator
        [a-zA-Z]{2,}            #   TLD (2+ chars)
    )
}x;

Before /x, regex was write-only line noise. With /x, regex became prose. Larry Wall got there first because he was a linguist who cared about readability.

Advanced Engine Features (Perl-First or Perl-Only)

# Backtracking control: match "bar" but NOT inside "foo"
$str =~ /foo(*SKIP)(*FAIL)|bar/;

# \K: reset match start (keep prefix out of $&)
$str =~ /prefix\Kwanted/;           # $& is "wanted"

# Recursive patterns: balanced parentheses
$str =~ /(?<paren> \( (?: [^()]++ | (?&paren) )* \) )/x;

# Atomic groups -- prevent backtracking
$str =~ /(?>a+)b/;

# Conditional patterns
$str =~ /(a)?(?(1)b|c)/;           # if group 1 matched, expect b

The One-Liner Culture

Perl’s command-line switches make it the ultimate shell companion: -e (execute), -n (loop over input), -p (loop + print), -a (autosplit), -F (field separator), -i (in-place edit).

Classic One-Liners

TaskOne-Liner
Find & replace in filesperl -pi -e 's/old/new/g' *.txt
Word frequency countperl -lane '$f{$_}++ for @F; END { print "$f{$_} $_" for sort { $f{$b} <=> $f{$a} } keys %f }' file
Extract email addressesperl -nle 'print $1 while /([a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,})/gi' file
Sum a columnperl -lane '$s += $F[2]; END { print $s }' data.tsv
Remove duplicate linesperl -ne 'print unless $seen{$_}++' file
Print lines 10–20perl -ne 'print if 10..20' file
Reverse linesperl -e 'print reverse <>' file
CSV → TSVperl -MText::CSV_XS -ne 'BEGIN{$c=Text::CSV_XS->new} $c->parse($_); print join("\t",$c->fields),"\n"' file.csv
JSON pretty-printperl -MJSON::PP -0e 'print JSON::PP->new->pretty->encode(decode_json(<>))' file.json
Count lines/words/charsperl -lne '$l++; $w+=split; $c+=length; END { print "$l $w $c" }' file

Unicode Support

Full Unicode 13+ support since Perl 5.32. The \p{} property escapes cover the entire Unicode Character Database:

use utf8;
use open ':std', ':encoding(UTF-8)';

$text =~ /\p{Script=Greek}/;        # Greek characters
$text =~ /\p{Emoji}/;               # emoji (5.26+)
$text =~ /\p{Lu}/;                  # uppercase letter (any script)
$text =~ /\p{Han}/;                 # CJK ideographs
$text =~ /\b\p{L}+\b/g;            # all "words" in any script
$text =~ /\N{GREEK SMALL LETTER ALPHA}/;  # named character

Native Unicode case folding, normalization-aware matching (Unicode::Normalize), and grapheme cluster matching with \X.
