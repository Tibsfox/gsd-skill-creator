“Perl is designed to give you several ways to do anything, so consider picking the most readable one.” — Larry Wall. Perl supports procedural, object-oriented, and functional paradigms simultaneously. It runs on 100+ platforms. The type system is dynamic, the philosophy is pragmatic, and the language meets you where you are.

Context sensitivity: Perl’s most distinctive feature. Expressions behave differently in scalar context vs. list context: @arr in scalar context returns the array length. localtime() in list context returns 9 values; in scalar context, a formatted string. This is a linguistic concept — words meaning different things in different contexts — applied to programming.

References & Complex Data Structures

# References: the key to complex data in Perl
my $scalar_ref = \$name;         # reference to scalar
my $array_ref  = [1, 2, 3];     # anonymous array reference
my $hash_ref   = { a => 1 };    # anonymous hash reference
my $code_ref   = sub { ... };    # anonymous subroutine reference

# Nested structures: hash of arrays of hashes
my %data = (
    servers => [
        { name => "web01", cpu => 45, mem => 2048 },
        { name => "web02", cpu => 32, mem => 4096 },
    ],
    databases => [
        { name => "pg01", connections => 120 },
    ],
);

# Arrow notation for dereferencing
say $data{servers}[0]{name};     # "web01"
say $data{servers}[1]{mem};      # 4096

Object-Oriented Perl

Classic OOP (bless)

package Point;

sub new {
    my ($class, %args) = @_;
    return bless { x => $args{x}, y => $args{y} }, $class;
}

sub distance_to {
    my ($self, $other) = @_;
    return sqrt(($self->{x} - $other->{x})**2 + ($self->{y} - $other->{y})**2);
}

Modern OOP with Moo (lightweight) or Moose (full meta-object protocol)

package Point;
use Moo;

has x => (is => 'ro', required => 1);
has y => (is => 'ro', required => 1);

sub distance_to {
    my ($self, $other) = @_;
    return sqrt(($self->x - $other->x)**2 + ($self->y - $other->y)**2);
}

# Usage:
my $p1 = Point->new(x => 3, y => 4);
my $p2 = Point->new(x => 0, y => 0);
say $p1->distance_to($p2);  # 5

Perl 5.38+ Class Keyword (Experimental)

use v5.38;
use feature 'class';

class Point {
    field $x :param;
    field $y :param;

    method distance_to ($other) {
        return sqrt(($x - $other->x)**2 + ($y - $other->y)**2);
    }
}

The class keyword is Perl’s path to built-in OOP — no CPAN module required. It’s experimental in 5.38–5.42 but progressing toward stable. The syntax draws from Moo/Moose lessons and provides field, method, ADJUST, and inheritance via :isa.

Functional Programming Features

# Closures
sub make_counter {
    my $count = 0;
    return sub { return ++$count };
}
my $counter = make_counter();
say $counter->();  # 1
say $counter->();  # 2

# Map, grep, sort — the functional trio
my @squares = map { $_ ** 2 } 1..10;
my @evens   = grep { $_ % 2 == 0 } @numbers;
my @sorted  = sort { $a->{name} cmp $b->{name} } @records;

# Dispatch tables — hashes of code refs
my %actions = (
    create => sub { create_record(@_) },
    delete => sub { delete_record(@_) },
    update => sub { update_record(@_) },
);
$actions{$command}->(@args) if exists $actions{$command};

Modern Perl: Subroutine Signatures (5.36+)

use v5.36;  # enables strict, warnings, signatures

sub greet ($name, $greeting = "Hello") {
    say "$greeting, $name!";
}

sub sum (@numbers) {
    my $total = 0;
    $total += $_ for @numbers;
    return $total;
}

greet("World");              # Hello, World!
greet("Perl", "Greetings");  # Greetings, Perl!
say sum(1, 2, 3, 4, 5);     # 15
