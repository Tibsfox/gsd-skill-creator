Overview Core PyPI Science ML/AI Web Rosetta Learn History
Python is the #1 most-taught programming language in the world. By 2014, eight of the top ten US computer science programs taught Python as their first language. The reasons are simple: readable syntax, fast feedback loops, and a gentle learning curve. You can write meaningful programs in minutes, not days. The community has built an enormous pool of high-quality, often free, learning materials.

Learning Strategies

The Jupyter Exploration Method

The best way to learn scientific Python is by exploring interactively in a notebook:

  1. Install Jupyter: pip install jupyterlab
  2. Start it: jupyter lab
  3. Create a notebook. Import NumPy, Pandas, Matplotlib
  4. Follow a tutorial as you type. Modify every example
  5. Use ? after any name to get docs: np.array?
  6. Use tab completion obsessively
  7. Use %matplotlib inline for inline plots

Notebooks combine code, output, markdown notes, and plots in one place. It’s the closest thing to Knuth’s literate programming vision, and it’s how a generation learned data science.

The Read-the-Source Method

When you depend on a package, read its source code. PyPI packages usually link to GitHub. Study:

  • The __init__.py — public API
  • The main module file — core logic
  • The tests/ directory — shows intended usage
  • The docs/ directory — Sphinx tutorials
  • Recent PRs — what the maintainers care about

Python’s readability makes this possible. Reading the source of requests, click, or pydantic is an education in idiomatic Python.
