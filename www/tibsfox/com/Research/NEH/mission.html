The Through-Line

The NeHe corpus is a perfect example of structured, layered knowledge that maps to skill-creator's architecture. Each lesson is a skill node with defined prerequisites, concepts introduced, and successors. The 48 lessons form a directed acyclic graph — the rendering knowledge graph — that can be traversed to answer graphics questions, generate code at the right complexity level, or build multi-lesson learning sequences.

The modernization mandate transforms legacy Win32/glaux code into cross-platform Python with ModernGL or PyOpenGL + GLFW. The deprecation map is not just a compatibility tool — it is a model of how APIs evolve, how fixed-function pipelines become programmable shaders, how the architecture of graphics itself has changed over two decades.
