Overview Taxonomy Foundation Intermediate Advanced Expert Upstream Mission Pack
Jeff Molofee's NeHe Productions — Neon Helium — was the corpus that defined an entire generation of 3D graphics programmers. Between 1997 and 2012, the 48 tutorials taught OpenGL to hundreds of thousands of developers: the ones who built the games, the simulations, the visualizers. Every lesson was a handshake between a beginner and a coordinate system, between a curious programmer and a rendering pipeline.

Research Modules

M1: Taxonomy & Skill Mapping

Taxonomy Track A · Knowledge graph

Machine-readable knowledge graph for skill-creator: YAML taxonomy of all 48 lessons, JSON-LD concept DAG with prerequisite edges, skill definition for the "opengl-lesson" skill type. The intelligence backbone that all other modules flow through.

M2: Foundation Track (L01–L12)

Foundation Track B · Tier 0–1

Window setup, first triangle, color, rotation, 3D shapes, texture mapping, filters and lighting, blending, moving bitmaps, 3D world loading, flag/wave effect, display lists. The bedrock: from blank window to textured, lit 3D scene.

M3: Intermediate Track (L13–L24)

Intermediate Track B · Tier 2

Bitmap fonts, outline fonts, texture-mapped fonts, cool-looking fog, 2D texture font, quadrics, particle engine, line-based extensions, morphing, picking, reflections/clipping, TGA loading. Building the visual vocabulary.

M4: Advanced Track (L25–L36)

Advanced Track C · Tier 3

Morphing and OBJ loading, stencil buffer reflections, shadow volumes, Bezier patches, collision detection, heightmap terrain, text-to-texture, multi-texturing, cel shading, compressed textures (DXT), spring-mass physics, translucent objects.

M5: Expert Track (L37–L48)

Expert Track C · Tier 4

Cel-shading techniques, compressed textures, physical simulation, rope physics (Verlet integration), virtual worlds with ODE, multiple viewports, FreeType fonts, 3D lens flare, vertex buffer objects, fullscreen antialiasing, CG shader translation, ArcBall rotation.

M6: Upstream Intelligence

Upstream Synthesis · Integration

The artifact that makes all other modules useful inside skill-creator: intelligence index for agent queries, DACP three-part bundle schemas for graphics handoffs, deprecation map (15+ legacy patterns flagged), documentation index searchable by concept/lesson/API function.
