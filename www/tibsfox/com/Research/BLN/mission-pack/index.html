🚀 How to Use This Package

Handing Off to GSD Orchestrator

  1. Download both files (PDF and .tex) and place them in your GSD project directory.
  2. Open Claude Code and hand the PDF to your GSD orchestrator agent: "Here is the mission package. Execute the Blender User Manual mission."
  3. The orchestrator runs Wave 0 (Foundation: schema, source index, outline) — Haiku model, sequential. Approve the outline before continuing.
  4. Wave 1 runs in parallel — Track A (Modules A, B, C) and Track B (Modules D, E, F) execute simultaneously. Six writing agents produce ~250 pages of chapter content.
  5. Human approval gate at end of Wave 1 — Review all six modules before authorizing synthesis. CAPCOM agent manages this checkpoint.
  6. Wave 2 synthesis — Opus model performs cross-module integration, cohesion pass, production workflow chapter, and appendix compilation.
  7. Wave 3 publication — Final PDF assembly, verification against blender.org, index generation, and delivery.
