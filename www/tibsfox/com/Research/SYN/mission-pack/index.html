Usage

  1. Download the PDF and .tex files
  2. Place in your GSD skill-creator .planning/staging/inbox/ directory
  3. The orchestrator reads the mission package and decomposes into wave execution
  4. Wave 0 (Foundation) runs first, establishing shared schemas
  5. Wave 1 runs 3 parallel tracks: Company+IP, Market+Competitive, Integration
  6. Wave 2 synthesizes across modules and produces the GSD bridge spec
  7. Wave 3 assembles, verifies, and publishes the final document

Recompilation

To modify and recompile the LaTeX source:

xelatex -interaction=nonstopmode synsor-mission.tex
xelatex -interaction=nonstopmode synsor-mission.tex
xelatex -interaction=nonstopmode synsor-mission.tex

Requires XeLaTeX with DejaVu fonts installed. Three passes resolve TOC, cross-references, and page counts.
