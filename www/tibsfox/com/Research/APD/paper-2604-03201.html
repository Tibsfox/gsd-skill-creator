SCRAT: Coupled Control, Structured Memory, Verifiable Action

arXiv: 2604.03201

Authors: Armesto, Kolb

Date: April 2026 · Categories: cs.AI, cs.RO

Abstract

SCRAT models agent architecture after squirrel locomotion biomechanics, implementing a hierarchical control framework with coupled sensory-motor loops. The system features episodic memory with observer-state tracking, role-differentiated architecture with fast feedback loops, and verifiable action sequences. The biological inspiration enables agents that adapt to environmental complexity through layered control rather than monolithic planning.

Key Findings

  1. Hierarchical control framework with coupled sensory-motor loops outperforms flat architectures
  2. Episodic memory with observer-state tracking enables context-dependent action selection
  3. Role-differentiated architecture separates planning from execution with fast feedback loops
  4. Biological model provides principled decomposition of complex agent behaviors
  5. Verifiable action sequences enable post-hoc analysis of agent decision quality
  6. Fast feedback loops prevent the overthinking patterns identified in other MCP research

Methodology

Biomechanical analysis of squirrel locomotion patterns, translated into agent architecture primitives. The hierarchical control framework decomposes into: high-level planning (route selection), mid-level coordination (limb sequencing), and low-level execution (motor commands). Each layer operates at different temporal scales with fast feedback between levels.

Implications for Our Architecture

Maps directly to our three-level agent hierarchy: mayor-coordinator (high-level), sling-dispatch (mid-level), and polecat-worker (low-level). The role-differentiated architecture validates our separation of planning (GSD planner) from execution (GSD executor) from verification (GSD verifier). The fast feedback loops align with our nudge-sync pattern. Episodic memory with observer-state tracking should inform our beads-state persistence model.
