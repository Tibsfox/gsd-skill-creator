Reflective Context Learning with Optimization Primitives

arXiv: 2604.03189

Authors: Vassilyev et al.

Date: April 2026 · Categories: cs.AI, cs.LG

Abstract

Introduces two operators for in-context learning optimization: reflection converts execution trajectories into update signals, and mutation applies those signals to improve agent behavior. The framework borrows classical optimization concepts (batching, auxiliary losses, variance reduction) for context-space optimization. Evaluated on AppWorld, BrowseComp+, and RewardBench2, demonstrating that reflective learning in context can approach the effectiveness of weight-space optimization.

Key Findings

  1. Reflection operator: converts execution trajectories into structured update signals
  2. Mutation operator: applies update signals to improve future behavior
  3. Classical optimization primitives (batching, auxiliary losses) transfer to context space
  4. Variance reduction techniques improve learning stability across episodes
  5. Effective on AppWorld (tool use), BrowseComp+ (browsing), RewardBench2 (reward)
  6. Context-space optimization approaches weight-space optimization effectiveness

Methodology

The reflection operator extracts structured lessons from agent execution traces: what worked, what failed, and why. The mutation operator modifies the agent's system prompt or context window using these lessons. Classical optimization concepts are adapted: batch reflections across multiple episodes, use auxiliary losses to preserve previously learned behaviors, and reduce variance through ensemble reflection.

Implications for Our Architecture

This is the theoretical foundation for our skill-creator's observation/suggestion cycle. Our sc:observe and sc:digest skills implement a version of the reflection operator — extracting patterns from session data. The mutation operator maps to our skill suggestion acceptance workflow. The classical optimization framing (batching, auxiliary losses) suggests we should batch observations across sessions rather than reacting to each individually, and preserve prior learned patterns when incorporating new ones.
