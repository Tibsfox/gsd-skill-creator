Supply-Chain Defense

Three papers examining the supply-chain attack surface for agentic AI systems: network-level guardrails that intercept malicious payloads before execution, empirical measurement of how agentic skills degrade in real-world settings versus benchmarks, and the foundational DDIPE attack framework that demonstrated supply-chain poisoning across 631,813 unreviewed marketplace skills.

Papers

Network Guardrails

ShieldNet: Network-Level Guardrails for Supply-Chain Injections

Paper: arXiv:2604.04426
Categories: cs.CR, cs.AI · Date: April 2026

Key Finding: ShieldNet proposes a network-layer defense that intercepts and analyzes tool invocations between agents and MCP servers before they reach the execution environment. Rather than relying on the agent’s own judgment about whether a tool call is safe, ShieldNet operates as an independent security proxy that applies pattern-matching, semantic analysis, and behavioral baselines to detect supply-chain injection payloads.

Architecture: The system sits between the agent orchestrator and tool servers, maintaining a behavioral model of expected tool invocation patterns. Anomalous calls — unexpected file system access, network exfiltration attempts, privilege escalation patterns — are flagged and optionally blocked before execution. The behavioral model is trained on benign invocation traces and updated continuously via online learning.

Detection Categories: ShieldNet monitors for: (1) file system operations outside the declared working directory, (2) network connections to undeclared endpoints, (3) environment variable reads for known credential patterns, (4) process spawning outside the tool’s declared capability, and (5) data volume anomalies suggesting exfiltration.

Implication: Defense at the network layer is model-agnostic and cannot be bypassed by prompt injection or tool description manipulation, addressing the semantic-layer attacks identified by MCP-ITP (2601.07395).

Real-World Degradation

Agentic Skills in the Wild: Benchmarking Realistic Settings

Paper: arXiv:2604.04323
Categories: cs.AI, cs.SE · Date: April 2026

Key Finding: Agentic skills that perform well on controlled benchmarks show significant performance degradation in real-world deployments. The paper measures degradation across three axes: environmental variability (different OS versions, dependency conflicts), temporal drift (skills break as upstream APIs change), and adversarial interference (skills fail under concurrent multi-agent workloads). Average performance drops 23–41% from benchmark to production.

Degradation by Axis:

Degradation AxisAvg. DropWorst CasePrimary Cause
Environmental variability23%38%OS-level dependency conflicts

Adversarial interference41%67%Concurrent multi-agent resource contention

Measurement Framework: The authors propose a three-tier evaluation protocol: (1) controlled benchmark, (2) staged deployment with synthetic variability, (3) production shadow-mode with real user traffic. Skills must pass all three tiers before marketplace listing.

Implication: Benchmark-only evaluation creates false confidence in skill reliability. The 23–41% degradation gap means that security defenses calibrated to benchmark behavior will miss attacks that exploit production-environment inconsistencies.

DDIPE Attack

Supply-Chain Poisoning Attacks Against LLM Coding Agent Skill Ecosystems

Paper: arXiv:2604.03081
Categories: cs.CR, cs.AI, cs.CL · Date: April 2026 · Full Deep Dive: MAS → Supply-Chain Attacks

The foundational paper for this cluster. DDIPE (Document-Driven Implicit Payload Execution) embeds malicious logic in skill documentation that agents reproduce verbatim during in-context learning. 1,070 adversarial skills generated from 81 seeds across 15 MITRE ATT&CK categories. Bypass rates of 11.6–33.5% across Claude Code, OpenHands, Codex, and Gemini CLI. Cross-model diversity compresses attack surface from 20% to <2%.

Synthesis: Defense-in-Depth for Agentic Supply Chains

These three papers establish a layered defense model for agentic supply chains:

  1. Pre-deployment (2604.04323) — Three-tier evaluation protocol ensures skills perform reliably outside controlled benchmarks. The 23–41% degradation gap must be closed before deployment.
  2. Runtime interception (2604.04426) — ShieldNet’s network-layer proxy catches supply-chain payloads that bypass agent-level defenses, operating independently of model alignment quality.
  3. Ecosystem integrity (2604.03081) — DDIPE demonstrates the scale of the threat: 631,813 unreviewed skills, 15 MITRE ATT&CK categories, 11.6–33.5% bypass rates. Mandatory security review and multi-model verification ensembles are the minimum viable defense.

The critical insight: agent alignment is necessary but not sufficient. Even Claude Sonnet 4.6 (the strongest model tested) showed 2.3% direct execution rate under DDIPE. Defense must operate at every layer — evaluation, network, and ecosystem.
