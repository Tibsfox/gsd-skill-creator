Agent Psychometrics: Task-Level Performance Prediction

arXiv: 2604.00594

Authors: Ge et al.

Date: April 2026 · Categories: cs.AI, cs.CL · Venue: ICLR 2026

Abstract

Applies Item Response Theory (IRT) to agent evaluation, decomposing performance into LLM ability and scaffold ability components. Augmented with rich features, the framework predicts task-level performance on unseen benchmarks, reducing the need for expensive evaluations. The key insight is that agents are not monolithic — the LLM and its scaffolding contribute independently measurable capabilities.

Key Findings

  1. IRT decomposes agent performance into LLM ability and scaffold ability components
  2. Rich features augment classical IRT for more accurate prediction
  3. Unseen benchmark prediction reduces expensive evaluation runs
  4. LLM and scaffold contributions are independently measurable and separable
  5. Task difficulty parameters transfer across agent configurations
  6. ICLR 2026 acceptance validates the psychometric approach to agent evaluation

Methodology

Item Response Theory models adapted from educational testing applied to agent benchmark results. Each benchmark task is treated as a “test item” with difficulty and discrimination parameters. Agent ability is decomposed into base LLM capability and scaffold-provided capability using a two-parameter model. Rich features (task type, domain, complexity) augment the model for cross-benchmark prediction.

Implications for Our Architecture

The LLM/scaffold decomposition is precisely our architecture: Claude (LLM) + gsd-skill-creator (scaffold). This paper gives us a framework to measure how much of our system's performance comes from Claude vs from our skills, hooks, and agents. We should implement IRT-based evaluation in our test suite — our 21,298 tests could be parameterized with difficulty/discrimination values. The scaffold ability metric could drive our skill improvement decisions.
