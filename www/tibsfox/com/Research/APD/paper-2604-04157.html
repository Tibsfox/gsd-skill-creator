Readable Minds: Theory of Mind Emergence in LLM Poker Agents

arXiv: 2604.04157

Authors: Lin, Hou

Date: April 2026 · Categories: cs.AI, cs.CL

Abstract

A factorial study demonstrating that persistent memory is both necessary and sufficient for Theory of Mind emergence in LLM poker agents. With memory enabled, agents develop strategic deception capabilities (p<0.001) and opponent modeling that generalizes across game contexts. Cross-model validation achieves kappa=0.81, confirming robust inter-rater agreement on Theory of Mind attribution.

Key Findings

  1. Memory is necessary and sufficient for ToM: p=0.008
  2. Strategic deception emerges with persistent memory: p<0.001
  3. Cross-model validation kappa: 0.81 (strong agreement)
  4. Persistent memory enables opponent modeling that generalizes across contexts
  5. Without memory, agents play myopically — each hand is independent
  6. Factorial design isolates memory as the causal variable, not model scale

Methodology

Factorial experimental design with memory as the primary independent variable. Three conditions: no memory (stateless), short-term memory (within-hand), and persistent memory (across hands). Multiple LLM backends tested to isolate memory effects from model effects. Theory of Mind assessed via opponent modeling accuracy and strategic adaptation metrics. Cross-model validation via Cohen's kappa for inter-rater agreement.

Implications for Our Architecture

Memory Architecture

The memory-as-necessary-for-ToM finding validates our beads-state persistence and session-awareness patterns. If persistent memory enables strategic reasoning in competitive contexts, it also enables the cooperative reasoning our multi-agent orchestration depends on.

Evaluation Design

  • The factorial design methodology — isolating single variables — should be applied to our agent evaluation
  • Which capabilities require which state persistence patterns? This paper provides the methodology to answer that
  • Kappa=0.81 cross-model validation suggests our multi-model testing strategy (Sonnet/Opus) is sound
  • Strategic deception emergence warns about emergent agent behaviors in competitive multi-agent settings
