1. Artemis II Mission Dashboard

LIVE NASA

Enhanced real-time telemetry dashboard for the Artemis II mission, displaying crew status, trajectory, and mission phase with automatic sweep updates.

Data Sources

  • NASA HORIZONS: ephemeris data for spacecraft position (API: ssd.jpl.nasa.gov)
  • NASA OSDR: mission timeline and phase markers
  • Space-Track.org: TLE data for orbit visualization

Implementation

  • Client-side JavaScript with fetch() to CORS-enabled NASA APIs
  • SVG trajectory visualization with Earth-Moon system to scale
  • Mission elapsed time (MET) counter with phase boundary markers
  • Sweep interval: 15 minutes during active mission phases

3. Earthquake Monitor — Cascadia Subduction Zone

LIVE ECO

Real-time earthquake monitoring focused on the Cascadia Subduction Zone and the South Whidbey Island Fault Zone (SWIFZ), which runs directly beneath the Mukilteo area.

Data Sources

  • USGS Earthquake API: real-time GeoJSON feed (earthquake.usgs.gov/fdsnws)
  • PNSN (Pacific Northwest Seismic Network): regional seismograph data from UW
  • ShakeAlert: early warning system status for Cascadia

Implementation

  • Leaflet.js map with magnitude-scaled markers and depth-based coloring
  • Filter by region: Puget Sound, Cascadia plate boundary, Juan de Fuca Ridge
  • 72-hour rolling window with configurable magnitude threshold (default M1.0+)
  • SWIFZ trace overlay showing the fault zone geometry
  • Sweep interval: 5 minutes for M3.0+, 30 minutes for M1.0+

4. Tide Prediction — Puget Sound

LIVE SPS BEACH

Tide predictions and real-time water level data for Puget Sound NOAA stations, essential for beach access, forage fish surveys, and marine ecology monitoring.

Data Sources

  • NOAA CO-OPS: tidesandcurrents.noaa.gov API for predictions and observations
  • Stations: Seattle (#9447130), Mukilteo, Edmonds, Everett, Port Townsend
  • Harmonic constituents: 37 tidal components for local prediction

Implementation

  • SVG tide chart with predicted vs observed water levels
  • Next high/low tide countdown with height prediction
  • King tide alerts when predicted height exceeds MHHW + 2 feet
  • Integration with forage fish spawning survey timing (DIY Science module)

5. Air Quality Index — PNW Wildfire Season

LIVE AWF

Real-time AQI monitoring for the PNW with wildfire smoke tracking, particularly important during August–October fire season when PM2.5 levels can exceed 300 (hazardous).

Data Sources

  • AirNow API: EPA official AQI data (airnow.gov)
  • PurpleAir: community sensor network for hyperlocal PM2.5
  • NOAA HMS (Hazard Mapping System): satellite-detected fire and smoke plumes
  • NIFC InciWeb: active wildfire locations and perimeters

Implementation

  • Color-coded AQI map: green (0–50) through maroon (301–500)
  • PM2.5 time series with 24-hour rolling average
  • Smoke plume overlay from NOAA HMS satellite data
  • Health advisory generation based on AQI bracket

6. ISS Tracker — Visible Passes for PNW

LIVE NASA MST

Track the International Space Station with predictions for visible passes over the Pacific Northwest, computed from TLE orbital elements.

Data Sources

  • CelesTrak: current TLE data for ISS (NORAD ID 25544)
  • Spot the Station: NASA visible pass predictions for verification
  • Clear Sky Chart: cloud cover overlay for viewing conditions

Implementation

  • SGP4 orbital propagation in JavaScript (satellite.js library)
  • Real-time ground track on Leaflet.js map with footprint circle
  • Next 7 days of visible passes with magnitude, direction, and elevation
  • Observer location: configurable, default Mukilteo (47.9445°N, 122.3046°W)
  • Brightness filter: only show passes brighter than magnitude -1.0

7. Solar Activity Monitor

LIVE NASA

Real-time solar activity monitoring from NOAA SWPC (Space Weather Prediction Center) with aurora probability for the PNW latitude.

Data Sources

  • NOAA SWPC: solar wind, sunspot number, X-ray flux, geomagnetic indices
  • SDO (Solar Dynamics Observatory): AIA 193Å and HMI magnetogram imagery
  • DSCOVR/ACE: real-time solar wind at L1 point (Bz component for aurora forecasting)

Implementation

  • Kp index display with aurora visibility threshold for 48°N latitude (Kp ≥ 6)
  • Solar X-ray flux chart with flare classification (C/M/X class)
  • SDO image carousel with latest solar disc imagery
  • OVATION aurora probability map for northern hemisphere
  • Sweep interval: 15 minutes for solar wind, 6 hours for SDO imagery
