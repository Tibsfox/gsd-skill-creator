Usage Instructions

  1. Open the PDF to review the complete three-stage mission package.
  2. To hand off to gsd-skill-creator: provide the PDF as context for a new milestone.
  3. The GSD Orchestrator should read Stage 3 for wave execution plan, component specs, and crew manifest.
  4. Each module spec (M1–M5) is self-contained — agents can execute from their spec alone.
  5. Wave 0 (Foundation) must complete before Wave 1 begins; Tracks A and B execute in parallel.

Recompiling the LaTeX Source

  1. Ensure texlive-xetex and fonts-dejavu are installed.
  2. Run three passes: xelatex -interaction=nonstopmode 555-timer-mission.tex (×3)
  3. Three passes resolve TOC, cross-references, and page count.
