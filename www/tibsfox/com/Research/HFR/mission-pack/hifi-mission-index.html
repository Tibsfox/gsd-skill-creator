Contents

  1. Stage 1 — Vision Document
    • Vision narrative, problem statement, core concept: The Wave and the Wire
    • Domain architecture: source, electronics, acoustic, perception layers
    • 5 research modules: Speakers, Amplifiers, DAC, Room Acoustics, Headphones
    • Chipset configuration, scope boundaries, 12 success criteria
  2. Stage 2 — Research Reference
    • Speaker physics: Thiele-Small, enclosure types, crossover theory, electrostatic/planar/ribbon
    • Amplifier topology: Class A/AB/D/tube, negative feedback, damping factor, distortion mechanisms
    • DAC architectures: R-2R, delta-sigma, DSD, oversampling, jitter analysis
    • Room acoustics: modes, RT60, Sabine/Eyring, absorption/diffusion
    • Psychoacoustics: Fletcher-Munson, masking, localization, Harman preference
    • Headphones: dynamic, planar, electrostatic, BA, HRTF, binaural rendering
    • Vinyl vs digital: complete technical comparison with scientific consensus
  3. Stage 3 — Mission Package
    • 10 deliverables, 3-wave execution plan with 3 parallel tracks
    • 38 tests (6 safety-critical), "The Listening Report" verification matrix
    • ~80K token budget, cross-references to 8 sibling projects

Usage Instructions

  1. Point GSD's new-project at the vision section of the PDF
  2. Wave 0 produces shared glossary and source index
  3. Wave 1 executes three parallel tracks (A: Speakers+Amplifiers, B: DAC+Room, C: Headphones)
  4. Wave 2 synthesizes integration patterns, verifies, and assembles final document
