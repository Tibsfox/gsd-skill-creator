Usage

  1. Feed the PDF to gsd-skill-creator via Claude Code as the mission brief
  2. Stage 2 (Research Reference) provides pre-loaded context for all survey agents — no independent web research needed for foundational data
  3. Wave 0 outputs serve as shared schemas; Wave 1 tracks run in parallel
  4. CAPCOM approvals required at wave boundaries (0→1, 1→2, 2→3)
  5. All safety-critical tests are mandatory-pass / BLOCK gates

Recompilation

  1. Ensure XeLaTeX and DejaVu fonts are installed
  2. Run three passes: xelatex -interaction=nonstopmode mission.tex (×3)
  3. Three passes resolve TOC, cross-references, and page count
