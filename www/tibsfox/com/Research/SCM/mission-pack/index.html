How to Use This Package

Handing Off to GSD Orchestrator

  1. Download both files (PDF and .tex) and place them in your GSD project directory.
  2. Open Claude Code and hand the PDF to your GSD orchestrator agent: "Here is the mission package. Execute the Scientific Method mission."
  3. The orchestrator runs Wave 0 (Foundation: schema, source index, outline) — Haiku model, sequential. Approve the outline before continuing.
  4. Wave 1 runs in parallel — Five writing agents produce the research modules. Historical arc, core mechanics, philosophy, integrity, and AI futures execute simultaneously.
  5. Human approval gate at end of Wave 1 — Review all five modules against the 12 success criteria before authorizing synthesis. CAPCOM agent manages this checkpoint.
  6. Wave 2 synthesis — Opus model performs cross-module integration, the GSD through-line pass, and cross-reference compilation.
  7. Wave 3 publication — Final verification, index generation, and delivery to Research/SCM/.
