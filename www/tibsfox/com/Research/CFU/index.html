ComfyUI embodies the Amiga Principle at the inference layer — a directed-acyclic-graph engine where node connections define data flow, partial re-execution prevents redundant computation, and a clean HTTP/WebSocket API makes the entire pipeline accessible from any framework. The meaning lives in the connection between intent, the workflow graph, and the artifact that emerges.

Research Modules

M1: ComfyUI API Bridge

API Track A · Foundation layer

Complete REST and WebSocket API mapping for ComfyUI server. Workflow JSON schema: nodes, links, execution order. Queue management via /prompt endpoint. Real-time progress tracking over WebSocket. Python client implementation with async dispatch and artifact retrieval.

M2: Image Production Pipeline

Image Track B · Generation layer

Model landscape: FLUX.1 Dev/Schnell (FP8 quantized), SDXL, SD 3.5. VRAM tier routing for 8/12/16/24GB GPUs. ControlNet integration for pose, depth, and edge-guided generation. Workflow templates: text-to-image, image-to-image, inpainting, upscaling. Batch generation with consistent seed management.

M3: Video Production Pipeline

Video Track B · Motion layer

Video model survey: LTX-2.3, Wan 2.2, CogVideoX, HunyuanVideo, AnimateDiff. Text-to-video and image-to-video workflows. Frame interpolation and temporal consistency. Multi-clip assembly with consistent seeds. VRAM requirements and quantization strategies for consumer GPUs.

M4: Skill-Creator Integration

Integration Synthesis · DACP layer

DACP three-part bundle structure for generation requests: human intent, workflow JSON, executable trigger. MCP bridge specification adapting the planning-bridge pattern. Registry feedback loop: recurring prompt patterns become reusable workflow templates. Model auto-discovery via /object_info endpoint parsing.

M5: Security & Sandboxing

Security Synthesis · Safety layer

Docker isolation for ComfyUI instances: mount-only access to /models, /input, /output. Custom node security audit: eval/exec scanning, runtime-pip detection. Audited node allowlist with 10+ verified nodes. Network policy restricting container to localhost API only. Cryptominer and credential stealer detection patterns.
