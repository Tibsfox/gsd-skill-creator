Usage Instructions
  1. Open comfyui_mission.pdf and review all three stages. Approve the node allowlist in Stage 3 Wave 0 before any ComfyUI custom node installation.
  2. Hand the PDF to the GSD Orchestrator in a Claude Code session: "Execute this mission package. Begin with Wave 0 foundation tasks."
  3. At each HITL Gate (post-Wave 0, post-Wave 1, post-Wave 3), review deliverables before proceeding. The node allowlist gate is mandatory before any container build.
  4. The DACP bundle schema (schemas/generation_bundle.json) produced in Wave 0 becomes the contract between Skill-Creator agents and the ComfyUI skill.
  5. After Wave 3, the skill lives at .claude/skills/comfyui/SKILL.md in the gsd-skill-creator dev branch.
