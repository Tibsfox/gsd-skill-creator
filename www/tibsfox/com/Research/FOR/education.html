Fortran has more high-quality books per active developer than perhaps any other language. The reason is its longevity: every era left a canonical text behind.

Learning Path — From Zero to Modern Fortran Developer

  1. Level 1: Basics — Install GFortran. Read fortran-lang.org/learn. Write PROGRAM main, IMPLICIT NONE, INTEGER, REAL, PRINT. Compile and run your first program. Try LFortran for an interactive REPL experience.
  2. Level 2: Structure — DO loops, IF/THEN/ELSE, SUBROUTINE, FUNCTION. Read the first chapters of Chapman or Modern Fortran Explained. Start using -std=f2018 -fimplicit-none -Wall.
  3. Level 3: Modules — MODULE, USE, PUBLIC/PRIVATE, INTENT(IN/OUT/INOUT). Organize code into reusable modules. The key transition from FORTRAN 77 thinking to modern Fortran.
  4. Level 4: Arrays — The killer feature. Whole-array operations, slicing, intrinsics (SUM, MATMUL, RESHAPE). Allocatable arrays. Stop writing explicit DO loops for elementwise math.
  5. Level 5: Numerical Libraries — BLAS and LAPACK. stdlib_linalg as the modern wrapper. Solve linear systems, find eigenvalues, do SVD. The reason Fortran exists.
  6. Level 6: Object-Oriented Fortran — Derived types with type-bound procedures. EXTENDS for inheritance. CLASS for polymorphism. Read Rouson’s Scientific Software Design.
  7. Level 7: Parallel — OpenMP directives. DO CONCURRENT. Coarrays. Pick one model, learn it deeply. Read Curcic’s parallel Fortran book.
  8. Level 8: HPC Production — MPI for distributed memory. GPU offloading via nvfortran. Real climate or CFD codebases. You’re now writing the code that runs supercomputers.

The unique advantage of learning Fortran in 2026: The barrier is gone. Free compilers (GFortran, ifx, nvfortran). Modern build tools (fpm). A standard library (stdlib). Interactive REPL (LFortran). LSP support (fortls). VS Code extension. A welcoming community (Fortran Discourse). And the language itself is mature, stable, and faster than ever. There has never been a better time to learn it.
