A language frequently declared dead by outsiders has undergone a quiet but determined renaissance. fpm, stdlib, LFortran, Flang — modern tooling for the first time in decades.

Five years ago the answer was “GFortran, and maybe Intel if you paid for it.” In 2026 there are nine compilers worth knowing, with two brand-new LLVM-based frontends (LFortran and Flang) genuinely competing with the long-established GFortran. Intel made ifx free for personal use in 2024; NVIDIA made nvfortran free for non-commercial use. The barriers to high-performance Fortran are gone.

fpm — The Fortran Package Manager

The single most transformative tool in modern Fortran. Modeled on Rust’s Cargo. Simple TOML manifest, fetches dependencies, builds modern Fortran projects. The way to start a new Fortran project in 2026.

# Install fpm
brew install fpm                         # macOS
cargo install fortran-fpm                # via Cargo (yes, really)
# Or download binary from github.com/fortran-lang/fpm

# Create a new project
fpm new myproject --app --lib --test
cd myproject

# Build, test, run
fpm build                                # builds with gfortran by default
fpm build --compiler ifx                 # build with Intel
fpm test                                 # run tests
fpm run                                  # run the executable
# fpm.toml
name = "myproject"
version = "0.1.0"
license = "MIT"
author = "Foxy"

[build]
auto-executables = true
auto-tests = true

[dependencies]
stdlib = "*"
test-drive = { git = "https://github.com/fortran-lang/test-drive" }

[[executable]]
name = "myapp"
source-dir = "app"
main = "main.f90"

stdlib — The Fortran Standard Library

Community effort to give Fortran a modern standard library. Inspired by Python’s stdlib and Rust’s std. Fixes the historical complaint that every Fortran project had to reinvent string manipulation, hash maps, and statistical primitives.

ModulePurpose
stdlib_ioText file I/O, TOML loading, NumPy npy/npz
stdlib_string_type / stdlib_stringsModern string type, split/join/replace/strip/find
stdlib_linalgBLAS/LAPACK-backed: inverse, det, eigenvalues, SVD, least squares
stdlib_statsMean, variance, covariance, median, quantile, correlation
stdlib_stats_distribution_*Random variates: normal, uniform, exponential, gamma
stdlib_randomModern PRNGs (xoshiro, PCG)
stdlib_quadratureNumerical integration: trapezoidal, Simpson, Gauss-Legendre
stdlib_specialfunctionsGamma, beta, error function, Bessel, Legendre
stdlib_hash_mapsHash map containers (finally!)
stdlib_loggerStructured logging
stdlib_ansiTerminal colors and formatting
