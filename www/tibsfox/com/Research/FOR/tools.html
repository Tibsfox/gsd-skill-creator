Choose your compiler by domain: free defaults (GFortran), performance leader (ifx), GPU (nvfortran), strict standards (NAG), modern interactivity (LFortran).

GFortran (GNU Fortran)

Part of GCC. Ships with every Linux distribution. As of GCC 15 (April 2025), supported versions are GCC 13, 14, 15.

Standards support: Full F95, most F2003/2008, partial F2018, early F2023.
License: GPLv3 with runtime exception.
Install: sudo apt install gfortran / brew install gcc

# Compile a single file
gfortran -O3 -march=native -Wall -Wextra prog.f90 -o prog

# With OpenMP
gfortran -O3 -fopenmp prog.f90 -o prog

# Strict standards mode
gfortran -std=f2018 -fimplicit-none -Wall prog.f90 -o prog

# Multi-file build
gfortran -c module.f90 main.f90
gfortran module.o main.o -o prog -lblas -llapack

# Common flags
-O0 / -O1 / -O2 / -O3      # optimization levels
-Og                         # optimize for debugging
-g                          # debug symbols
-ffree-form / -ffixed-form  # source form override
-ffree-line-length-none     # no line length limit
-fcheck=all                 # runtime bounds checking
-finit-real=nan             # initialize reals to NaN (catches uninitialized use)
-fbacktrace                 # backtrace on crash

Intel ifx (oneAPI)

Intel’s LLVM-based Fortran compiler. The successor to ifort (end-of-life late 2024). Free for personal use since 2024.

Best on: x86-64 (AVX-512), Intel GPUs.
Bundled with: MKL (Math Kernel Library), VTune, Advisor, Inspector.
Install: Intel oneAPI Base + HPC Toolkit.

# Standard build
ifx -O3 -xHost prog.f90 -o prog

# With MKL (BLAS/LAPACK)
ifx -O3 prog.f90 -o prog -qmkl

# OpenMP
ifx -O3 -qopenmp prog.f90 -o prog

# OpenMP target offload (Intel GPU)
ifx -O3 -fiopenmp -fopenmp-targets=spir64 prog.f90 -o prog

nvfortran (NVIDIA HPC SDK)

NVIDIA’s Fortran compiler. Successor to PGI Fortran (acquired 2013). LLVM-based with NVIDIA backends.

Best for: GPU offloading. CUDA Fortran. OpenACC. do concurrent → GPU kernels.
License: Free for non-commercial use.

# Standard CPU build
nvfortran -O3 -fast prog.f90 -o prog

# OpenACC GPU offload
nvfortran -O3 -acc -gpu=cc80 prog.f90 -o prog

# DO CONCURRENT to GPU (no annotations needed!)
nvfortran -O3 -stdpar=gpu prog.f90 -o prog

# CUDA Fortran (kernel attribute)
nvfortran -O3 -cuda prog.cuf -o prog

# OpenMP target offload
nvfortran -O3 -mp=gpu prog.f90 -o prog

NAG Fortran

Commercial. The reference for standards conformance and diagnostics. Used to validate that code is portable, not GFortran-specific.

Role: Compliance oracle. Run your code through NAG to find non-standard usage.
Strength: Catches more bugs at compile time than any other Fortran compiler.

LFortran

The new hotness. LLVM-based. Interactive REPL. Jupyter kernel. Lazy compilation. Built by Ondřej Čertík and the fortran-lang community.

Status (2026): 1.0.0 released late 2025. Production-ready for many use cases. Active development.
Unique: Type-in-and-run REPL for Fortran. First time in 69 years.

# Install LFortran
pip install lfortran                     # via pip
# Or download from lfortran.org

# Interactive REPL
lfortran
>>> integer :: i
>>> do i = 1, 5
...   print *, i*i
... end do
            1
            4
            9
           16
           25

# Compile
lfortran prog.f90 -o prog

# Run as Jupyter kernel
jupyter notebook  # select Fortran kernel

Flang (LLVM)

LLVM’s official Fortran frontend. The long-term replacement strategy for many backends. Apache 2.0 license. Backed by Arm, AMD, NVIDIA, and others.

Status (2026): Production for most F95 code, increasing F2018/2023 coverage. Used by AMD AOCC and partially by nvfortran.

The Recommended Modern Setup (2026)

  1. Compiler: GFortran (free) or ifx (free for personal, fastest on Intel)
  2. Build: fpm for new projects, CMake for mixed-language projects
  3. Editor: VS Code with Modern Fortran extension + fortls
  4. Library: stdlib for modern data structures and numerical primitives
  5. Tests: test-drive for unit tests
  6. Docs: FORD for HTML documentation
  7. Standards: -std=f2018 -fimplicit-none as the floor; validate with NAG for portability
  8. GPU (if needed): nvfortran with do concurrent or OpenACC
# The full modern Fortran greenfield project
fpm new myproject --app --lib --test
cd myproject
$EDITOR fpm.toml                         # add stdlib, test-drive
fpm build --compiler gfortran --flag "-O3 -fimplicit-none -Wall"
fpm test
fpm run
