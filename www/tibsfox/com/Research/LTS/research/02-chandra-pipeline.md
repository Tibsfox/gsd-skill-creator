## 1. The CXC Program
Launched 2020. First ongoing sustained sonification initiative at NASA. 35+ sonified objects. 4M+ listens/views across NASA media accounts (Arcand et al. 2024, Frontiers).

## 3. Inverse Spectrogram Technique

| Data Dimension | Audio Mapping |
|---------------|--------------|
| Horizontal position (image) | Time (left-to-right scan) |
| Vertical position (image) | Pitch (bottom=lower, top=higher) |
| Pixel brightness | Volume/loudness |
| Radio wavelength | Lowest frequency tones |
| Optical wavelength | Mid-range tones |
| X-ray wavelength | Highest frequency tones |
| Distinct object types | Distinct instrument families |

Each sonification caption explicitly describes the mapping used -- intelligibility requires understanding the translation conventions.

## 4. Multi-wavelength Integration
The M87 sonification (2022) demonstrates multi-telescope layering: X-ray (Chandra), optical (Hubble), radio (ALMA) mapped simultaneously. Released alongside Perseus, together accumulating 4M+ listens.
