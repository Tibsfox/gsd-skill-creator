How to Use This Package

  1. 1
    Download the PDF. Read Stage 1 (Vision) to orient yourself to the research scope and GSD ecosystem connections. Review the five modules and success criteria.
  2. 2
    Open Stage 2 (Research Reference) for the compiled findings. All instrument tables, hardware specs, and extensibility documentation are ready to reference during execution.
  3. 3
    Hand Stage 3 to gsd-skill-creator via the planning bridge. Deposit to .planning/staging/inbox/. The Wave Execution Plan and Component Breakdown are the primary execution spec.
  4. 4
    Start with Wave 0 (Foundation). Launch Waves 1-A, 1-B, and 1-C in parallel within the same session to exploit the 5-minute TTL cache window on the shared schema context.
  5. 5
    At the CAPCOM gate (post Wave 1), review all five module documents before approving Wave 2 (Synthesis). The Opus-powered synthesis document will draw the GSD ecosystem connections.
