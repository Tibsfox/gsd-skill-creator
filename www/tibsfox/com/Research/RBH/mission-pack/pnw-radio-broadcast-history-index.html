Usage Instructions

  1. Feed mission.pdf to gsd-skill-creator as the mission specification
  2. The orchestrator reads the wave execution plan and assigns agents per the crew manifest
  3. Wave 0 produces shared schemas; Wave 1 runs two parallel research tracks
  4. Wave 2 synthesizes cross-module connections; Wave 3 assembles and verifies the publication
  5. CAPCOM checkpoints at each wave boundary require human approval before proceeding

Recompiling the LaTeX Source

  1. Ensure XeLaTeX and DejaVu fonts are installed
  2. Run three compilation passes:
    xelatex -interaction=nonstopmode mission.tex
    xelatex -interaction=nonstopmode mission.tex
    xelatex -interaction=nonstopmode mission.tex
  3. Three passes resolve table of contents, cross-references, and page counts
