Usage Instructions

  1. Hand the mission.pdf to the GSD skill-creator orchestrator as the mission brief
  2. The orchestrator reads all three stages and initializes the Squadron crew manifest
  3. Wave 0 (Foundation) executes first — schemas and source index
  4. Wave 1 runs two parallel tracks: Track A (Hydro + Tribal) and Track B (Geothermal + Regulatory)
  5. Wave 2 synthesizes: Grid Integration depends on both M1 and M2 outputs
  6. Wave 3 verifies all sources and produces the final formatted publication
  7. CAPCOM checkpoints at each wave boundary for human go/no-go

Recompilation

  1. Requires XeLaTeX with DejaVu fonts (pre-installed on Ubuntu)
  2. Run three passes: xelatex -interaction=nonstopmode mission.tex (×3)
  3. Three passes resolve TOC, cross-references, and page counts
