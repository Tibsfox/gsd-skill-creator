How to Use This Package

  1. Hand vbrc_mission.pdf to GSD orchestrator or paste into a fresh Claude Code session on branch wasteland/skill-creator-integration
  2. Tell GSD: gsd new-project "Virtual Black Rock City v1.44" and load this document as the mission spec
  3. Execute Wave 0 first (Foundation): schema, PNW naming glossary, source index — these seed the cache for all parallel tracks
  4. Launch Waves 1A and 1B simultaneously in separate Claude Code sessions; each produces its deliverables independently
  5. Run Wave 2 Synthesis (Opus) after both 1A and 1B complete; this is where Wasteland integration, art patterns, and safety gates are finalized
  6. Run Wave 3 Publication (Sonnet) to produce the role-chooser catalog and execute all 6 safety-critical tests — all must BLOCK-pass before milestone close

Recompile LaTeX

  1. Install XeLaTeX: sudo apt-get install texlive-xetex texlive-fonts-recommended
  2. Run three passes: xelatex -interaction=nonstopmode vbrc_mission.tex (repeat 3×)
  3. Three passes resolve: table of contents, cross-references, lastpage counter, hyperlink destinations
