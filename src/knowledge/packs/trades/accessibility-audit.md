# TRADE-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Trades involve real tools, real hazards, and physical work. This audit ensures all learners can participate safely while developing competency, with necessary accommodations.

| Feature | Status | Notes |

## Safety-Critical Accessibility

**Critical Principle:** All safety information must be **multi-modal**, not visual-only.

### Safety Warnings

- ✓ **Written warnings** on all tools and materials
- ✓ **Color-coded safety labels** PLUS text labels
- [ ] **Audio warnings** at start of activity
- ✓ **In-person supervision** ensuring safety awareness
- [ ] **Verbal confirmation** that hazards understood
- [ ] **Vibration alerts** for dangerous conditions

### Hazard Identification

**Challenge:** Many hazards are visual (exposed blade, hot surface, electrical danger).

**Remediation:**
- [ ] **Audio descriptions** of hazard types
- [ ] **Tactile identification methods** (feel heat, vibration)
- [ ] **Scent warnings** for some chemicals (gas smell, etc.)
- [ ] **Texture mapping** of danger zones
- [ ] **Physical barriers** preventing access to hazards

## Vision Accessibility

### For Blind Learners

**Challenge:** Tool identification, precise measurement, detecting hazards require sight.

**Remediation:**
- [ ] **Tactile tool identification:** Labels, shapes, distinctive features
- [ ] **Audio measurement tools:** Speaking digital calipers, laser measuring
- [ ] **Measurement assistance devices:** Guides, jigs reducing precision required
- [ ] **Auditory feedback:** Tone indicating correct positioning
- [ ] **Hand-over-hand instruction:** Feeling correct technique
- [ ] **Tool modifications:** Marked gauges, tactile stopping points

### For Low-Vision Learners

- ✓ **Large-print instruction materials**
- ✓ **High-contrast markings** on tools and materials
- [ ] **Magnification lamps** for precision work
- [ ] **Enhanced lighting** in workspace

## Motor Accessibility

### Limited Hand/Arm Function

**Challenge:** Most trades require bilateral hand use, grip strength, fine motor control.

**Adaptations:**
- [ ] **One-handed tool techniques:** Clamps, vises as substitute hand
- [ ] **Adaptive tools:** Ergonomic grips, lighter tools, powered tools
- [ ] **Modified workstations:** Optimal height, accessibility
- [ ] **Delegation role:** Supervising others, quality control
- [ ] **Technologies:** Cordless tools, power assists

### Limited Mobility/Standing

**Challenge:** Many trade skills require standing, reaching, bending.

**Adaptations:**
- ✓ **Seated variations:** Workbench height adaptable
- ✓ **Modified reach:** Bringing materials closer
- ✓ **Reduced duration:** Shorter work sessions
- [ ] **Assistive devices:** Reaching aids, mobility aids
- [ ] **Supervision role:** Planning, instruction, inspection

## Hearing Accessibility

**Challenge:** Many auditory cues (power tool sounds, alarms, instructions).

**Accommodations:**
- ✓ **Written instructions** accompanying verbal
- ✓ **Visual demonstrations** (can be followed without audio)
- [ ] **Vibration alerts** for machinery sounds
- [ ] **Flashing lights** for alarms
- [ ] **Text-to-speech** of audio instructions
- [ ] **ASL interpretation** for group instruction

## Neurodiversity Considerations

### ADHD-Friendly Features

- ✓ Hands-on, concrete, result-oriented
- ✓ Immediate feedback (tool works or doesn't)
- [ ] Add: **Clear step-by-step instructions**
- [ ] Add: **Shorter projects** with visible completion
- [ ] Add: **Frequent breaks** and movement

### Autism Spectrum

- ✓ Systematic safety protocols
- ✓ Predictable sequences
- ✓ Precise technical requirements
- [ ] Add: **Explicit safety rules** (non-negotiable boundaries)
- [ ] Add: **Sensory management** (hearing protection, ventilation)
- [ ] Add: **Calm, structured environment**

### Anxiety About Tools/Hazards

- ✓ Gradual skill-building with safe materials
- ✓ Supervised practice, not unsafe autonomy
- ✓ Success-oriented progression
- [ ] Add: **Breathing/calming techniques** for anxiety
- [ ] Add: **Acknowledgment** that fear is normal
- [ ] Add: **Flexible pacing** (not rushed)

## Sensory Accessibility

### Proprioceptive & Kinesthetic Learning

- ✓ Trades are inherently hands-on
- ✓ Excellent for proprioceptive learners
- ✓ Feel and movement critical to learning
- [ ] Add: **Weighted tools** for those needing proprioceptive input

### Audio-Visual Trade-offs

- Deaf learners can't hear machinery but CAN see hazards
- Blind learners can't see hazards but CAN hear machinery
- **Solution:** Multiple safety systems (never one-modality-only)
