# WRIT-101: Standards Alignment

## Common Core State Standards (ELA Writing & Reading)

**RL.K.1** (K): Ask and answer questions about key details in a text
**RL.2.1** (2): Ask and answer questions about key details
**RL.4.3** (4): Describe characters, settings, events with details
**RL.6.2** (6): Determine theme and central idea
**RL.9-10.3** (9-10): Analyze complex character development
**W.K.3** (K): Attempt to write simple stories
**W.2.3** (2): Write narratives recounting events
**W.4.3** (4): Write narratives about real/imagined experiences
**W.6.3** (6): Write narratives to develop real/imagined experiences
**W.9-10.3** (9-10): Write narratives to develop real/imagined experiences

## NCTE (National Council of Teachers of English) Standards

**Standard 1**: Read widely and fluently to build knowledge
**Standard 3**: Use other inquiry tools and information sources
**Standard 4**: Adopt a variety of reading strategies
**Standard 6**: Use spoken, written, and visual language to accomplish purposes

## IB MYP Language & Literature

**Criterion A**: Analyzing literature and textual features
**Criterion B**: Organizing information to support response
**Criterion C**: Using language to support understanding
**Criterion D**: Using language to create effect

## Curriculum Connections

- **K-2 Emphasis**: Story understanding, initial writing attempts, read-alouds
- **3-5 Emphasis**: Character analysis, narrative writing, text selection
- **6-8 Emphasis**: Theme identification, literary techniques, longer works
- **9-12 Emphasis**: Complex analysis, literary criticism, authentic writing projects
