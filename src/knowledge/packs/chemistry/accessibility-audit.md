# CHEM-101: Accessibility Audit

**Pack:** Chemistry — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Partially Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Partial | Text-based kitchen chemistry instructions readable; periodic table and molecular diagrams need descriptions |
| Large Text Support | Implemented | Instructions and activity descriptions scale well |
| High Contrast Mode | Implemented | High-contrast periodic table layout available |
| Keyboard Navigation | Implemented | Interactive periodic table supports keyboard exploration |
| Alt Text for Images | Partial | Chemical reaction illustrations included; molecular structure diagrams need enhancement |
| Media Captions | Not Started | No video demonstrations; captions planned for lab technique videos |

## Content-Specific Considerations

- **Periodic Table:** Visual layout is organizing principle but accessibility-challenged. Interactive version with searchable properties (atomic number, mass, family) provided; screen reader descriptions available for element cards.
- **Molecular Structures:** Ball-and-stick and space-filling models are visual. Text descriptions of geometry and bonding provided; interactive 3D models need enhanced accessibility.
- **Lab Safety Diagrams:** Visual safety warnings require text-based alternatives; safety information is critical and needs extra detail.
- **Kitchen Chemistry:** Accessible by design — sensory observations (color change, temperature, smell) form basis of understanding; hands-on participation is inclusive.

## Remediation Path

### Priority 1 (Critical)
1. Enhance periodic table accessibility with searchable fields and detailed element descriptions
2. Create text-based molecular structure descriptions including geometry and bond angles
3. Add detailed descriptions of chemical reaction visuals (color changes, precipitate formation)
4. Ensure safety warnings are accessible in text form with extra clarity
5. Provide olfactory and tactile descriptions where safe (e.g., "vinegar smell," "fizzing sensation")

### Priority 2 (Important)
1. Add captions to chemistry demonstration videos with detailed visual descriptions
2. Create accessible versions of electron configuration and bonding diagrams
3. Develop keyboard shortcuts for interactive periodic table and molecular viewers
4. Build collaborative lab guides for mixed-ability teams
5. Create accessible assessment materials for chemistry concepts

### Priority 3 (Enhancement)
1. Develop tactile 3D molecular models for hands-on learning
2. Create chemical property charts with alternative data representations
3. Build audio descriptions for chemical phenomena and reactions
4. Develop customizable periodic table layouts (colors, font sizes, grouping)

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level A for text instructions; Level AA for interactive elements
- **Major Gaps:**
  - Periodic table accessibility incomplete
  - Molecular structure descriptions need expansion
  - Chemical reaction visuals lack detailed descriptions
  - Lab demonstration videos not captioned
- **Strengths:**
  - Kitchen chemistry hands-on nature is accessible
  - Clear text-based procedures
  - Good contrast in interactive elements
  - Sensory descriptions (smell, temperature) naturally accessible
