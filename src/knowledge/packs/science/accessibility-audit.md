# SCI-101: Accessibility Audit

**Pack:** Science Method & Nature — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Mostly Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Partial | Text content and experiment instructions are readable; observation diagrams need detailed descriptions |
| Large Text Support | Implemented | Content renders well at larger text sizes |
| High Contrast Mode | Implemented | High-contrast styling available for visual clarity |
| Keyboard Navigation | Implemented | Interactive simulation tools support keyboard control |
| Alt Text for Images | Partial | Experiment diagrams and data visualization charts included; observation journal prompts need image descriptions |
| Media Captions | Not Started | No video content; captions planned for future demonstration videos |

## Content-Specific Considerations

- **Observation Activities:** Nature walk journals and sensory observation tasks are inherently visual. Multi-sensory descriptions provided; activities can be adapted for learners who are blind/low vision by emphasizing auditory, tactile, and kinesthetic observation methods.
- **Experiment Diagrams:** Setup diagrams for investigations (pendulum, plant growth, etc.) include text descriptions but would benefit from more detailed spatial relationships.
- **Data Representations:** Graphs and charts in Module 3 include alt text; data table alternatives provided for visual representations.
- **Variable Identification:** Systems diagrams showing independent, dependent, and controlled variables include text descriptions of relationships.

## Remediation Path

### Priority 1 (Critical)
1. Add detailed descriptions for experiment setup diagrams showing spatial relationships
2. Create tactile/auditory alternatives for observation activities (e.g., listening for sound patterns instead of visual observation)
3. Ensure all graphs have accessible data tables with specific values
4. Add keyboard shortcuts for interactive simulation controls
5. Provide text descriptions for scientific tool visuals (microscopes, thermometers, etc.)

### Priority 2 (Important)
1. Create accessible versions of claim-evidence-reasoning (CER) argument templates
2. Add captions and transcripts for any instructional demonstration videos
3. Develop accessible alternatives for peer review discussion protocols
4. Enhance descriptions of scientific poster layouts for visually impaired learners

### Priority 3 (Enhancement)
1. Create multi-sensory exploration kits that pair visual observation with tactile and auditory components
2. Develop audio descriptions for nature photography in observation activities
3. Build customizable observation journal templates for diverse sensory preferences

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level A for text content; Level AA for interactive tools
- **Major Gaps:**
  - Observation diagram descriptions need enhancement
  - Some interactive simulations lack full keyboard navigation
  - Data visualization alternatives incomplete
  - Demonstration video captions needed
- **Strengths:**
  - Clear text-based experiment instructions
  - Good contrast in interactive simulation interfaces
  - Semantic markup for scientific reports
  - Logical flow of inquiry process
