# PROB-101: Problem Solving

**Classification:** Core Academic
**Status:** Alpha
**Version:** 1.0.0

Problem solving as the universal skill of turning "stuck" into "solved" -- general-purpose strategies, heuristics, and transfer skills that apply across every domain.

## Quick Start

This pack teaches **problem-solving strategies** explicitly as domain-independent skills. It is the meta-cognitive engine for all packs that involve structured reasoning (MATH-101, CODE-101, ENGR-101, DATA-101).

### Modules

| Module | Name | Focus |
|--------|------|-------|
| M1 | Understanding Problems | Restating, knowns/unknowns, constraints |
| M2 | Problem-Solving Strategies | Heuristics toolkit (Polya, computational thinking) |
| M3 | Creative & Lateral Thinking | Brainstorming, SCAMPER, reframing, analogy |
| M4 | Collaborative Problem Solving | Group dynamics, distributed cognition |
| M5 | Complex & Wicked Problems | Systems thinking, tradeoffs, uncertainty |

### Prerequisites

- None required (this is a foundational pack)
- MATH-101 recommended for mathematical problem-solving contexts

### What This Pack Enables

- CODE-101 (Coding & Computational Thinking)
- ENGR-101 (Engineering Design)
- DATA-101 (Data Literacy)
