# HIST-101: Accessibility Audit

**Pack:** History — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Mostly Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Implemented | Historical narratives and primary sources in text |
| Large Text Support | Implemented | Text documents scale well for larger fonts |
| High Contrast Mode | Implemented | High-contrast interface available |
| Keyboard Navigation | Implemented | Timeline and archive tools support keyboard |
| Alt Text for Images | Partial | Historical photographs and documents included; detailed descriptions provided |
| Media Captions | Partial | Some historical video documentaries; many lack captions |

## Content-Specific Considerations

- **Primary Sources:** Documents are text-based and naturally accessible. Photographs and visual primary sources have detailed alt text describing historical context, people, and events.
- **Timelines:** Visual chronological representations described verbally and in text. Interactive timelines have keyboard navigation and list view alternatives.
- **Maps & Diagrams:** Historical maps and warfare diagrams have text descriptions of spatial relationships and significance. Tactile map alternatives recommended for blind learners.
- **Perspectives & Empathy (M4):** Discussion-based and narrative-based; naturally accessible through dialogue and reflection writing.

## Remediation Path

### Priority 1 (Critical)
1. Add detailed descriptions of historical photographs emphasizing people, context, and significance
2. Ensure timeline representations have accessible keyboard navigation and list alternatives
3. Create detailed descriptions of historical maps and battle diagrams
4. Add captions to all historical documentary video content
5. Provide text transcripts for all oral history recordings

### Priority 2 (Important)
1. Create text-based alternatives for timeline visualizations
2. Develop accessible primary source analysis worksheets
3. Build collaborative historical inquiry guides
4. Create accessible historical narrative templates
5. Develop captions for any historical reenactment videos

### Priority 3 (Enhancement)
1. Develop audio descriptions of historical photographs
2. Create tactile historical maps for hands-on learning
3. Build audio versions of primary source excerpts
4. Develop customizable timeline presentation options

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level AA for text and documents; Level A for some video content
- **Major Strengths:**
  - Primary sources are text-based
  - Historical narratives naturally accessible
  - Discussion format is inclusive
  - Perspective-taking activities flexible
- **Minor Gaps:**
  - Historical photo descriptions could be more detailed
  - Documentary videos lack captions
  - Timeline visualizations need better keyboard support
  - Map descriptions need enhancement
