# NATURE-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Nature studies is outdoor-focused, but nature is accessible through multiple modalities: observation, listening, touch, smell. This audit ensures that nature education reaches all learners.

## WCAG 2.1 AA Accessibility Status

| Feature | Status | Notes |
|---------|--------|-------|
| Outdoor Alternative | ✓ Yes | Indoor/virtual alternatives available |
| Field Guide Images | ✓ Yes | Alt text for all species identification |
| Audio Descriptions | ✓ Partial | Nature sounds, species calls described |
| Keyboard Navigable | ✓ Yes | iNaturalist, digital tools keyboard-accessible |
| Large Text Available | ✓ Yes | Field guide magnification supported |
| Captions Available | ✓ Yes | Video nature observations captioned |

## Physical Accessibility Challenges

### Outdoor Activity Limitations

**Challenge:** Nature observation outdoors may not be physically possible for all learners due to mobility, stamina, or environmental factors.

**Remediation:**
- ✓ **Indoor alternatives:** Museum collections, herbarium, preserved specimens
- ✓ **Virtual options:** 360° nature videos, planetarium programs
- ✓ **Window watching:** Observation from stationary position
- [ ] **Tactile nature kits:** Touch specimens in controlled settings
- [ ] **Home-based observation:** Backyard, balcony, potted plants
- [ ] **Virtual field trips:** Nature documentaries, live webcam feeds

**Timeline:** 2026-Q2

### Sensory-Focused Learning

**Challenge:** Outdoor observation relies heavily on sight; other senses are equally important.

**Adaptations:**
- ✓ **Audio nature walks:** Soundscape recordings with natural sounds
- ✓ **Tactile identification:** Leaf rubbings, bark textures, flower shapes
- ✓ **Scent identification:** Flower scents, plant smells (carefully selected)
- ✓ **Taste identification:** (Some plants only—under supervision)
- ✓ **Sound identification:** Bird calls, insect sounds, water sounds

**Enhancements:**
- [ ] **Sensory garden kits** for home observation
- [ ] **Multisensory field guide** (describe texture, sound, smell)
- [ ] **Scent samples** for common species

### Weather & Environmental Barriers

**Challenge:** Bad weather, pollen, heat, cold can prevent outdoor observation.

**Accommodations:**
- ✓ Flexible timing (observe in optimal weather)
- ✓ Indoor documentation of outdoor observations
- ✓ Seasonal flexibility (different species at different times)
- ✓ Climate-controlled alternatives

## Vision Impairment Accessibility

### For Blind Learners

**Challenges:** Species identification, field observation require detailed visual information.

**Remediation:**
- [ ] **Detailed species descriptions** (texture, sound, smell, habitat)
- [ ] **Tactile identification kits:** Leaves, flowers, seeds for touch
- [ ] **Audio field guides:** Recorded descriptions of common species
- [ ] **Braille field guides:** Species name, characteristics in Braille
- [ ] **Tactile maps:** Local habitat maps in raised format
- [ ] **Audio recordings:** Sounds from different ecosystems

**Timeline:** 2026-Q2-Q3

### For Low-Vision Learners

- ✓ Large-print field guides available
- ✓ High-contrast printed materials
- [ ] **Digital magnification** of field guide images
- [ ] **Audio descriptions** of habitat photographs

## Hearing Impairment Accessibility

**Challenge:** Field observation often includes auditory cues (bird calls, insect sounds, water flow).

**Accommodations:**
- [ ] **Visual bird identification** guides (without requiring sound)
- [ ] **Vibration sensors** for detecting animal movement
- [ ] **Video demonstrations** with captions and visual indicators
- [ ] **ASL interpretation** for nature walks
- [ ] **Text descriptions** of sounds encountered

## Neurodiversity Considerations

### ADHD-Friendly Features

- ✓ High engagement (active observation)
- ✓ Nature as sensory/motor outlet
- [ ] Add: **Structured observation protocols** (specific species to find)
- [ ] Add: **Shorter sessions** with clear start/end times
- [ ] Add: **Reward systems** for observation goals

### Autism Spectrum

- ✓ Systematic taxonomy (species classification)
- ✓ Detailed nature journaling
- ✓ Pattern recognition (seasonal cycles)
- [ ] Add: **Field guide organization** (by color, habitat, etc.)
- [ ] Add: **Scripted nature walks** with predictable routes
- [ ] Add: **Options for solitary observation** (not always group)

### Anxiety & Trauma

- ✓ Flexible, non-pressured observation
- ✓ Options for staying close to base/shelter
- [ ] Add: **Discussion of outdoor safety** to reduce anxiety
- [ ] Add: **Trauma-informed facilitation** for nature activities

## Technology Accessibility

### iNaturalist Platform

**Accessibility:** Mostly good; some enhancements needed

**Current:**
- ✓ Text-based species search
- ✓ Photo upload
- [ ] **Improve:** Image crop/positioning keyboard navigation
- [ ] **Improve:** AI ID result presentation for screen readers
- [ ] **Improve:** Comment section keyboard accessibility

### eBird Platform

**Accessibility:** Good bird identification text-based search

**Current:**
- ✓ Keyboard-navigable forms
- ✓ Text descriptions
- [ ] **Improve:** Sonogram audio descriptions

### Virtual Nature Tours

**Current Status:** Need audio descriptions

- [ ] 360° video captions
- [ ] Audio guidance through virtual landscape
- [ ] Detailed description of visible features
- [ ] Audio identification of species in view
