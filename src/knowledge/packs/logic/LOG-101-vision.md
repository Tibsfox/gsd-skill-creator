# LOG-101: Logic & Formal Systems — Foundational Knowledge Pack

## Vision Statement

Logic is the science of valid reasoning. It teaches us to distinguish sound arguments from fallacies, to manipulate abstract symbols with precision, and to build formal systems of thought. From the toddler learning "if-then" reasoning to the college student constructing rigorous proofs, logic is the backbone of clear thinking in mathematics, computer science, philosophy, law, and daily life.

This pack teaches logic not as a collection of rules to memorize, but as a set of reasoning skills to practice, internalize, and apply. Learners will experience the joy of spotting a fallacy in a political speech, constructing a watertight proof, and understanding why certain arguments work and others fail.

---

## Problem Statement

Logic is rarely taught as its own coherent subject. Instead, it appears in fragments:

- **Mathematics classes** teach logical proof but call it "math" rather than logic
- **Computer science** teaches boolean algebra and logic gates without connecting them to formal reasoning
- **Philosophy classes** dive into logical fallacies but often assume formal logic background students don't have
- **Everyday curricula** might address critical thinking or media literacy without explicit instruction in how arguments actually work

Students learn piecemeal, without seeing the unified discipline. Fallacy identification becomes a list to memorize rather than a skill to practice. Formal proof appears mysterious rather than mechanical. The connections between "if-then" thinking, symbolic logic, and rigorous argumentation remain invisible.

Additionally, logic is often taught as dry, removed from the world:
- Formal systems feel abstract and disconnected from real communication
- Real arguments in politics, persuasion, and rhetoric remain unexamined through a logical lens
- The history of logic—why these systems evolved—is omitted
- Applications to actual problem-solving (in law, science, engineering, code) are left unstated

This pack addresses the gap. Logic belongs in the foundation of every learner's education, presented as a coherent discipline with clear progression from everyday reasoning to formal systems.

## Core Concepts

### 1. Propositions & Logical Connectives

A proposition is a statement that is either true or false. Propositions are combined using logical connectives: AND, OR, NOT, and IF-THEN.

- **Conjunction (AND)**: "It is raining AND cold" is true only if both parts are true
- **Disjunction (OR)**: "I will study OR exercise" is true if at least one part is true
- **Negation (NOT)**: "It is NOT raining" reverses the truth value
- **Conditional (IF-THEN)**: "If it rains, then the game is canceled" — false only when antecedent is true and consequent is false
- **Biconditional (IF-AND-ONLY-IF)**: Both sides must match in truth value

From foundation level (understanding "if-then" in everyday language) through college (evaluating complex conditional statements and logical equivalences), this concept grounds all logical reasoning.

### 2. Argument Structure & Validity

An argument consists of premises (assumed true) and a conclusion (claimed to follow). An argument is valid if the conclusion necessarily follows from the premises; it is sound if it is valid AND all premises are actually true.

- **Deductive reasoning** starts with general principles and derives specific conclusions
- **Inductive reasoning** observes specific cases and generalizes to broader principles
- **Abductive reasoning** infers the best explanation for observed facts

Learning to diagram arguments, identify implicit premises, and distinguish valid structure from true content is foundational to all critical thinking.

### 3. Fallacies & Critical Evaluation

A fallacy is a failure of reasoning—either formally (invalid argument structure) or informally (true-sounding but misleading claims).

Formal fallacies include:
- **Affirming the consequent**: "If A then B. B is true, so A is true." (Invalid)
- **Denying the antecedent**: "If A then B. A is false, so B is false." (Invalid)

Informal fallacies include:
- **Ad hominem**: Attacking the speaker instead of the argument
- **Begging the question**: Assuming the conclusion in the premises
- **False dilemma**: Presenting only two options when more exist
- **Appeal to authority**: Using an expert inappropriately (appeal to authority from a non-expert)
- **Slippery slope**: Claiming inevitable negative consequences without evidence
- **Hasty generalization**: Drawing broad conclusions from limited cases
- **Post hoc ergo propter hoc**: Confusing sequence with causation

Learning to name and recognize these patterns inoculates against manipulation and clarifies thinking.

### 4. Symbolic Logic & Truth Tables

Propositional logic uses symbols to represent propositions and connectives:
- **p, q, r, ...** represent propositions
- **¬** represents NOT
- **∧** represents AND
- **∨** represents OR
- **→** represents IF-THEN
- **↔** represents IF-AND-ONLY-IF

A truth table shows all possible truth value combinations and the resulting truth value of a compound proposition. For example, (p ∧ q) → r requires 8 rows (2³ combinations).

Truth tables reveal logical equivalences:
- **De Morgan's Laws**: ¬(p ∧ q) ≡ (¬p ∨ ¬q)
- **Contrapositive**: (p → q) ≡ (¬q → ¬p)
- **Exportation**: ((p ∧ q) → r) ≡ (p → (q → r))

Mastering symbolic notation and truth tables gives learners precise tools for evaluating complex claims and discovering non-obvious logical truths.

### 5. Proof & Formal Systems

A proof is a finite sequence of statements, each either an axiom (assumed true) or derived from previous statements using inference rules. A statement is provable in a system if such a sequence exists.

Common proof techniques:
- **Direct proof**: Assume antecedent, derive consequent step-by-step
- **Proof by contradiction**: Assume negation of desired conclusion, derive a contradiction, conclude original statement
- **Mathematical induction**: Prove base case, then prove that if true for n, it's true for n+1
- **Proof by cases**: Divide into exhaustive cases, prove each

Learners progress from simple direct proofs (in high school geometry) to induction (in algebra and combinatorics) to formal systems (in college logic and discrete math).

## Skill Tree Architecture

Logic builds on itself in a clear progression:

### Foundation (PreK–2): Thinking in Patterns
- Sorting and classifying objects by properties
- Understanding "if-then" relationships ("If it's raining, then we play inside")
- Recognizing patterns and rules
- Using logic to solve simple puzzles

### Elementary (3–5): Arguments and Reasoning
- Distinguishing reasons from conclusions
- Identifying valid vs. invalid reasoning in stories and explanations
- Using "because" to explain reasoning
- Understanding "all," "some," and "none"
- Spotting obvious fallacies ("That's not true because you said it!")

### Middle School (6–8): Formal Thinking
- Diagramming arguments and identifying premises and conclusions
- Named fallacies (ad hominem, false dilemma, hasty generalization)
- Introduction to symbolic notation and truth tables
- Proof by contradiction and counterexample
- Logic in mathematics (geometric proofs, equations as logical statements)

### High School (9–12): Symbolic and Formal Logic
- Propositional logic: complete truth tables and logical equivalences
- Predicate logic: quantifiers (all, some, none) and properties
- Inference rules and rule-based reasoning
- Mathematical induction
- Applications in computer science and mathematics

### College & Beyond: Advanced Systems
- First-order logic and predicate calculus
- Completeness, soundness, and decidability
- Modal logic and temporal reasoning
- Proof assistants and formal verification
- Philosophical logic: relevance, paraconsistency, many-valued logic

## Modules

### Module 1: Logical Thinking Foundations
**Age Range**: PreK–5 | **Time**: 30–50 hours foundation; 25 hours elementary

Logic begins with play. Young learners sort objects by properties, recognize patterns, and understand cause-and-effect relationships. "If we add more water to the plant, then it will grow." The foundations of logical thinking are laid through activities that feel natural and playful.

**Core Topics**:
- Sorting and classifying by one or multiple properties
- If-then relationships in everyday contexts
- Recognizing patterns and rules
- Understanding relationships between objects and ideas
- Basic logical puzzles and games

**Learning Outcomes**:
- Classify objects systematically by shared properties
- Understand and use "if-then" language in context
- Recognize repeating patterns and predict what comes next
- Follow and create rules for games and activities
- Use logic to solve simple puzzles and riddles

---

### Module 2: Arguments & Reasoning
**Age Range**: 3–8 | **Time**: 25–35 hours across elementary and middle

As learners mature, they begin to understand argument structure. They learn to identify reasons and conclusions, distinguish deductive from inductive reasoning, and recognize valid versus invalid argument forms.

**Core Topics**:
- Premises and conclusions
- Reasons vs. conclusions in text and conversation
- Valid vs. invalid reasoning
- Deductive vs. inductive reasoning
- Soundness: Are the premises actually true?
- How to respond to an argument: agree with premises, disagree with premises, or show invalid reasoning

**Learning Outcomes**:
- Identify the main claim and supporting reasons in a paragraph
- Distinguish between a logical error and a false premise
- Evaluate whether an argument is deductively valid
- Recognize inductive arguments and assess their strength
- Construct clear, valid arguments supporting their own claims

---

### Module 3: Fallacies & Persuasion
**Age Range**: 6–12 | **Time**: 15–25 hours

Fallacies are failures of reasoning. Some are formally invalid (the structure doesn't work). Others are informally misleading (they sound good but rely on tricks). Learning to name and recognize fallacies is both intellectually satisfying and practically powerful—it inoculates against manipulation in advertising, politics, and rhetoric.

**Core Topics**:
- Formal fallacies (affirming the consequent, denying the antecedent)
- Informal fallacies: Ad hominem, straw man, false dilemma, begging the question, appeal to authority, hasty generalization, post hoc reasoning, slippery slope
- Propaganda techniques and persuasive tricks
- Media literacy: recognizing manipulation in advertising and news
- How to respond to fallacies: name them, show the error, offer better reasoning

**Learning Outcomes**:
- Name and identify at least 8 common fallacies
- Explain why a fallacious argument fails
- Recognize fallacies in real arguments (political speeches, advertisements, social media)
- Construct advertisements and arguments without fallacies
- Teach others to recognize and avoid fallacies

---

### Module 4: Symbolic Logic
**Age Range**: 9–16 | **Time**: 10–30 hours

The elegance of symbolic logic is that it strips away the complexity of natural language and reveals the structure beneath. In this module, learners become fluent with propositional logic notation and truth tables, discovering non-obvious logical truths and equivalences.

**Core Topics**:
- Propositions and truth values
- Logical connectives: AND, OR, NOT, IF-THEN, IF-AND-ONLY-IF
- Symbolic notation (∧, ∨, ¬, →, ↔)
- Truth tables and truth conditions
- Logical equivalence and De Morgan's Laws
- Predicate logic: quantifiers and properties
- From English to logic notation and back

**Learning Outcomes**:
- Translate between English and symbolic logic
- Construct complete truth tables for compound propositions
- Identify logical equivalences using truth tables
- Recognize valid argument forms (modus ponens, modus tollens, hypothetical syllogism)
- Apply symbolic logic to evaluate complex claims

---

### Module 5: Proof & Formal Systems
**Age Range**: 12–16+ | **Time**: 10–30 hours

The capstone module is proof. Learners experience the power of formal reasoning: starting from axioms (assumed truths) and inference rules, they derive necessarily true statements. Proof is logic in action.

**Core Topics**:
- What is a proof? Axioms, inference rules, derivation
- Direct proof: assume antecedent, derive consequent
- Proof by contradiction: assume negation, reach contradiction
- Mathematical induction: base case and inductive step
- Proof by cases: partition domain, prove each piece
- Logical consequence and satisfiability
- Introduction to formal systems (axiomatic systems, rules of inference)

**Learning Outcomes**:
- Prove simple mathematical statements using direct proof
- Use proof by contradiction effectively
- Apply mathematical induction to prove properties
- Understand the relationship between logical validity and mathematical proof
- Begin to reason about formal systems themselves

## Pedagogical Approach

Logic is best learned through activity, not lecture.

### **Principles**:

1. **Concrete before abstract**: Start with real arguments from everyday life. Only move to symbolic notation after learners understand the underlying ideas.

2. **Play as practice**: Logic puzzles, games, and creative activities develop intuition faster than drills.

3. **Dialogue and debate**: Have learners construct arguments, attack each other's reasoning, defend against criticism. Argument is a social, interactive process.

4. **Multiple modalities**: Use diagrams, charts, games, stories, and symbolic notation. Different brains reach logic through different doors.

5. **Application before formalism**: Show why logic matters before teaching the formal system. Then formalism becomes a tool, not a mystery.

6. **Errors as learning**: Fallacies and invalid arguments are teaching moments. Celebrate finding errors; don't penalize them.

7. **Progress through levels**: The same concepts return at higher levels of sophistication. Revisit "if-then" reasoning from elementary through college, each time adding rigor and complexity.

## Assessment Framework

Logic learning is assessed through reasoning quality, not answer memorization.

### **Formative Assessment** (ongoing, informal):
- Classroom discussions: Are students spotting fallacies? Defending their claims? Questioning reasoning?
- Problem-solving activities: Can students construct valid arguments for their own claims?
- Peer critique: Do students give constructive feedback on each other's reasoning?
- Reflection: Can students articulate why a given argument works or fails?

### **Summative Assessment** (formal, end-of-unit):
- Argument construction: Write a multi-premise argument and defend it against counterargument
- Fallacy identification: Recognize and name fallacies in provided texts
- Symbolic logic: Construct truth tables, prove equivalences, translate between English and logic
- Proof writing: Prove a statement using appropriate techniques
- Real-world application: Analyze an advertisement, political claim, or scientific argument for logical structure and fallacies

### **Rubric Levels**:

**Beginning**: Student identifies obvious logical errors, attempts to construct basic arguments, recognizes simple fallacies when named.

**Developing**: Student consistently identifies logical structure, names fallacies independently, constructs clear arguments with minor gaps, understands basic symbolic logic.

**Proficient**: Student constructs rigorous, multi-premise arguments; identifies subtle fallacies; fluently translates between English and logic; writes simple proofs.

**Advanced**: Student creates arguments addressing counterarguments; analyzes logical structure in complex real-world texts; teaches fallacy recognition to others; applies formal logic to novel domains.

## Skill Progression by Age

### PreK–2: "Logic is Sorting and Patterns"
Learners engage with sorting games, pattern recognition, if-then stories, and simple logic puzzles. The focus is on the joy of order and prediction.

### 3–5: "Logic is Reasons and Conclusions"
Learners learn to identify what someone claims and why. They practice supporting their own claims with reasons. They spot when reasoning fails ("That doesn't make sense because...").

### 6–8: "Logic is Fallacies and Valid Argument"
Learners name common fallacies and learn to explain why arguments fail. They begin to see logical structure in texts. They prove simple statements (e.g., geometric proofs).

### 9–12: "Logic is Formal Proof"
Learners master symbolic notation, construct truth tables, prove statements, and understand formal systems. They apply logic to mathematics, computer science, and philosophy.

### College+: "Logic is a Language for Reasoning"
Learners use logic as a tool in specialized domains: formal verification in CS, logical reasoning in philosophy and law, proof in mathematics. They may explore advanced topics like modal logic or proof assistants.

## Parent & Mentor Guidance

### Why Logic Matters for Your Learner

Logic is not abstract philosophy. It's the foundation of:
- **Clear communication**: Being able to explain your thinking persuasively
- **Critical thinking**: Spotting bad reasoning in advertising, media, and public discourse
- **Problem-solving**: Breaking complex problems into logical pieces
- **Mathematics & science**: Understanding proof and experimental reasoning
- **Computer science**: Boolean logic and algorithms
- **Philosophy & ethics**: Reasoning about right and wrong

A learner who can think logically is equipped to understand nearly any field and to navigate a world full of claims competing for attention.

### How to Support Logic Learning at Home

1. **Play logic games**: Sudoku, chess, logic puzzles, pattern games. These are logic training in disguise.

2. **Ask "Why?" and "How do you know?"**: Encourage your learner to explain their thinking. Push back respectfully. "That's a good reason. Can you think of a case where it might not work?"

3. **Spot fallacies together**: When watching TV, reading news, or discussing politics, ask: "Is that a fair argument? Do you see any fallacies?"

4. **Construct arguments together**: "You want to stay up late. Give me three reasons why I should allow it. Make them strong reasons—not 'because.'"

5. **Play devil's advocate**: Help your learner defend against criticism. "Okay, but what if someone says...?"

6. **Read and discuss stories**: Stories contain implicit logical reasoning. "Why did the character make that choice? Was it logical?"

### Red Flags

- Learner struggles to give reasons for claims ("I don't know why, I just think so")
- Learner assumes if something sounds good, it must be true
- Learner dismisses counterarguments without engaging with them
- Learner confuses "I believe it" with "It's true"

These are not failures—they're growth opportunities. Logic is learned through practice, and even adults improve with attention.

## Frequently Asked Questions

**Q: Isn't logic just philosophy?**
A: Logic appears in philosophy, but also in mathematics (proof), computer science (algorithms), law (reasoning), and everyday persuasion. It's a tool used across domains.

**Q: Can young children really learn logic?**
A: Yes. Young learners reason logically all the time. We make that reasoning explicit and systematic through games and activities.

**Q: Will learning formal logic make my learner overly pedantic?**
A: The opposite. Good logic instruction teaches when to be precise and when to communicate naturally. It's a tool, not a straitjacket.

**Q: How does logic relate to mathematics?**
A: Logic is the language and tool of mathematics. Proof—the heart of mathematics—is logic in action. Both packs reinforce each other.

**Q: Is logic useful in real life?**
A: Absolutely. It helps with clear communication, detecting manipulation, problem-solving, and understanding nearly any complex topic.
