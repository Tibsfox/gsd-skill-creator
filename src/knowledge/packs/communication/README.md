# COMM-101: Communication & Speech

**Classification:** Core Academic
**Status:** Alpha
**Version:** 1.0.0

Communication as the skill of making ideas travel between minds -- speaking clearly, listening deeply, reading nonverbal signals, presenting ideas, and collaborating through dialogue.

## Quick Start

This pack teaches the full spectrum of **human communication**, from one-on-one conversation to formal presentation, from active listening to structured debate. It is the human interface layer for all other knowledge packs.

### Modules

| Module | Name | Focus |
|--------|------|-------|
| M1 | Foundations of Speaking & Listening | Clarity, turn-taking, active listening, questioning |
| M2 | Nonverbal Communication & Presence | Body language, voice, eye contact, presence |
| M3 | Structured Presentation | Audience analysis, structure, delivery, anxiety |
| M4 | Discussion, Debate & Collaboration | Protocols, argumentation, feedback, consensus |
| M5 | Communication Across Contexts | Digital, cross-cultural, professional, ethics |

### Prerequisites

- None required (this is a foundational pack)
- READ-101 recommended for vocabulary and comprehension support

### What This Pack Enables

- WRIT-101 (Writing & Composition)
- BUS-101 (Business & Entrepreneurship)
- LANG-101 (World Languages)
