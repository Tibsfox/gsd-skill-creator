# PE-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Physical Education's challenge is inherent: not all learners can move the same way. PE-101 explicitly teaches that physical education is for **all bodies, all abilities**. This audit ensures that every learner can participate meaningfully.

## Physical Ability Considerations

### Mobility-Limited Learners

**Challenge:** Many PE-101 activities involve locomotion, balance, or fine motor control that wheelchair users or people with limited mobility cannot perform in standard ways.

**Adaptations Provided:**
- ✓ Seated alternatives for all activities
- ✓ Visual/strategic roles (coaching, scorekeeping, officiating)
- ✓ Partner-based modifications
- ✓ Equipment adaptations (seated volleyball, wheelchair sports)
- ✓ Adapted PE specialist resources

**Enhancements Needed:**
- [ ] Wheelchair dance choreography resources
- [ ] Seated fitness routines with progressive complexity
- [ ] Accessible exercise equipment descriptions
- [ ] Partnerships with wheelchair sports organizations

**Timeline:** 2026-Q2

### Vision Impairment

**Challenge:** PE activities require spatial awareness and often visual demonstrations.

**Adaptations:**
- ✓ Verbal descriptions of movements and formations
- ✓ Tactile guidance (hand-over-hand instruction)
- ✓ Audio cues and feedback
- ✓ Modified equipment (e.g., goalball, beep baseball)
- ✓ Accessible fitness facilities (treadmill with verbal feedback)

**Enhancements Needed:**
- [ ] Audio description tracks for all video demonstrations (30-40 seconds per clip)
- [ ] Tactile diagram guides for game strategies
- [ ] Partnerships with blind sports organizations
- [ ] Accessible outdoor space assessment

**Timeline:** 2026-Q2

### Hearing Impairment

**Challenge:** Many PE instructions are audio-only; communication on court/field is verbal.

**Adaptations:**
- ✓ Video captions for all demonstrations
- ✓ Visual signals and hand signals
- ✓ Written activity descriptions
- ✓ ASL interpretation available
- ✓ Visual timer displays

**Enhancements Needed:**
- [ ] Certified ASL interpreters for workshops
- [ ] Visual communication protocols for group activities
- [ ] Captions for live demonstrations
- [ ] Accessible audio descriptions (for those with usable hearing)

### Neuromuscular Disorders (CP, MS, SMA, etc.)

**Challenge:** Fine motor control, endurance, or muscle tone issues affect standard movement.

**Adaptations:**
- ✓ Activity modifications by muscle group/motor control level
- ✓ Modified equipment (lighter balls, shorter distances)
- ✓ Fatigue management strategies
- ✓ Frequent breaks built into activities
- ✓ Energy conservation techniques

**Enhancements Needed:**
- [ ] Individualized adaptation guides
- [ ] Partnerships with physical therapists
- [ ] Peer support networks

## Neurodiversity Considerations

### Sensory Processing Differences

- ✓ Options for proprioceptive activities
- ✓ Vestibular activity alternatives
- ✓ Low-stimulus environments available
- [ ] Add: **Sensory breaks** between activities
- [ ] Add: **Weighted clothing/equipment** options
- [ ] Add: **Visual predictability** (activity schedules)

### ADHD-Friendly Features

- ✓ High-engagement physical activities
- ✓ Clear instructions with demonstration
- ✓ Movement breaks integrated
- [ ] Add: **Shorter activity segments** with transitions
- [ ] Add: **Choice** in activity selection
- [ ] Add: **Immediate feedback** on performance

### Autism Spectrum

- ✓ Structured activities with clear rules
- ✓ Predictable patterns and transitions
- [ ] Add: **Social scripts** for group activities
- [ ] Add: **Options for non-contact participation**
- [ ] Add: **Advance warning** of activity changes

## Mental Health & Psychological Accessibility

### Body Image Sensitivity

**Challenge:** Some learners avoid PE due to body image anxiety, especially adolescents.

**Protections:**
- ✓ No weight or body measurements as metrics
- ✓ Focus on personal improvement, not comparison
- ✓ Inclusive language throughout
- ✓ Gender-neutral language for all activities
- ✓ Private changing areas

**Enhancements:**
- [ ] Mental health support resources
- [ ] Discussion of social media and body image
- [ ] Celebration of body diversity

### Trauma-Informed Practices

- ✓ Consent-based participation (can opt-out)
- ✓ No forced physical contact
- ✓ Respectful of personal space
- ✓ Trigger warnings for contact sports
- [ ] Add: **Trauma-informed facilitation guide** for teachers
