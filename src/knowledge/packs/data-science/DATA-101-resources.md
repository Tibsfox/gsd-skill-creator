# DATA-101 Resources: Tools, Data Sets, and References

This document curates tools, data sets, and references useful for teaching and learning data literacy with DATA-101.

---

## Foundational Texts & Curricula

### Books

- [Naked Statistics: Stripping the Dread from the Data](https://www.amazon.com/Naked-Statistics-Stripping-Dread-Data/dp/0393071669) by Charles Wheelan — Accessible introduction to statistical thinking with real examples
- [Freakonomics](https://freakonomics.com/) by Steven D. Levitt and Stephen J. Dubner — Economics and data analysis applied to unexpected questions
- [Thinking with Data](https://www.oreilly.com/library/view/thinking-with-data/9781491956007/) by Max Shron — Guide to data practice for non-specialists
- [The Truthful Art](https://www.amazon.com/Truthful-Art-Data-Charts-Communication/dp/0321934075) by Alberto Cairo — Data visualization principles and critique
- [How to Lie with Statistics](https://www.amazon.com/How-Lie-Statistics-Darrell-Huff/dp/0393310728) by Darrell Huff — Classic guide to misleading graphs and claims (with solutions)
- [The Model Thinker](https://www.amazon.com/Model-Thinker-What-You-Need/dp/0374519944) by Scott Page — Using data and models to understand systems

### Curricula & Lesson Plans

- [Data Science for Good](https://www.kaggle.com/learn/intro-to-sql) from Kaggle — Free interactive tutorials
- [Khan Academy Statistics & Probability](https://www.khanacademy.org/math/statistics-probability) — Video lessons and practice
- [Statistics Online Computational Resource (SOCR)](http://www.socr.ucla.edu/) — Interactive demonstrations and applets
- [Stat Trek](https://stattrek.com/) — Statistics lessons and tutorials

## For Learners: Tools & Platforms

### Spreadsheets & Data Entry

- [Google Sheets](https://sheets.google.com) — Free, web-based spreadsheet with graphing and sharing (Recommended start here)
- [LibreOffice Calc](https://www.libreoffice.org) — Free desktop spreadsheet alternative to Excel
- [Excel Online](https://www.office.com) — Microsoft Excel in browser (requires account)

### Visualization & Dashboard Tools

- [Gapminder](https://www.gapminder.org) — Interactive, animated visualizations of global trends with narrative context (Excellent for K-12)
- [Our World in Data](https://ourworldindata.org) — High-quality, open-source data visualizations on global topics
- [Google Charts](https://www.google.com/charts/gallery) — Free charting library; integrates with Sheets
- [Plotly for Spreadsheets](https://plotly.com/spreadsheets/excel/) — Interactive charts from spreadsheet data
- [Tableau Public](https://public.tableau.com/) — Free data visualization tool (more complex; for older students)
- [Flourish](https://flourish.studio/) — Modern visualization and storytelling with data

### Data Analysis & Programming

- [Google Colaboratory](https://colab.research.google.com/) — Free Python environment in browser; no installation needed
- [Replit](https://replit.com/) — Online code editor; supports Python, R, JavaScript
- [RStudio Cloud](https://posit.cloud/) — Free cloud-based R programming environment

### Interactive Simulations

- [PhET Interactive Simulations](https://phet.colorado.edu) — Free simulations including probability and statistics
- [Desmos](https://www.desmos.com) — Free graphing calculator and tools
- [StatKey](http://www.lock5stat.com/StatKey/index.html) — Interactive statistics simulations and bootstrap methods

## For Parents & Mentors

### Introductions to Data Literacy

- [Khan Academy: Understanding Graphs](https://www.khanacademy.org/math/statistics-probability) — Video basics
- [Pew Research Center: How to Evaluate a Poll](https://www.pewresearch.org/methodology/) — Guide to understanding surveys
- [How to Read Scientific Papers](https://www.youtube.com/watch?v=ggCR5QhYmDc) — Michael Eisen on evaluating studies
- [TED Talks on Data](https://www.ted.com/search?q=data%20visualization) — Entertaining talks about data and visualization

### Supporting at Home

- Use data from your family: "How much do we spend on groceries?" "How have temperatures changed?" "How much screen time does each family member use?"
- Notice data in news: "That article cites a study. What data support that claim?"
- Play with visualizations together: Explore Gapminder or Our World in Data
- Discuss decisions: "The city is building a new park. What data would help them decide where?"

## How to Use These Resources

1. **Get started:** Begin with [Google Sheets](https://sheets.google.com) for data entry and [Gapminder](https://www.gapminder.org) for exploring data
2. **Find data:** Browse [Data.gov](https://www.data.gov) or [Kaggle](https://www.kaggle.com/datasets) for real data
3. **Learn:** Watch Khan Academy videos or Crash Course Statistics videos
4. **Teach:** Use lesson plans from ASA or scholastic.com
5. **Go deeper:** Read books on data ethics and visualization
6. **Stay current:** Follow FiveThirtyEight or ProPublica for data journalism examples

## Recommendations by Grade Level

**Elementary (3-5):**
- Start with: Google Sheets, Gapminder
- Data sets: Census at School, simple class surveys
- Videos: Khan Academy basics, TED-Ed lessons

**Middle School (6-8):**
- Tools: Google Sheets, Plotly, Desmos
- Data sets: Kaggle, government data, sports data
- Resources: Crash Course Statistics, How to Lie with Statistics
- Discussions: Data ethics, bias in visualization

**High School (9-12):**
- Tools: Google Colaboratory (Python), RStudio
- Resources: OpenStax Statistics, FiveThirtyEight, ProPublica
- Read: Articles on algorithmic bias, data ethics
- Create: Original data analysis projects

**College+:**
- Advanced texts: ISLR, The Book of Why, Weapons of Math Destruction
- Programming: Python (pandas, scikit-learn), R
- Research: Academic papers, machine learning, causal inference
