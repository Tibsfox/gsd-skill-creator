# CRIT-101: Critical Thinking

**Classification:** Core Academic
**Status:** Alpha
**Version:** 1.0.0

Critical thinking as the discipline of thinking well -- analyzing arguments, evaluating evidence, recognizing fallacies, identifying cognitive biases, and making reasoned judgments. The meta-skill that improves performance in every other domain.

## Quick Start

This pack teaches **how to think**, not what to think. It provides the reasoning tools that improve learning in every other domain: evidence evaluation, argument analysis, fallacy detection, bias awareness, and metacognition.

### Modules

| Module | Name | Focus |
|--------|------|-------|
| M1 | Claims & Evidence | Evaluating evidence, source reliability, burden of proof |
| M2 | Arguments & Logical Reasoning | Argument structure, deduction, induction, counterarguments |
| M3 | Fallacies & Biases | Recognizing reasoning errors and cognitive biases |
| M4 | Applied Critical Thinking | Media literacy, decision making, ethics, Socratic method |
| M5 | Metacognition & Intellectual Humility | Thinking about thinking, calibration, intellectual courage |

### Prerequisites

- None required (this is a foundational pack)
- READ-101 recommended for textual analysis skills

### What This Pack Enables

- LOG-101 (Formal Logic)
- PHILO-101 (Philosophy)
- PROB-101 (Problem Solving)
