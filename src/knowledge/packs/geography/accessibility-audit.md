# GEO-101: Accessibility Audit

**Pack:** Geography — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Partially Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Partial | Text descriptions of maps and places provided; detailed spatial relationships need work |
| Large Text Support | Implemented | Map labels and text scale well for larger fonts |
| High Contrast Mode | Implemented | High-contrast map backgrounds available |
| Keyboard Navigation | Implemented | Interactive maps support keyboard exploration |
| Alt Text for Images | Partial | Map and satellite image alt text included; spatial relationships not fully described |
| Media Captions | Not Started | No video content; captions needed for geographic demonstrations |

## Content-Specific Considerations

- **Maps:** Visual centerpiece of geography. Text-based map descriptions and legends provided; interactive maps include keyboard navigation and searchable landmarks/features.
- **Spatial Relationships:** Directional language used (north, south, relative position); visual maps described in text. Tactile maps recommended for blind learners.
- **Observation Activities:** Outdoor observation tasks include multi-sensory elements (temperature, sounds, textures) naturally accessible.
- **Astronomy (M4):** Celestial relationships are visual; descriptions of star positions, constellations, planetary movements provided with spatial detail.

## Remediation Path

### Priority 1 (Critical)
1. Create detailed text descriptions of all maps with spatial relationships
2. Develop tactile/raised-relief maps for hands-on geographic exploration
3. Add comprehensive legend descriptions for map symbols and colors
4. Ensure interactive maps have full keyboard navigation and search functionality
5. Create detailed descriptions of satellite imagery and geographic features

### Priority 2 (Important)
1. Add captions to any geographic video demonstrations
2. Create alternative data representations (tables, lists) for map-based information
3. Develop audio descriptions of geographic phenomena (weather patterns, water cycles)
4. Build collaborative geographic investigation guides
5. Create accessible assessment alternatives for map-based activities

### Priority 3 (Enhancement)
1. Develop 3D tactile map models for hands-on learning
2. Create audio descriptions of geographic field trips
3. Build customizable map visualization options (colors, detail levels)
4. Develop sound-based representations of weather patterns

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level A for text descriptions; Level AA for interactive maps
- **Major Gaps:**
  - Map descriptions need more spatial detail
  - Satellite imagery descriptions incomplete
  - Tactile map alternatives not yet available
  - Geographic video content not captioned
- **Strengths:**
  - Outdoor observation is naturally multi-sensory
  - Interactive maps have keyboard support
  - Good contrast in map interfaces
  - Text-based geographic information available
