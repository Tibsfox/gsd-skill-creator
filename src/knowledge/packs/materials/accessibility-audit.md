# MFAB-101: Accessibility Audit

**Pack:** Materials & Fabrication — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Partially Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Partial | Material testing instructions are clear; material property charts need descriptions |
| Large Text Support | Implemented | Property tables and specifications scale well |
| High Contrast Mode | Implemented | High-contrast material comparison interface available |
| Keyboard Navigation | Implemented | Material testing tools support keyboard input |
| Alt Text for Images | Partial | Material samples and fabrication diagrams included; detailed descriptions provided |
| Media Captions | Not Started | No video content; captions needed for fabrication demonstrations |

## Content-Specific Considerations

- **Hands-on Material Testing:** Inherently tactile and accessible. Learners with visual impairments can explore materials through touch (texture, weight, flexibility, temperature change). Verbal descriptions support understanding.
- **Material Properties:** Charts comparing strength, flexibility, durability include text descriptions and accessible data tables. Visual indicators (color-coded) have text-based alternatives.
- **Fabrication Techniques:** Step-by-step instructions are text-based and detailed. Safety warnings emphasized with extra clarity.
- **Sustainability (M5):** Discussion-based and narrative; naturally accessible through reading and reflection.

## Remediation Path

### Priority 1 (Critical)
1. Create detailed tactile property descriptions for each material (texture, warmth, weight, feel)
2. Ensure material comparison tables have clear headers and accessible structure
3. Provide keyboard shortcuts for all material testing tool interfaces
4. Add detailed descriptions of fabrication technique diagrams
5. Create safety information in highly accessible, redundant formats

### Priority 2 (Important)
1. Add captions to material testing and fabrication demonstration videos
2. Develop accessible alternative formats for material specification sheets
3. Create text-based alternatives for property visualizations
4. Build collaborative material investigation guides
5. Develop accessible sustainability assessment tools

### Priority 3 (Enhancement)
1. Develop tactile/3D material sample kits for reference
2. Create audio descriptions of material transformations
3. Build multi-sensory material property exploration activities
4. Develop material sustainability audio guides

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level A for text instructions; Level AA for interactive tools
- **Major Gaps:**
  - Material property descriptions could be more tactile/sensory
  - Some visualization descriptions incomplete
  - Fabrication demonstration videos not captioned
  - Safety information could be more multi-modal
- **Strengths:**
  - Hands-on testing is inherently tactile
  - Clear text-based instructions
  - Good contrast in material interfaces
  - Property tables are structured text
