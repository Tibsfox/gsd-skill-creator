# MUSIC-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Music is uniquely accessible: deaf/hard-of-hearing learners can feel vibrations and see sheet music; blind learners can hear and feel instruments. However, audio-visual balance matters.

## Hearing Accessibility

### For Deaf Learners

**Challenge:** Music is audio; many activities require listening.

**Remediation:**
- ✓ **Visual rhythm notation** (see rhythm without hearing)
- ✓ **Sheet music** (read the composition)
- ✓ **Vibration exploration** (feel the beat, bass frequencies)
- [ ] **ASL Interpreter** for group instruction
- [ ] **Tactile vibration sensors** for frequency exploration
- [ ] **Video with captions + visual music analysis**
- [ ] **Sign-sung music** (BSL, ASL versions of songs)

**Timeline:** 2026-Q2

### For Hard-of-Hearing Learners

- ✓ **Video captions** for listening-based activities
- ✓ **Assisted listening devices** available
- [ ] **Visual demonstration** of how sounds are made
- [ ] **Vibration feedback** for beat/rhythm

## Vision Accessibility

### For Blind Learners

**Challenge:** Sheet music and music notation are visual; demonstrations show hand positions.

**Remediation:**
- ✓ **Braille music notation** for those who know Braille music
- ✓ **Audio descriptions** of demonstrations
- ✓ **Learning by ear** (traditional music pedagogy)
- [ ] **Braille note names and symbols** resources
- [ ] **Tactile instrument exploration** (feel how it produces sound)
- [ ] **Hand-over-hand instruction** (feel correct technique)
- [ ] **Audio score guides:** Narrated description of musical form

### For Low-Vision Learners

- ✓ **Large-print sheet music** available
- ✓ **High-contrast notation**
- [ ] **Magnified demonstration videos**
- [ ] **Digital magnification** of sheet music

## Music Participation Accessibility

### Singing & Vocal

**Status:** Excellent accessibility ✓

- ✓ No equipment needed
- ✓ Can participate at any volume level
- ✓ Doesn't require reading notation
- [ ] Add: **Recorded backing tracks** (not requiring others to accompany)

### Playing Instruments

**Challenge:** Some learners lack fine/gross motor control for traditional instruments.

**Adaptations:**
- ✓ **Adaptive instruments:** Larger keys, lighter keys, one-handed options
- ✓ **Glockenspiel** (easier than piano)
- ✓ **Drums/percussion** (accessible to many ability levels)
- [ ] **Electronic instruments** with accessibility features
- [ ] **Adapted drumsticks** for limited grip strength
- [ ] **Foot pedals** as input method

### Movement & Dance

- ✓ All movement can be adapted
- ✓ Seated movement options
- ✓ Minimal movement alternatives
- [ ] Add: **Choreography** using only hand movements

## Digital Music Tools Accessibility

### Music Notation Software (MuseScore)

**Accessibility:**
- ✓ Keyboard navigation possible
- [ ] Improve: Screen reader support for playback
- [ ] Improve: Keyboard shortcuts for all common tasks
- [ ] Improve: Note entry without mouse

### Digital Audio Workstation (GarageBand, Audacity)

**Accessibility:**
- ✓ Most functions keyboard-accessible
- [ ] Improve: Screen reader labeling of controls
- [ ] Improve: Zoom and magnification options
- [ ] Improve: Voice control compatibility

### Listening/Analysis Tools

- ✓ Text descriptions of compositions
- ✓ Transcrip of lyrics/liner notes
- [ ] Add: **Interactive score** with audio (follow along)
- [ ] Add: **Timeline annotations** (what happens at 0:45)
