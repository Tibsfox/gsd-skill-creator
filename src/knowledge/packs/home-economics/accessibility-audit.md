# DOMESTIC-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Home economics is deeply practical: cooking, sewing, cleaning, budgeting. This audit ensures all activities have accessible alternatives and multi-modal instruction.

## WCAG 2.1 AA Accessibility Status

| Feature | Status | Notes |
|---------|--------|-------|
| Video Captions | ✓ Yes | All demonstrations captioned |
| Audio Descriptions | ✓ Partial | Primary activities described |
| Text Instructions | ✓ Yes | All activities have written steps |

| Keyboard Navigable | ✓ Yes | Digital tools (calculators, meal planners) |

## Motor Accessibility Challenges

### Cooking & Kitchen Activities

**Challenge:** Cooking requires fine/gross motor control, standing, hand strength (chopping, mixing).

**Adaptations:**
- ✓ **Seated cooking** at accessible height (lower counters, seated chairs)
- ✓ **Adaptive kitchen equipment:** One-handed utensils, jar openers, grippers
- ✓ **Measured ingredients:** Pre-measured for simplified recipes
- ✓ **Delegation:** Group cooking with role distribution
- [ ] **Adaptive equipment catalog:** Links to specialized cooking tools
- [ ] **Video demonstrations** of adaptive techniques

### Textiles & Sewing

**Challenge:** Sewing requires fine motor control, vision, dexterity.

**Adaptations:**
- ✓ **Large-stitch projects** (reduces precision required)
- ✓ **Adaptive sewing tools:** Larger needles, magnification
- ✓ **Hand-sewing + machine alternatives:** Choose method that works
- ✓ **Repurposing activities:** Creative use instead of constructing from scratch
- [ ] **One-handed sewing techniques**
- [ ] **Texture-feeling fabric exploration** for blind learners

### Home Maintenance

**Challenge:** Cleaning, organizing, repair require reaching, bending, physical exertion.

**Adaptations:**
- ✓ **Seated organization:** Drawer, shelf organization at accessible height
- ✓ **Minimal-movement cleaning:** Spot-cleaning vs. full deep clean
- ✓ **Adaptive equipment:** Long-handled tools, lightweight equipment
- ✓ **Instruction-based participation:** Plan cleaning, supervise others
- [ ] **Energy conservation techniques** for people with chronic fatigue

## Vision Accessibility

### For Blind Learners

**Challenges:** Cooking, sewing, budgeting rely on visual information.

**Remediation:**
- [ ] **Audio recipe instruction:** Step-by-step verbal guidance
- [ ] **Tactile measurement tools:** Raised-line measuring cups (feel level)
- [ ] **Audio calculators/budgeting apps:** Voice output for financial planning
- [ ] **Fabric identification by touch:** Texture, weight, thickness
- [ ] **Kitchen layout guides:** Tactile maps of safe organization
- [ ] **Speaking labels:** For organizing items (label maker with voice output)

### For Low-Vision Learners

- ✓ Large-print recipes available
- ✓ High-contrast measuring cups
- [ ] **Digital magnification** of instructional videos
- [ ] **Increased font size** on digital tools

## Safety-Critical Accessibility

**Challenge:** Home economics involves hazards (sharp tools, heat, chemicals).

**Critical Accessibility Need:** Safety warnings must be **multi-modal** (not visual-only):

- ✓ **Written warnings** on all materials
- [ ] **Audio warnings** at start of activity
- [ ] **Video captions + audio descriptions** for hazard identification
- ✓ **Supervised instruction** ensuring hazard awareness
- ✓ **Adapted tools** reducing injury risk (e.g., child-safe knives)

## Hearing Accessibility

**Challenge:** Demonstration videos, group instruction may be audio-only.

**Accommodations:**
- ✓ **Video captions** for all demonstrations
- ✓ **Written recipe cards** accompanying video
- ✓ **Visual demonstrations** (can be followed without audio)
- [ ] **ASL interpretation** for live workshops
- [ ] **Vibration alerts** for kitchen timers

## Neurodiversity Considerations

### ADHD-Friendly Features

- ✓ Hands-on, concrete activities
- ✓ Immediate results (cooking produces edible food)
- [ ] Add: **Shorter recipes** (30 minutes or less)
- [ ] Add: **Pre-prep options** (ingredients gathered, measured)
- [ ] Add: **Checklist reminders** for multi-step processes

### Autism Spectrum

- ✓ Systematic, step-by-step instructions
- ✓ Predictable outcomes
- ✓ Detailed recipe precision
- [ ] Add: **Exact measurements** (not "a pinch" or "to taste")
- [ ] Add: **Social scripts** for group cooking
- [ ] Add: **Sensory considerations** (texture preferences, smell)

### Anxiety

- ✓ Low-pressure exploration (no grading)
- ✓ Options for private practice
- [ ] Add: **Success-guaranteed recipes** (very hard to fail)
- [ ] Add: **Discussion of cooking mistakes** as normal

## Sensory Accessibility

### Cooking

- ✓ Taste exploration (sensory rich!)
- ✓ Smell description important for blind learners
- ✓ Texture identification (crunchy, smooth, etc.)
- [ ] Add: **Multisensory recipe descriptions**
- [ ] Add: **Sensory preparation guides** (what to expect)

### Sewing

- ✓ Tactile fabric exploration
- ✓ Texture-based discrimination
- [ ] Add: **Fabric sample kit** (feel different weaves)
- [ ] Add: **Stitch texture descriptions**

## Technology Accessibility
