# PHILO-101: Philosophy Standards Alignment

This document maps PHILO-101 (Philosophy: Foundational Knowledge Pack) to major educational standards frameworks.

## Philosophy for Children (P4C) Standards

The **Philosophy for Children curriculum** provides explicit alignment with the Community of Inquiry model:

| Standard | Module Alignment | Description |
|----------|------------------|-------------|
| Community of Inquiry (COI) | M1, M2, M3, M5 | Socratic dialogue, collective meaning-making, shared responsibility for inquiry |
| Reasoning Skills | M2, M4 | Logical thinking, argument analysis, inference, critical evaluation |
| Ethical Reasoning | M3 | Moral dilemmas, value exploration, ethical frameworks |
| Dialogue & Questioning | M1, M2, M4 | Asking probing questions, deep listening, response scaffolding |
| Metacognition | M3, M5 | Thinking about thinking, self-reflection, examining assumptions |

## IB Theory of Knowledge (TOK)

PHILO-101 aligns with **IB TOK**, the core IB diploma course on knowledge itself:

| TOK Element | PHILO-101 Module | Application |
|-------------|-----------------|-------------|
| Knowledge Questions | M4 | Epistemological inquiry: "How do we know?" |
| Knowing vs. Believing | M2, M4 | Distinction between justified belief and mere opinion |
| Ways of Knowing | M1-M5 | Sense perception, reason, emotion, intuition, testimony across modules |
| Areas of Knowledge | M3, M4, M5 | Ethics, aesthetics, sciences, humanities |
| Knower's Perspective | M5 | Consciousness, subjectivity, self-awareness |

## Pedagogical Note

Philosophy education standards emphasize **process over content** — the quality of thinking, dialogue, and inquiry matters more than memorized doctrines. PHILO-101 assesses reasoning quality, perspective-taking ability, and genuine engagement with ideas rather than recall of specific philosophers or theories.
