# LEARN-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

**LEARN-101 is meta-pack about learning itself.** As such, it should **model best accessibility practices** in every dimension. This audit ensures that the pack about learning is itself accessible to all learners.

## Multi-Modal Accessibility (Core to LEARN-101)

**Principle:** Since LEARN-101 teaches multiple learning modalities, it MUST implement multiple modalities throughout:

| Learning Style | LEARN-101 Implementation |
|---|---|
| **Visual** | Infographics, diagrams, color-coded information, videos |
| **Auditory** | Lectures, podcasts, discussion recordings, verbal explanations |
| **Kinesthetic** | Hands-on experiments with learning science (timing memory tests, etc.) |
| **Reading/Writing** | Written guides, articles, journaling, note-taking |

**Critical:** All core concepts taught in **at least 3 modalities**.

## Growth Mindset Messaging & Accessibility

**Critical Principle:** Growth mindset messaging must NEVER equate disability with fixed limitation.

### Language Review

- [ ] **Audit all content** for ableist language
- [ ] **Never use disability metaphors** for limitation ("deaf to feedback," "blind spot")
- [ ] **Affirm that neurotypes differ**, not that "learning differences" mean deficit
- [ ] **Celebrate different learning modalities** as diversity, not accommodation

### Inclusive Examples

- [ ] Include **disabled people as role models** of growth and learning
- [ ] Feature **neurodivergent professionals** and their learning strategies
- [ ] Provide **examples of successful people with disabilities**
- [ ] Discuss **how people with disabilities learned their skills**
