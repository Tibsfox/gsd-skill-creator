The documentation section provides technical reference material for the gsd-skill-creator platform and its components. Topics span the desktop shell environment built on Tauri, the skill validation pipeline that ensures quality and correctness, the conflict detection system that prevents overlapping definitions, and the pattern discovery engine that identifies reusable abstractions from real project work.

Additional guides cover team validator configuration, agent orchestration, the holomorphic composition layer, and integration points with external services. Each document is written for practitioners who need to understand internals, extend functionality, or troubleshoot specific subsystems.

Whether you are building custom skills, configuring agent teams, wiring up MCP connections, or exploring the electronics and retro computing educational packs, these pages serve as the authoritative reference for how things work and why certain design choices were made.
