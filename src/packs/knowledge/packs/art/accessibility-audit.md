# ART-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Visual art is fundamentally visual, but visual experiences are not the only way to engage with art. This audit ensures blind/low-vision learners can engage with artmaking and art appreciation meaningfully through description, tactile experience, and alternative media.

## WCAG 2.1 AA Accessibility Status

| Feature | Status | Notes |
|---------|--------|-------|
| Artwork Descriptions | ✓ Yes | All images have comprehensive alt text |
| Audio Descriptions | ✓ Partial | Primary artworks described; expanding |
| Color Independence | ✓ Yes | Patterns/labels never rely on color alone |
| Keyboard Navigable | ✓ Yes | Digital tools (drawing programs) keyboard-accessible |
| High Contrast | ✓ Yes | Instructional text uses WCAG AA ratios |
| Video Captions | ✓ Yes | All demonstrations captioned |

## Vision Impairment Accessibility

### For Blind Learners

**Challenge:** Visual art is inherently visual. For blind learners, engagement requires alternative modalities.

**Remediation:**
- ✓ **Detailed verbal descriptions** of all artworks (300+ words each)
- ✓ **Tactile 3D models** of famous sculptures/architecture
- ✓ **Texture exploration activities** (making art with materials)
- ✓ **Audio descriptions** of techniques and visual elements
- [ ] **3D printing** of famous artworks for touch exploration
- [ ] **Tactile art kits:** Bas-reliefs, raised-line drawings
- [ ] **Braille labels** on exhibitions and descriptions

**Timeline:** 2026-Q2-Q3

### For Low-Vision Learners

- ✓ **Large reproductions** of artworks
- ✓ **High-contrast images** (avoid subtle color gradations)
- ✓ **Zoomable digital images** (can magnify to detail)
- [ ] **Video magnification** of demonstration techniques
- [ ] **Simplified color palettes** for basic exercises

## Accessibility of Artmaking

### Drawing & Painting Accessibility

**Current Status:** Good with adaptations

**Accommodations:**
- ✓ **Textured paper** provides tactile feedback
- ✓ **Large brushes** for those with fine motor limitations
- ✓ **Mixed media** approach (not just painting)
- [ ] **Adaptive brush holders** for limited grip strength
- [ ] **One-handed drawing techniques** and tools
- [ ] **Non-visual color identification** (scent, texture markers)

### Printmaking Accessibility

- ✓ Inherently tactile (carving, pressing)
- ✓ Can be performed seated
- [ ] **Adaptive tools** for carving (reduced hand strength)
- [ ] **Scent-marked inks** for color distinction

### Sculpture & 3D Work

- ✓ Highly tactile
- ✓ Accessible to blind learners
- ✓ Adaptive tools available
- [ ] **Safety modifications** for certain tools

## Color Accessibility

### Color Blindness Considerations

**Challenge:** Many activities involve color mixing, color theory, color-coded instructions.

**Remediation:**
- ✓ **Never use color as only differentiator** (also use pattern, label, position)
- ✓ **Inclusive color palettes** tested for colorblind visibility
- ✓ **Pattern in addition to color** (stripes, polka dots, solid)
- [ ] **Color simulation tools:** Show how colorblind people see colors
- [ ] **Alternative color descriptors:** "warm vs. cool" not just red/blue

### High Contrast Consideration

- ✓ **High-contrast color combinations** in demonstrations
- ✓ **Solid colors** preferred over gradients
- [ ] **Adjustable contrast** in digital tools

## Neurodiversity Considerations

### ADHD-Friendly Features

- ✓ Hands-on, engaging activity
- ✓ Immediate sensory feedback
- ✓ Creative freedom
- [ ] Add: **Structured projects** with clear end goals
- [ ] Add: **Shorter work sessions** (30 min max)
- [ ] Add: **Sensory breaks** between activities

### Autism Spectrum

- ✓ Sensory-rich activities (tactile, visual, possibly auditory)
- ✓ Systematic exploration of color, form, composition
- ✓ Pattern-based creative work
- [ ] Add: **Preference options** (realistic vs. abstract)
- [ ] Add: **Clear material expectations** (what to create)
- [ ] Add: **Quiet work space** options

### Perfectionistic/Anxiety Tendencies

- ✓ Process-focused assessment (not product quality)
- ✓ Emphasis on experimentation
- ✓ Gallery context celebrates all artwork
- [ ] Add: **Discussion of happy accidents** in art-making
- [ ] Add: **Gallery hang** including "failed" experiments

## Motor Accessibility

### Limited Fine Motor Control

**Challenge:** Drawing, painting, precise work difficult for those with cerebral palsy, tremors, arthritis.

**Adaptations:**
- ✓ **Large-scale work** (bigger is easier)
- ✓ **Wider tools** (thicker brushes, chunky markers)
- ✓ **Mixed media** approach (not just precision)
- [ ] **Adaptive tool grips** for arthritic hands
- [ ] **One-handed techniques** documentation
- [ ] **Mounted easels** reducing reaching requirements

### Limited Gross Motor Control

- ✓ **Seated activities** available
- ✓ **Lightweight materials**
- ✓ **Accessible furniture** (table height, chair stability)
- [ ] **Stilt/riser systems** for standing work

## Core Principle

**ART-101** believes that **everyone is creative**. Art is not just for those with typical vision. Blind artists, d/Deaf artists, disabled artists have created powerful, profound work. ART-101 ensures all learners can explore their creative potential.
