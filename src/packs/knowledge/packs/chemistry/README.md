# CHEM-101: Chemistry

**Classification:** Core Academic
**Status:** Alpha
**Version:** 1.0.0

Chemistry as the central science -- understanding what things are made of, how atoms combine into molecules, why reactions happen, and how chemistry connects the microscopic to the macroscopic.

## Quick Start

This pack teaches **chemistry through hands-on experience**: kitchen experiments, material investigations, and observable reactions before formulas and equations. It is the material science engine for all applied STEM packs.

### Modules

| Module | Name | Focus |
|--------|------|-------|
| M1 | Matter & Properties | States, mixtures, physical vs chemical changes |
| M2 | Atoms & the Periodic Table | Atomic structure, element families, trends |
| M3 | Bonds & Molecular Structure | Ionic, covalent, metallic bonding, molecular shape |
| M4 | Chemical Reactions & Energy | Conservation of mass, reaction types, rates, pH |
| M5 | Applied Chemistry | Materials, biochemistry, environment, food, energy |

### Prerequisites

- None required (Module 1 starts from everyday observation)
- MATH-101 recommended for stoichiometry and quantitative analysis
- SCI-101 recommended for scientific inquiry skills

### What This Pack Enables

- ENVR-101 (Environmental Science)
- NUTR-101 (Nutrition)
- NATURE-101 (Nature Study)
