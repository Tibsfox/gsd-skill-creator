# TECH-101: Accessibility Audit

**Pack:** Technology & Tools — Foundational Knowledge Pack
**Audit Date:** 2026-02-20
**Overall Status:** Partially Accessible

## Feature Status

| Feature | Status | Notes |
|---------|--------|-------|
| Screen Reader Compatible | Partial | Text-based activity descriptions are readable; hands-on tool demonstrations need detailed descriptions |
| Large Text Support | Implemented | Instructions render clearly at larger text sizes |
| High Contrast Mode | Implemented | High-contrast interface available for visual clarity |
| Keyboard Navigation | Implemented | Digital design tools support keyboard interaction |
| Alt Text for Images | Partial | Tool diagrams and systems drawings included; assembly instructions need enhanced descriptions |
| Media Captions | Not Started | No video demonstrations currently included; captions planned for tool use videos |

## Content-Specific Considerations

- **Hands-on Activities:** Tool use and simple machine building activities are inherently tactile. Detailed instructions provided; learners with visual impairments can explore through touch and verbal guidance.
- **Systems Diagrams:** Illustrations of simple machines, pulley systems, and lever mechanisms include text descriptions; interactive 3D visualizations would enhance understanding.
- **Digital Design Tools:** CAD and circuit simulation activities include keyboard navigation; screen reader descriptions of visual outputs need enhancement.
- **Safety Considerations:** Tool safety warnings are critical for all learners but especially important for those with visual impairments; extra detail recommended.

## Remediation Path

### Priority 1 (Critical)
1. Enhance descriptions of tool anatomy and safe handling for each activity
2. Create detailed verbal descriptions of machine assembly and function
3. Add tactile exploration guides for simple machines (what to feel for, how mechanisms work)
4. Ensure digital design tool outputs are described in text (e.g., "lever extends 10 cm")
5. Provide alternative assessment methods for tool-based learning (verbal explanation of design choices)

### Priority 2 (Important)
1. Add captions and audio descriptions for tool demonstration videos
2. Create accessible digital versions of assembly instruction diagrams
3. Develop keyboard shortcuts guide for all interactive design tools
4. Build peer teaching guides for collaborative hands-on activities
5. Create accessible alternatives for technology troubleshooting activities

### Priority 3 (Enhancement)
1. Develop 3D printed or tactile models of machines for hands-on exploration
2. Create audio descriptions of technology design case studies
3. Build customizable visualization settings for systems diagrams
4. Partner with occupational therapists to adapt tool grips and use

## WCAG 2.1 Compliance Notes

- **Target:** WCAG 2.1 Level AA
- **Current Status:** Level A for text instructions; Level AA for digital design tools
- **Major Gaps:**
  - Tool and machine descriptions need more detail
  - Safety information could be more accessible
  - Digital tool visual outputs lack text descriptions
  - Instructional videos not yet captioned
- **Strengths:**
  - Clear text-based instructions
  - Good contrast in interactive tools
  - Hands-on nature allows tactile learning
  - Logical sequence of activities
