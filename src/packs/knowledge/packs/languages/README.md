# LANG-101: World Languages — Foundational Knowledge Pack

## Quick Start

Welcome to LANG-101! This pack teaches the meta-skill of language learning — how to learn any language, at any age.

**Not included:** How to speak Spanish, French, Mandarin, or any specific language.

**Included:** How language works (sound systems, grammar patterns, vocabulary acquisition, communication strategies, cultural context) so you can effectively learn any language.

## What You'll Learn

**Module 1: Sound Systems & Phonetics** — How languages use different sounds; ear training; pronunciation

**Module 2: Grammar Patterns & Sentence Structure** — Grammar as systematic pattern; word order; morphology; agreement systems

**Module 3: Vocabulary & Reading Comprehension** — Efficient vocabulary acquisition; spaced repetition; reading progression

**Module 4: Speaking & Listening** — Conversation strategies; listening comprehension; communication in the target language

**Module 5: Language, Culture & Identity** — How language shapes thought; multilingualism; code-switching; linguistic diversity

## Who This Is For

- Children learning a second language in school
- Heritage language learners maintaining family languages
- Teenagers and adults learning languages for travel, work, or personal growth
- Teachers and parents supporting language learners
- Anyone curious about how languages work

## How to Use This Pack

Each module includes learning outcomes, topics, activities, and assessment guidance. Modules are designed to work together but are not strictly sequential:

- **Foundation/Elementary learners**: Focus on M1 (sounds), M3 (vocabulary), M5 (culture)
- **Middle school learners**: Build grammar (M2), reading (M3), speaking (M4), cultural understanding (M5)
- **High school/College learners**: Deepen all modules toward fluency

Activities include:
- Phonetic and pronunciation exercises
- Grammar pattern analysis
- Vocabulary and reading comprehension practice
- Conversation simulation and role plays
- Cultural exploration and reflection

## Key Principles

1. **Language is pattern, not rules** — Grammar is systematic and can be learned, not arbitrary to memorize
2. **Sound matters** — Different languages use different sounds; training ears and mouths is essential
3. **Vocabulary needs spacing** — Repeated review at expanding intervals works; cramming doesn't
4. **Communication is the goal** — Accuracy is secondary to being understood and understanding others
5. **Culture is part of language** — Language reflects culture; learning a language means learning a culture
6. **Multilingualism is strength** — Knowing multiple languages develops cognitive flexibility and perspective-taking

## Time Commitment

- **Conversational ability**: 500-1000 hours of study + immersion
- **Intermediate proficiency**: 300-500 hours
- **Advanced fluency**: 1500+ hours

These are minimums with effective instruction. Self-study requires consistent daily practice.

## Getting Started

1. **Start with Module 1** — Understand how language sounds work; train your ears to distinguish new sounds
2. **Engage Module 2 alongside M3** — As you encounter vocabulary (M3), also learn the grammar patterns (M2) that organize it
3. **Add Module 4 early** — Listening to authentic native speech (with subtitles/transcripts) accelerates learning
4. **Return to Module 5 throughout** — Culture provides motivation and context for all other learning
