# ASTRO-101: Accessibility Audit

Last Updated: 2026-02-20

## Overview

Astronomy is visual and conceptual. This audit ensures blind/low-vision learners can engage with celestial observation and cosmological thinking through alternative modalities.

## WCAG 2.1 AA Accessibility Status

| Feature | Status | Notes |
|---------|--------|-------|
| Star Chart Descriptions | ✓ Yes | Audio guides, tactile maps |
| Astronomical Images | ✓ Yes | Alt text for all images |
| Video Captions | ✓ Yes | Planetarium videos, documentaries |
| Audio Descriptions | ✓ Partial | Primary content; expanding |
| Keyboard Navigable | ✓ Yes | Digital tools (Stellarium, NASA Eyes) |
| High Contrast | ✓ Yes | Star charts, diagrams |

## Vision Impairment Accessibility

### For Blind Learners

**Challenge:** Astronomy is deeply visual—stars, planets, galaxies appear only to sight.

**Remediation:**
- ✓ **Audio star guides:** Recorded descriptions of constellations, planets
- ✓ **Tactile celestial globes:** Feel Earth orientation, major constellations
- [ ] **Braille star charts:** Major constellations and bright stars in Braille
- [ ] **Tactile sky maps:** Raised-line maps of night sky
- [ ] **Digital audio descriptions:** Detailed descriptions of astronomical images
- [ ] **Audio descriptions of planetarium shows:** Narrated cosmic journey
- [ ] **Tactile 3D models** of planetary orbits and solar system

**Timeline:** 2026-Q2-Q3

### For Low-Vision Learners

- ✓ **Large-print star charts**
- ✓ **High-contrast star maps** (white stars on black)
- [ ] **Magnified planetarium software** (accessible zoom)
- [ ] **Simplified star charts** (showing major constellations only)

## Technology Accessibility

### Stellarium (Planetarium Software)

**Accessibility:** Good; improvements needed

**Current:**
- ✓ Visual simulation of night sky
- [ ] **Improve:** Keyboard navigation for sky navigation
- [ ] **Improve:** Screen reader support for object information
- [ ] **Improve:** Audio descriptions of celestial events

**Goal:** Make Stellarium fully accessible for blind astronomy education

### NASA Eyes (Space Exploration)

**Accessibility:** Needs improvement

- [ ] **Keyboard-only navigation** through solar system
- [ ] **Audio descriptions** of space missions and planets
- [ ] **Screen reader labels** for interactive elements

### WorldWide Telescope

**Accessibility:** Needs improvement

- [ ] **Accessible query interface** for finding objects
- [ ] **Audio descriptions** of astronomical phenomena
- [ ] **Keyboard navigation** improvements

## Digital & Mobile Accessibility

### Smartphone Apps

**Challenge:** Many astronomical apps are visual-only.

**Remediation:**
- [ ] **Audio companion apps:** Voice descriptions of astronomical objects
- [ ] **Haptic feedback:** Vibration indicating object identification
- [ ] **Voice-controlled sky guides:** "What's north of Orion?"
- [ ] **Text-based object finder:** Search for objects by name

### Online Planetariums

- [ ] All planetarium shows include **captions + audio descriptions**
- [ ] **Virtual tours** have **audio narration**
- [ ] **Educational videos** have **full accessibility features**

## Observing Accessibility

### Naked-Eye Observation

**Current Status:** Requires accommodation planning

**For Blind Learners:**
- [ ] **Audio observation guides:** Seasonal constellation recordings
- [ ] **Community observation events:** Describe visible objects
- [ ] **Local librarians/educators:** Read constellation mythology, data
- [ ] **Virtual planetarium** substitute for naked-eye observation

### Telescopic Observation

**Challenge:** Telescopes are visual instruments.

**Alternatives:**
- [ ] **Radio astronomy:** Listen to cosmic radio emissions
- [ ] **Infrared detection:** Felt as heat from astronomical objects
- [ ] **Data analysis:** Work with real astronomical datasets (light curves, spectra)
- [ ] **Spectroscopy interpretation:** Analyze data about distant objects

## Neurodiversity Considerations

### ADHD-Friendly Features

- ✓ Vast scale captures imagination
- ✓ Multiple modalities (visual, audio, kinesthetic)
- [ ] Add: **Structured observation goals** (find 5 constellations)
- [ ] Add: **Time-limited activities** (30 min sessions)
- [ ] Add: **Frequent topic changes** (planets, stars, galaxies, cosmology)

### Autism Spectrum

- ✓ Systematic knowledge (catalogued objects, precise measurements)
- ✓ Pattern recognition (orbital mechanics, cosmic structure)
- ✓ Detailed data analysis
- [ ] Add: **Catalog-based learning** (specific star lists)
- [ ] Add: **Predictable observation protocols**
- [ ] Add: **Detailed specifications** (distance, magnitude, color)

### Anxiety About Scale/Cosmology

- ✓ Can cause existential anxiety in some learners
- [ ] Add: **Discussion of "cosmic perspective" as wonder not dread**
- [ ] Add: **Mental health resources** if needed
- [ ] Add: **Opt-out option** for cosmological philosophy discussions
