# DATA-101: Standards Alignment

## GAISE Framework (Statistics Education)

**GAISE.1**: Formulating Questions — Asking questions about data and creating hypotheses

**GAISE.2**: Data Collection & Management — Collecting data and organizing it appropriately

**GAISE.3**: Analysis & Interpretation — Analyzing data to answer questions and making inferences

## Common Core Mathematics Standards (Statistics & Probability)

**CCSS.MATH.CONTENT.6.SP.A**: Develop understanding of statistical variability

**CCSS.MATH.CONTENT.7.SP.A**: Use random sampling to draw inferences about populations

**CCSS.MATH.CONTENT.HS.S.ID.A**: Summarize, represent, and interpret data on a single count or measurement variable

**CCSS.MATH.CONTENT.HS.S.CP.A**: Understand independence and conditional probability

## NCTM Data Analysis & Probability Standards

**Standard 1**: Formulate questions that can be addressed with data and collect, organize, and display relevant data to answer them

**Standard 2**: Select and use appropriate statistical methods to analyze data

**Standard 3**: Develop and evaluate inferences and predictions based on data

## IB MYP Mathematics (Statistics Module)

**MYP.STAT.1**: Organizing and displaying data — Creating appropriate visual representations

**MYP.STAT.2**: Interpreting statistical representations — Reading and understanding graphs, charts, tables

**MYP.STAT.3**: Statistical reasoning — Drawing conclusions from data with justified reasoning

## NGSS Science Practices (Data Analysis)

**SEP-4**: Analyzing and Interpreting Data — Processing data to identify patterns and relationships

**SEP-5**: Using Mathematics and Computational Thinking — Applying statistical methods and modeling

## Curriculum Connection Notes

- **K-2 Emphasis**: Collecting data through observation, simple graphs (picture graphs, tally marks)
- **3-5 Emphasis**: Bar graphs, line plots, understanding mean/median, drawing conclusions
- **6-8 Emphasis**: Statistical variability, probability experiments, creating/interpreting graphs
- **9-12 Emphasis**: Sampling distributions, hypothesis testing, probability calculations, data modeling

## Integration Pathways

**Prerequisite**: MATH-101 (quantitative reasoning foundation)

**Enables**: ECON-101 (analyzing economic data), college statistics

**Bridges to**: Scientific method (SCI-101), evidence evaluation (CRIT-101), computer science (CODE-101)
