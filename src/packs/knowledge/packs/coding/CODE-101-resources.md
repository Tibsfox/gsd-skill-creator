# CODE-101 Resources

Curated links to help learners, parents, and educators at every level and interest.

## Foundational Texts

Read these to understand the philosophy behind the CODE-101 pack:

- [Structure and Interpretation of Computer Programs (SICP)](https://mitpress.mit.edu/sites/default/files/sicp/full-text/book/book.html) — The classic. Teaches you to think computationally. Free online.
- [Code: The Hidden Language](https://codehiddenlanguage.com/) — Charles Petzold. How codes actually work, from ancient times to modern computing.
- [Algorithms Unlocked](https://mitpress.mit.edu/9780262525121/algorithms-unlocked/) — Thomas Cormen. Algorithms without the heavy math.
- [The Pragmatic Programmer](https://pragprog.com/titles/tpp20/the-pragmatic-programmer-your-journey-to-mastery-20th-anniversary-edition/) — Hunt & Thomas. How to think like a professional developer.

## When You Get Stuck

Remember: every programmer gets stuck. It's not a sign you're bad at programming; it's how programming works.

**Steps when stuck:**
1. **Read the error message** — It usually tells you what's wrong
2. **Search for your error** — Odds are others hit it before
3. **Ask specifically** — "My function returns None instead of a list" is better than "It doesn't work"
4. **Try a simple case** — Strip down to the smallest example
5. **Take a break** — Fresh eyes often see the solution
6. **Explain to a rubber duck** — Seriously. Explaining your code aloud often reveals the bug.
