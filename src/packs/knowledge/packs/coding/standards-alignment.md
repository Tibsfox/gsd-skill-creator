# CODE-101: Standards Alignment

## CSTA (Computer Science Standards)

**Standard 1A-CS-01** (K-2): Understand that computing devices are used for different purposes.

**Standard 1B-AP-09** (K-2): Create programs with sequences and simple loops, using block-based or visual programming.

**Standard 2-AP-10** (3-5): Model the way programs store and manipulate data by using numbers or other symbols to represent information.

**Standard 2-AP-13** (3-5): Create procedures with parameters to organize code and make it more modular and reusable.

**Standard 3A-AP-13** (6-8): Create procedures with parameters and return values to generalize solutions to a problem.

**Standard 3B-AP-08** (6-8): Describe how artificial intelligence drives many software and physical systems.

## IB MYP Design Standards

**Design 1**: Inquiring and analyzing — Understanding the design problem and research context for computational thinking

**Design 2**: Developing ideas — Planning and designing algorithms and code structure

**Design 3**: Creating the solution — Implementation through code and testing of programs

## UK Computing National Curriculum

**KS2 (Key Stage 2)**: Pupils learn to understand and predict the behavior of computer programs; design, write and debug programs that accomplish goals including controlling physical systems

**KS3 (Key Stage 3)**: Pupils deepen understanding of algorithms, programming concepts, and computing systems

**KS4 (Key Stage 4)**: Pupils understand computing systems at a deeper level; apply programming to solve real-world problems

## ACM/IEEE CS2013

**AL/Algorithms**: Understanding algorithm design, analysis, and complexity

**PL/Programming Languages**: Understanding language syntax, control structures, and data types

**SE/Software Engineering**: Understanding software development lifecycle, testing, and collaboration

## Curriculum Connection Notes

- **Computational Thinking** forms the foundation across all standards frameworks
- **K-2 Emphasis**: Unplugged activities, visual programming, step-by-step thinking
- **3-5 Emphasis**: Block-based languages (Scratch), basic loops and sequences
- **6-8 Emphasis**: Transition to text-based languages, procedures and parameters
- **9-12 Emphasis**: Advanced algorithms, complexity analysis, computing ethics
