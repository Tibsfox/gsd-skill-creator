# Find Your First Wasteland Win

**Type:** Mission
**Author:** MapleFoxyBells
**Duration:** 2-4 hours
**Difficulty:** Beginner

## What You're Getting Into

The Wasteland is a federated work economy powered by Dolt (a database with git-style branching). It's like GitHub Issues, but distributed, versioned, and reputation-backed.

This pack walks you through:
1. **Setup** — Install Dolt and register as a rig (participant)
2. **Browse** — See what work is available
3. **Claim** — Signal that you're working on a task
4. **Work** — Do the work (the fun part!)
5. **Submit** — Show evidence of completion
6. **Celebrate** — Get validated and earn your first stamp

## For Whom?

You're perfect for this pack if you:
- Are curious (that's all you need to start)
- Want to explore how the Wasteland works
- Like learning by doing
- Are willing to ask questions when stuck
- Have a few hours of free time

**You do NOT need:**
- Deep technical knowledge
- Prior experience with Dolt or Git
- Perfect code or writing
- Confidence about what you're doing

We'll walk you through every command. Your curiosity is enough.

**What will happen:**
- You'll start exploring (just because it's interesting)
- You'll do some real work (and be surprised you could)
- You'll help someone else (and realize you know something)
- You'll make friends (who were also just curious once)
- You'll teach (without planning to)

## What You'll Learn

- **How Dolt works** — Git-style versioning for databases
- **How the Wasteland works** — Claim → Work → Submit → Get Stamped
- **How to find your strength** — What kind of work suits you?
- **How to collaborate** — Community, validators, reputation

## Getting Started

No prerequisites except:
- A computer (Mac/Linux)
- 2-4 hours of uninterrupted time
- A willingness to try something new
- Sam (the center at 0.50, 0.50) as your safe retreat if overwhelmed
