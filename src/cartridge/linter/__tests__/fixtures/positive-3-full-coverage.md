# Tiny Skill

A small skill that stays compliant.

- Algorithm complexity: O(n) decidable; no recursive halting concerns.
- If precondition holds then postcondition follows; invariant preserved
  at each step (proof by induction).
- Estimate uncertainty with a Bayesian posterior; report 90% confidence
  interval.
- Data classified as **public**; no PII; redaction unnecessary.
- Quality assessed via rubric: passes if metric ≥ 0.9 (acceptance criteria
  documented in the checklist).
