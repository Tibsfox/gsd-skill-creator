# Quick Start: Power Distribution Design

Design a complete data center power distribution system from a natural language requirement. This walkthrough covers load calculations, transformer sizing, solar PV, battery storage, and safety review.

## What You Will Produce

- NEC Article 220 load calculations (IT load, PUE, total facility load)
- Conductor sizing per NEC Table 310.16
- Transformer sizing (next standard kVA size)
- Solar PV sizing per NEC 690 (optional, when solar area provided)
- BESS capacity sizing (optional, when battery runtime provided)
- SLD (Single Line Diagram) with IEEE symbols
- Safety review including arc flash (NFPA 70E), rapid shutdown (NEC 690.12), BESS thermal (NFPA 855)

## Prerequisites

Gather the following before starting:
- Rack count and per-rack density (kW/rack)
- Voltage class: 120V, 208V, 240V, 277V, 400V, or 480V (typically 480V for data centers)
- Redundancy level (N, N+1, 2N, 2N+1)
- Solar availability: roof area in m2 (optional)
- Battery backup runtime in minutes (optional)
- Safety class: residential, commercial, industrial, or data-center

## Step 1: Invoke the Design-Review Team

```
/physical-infrastructure:design-review "Design 480V three-phase power distribution for [N] racks at [X]kW with [Y]m2 solar and [Z]-minute battery backup"
```

Example:
```
/physical-infrastructure:design-review "Design 480V three-phase power for 100 racks at 10kW each with 5000m2 solar and 30-minute battery backup, data-center safety class"
```

## Step 2: Review the Architect's Decomposition

The Architect agent identifies:
- IT load: 100 racks x 10kW = 1000kW
- PUE: 1.2 (default for N+1 data center)
- Total facility load: 1200kW
- Voltage: 480V, 3-phase
- Power factor: 0.9
- Solar and BESS subsystems requested

## Step 3: Review Calculation Results

The Calculator agent produces multiple `CalculationRecord` entries:

**Power Systems (NEC 220):**
- **it_load_kW:** 1000kW
- **total_load_kW:** 1200kW (1000kW x 1.2 PUE)
- **load_current_A:** ~1604A (for 480V, 3-phase, 0.9 PF)
- **transformer_kVA:** 1500kVA (next standard size above 1333kVA)

**Solar PV (NEC 690):**
- **solar_kWp:** calculated from target annual production, 5.5 PSH, 80% system efficiency

**BESS:**
- **bess_capacity_kWh:** ~327kWh (for 500kW, 30min, 85% DoD, 90% roundtrip efficiency)

## Step 4: Safety Warden Review

The Safety Warden checks all power-specific hazards:

- **SC-11 (Arc flash):** 480V three-phase always triggers an arc flash warning with NFPA 70E PPE Category 2 minimum requirement
- **SC-12 (Rapid shutdown):** Solar PV designs include NEC 690.12 rapid shutdown requirement
- **SC-16 (BESS thermal):** Lithium battery designs include thermal management requirement and NFPA 855 reference
- **PE disclaimer:** Present on all outputs

For data-center safety class, the warden operates in **gate mode**.

## Step 5: Blueprint Output

The Draftsman agent generates:
- **SLD (Single Line Diagram)** in SVG format with IEEE symbols
- **Title block** referencing voltage class and transformer size
- **PE disclaimer** embedded in SVG: "AI GENERATED - VERIFY WITH LICENSED PE BEFORE USE"
