# Hello World -- Deep Reference

## History

The "Hello, World!" program originated in Brian Kernighan's 1978 book
"The C Programming Language" (K&R). It has since become the universal
first program in every language tutorial.

## Why It Matters

Hello World is the minimal viable program. It proves:
1. The toolchain works (compiler/interpreter installed correctly)
2. Output is possible (the program communicates with the outside world)
3. The basic program structure is understood (entry point, statements, termination)

## Cross-References

- **Mathematics:** The concept of a function (input -> output) is analogous to a program (run -> display)
- **Culinary Arts:** Following a recipe for the first time -- Hello World is the "boil water" of programming
