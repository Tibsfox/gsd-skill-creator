# Coding Department

**Domain:** coding
**Source:** CODE-101 Computer Science & Coding Foundational Knowledge Pack
**Status:** Active
**Purpose:** Problem-solving and clear thinking through code, from foundations to complexity analysis -- five wings covering computational thinking, programming fundamentals, project building, algorithms, and computing in society

## Wings

- Computational Thinking -- Sequential thinking, algorithm design, pattern recognition, abstraction, debugging mindset
- Programming Fundamentals -- Variables, data types, control flow, input/output, syntax, block-based and text-based environments
- Building Projects -- Project planning, iterative development, data analysis, game design, testing, code organization, peer review
- Algorithms & Efficiency -- Sorting, searching, Big-O complexity, greedy algorithms, dynamic programming, graph algorithms
- Computing & Society -- Ethics, AI/ML basics, cybersecurity, digital citizenship, open source, accessibility, environmental impact

code-sequential-thinking

## Concepts

### Computational Thinking (4 concepts)
- code-sequential-thinking -- Step-by-step logic and algorithm design before implementation
- code-decomposition -- Breaking complex problems into manageable sub-problems
- code-pattern-recognition -- Identifying and generalizing patterns in problems and data
- code-abstraction -- Hiding complexity behind clean interfaces and representations

### Programming Fundamentals (4 concepts)
- code-variables-data-types -- Variables, numbers, strings, lists, objects and their behaviors
- code-control-flow -- If/else branching, loops, and function definitions
- code-input-output -- Reading user input and producing output in multiple environments
- code-syntax-style -- Grammar rules of programming languages and code organization

### Building Projects (4 concepts)
- code-iterative-development -- Build basic version, test, improve -- the core development loop
- code-debugging-strategies -- Systematic error finding using tools, print statements, and reasoning
- code-code-organization -- Functions, modules, classes -- structuring code for maintainability
- code-peer-review -- Reading and improving others' code; giving and receiving feedback

### Algorithms & Efficiency (4 concepts)
- code-sorting-algorithms -- Bubble sort, merge sort, quicksort -- tradeoffs and visualizations
- code-searching-algorithms -- Linear and binary search -- when each applies
- code-big-o-notation -- Time and space complexity analysis for comparing algorithms
- code-dynamic-programming -- Memoization and optimal substructure for hard problems

### Computing & Society (4 concepts)
- code-computing-ethics -- Responsibility, bias in algorithms, accessibility requirements
- code-cybersecurity-basics -- Password security, encryption concepts, threat models
- code-ai-ml-fundamentals -- How machine learning systems work and their limitations
- code-open-source -- Collaboration, licensing, and the open source ecosystem
