# reproducibility-and-isolation

Get the failure to happen on demand. Strip variables until the smallest set that still reproduces the fault is found. Essential for any intermittent or context-sensitive problem — an unreproducible fault cannot be fixed with confidence.

**Triggers:** `reproduce the issue`, `make it happen`, `minimal reproduction`, `strip variables`, `isolate the cause`, `reliable repro`, `steps to reproduce`

**Affinity:** `hypothesis-tester`
