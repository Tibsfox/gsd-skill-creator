# hypothesis-and-test

Apply the scientific method to faults — state a falsifiable hypothesis, design the cheapest experiment that would disprove it, run the experiment, record the result, update or replace the hypothesis. Avoids the "keep changing things" anti-pattern.

**Triggers:** `form a hypothesis`, `test a theory`, `falsify`, `run an experiment`, `scientific method`, `what would disprove this`

**Affinity:** `hypothesis-tester`, `troubleshoot-lead`
