# fault-tree-and-failure-mode

Enumerate every plausible failure mode and organize them as a fault tree — root symptom at the top, OR / AND branches downward to every conceivable cause. Drives systematic coverage where intuition alone would miss a branch. Pairs naturally with FMEA when the system is safety-critical.

**Triggers:** `fault tree`, `failure mode`, `fmea`, `fta`, `enumerate causes`, `all possible reasons`, `systematic coverage`

**Affinity:** `comparative-analyst`, `troubleshoot-lead`
