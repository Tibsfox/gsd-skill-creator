# netops-change-window-team

Execute a scheduled network change. Architect reviews the MOP and approves the risk class, engineer performs the cutover, monitoring-engineer watches telemetry through the window, security-officer signs off on any policy changes in scope.

**Roster:** `netops-architect`, `netops-engineer`, `netops-monitoring-engineer`, `netops-security-officer`.
**Use when:** executing a scheduled maintenance window on production network devices.
