# netops-incident-team

Live network outage response. Incident-responder drives triage, engineer executes stabilization steps, monitoring-engineer surfaces the telemetry signal, architect handles escalation and vendor/carrier engagement.

**Roster:** `netops-incident-responder`, `netops-engineer`, `netops-monitoring-engineer`, `netops-architect`.
**Use when:** a production network outage or severe degradation is observed.
