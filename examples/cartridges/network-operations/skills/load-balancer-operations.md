# load-balancer-operations

Run L4 and L7 load balancers — VIPs, pool members, health checks, persistence, TLS termination, certificate rotation, policy routing. Covers hardware (F5, A10), cloud (ALB/NLB, GCLB), and software (haproxy, envoy) with the same grammar. Triggers on `vip`, `load balancer`, `health check`, `tls termination`, `haproxy`, `envoy`, `f5`, `alb`. Affinity: `netops-engineer`, `netops-architect`.
