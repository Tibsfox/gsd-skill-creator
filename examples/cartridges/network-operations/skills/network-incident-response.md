# network-incident-response

Triage live network outages — link flaps, routing black holes, BGP session drops, broadcast storms, MTU mismatches, asymmetric routing. Runs show commands, captures counters, correlates telemetry, and stabilizes before handing to postmortem. Triggers on `network outage`, `link flap`, `bgp flap`, `routing loop`, `broadcast storm`, `mtu issue`, `packet loss`. Affinity: `netops-incident-responder`, `netops-engineer`.
