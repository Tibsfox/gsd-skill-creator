# dns-operations

Operate authoritative and recursive DNS — zone files, record management, delegation, DNSSEC signing and key rollover, anycast, RPZ policy. Every change is version-controlled with rollback in seconds. Triggers on `dns zone`, `dns record`, `dnssec`, `ksk rollover`, `authoritative dns`, `rpz`. Affinity: `netops-engineer`, `netops-security-officer`.
