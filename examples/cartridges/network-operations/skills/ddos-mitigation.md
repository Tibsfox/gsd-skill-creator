# ddos-mitigation

Detect and mitigate volumetric and application-layer DDoS — scrubbing-center failover, BGP flowspec, edge rate limiting, anycast absorption, upstream ACL requests. Owns the null-route vs scrub-in-place decision. Triggers on `ddos attack`, `volumetric attack`, `flowspec`, `rtbh`, `null route`, `scrubbing center`. Affinity: `netops-security-officer`, `netops-incident-responder`.
