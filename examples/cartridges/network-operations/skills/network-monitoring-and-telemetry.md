# network-monitoring-and-telemetry

Collect and analyze network telemetry — SNMP, streaming telemetry (gNMI, OpenConfig), NetFlow/sFlow/IPFIX, syslog, trap handling. Feeds the NMS, traffic-analysis stack, and alerting. Owns collector health. Triggers on `snmp poll`, `streaming telemetry`, `gnmi`, `netflow`, `sflow`, `syslog`, `nms`, `interface counters`. Affinity: `netops-monitoring-engineer`, `netops-engineer`.
