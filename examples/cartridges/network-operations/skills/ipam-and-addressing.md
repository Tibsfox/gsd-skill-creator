# ipam-and-addressing

Manage IPv4/IPv6 allocation — prefix assignment, subnet planning, reservation registries, reverse-DNS alignment. Enforces the rule that no IP is assigned outside the IPAM of record. Triggers on `ip allocation`, `subnet planning`, `prefix assignment`, `ipam`, `ipv6 rollout`, `address reclaim`. Affinity: `netops-engineer`, `netops-architect`.
