# gsd-phase-cycle-team

Run one full phase cycle — research → plan → execute → verify — without leaving the cartridge. Capcom orchestrates, phase-researcher runs the upstream pass, planner authors PLAN.md, executor ships waves, verifier closes the loop. This is the default team for day-to-day GSD work.

**Roster:** `gsd-capcom`, `gsd-phase-researcher`, `gsd-planner`, `gsd-executor`, `gsd-verifier`.

**Use when:** running a normal phase from research through verification
