# gsd-project-bootstrap-team

Stand up a new GSD project from zero. Capcom routes, project-researcher gathers domain context, roadmapper drafts the phase graph, planner stress-tests the first phase. Outcome: a ROADMAP.md + a planned-and-checked Phase 1, plus the initial grove namespace.

**Roster:** `gsd-capcom`, `gsd-roadmapper`, `gsd-phase-researcher`, `gsd-planner`.

**Use when:** bootstrapping a new GSD project or milestone
