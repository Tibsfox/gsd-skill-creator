# user-and-project-profile

Manage per-user and per-project settings via `/gsd profile-user`, `/gsd set-profile`, `/gsd settings`, `/gsd stats`, and `/gsd progress`. Keeps the cartridge's behavior tuned to developer preferences (verbosity, autonomy, cadence) and exposes project-wide progress telemetry.

**Triggers:** `profile user`, `set profile`, `gsd settings`, `gsd stats`, `gsd progress`, `developer preferences`

**Affinity:** `gsd-capcom`, `gsd-user-profiler`
