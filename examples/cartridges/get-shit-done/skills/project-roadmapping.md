# project-roadmapping

Bootstrap a new GSD project and author its roadmap. Covers `/gsd new-project`, domain research, phase decomposition, requirement-to-phase mapping, success criteria derivation, dependency graphing, and the coverage-validation checklist that keeps a roadmap honest. The roadmap is the spine every later phase hangs off of.

**Triggers:** `new gsd project`, `bootstrap project`, `gsd new-project`, `author roadmap`, `decompose into phases`, `requirement to phase mapping`, `phase dependency graph`

**Affinity:** `gsd-capcom`, `gsd-roadmapper`
