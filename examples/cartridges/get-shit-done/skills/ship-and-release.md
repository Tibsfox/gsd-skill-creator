# ship-and-release

Prepare and execute release via `/gsd ship`, `/gsd pr-branch`, and `/gsd review`. Covers branch hygiene, PR composition, release-note authoring, and the final go/no-go against milestone success criteria. The last-mile skill of the cartridge.

**Triggers:** `gsd ship`, `ship milestone`, `pr branch`, `release notes`, `go no-go`, `final review`

**Affinity:** `gsd-capcom`, `gsd-verifier`
