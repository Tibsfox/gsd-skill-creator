# backlog-and-todos

Groom the backlog and per-phase todo list via `/gsd add-backlog`, `/gsd review-backlog`, `/gsd add-todo`, `/gsd check-todos`, and `/gsd plant-seed`. Each entry gets a durable grove record with provenance, so nothing falls through the cracks when a phase closes or a session hands off.

**Triggers:** `add backlog`, `review backlog`, `add todo`, `check todos`, `plant seed`, `groom backlog`, `pending work`

**Affinity:** `gsd-capcom`
