# phase-planning

Author, insert, update, and retire phase plans via `/gsd plan-phase`, `/gsd add-phase`, `/gsd insert-phase`, and `/gsd remove-phase`. Produces a PLAN.md with task breakdown, dependency analysis, files-modified list, deviation-handling rules, and the goal-backward gate that gsd-plan-checker enforces before execution.

**Triggers:** `plan phase`, `new phase plan`, `insert phase`, `add phase`, `remove phase`, `update phase plan`, `phase task breakdown`, `plan.md`

**Affinity:** `gsd-capcom`, `gsd-planner`, `gsd-plan-checker`
