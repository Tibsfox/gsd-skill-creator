# phase-research

Run the upstream research pass for a phase via `/gsd research-phase`. Fans out a researcher fleet, aggregates findings into RESEARCH.md, captures research grounding citations, and makes the output addressable in grove so the planner and executor can cite it.

**Triggers:** `research phase`, `phase research`, `research.md`, `research pass`, `researcher fleet`, `phase-researcher`

**Affinity:** `gsd-capcom`, `gsd-phase-researcher`, `gsd-project-researcher`, `gsd-research-synthesizer`
