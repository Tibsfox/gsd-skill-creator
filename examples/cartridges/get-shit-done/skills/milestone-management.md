# milestone-management

Open, track, summarize, audit, and close GSD milestones. Covers `/gsd new-milestone`, `/gsd milestone-summary`, `/gsd audit-milestone`, `/gsd plan-milestone-gaps`, and `/gsd complete-milestone`. A milestone is a phase cluster with its own success criteria and ship gate; this skill owns the whole lifecycle and writes a clean closeout report into grove.

**Triggers:** `new milestone`, `complete milestone`, `close milestone`, `milestone summary`, `audit milestone`, `plan milestone gaps`, `milestone health`

**Affinity:** `gsd-capcom`, `gsd-verifier`
