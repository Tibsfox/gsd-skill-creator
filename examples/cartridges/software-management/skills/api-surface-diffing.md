# api-surface-diffing

Detect breaking changes across versions — exported symbols, type signatures, default behaviors, wire protocols, serialization shapes. Separates additive (safe) from breaking (not safe) changes and flags the ones that need a semver major or a deprecation period.

**Triggers:** `api diff`, `breaking change`, `semver major`, `abi compatibility`, `public surface`, `deprecation`, `exported symbols`

**Affinity:** `sm-reviewer`, `sm-porter-translator`
