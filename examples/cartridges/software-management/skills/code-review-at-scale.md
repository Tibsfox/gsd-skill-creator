# code-review-at-scale

Review large or unfamiliar changesets — full PRs, multi-commit branches, or entire vendored drops. Covers reading code by intent first, grouping comments by theme, distinguishing blocking concerns from stylistic nits, and producing a written review summary the author can act on.

**Triggers:** `review a large pr`, `review this branch`, `review the vendored drop`, `big review`, `unfamiliar code review`, `write a review summary`

**Affinity:** `sm-reviewer`, `sm-lead`
