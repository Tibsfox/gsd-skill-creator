# architecture-extraction

Derive the de facto architecture from the code as it stands — layers, modules, bounded contexts, shared kernels, and the implicit contracts between them. Distinguishes the architecture that was designed from the one that actually exists. Produces diagrams and ADR-shaped notes.

**Triggers:** `extract architecture`, `as-built architecture`, `de facto design`, `layer diagram`, `bounded contexts`, `reverse engineer design`

**Affinity:** `sm-codebase-cartographer`, `sm-lead`
