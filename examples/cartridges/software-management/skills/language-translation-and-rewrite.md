# language-translation-and-rewrite

Translate a module from one language to another — typescript to rust, python to go, c to zig. Works in small, test-backed slices that keep both versions runnable during the cutover. Produces a translation plan, a correspondence table, and a shared behavioral test suite that both implementations pass.

**Triggers:** `rewrite in rust`, `translate to go`, `port from python`, `rewrite this module`, `cross language`, `language rewrite`, `correspondence table`

**Affinity:** `sm-porter-translator`
