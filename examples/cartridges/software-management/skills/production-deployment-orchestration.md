# production-deployment-orchestration

Drive a production deployment end-to-end — artifact promotion from staging, change-window scheduling, approval chain, running the deploy pipeline, verifying each stage, and closing out the change record. The controlled path from "green build" to "running in prod."

**Triggers:** `deploy to prod`, `production rollout`, `promote artifact`, `change window`, `cut the deploy`, `deploy pipeline`, `production push`

**Affinity:** `sm-deploy-engineer`, `sm-lead`
