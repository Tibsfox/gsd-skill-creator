# code-porting-between-platforms

Port existing code to a new platform while preserving behavior — different OS, different architecture, different runtime, or different deployment target. Covers conditional compilation, platform-specific shims, CI matrix extension, and a regression plan against the original platform.

**Triggers:** `port to linux`, `port to windows`, `port to arm`, `cross platform`, `new target`, `platform shim`, `conditional compilation`

**Affinity:** `sm-porter-translator`, `sm-lead`
