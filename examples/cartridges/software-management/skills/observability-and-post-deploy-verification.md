# observability-and-post-deploy-verification

Watch the system during and after a deploy. Covers metric / log / trace dashboards, error-rate SLOs, latency histograms, smoke-test probes, and the written verification checklist that has to be green before a deploy is declared complete.

**Triggers:** `post deploy checks`, `smoke tests`, `slo`, `error rate`, `latency histogram`, `deploy verification`, `watch the dashboards`

**Affinity:** `sm-deploy-engineer`, `sm-lead`
