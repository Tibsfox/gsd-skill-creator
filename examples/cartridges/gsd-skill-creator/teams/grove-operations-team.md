# grove-operations-team

Any end-to-end grove task — import a filesystem library, design record types for a new cartridge, write a multi-hop retrieval query, migrate between format versions, produce a snapshot. Grove-keeper owns correctness; skill-forger writes the code; capcom approves schema changes that cross cartridge boundaries.

**Roster:** `grove-keeper`, `skill-forger`, `skill-creator-capcom`.

**Use when:** anything touching .grove/arena.json, grove record types, or content-addressed retrieval
