# publish-release-team

Ship a new gsd-skill-creator release. Publish-engineer drives version bumps, release notes, and install wiring; cartridge-smith re-runs forge gates on every shipped cartridge; capcom signs off on the final release shape and writes the milestone retro.

**Roster:** `publish-engineer`, `cartridge-smith`, `skill-creator-capcom`.

**Use when:** cutting a release, bumping versions, publishing to npm/npx
