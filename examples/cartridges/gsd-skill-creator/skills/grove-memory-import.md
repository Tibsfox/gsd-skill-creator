# grove-memory-import

Import filesystem skills, agents, teams, and chipsets into the grove content-addressed arena via tools/import-filesystem-skills.ts. Covers source directory selection, exclusion rules (wasteland/, muses/), dedup on unique name binding, and the resulting .grove/arena.json snapshot. Grove import is the on-ramp from flat files to content-addressed memory.

**Triggers:** `import skills into grove`, `grove arena snapshot`, `import-filesystem-skills`, `arena.json`, `grove ingest`, `populate grove`, `grove bindings`

**Affinity:** `grove-keeper`, `skill-forger`
