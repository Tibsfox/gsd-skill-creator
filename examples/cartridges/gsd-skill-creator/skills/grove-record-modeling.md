# grove-record-modeling

Design grove record types and namespaces for a cartridge. Record types capture the durable artifacts a cartridge produces (authored skill, forged cartridge, validation result, migration log). Each type has a name, description, and lives under a cartridge-scoped namespace. Good record types outlive the skills that write them.

**Triggers:** `design grove records`, `grove record type`, `grove namespace`, `grove schema`, `content-addressed record`, `durable artifact schema`, `record type descriptions`

**Affinity:** `grove-keeper`
