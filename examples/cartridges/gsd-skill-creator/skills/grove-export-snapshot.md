# grove-export-snapshot

Export grove state — write a .grove/arena.json snapshot, serialize a SkillView or SkillCodebase projection, produce a GroveNamespace manifest. Snapshots are the portable, content-addressed handoff format between machines, sessions, and migrations. Also the fixture format CI consumes for grove-dependent tests.

**Triggers:** `export grove`, `grove snapshot`, `write arena.json`, `grove namespace export`, `grove manifest`, `serialize skillcodebase`

**Affinity:** `grove-keeper`
