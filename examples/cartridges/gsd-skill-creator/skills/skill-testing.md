# skill-testing

Write vitest tests for skills, agents, and cartridges — trigger accuracy tests, round-trip load/validate tests, gate assertions, and grove round-trip property tests. Includes the test-first discipline for publish gating (Phase B skill-author-discipline), and the pattern of skipIf(CI) for tests that need richer fixtures than CI can cheaply generate.

**Triggers:** `test a skill`, `vitest for skill`, `skill trigger accuracy`, `cartridge round-trip test`, `grove test fixture`, `publish gate test`, `skipif ci`

**Affinity:** `skill-forger`, `cartridge-smith`
