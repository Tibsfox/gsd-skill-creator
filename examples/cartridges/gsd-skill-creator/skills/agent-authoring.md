# agent-authoring

Create a new agent — role prose, model selection (opus/sonnet/haiku), tool allowlist, isolation mode, and memory scope. Covers when to write an agent vs. a skill, the capcom pattern (is_capcom flag), router topology, and the tradeoffs between single-shot agents and long-running orchestrators.

**Triggers:** `create an agent`, `new agent`, `author an agent`, `agent frontmatter`, `agent role`, `pick a model for this agent`, `agent tool allowlist`

**Affinity:** `skill-forger`, `skill-creator-capcom`
