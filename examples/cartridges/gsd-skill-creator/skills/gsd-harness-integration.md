# gsd-harness-integration

Wire skills, agents, and cartridges into the Claude Code harness — .claude/settings.json permissions, hooks (PreToolUse, SessionStart, UserPromptSubmit), .claude/commands/gsd/ command definitions, and the project-claude install flow (node project-claude/install.cjs). Covers harness-integrity invariants (hook files exist, version sync, MCP binary gating) and the read-before-edit discipline the harness enforces.

**Triggers:** `harness integration`, `claude settings.json`, `claude hooks`, `pretooluse hook`, `project-claude install`, `gsd command`, `harness integrity`

**Affinity:** `skill-creator-capcom`, `publish-engineer`
