# skill-authoring

Create a new skill from scratch — frontmatter (name, description, triggers, tools), the skill body prose, and the invocation contract. Covers the SKILL.md convention, trigger phrase design, scoping rules (when to invoke vs. when NOT), and the read-before-edit discipline that production skills must respect.

**Triggers:** `create a skill`, `new skill`, `author a skill`, `write skill frontmatter`, `skill.md`, `draft a skill`, `skill authoring`

**Affinity:** `skill-forger`, `skill-creator-capcom`
