# package-publishing

Publish gsd-skill-creator via npx/npm. Covers version bumps across package.json / Cargo.toml / tauri.conf.json, the skill-creator binary (future skill-center), pre-publish validation, release notes authoring in docs/release-notes/, and the distinction between dev-only (source) and installed (binary) invocation of the cartridge CLI.

**Triggers:** `publish skill-creator`, `bump version`, `release notes`, `npx publish`, `version drift`, `cartridge cli not installed`, `skill-creator binary`

**Affinity:** `publish-engineer`, `skill-creator-capcom`
