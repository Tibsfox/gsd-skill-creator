# type-theory-and-type-systems

Simply typed lambda calculus, System F / polymorphism, dependent types, subtyping, type inference (Hindley-Milner), the Curry-Howard correspondence, and the design space of real type systems (ML, Haskell, Rust, Idris, Lean). Explains why a type error catches a bug before it happens.

**Triggers:** `type theory`, `type system`, `hindley milner`, `system f`, `dependent types`, `curry howard`, `type inference`, `subtyping`

**Affinity:** `language-theorist`, `proof-theorist`
