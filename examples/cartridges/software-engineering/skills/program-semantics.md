# program-semantics

Operational (small-step and big-step), denotational, and axiomatic semantics. How to say precisely what a program means — not just what it does when you run it. The foundation for reasoning about equivalence, refinement, and correctness.

**Triggers:** `operational semantics`, `denotational semantics`, `axiomatic semantics`, `small step`, `big step`, `program meaning`, `refinement`

**Affinity:** `proof-theorist`, `language-theorist`
