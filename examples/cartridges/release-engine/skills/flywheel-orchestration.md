# flywheel-orchestration

Orchestrate the release flywheel — the feedback loop that makes each release improve on the last. Reads the prior release's ReleaseRetrospective, derives a FlywheelDelta (what we changed between N and N+1 and why), and feeds the delta into the next cadence's planning inputs. The flywheel is what separates a release series from a list of releases.

**Triggers:** `release flywheel`, `carry forward`, `feed retro forward`, `flywheel delta`, `improve next release`, `series momentum`

**Affinity:** `release-capcom`, `release-retrospective-analyst`
