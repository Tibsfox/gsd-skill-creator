# changelog-generation

Build changelogs from conventional-commits git history. Groups commits by type (feat, fix, docs, refactor, perf, test, chore), highlights breaking changes, and suggests the semver bump to version-policy. Runs per release and produces a ChangelogEntry grove record linked to the ReleaseRecord.

**Triggers:** `generate changelog`, `conventional commits`, `changelog entry`, `breaking changes`, `release changelog`, `what changed`

**Affinity:** `release-capcom`, `release-engineer`
