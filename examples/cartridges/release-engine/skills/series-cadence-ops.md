# series-cadence-ops

Run the per-release cadence loop inside an active series. Picks the next item off the series catalog (next degree, next mission, next chapter), drives it through the plan → build → ship → retro loop, carries the prior release's retrospective forward so the next one can improve on it, and honors the "one release at a time, sequential, no batching" rule.

**Triggers:** `next release`, `next degree`, `next mission`, `advance cadence`, `series loop`, `one at a time`, `sequential release`

**Affinity:** `release-capcom`, `release-engineer`
