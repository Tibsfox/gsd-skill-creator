# series-definition

Define a new release series. Covers the three canonical shapes — daily-cadence degree series (Seattle 360: one release per degree for 360 degrees), mission-catalog series (NASA: 720 missions tracked in a CSV-driven catalog), and book-as-release series (Thicc Splines Save Lives / Blender book: chapter flow published as a single shipped version with a research corpus). Produces a ReleaseSeries record with id, scope, cadence, arc, catalog source, and success criteria.

**Triggers:** `new release series`, `define series`, `360 series`, `nasa series`, `book series`, `series arc`, `release catalog`, `series scope`

**Affinity:** `release-capcom`, `release-roadmapper`
