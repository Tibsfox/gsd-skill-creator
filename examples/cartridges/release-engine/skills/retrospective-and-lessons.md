# retrospective-and-lessons

Run the per-release retrospective — what worked, what surprised us, what we'd change, and the carry-forward item for the next release. Writes a ReleaseRetrospective record and updates the series-level lessons-learned chain. Enforces the "retrospective + lessons learned" section requirement in release notes.

**Triggers:** `release retrospective`, `lessons learned`, `post release retro`, `what went well`, `what to change`, `carry forward`

**Affinity:** `release-capcom`, `release-retrospective-analyst`
