# release-communications

Compose and route release announcements — site posts, Discord, social, mailing list. Stores each announcement as a ReleaseCommunication record with its distribution targets so the same text can be re-sent, versioned, or retracted. Also owns deprecation notices for retired releases.

**Triggers:** `release announcement`, `announce release`, `discord post`, `release social`, `deprecation notice`, `release comms`

**Affinity:** `release-capcom`, `release-publisher`
