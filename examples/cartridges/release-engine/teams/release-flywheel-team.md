# release-flywheel-team

Drive the flywheel between releases. Retrospective-analyst reads the prior retro, computes the delta, capcom feeds it into the next cadence's planning inputs. Runs between every release in an active series, not just at series boundaries.

**Roster:** `release-capcom`, `release-retrospective-analyst`.

**Use when:** the previous release has shipped and the next is about to start
