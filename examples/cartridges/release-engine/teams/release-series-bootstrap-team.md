# release-series-bootstrap-team

Stand up a new release series from zero. Capcom routes, roadmapper picks the shape (degree / mission / book / custom) and authors the catalog, engineer verifies that versioning and branch strategy are sound. Outcome: a ReleaseSeries + a fully populated SeriesCatalog + the flywheel initialized.

**Roster:** `release-capcom`, `release-roadmapper`, `release-engineer`.

**Use when:** defining a new release series (360-style, NASA-style, book, or custom)
