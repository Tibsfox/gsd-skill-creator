# release-cycle-team

Run one full release cycle inside an active series — bump version, build artifacts, ship, verify, retro, feed the flywheel forward. This is the default team for day-to-day release work.

**Roster:** `release-capcom`, `release-engineer`, `release-publisher`, `release-verifier`, `release-retrospective-analyst`.

**Use when:** shipping the next release in an active series
