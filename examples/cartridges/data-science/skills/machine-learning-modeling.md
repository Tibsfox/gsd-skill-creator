# machine-learning-modeling

Build predictive models — supervised (classification, regression, ranking), unsupervised (clustering, dimensionality reduction), and deep learning. Covers model selection, hyperparameter tuning, regularization, and the discipline of leaving a clean test set alone until the very end.

**Triggers:** `train a model`, `classification`, `regression model`, `clustering`, `hyperparameter tuning`, `cross validation`, `deep learning`, `xgboost`

**Affinity:** `ml-modeler`
