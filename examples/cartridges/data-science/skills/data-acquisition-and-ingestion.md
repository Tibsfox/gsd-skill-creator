# data-acquisition-and-ingestion

Pull data from its source — databases, warehouses, APIs, files, event streams, web scrapes, third-party vendors. Handles auth, rate limits, schema discovery, incremental pulls, and the provenance record that links every row back to where it came from.

**Triggers:** `pull data`, `ingest`, `extract`, `api data`, `database query`, `web scrape`, `provenance`, `data source`

**Affinity:** `data-wrangler`
