# data-cleaning-and-validation

Turn a raw dataset into a trusted one. Covers missing values, outliers, type coercion, deduplication, schema enforcement, referential integrity, unit consistency, and the validation suite that runs every time the data is refreshed.

**Triggers:** `clean the data`, `missing values`, `outliers`, `deduplicate`, `schema validation`, `type coercion`, `data quality`

**Affinity:** `data-wrangler`
