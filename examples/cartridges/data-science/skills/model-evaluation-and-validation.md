# model-evaluation-and-validation

Measure whether a model is good enough to ship. Covers task-appropriate metrics (AUC, F1, RMSE, MAPE, NDCG), calibration, slice-level performance, error analysis, train / val / test discipline, and bias / fairness audits on protected subgroups.

**Triggers:** `evaluate the model`, `confusion matrix`, `auc`, `rmse`, `calibration`, `slice evaluation`, `error analysis`, `fairness audit`

**Affinity:** `ml-modeler`, `statistician`
