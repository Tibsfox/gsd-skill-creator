# sd-bugfix-team

Diagnose and fix a defect. Developer reproduces and bisects, test engineer writes the failing regression test that would have caught it, tech lead approves the scope of the fix so it stays minimal.

**Roster:** `sd-developer`, `sd-test-engineer`, `sd-tech-lead`.

**Use when:** a bug report is confirmed and the team is ready to reproduce, regression-test, and fix it
