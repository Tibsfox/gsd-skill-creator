# debugging-and-diagnosis

Locate and fix defects in running code. Covers reading stack traces, minimal reproduction, print / logger / debugger discipline, bisecting regressions with git, and writing a failing test before the fix so the bug cannot return silently.

**Triggers:** `debug this`, `fix the bug`, `stack trace`, `repro steps`, `git bisect`, `regression test`, `why is this broken`

**Affinity:** `sd-developer`, `sd-tech-lead`
