# sysops-game-day-team

Run a chaos experiment or scheduled game day. Reliability-engineer designs the experiment and blast-radius review, commander owns abort criteria, runner executes the injection, observability-engineer verifies signal integrity throughout.

**Roster:** `sysops-reliability-engineer`, `sysops-commander`, `sysops-runner`, `sysops-observability-engineer`.
**Use when:** running a scheduled game day or chaos experiment in any environment.
