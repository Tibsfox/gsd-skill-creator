# capacity-and-load-management

Runtime capacity controls — autoscaling policies, rate limits, load-shedding, connection pool sizing, hot-path back-pressure. Distinct from sysadmin capacity planning: this owns live traffic controls, not host rightsizing. Triggers on `autoscale`, `rate limit`, `load shed`, `overload`, `back pressure`, `traffic surge`. Affinity: `sysops-reliability-engineer`, `sysops-runner`.
