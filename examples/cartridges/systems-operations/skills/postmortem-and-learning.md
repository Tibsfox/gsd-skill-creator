# postmortem-and-learning

Run blameless postmortems after incidents and near-misses. Reconstructs timeline, identifies contributing factors without naming scapegoats, extracts action items with owners and due dates, files the writeup into the learning library. Triggers on `postmortem`, `blameless review`, `incident retro`, `root cause`, `action items`, `lessons learned`. Affinity: `sysops-reliability-engineer`, `sysops-commander`.
