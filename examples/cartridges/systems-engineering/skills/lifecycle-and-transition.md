# lifecycle-and-transition

Own the handoffs between lifecycle stages — concept, development, production, utilization, support, retirement. Plans transition activities, defines entry and exit criteria for each stage, and keeps the technical baseline aligned with the program schedule.

**Triggers:** `lifecycle stage`, `transition plan`, `entry criteria`, `exit criteria`, `retirement`, `sustainment`, `operations handoff`

**Affinity:** `se-chief-engineer`, `se-configuration-manager`
