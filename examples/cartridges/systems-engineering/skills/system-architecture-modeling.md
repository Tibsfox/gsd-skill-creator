# system-architecture-modeling

Build and maintain logical and physical system architectures using MBSE tools and views (functional, behavioral, structural). Produces block definition, internal block, activity, state, and sequence diagrams, and keeps them consistent with the requirements baseline.

**Triggers:** `system architecture`, `mbse`, `sysml`, `block diagram`, `functional decomposition`, `physical architecture`, `logical architecture`, `architecture view`

**Affinity:** `se-chief-engineer`
