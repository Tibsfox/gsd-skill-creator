# interface-control

Define and control interfaces between subsystems, external systems, and the operating environment. Owns the Interface Control Document (ICD), interface registry, and change-impact analysis when an interface needs to move. Prevents silent breakage at integration.

**Triggers:** `interface control`, `icd`, `define interface`, `interface change`, `api contract between subsystems`, `integration point`

**Affinity:** `se-chief-engineer`, `se-configuration-manager`
