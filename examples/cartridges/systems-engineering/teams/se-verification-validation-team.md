# se-verification-validation-team

Plan and execute the V&V campaign. Verification lead owns the plan and matrix, chief engineer approves scope and tailoring, configuration manager ties each piece of evidence back to the configuration that was tested.

**Roster:** `se-verification-lead`, `se-chief-engineer`, `se-configuration-manager`.

**Use when:** entering a verification or validation phase, standing up a test campaign, or preparing an acceptance package
