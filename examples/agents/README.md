# examples/agents/

Organized into 7 category subfolders plus `deprecated/`. See [`../CATEGORIES.md`](../CATEGORIES.md) for taxonomy definitions.

## Categories

| Category | Directory | Description |
|----------|-----------|-------------|
| GSD | [`gsd/`](gsd/) | GSD workflow agents (planner, executor, verifier, etc.) |
| Research | [`research/`](research/) | Research agents (project researcher, fact checker, market researcher) |
| Media | [`media/`](media/) | Media production agents (audio engineer, video editor, podcast producer) |
| Dev | [`dev/`](dev/) | Developer-facing agents (code reviewer, test orchestrator, security reviewer) |
| Ops | [`ops/`](ops/) | Operations agents (incident analyzer, deployment validator, SLO monitor) |
| UI | [`ui/`](ui/) | UI-facing agents (UI researcher, UI checker, UI auditor) |
| Audit | [`audit/`](audit/) | Audit and analysis agents (compliance, infrastructure, pipeline, performance) |
| Deprecated | [`deprecated/`](deprecated/) | Superseded agents, preserved for reference |

Per-category README files are populated during Stage 2 classification.

See [`../CHANGELOG.md`](../CHANGELOG.md) for the history of how agents arrived here.
