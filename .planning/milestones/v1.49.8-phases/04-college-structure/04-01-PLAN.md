**`.college/college/token-counter.ts`** -- Use the same 4 chars/token heuristic as ExpressionRenderer (CHARS_PER_TOKEN = 4):

```typescript
const CHARS_PER_TOKEN = 4;

export function countTokens(content: string): number {
  if (!content) return 0;
  return Math.ceil(content.length / CHARS_PER_TOKEN);
}

export function truncateToTokenBudget(content: string, tokenBudget: number): { content: string; truncated: boolean; tokenCost: number } {
  const totalTokens = countTokens(content);
  if (totalTokens <= tokenBudget) {
    return { content, truncated: false, tokenCost: totalTokens };
  }
  const suffix = '\n\n... [load more with loadDeep()]';
  const suffixTokens = countTokens(suffix);
  const availableChars = (tokenBudget - suffixTokens) * CHARS_PER_TOKEN;
  const truncated = content.slice(0, Math.max(0, availableChars)) + suffix;
  return { content: truncated, truncated: true, tokenCost: countTokens(truncated) };
}
```

Implement CollegeLoader class. The loader reads from the `.college/departments/` filesystem and builds in-memory representations.

**Key design decisions:**
- Constructor takes a `basePath` parameter (defaults to `.college/departments`) for testability -- tests use a temp directory
- Department discovery uses `fs.readdirSync` to list directories under basePath -- no hardcoded department list
- DEPARTMENT.md is parsed for department metadata (name, description, wings)
- Token budgets from `TokenBudgetConfig` default to `{ summaryLimit: 3000, activeLimit: 12000, deepLimit: 50000 }`
- Uses `countTokens` and `truncateToTokenBudget` from token-counter.ts for budget enforcement

**CollegeLoader methods:**

```typescript
import { readFileSync, readdirSync, existsSync, mkdtempSync } from 'node:fs';
import { join, basename } from 'node:path';
import type { RosettaConcept, DepartmentWing, TrySession, TokenBudgetConfig } from '../rosetta-core/types.js';
import type { DepartmentSummary, WingContent, DeepReference } from './types.js';
import { countTokens, truncateToTokenBudget } from './token-counter.js';

export class DepartmentNotFoundError extends Error {
  constructor(id: string) {
    super(`Department not found: '${id}'`);
    this.name = 'DepartmentNotFoundError';
  }
}

export class WingNotFoundError extends Error {
  constructor(deptId: string, wingId: string) {
    super(`Wing '${wingId}' not found in department '${deptId}'`);
    this.name = 'WingNotFoundError';
  }
}

export class CollegeLoader {
  private basePath: string;
  private defaultBudget: TokenBudgetConfig = { summaryLimit: 3000, activeLimit: 12000, deepLimit: 50000 };

  constructor(basePath: string = '.college/departments') {
    this.basePath = basePath;
  }

  /** List all discovered department IDs from the filesystem */
  listDepartments(): string[] { ... }

  /** Load summary tier (<3K tokens) */
  async loadSummary(departmentId: string): Promise<DepartmentSummary> { ... }

  /** Load wing at active tier (<12K tokens) */
  async loadWing(departmentId: string, wingId: string): Promise<WingContent> { ... }

  /** Load deep reference material (on explicit request) */
  async loadDeep(departmentId: string, topic: string): Promise<DeepReference> { ... }
}
```

**listDepartments()** reads `basePath` directory entries, filters for directories containing DEPARTMENT.md.

**loadSummary()** reads DEPARTMENT.md, parses wing list, counts concepts per wing, builds DepartmentSummary. Enforces `summaryLimit` using `truncateToTokenBudget` on the description field if total exceeds budget.

**loadWing()** reads concept files (`.ts` files) from the wing's concept directory. Each concept file exports a `RosettaConcept`. Token cost = sum of all concept descriptions + code content. Enforces `activeLimit`.

**loadDeep()** reads from `references/` directory, looking for `{topic}.md` files. Returns the full reference content. No token truncation (deep tier is on explicit request), but reports `tokenCost`.
