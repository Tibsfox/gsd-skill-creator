<objective>
Create the ChipsetAdapter that maps Rosetta panels to chipset specialist agents/engines for domain-specific translation routing.

Purpose: When the Rosetta Core routes a concept to a specific panel (e.g., Python, Lisp), the chipset adapter determines which specialist engine in the chipset architecture should handle the request. Systems panels (Python, C++, Java) route to the context engine, heritage panels (Lisp, Pascal, etc.) route to the render engine, and so on. This bridges the Rosetta panel world with the chipset agent world.

Output: ChipsetAdapter class with declarative configuration and tests proving correct routing.
</objective>
