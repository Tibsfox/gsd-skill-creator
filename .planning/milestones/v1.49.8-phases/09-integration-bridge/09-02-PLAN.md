Create the TokenBudgetAdapter that wraps CollegeLoader to enforce the 2-5% context window ceiling across all progressive disclosure tiers.

Purpose: The CollegeLoader enforces per-tier absolute limits (3K/12K/50K tokens), but the integration bridge must also enforce a relative ceiling: no single panel load should consume more than 2-5% of the total context window. This adapter wraps the loader and rejects or truncates loads that would exceed the budget.

Output: TokenBudgetAdapter class with tests proving budget enforcement at every tier.
