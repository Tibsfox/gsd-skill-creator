ProfileSynthesizer logic (to be implemented and exported):
```typescript
export class ProfileSynthesizer {
  synthesize(deltas: CalibrationDelta[]): CalibrationProfile;
  // synthesize() derives:
  //   - domain: from deltas[0].domainModel (or 'unknown' if empty)
  //   - deltas: the full array as-is
  //   - confidenceScore: computed by scoreConfidence()
  //   - lastUpdated: max(delta.timestamp) across all deltas

  scoreConfidence(deltas: CalibrationDelta[]): number;
  // Confidence algorithm (simple, deterministic, testable):
  //   If deltas.length === 0 → return 0
  //   Base score = average of delta.confidence values
  //   Consistency bonus: for each parameter across all deltas, count
  //     "consistent" if all non-zero adjustments have the same sign
  //     consistent_params / total_params → multiplier in [0, 1]
  //   Final = clamp(base_score * (1 + consistency_multiplier * 0.5), 0, 1)
  //   Rationale: consistent feedback boosts score by up to 50%, contradictions keep base score
}
```

    - Test 6 (single consistent delta): 1 delta with confidence 0.5, consistent parameter

    - Test 7 (score always in [0,1]): even with confidence=1.0 on all deltas, score never > 1.0

  <implementation>
    After tests pass:

    1. Add ProfileSynthesizer class to engine.ts:
       - synthesize(deltas: CalibrationDelta[]): CalibrationProfile
         - domain: deltas.length > 0 ? deltas[0].domainModel : 'unknown'
         - deltas: [...deltas] (copy, preserve all)
         - confidenceScore: this.scoreConfidence(deltas)
         - lastUpdated: deltas.length > 0 ? new Date(Math.max(...deltas.map(d => d.timestamp.getTime()))) : new Date()
       - scoreConfidence(deltas: CalibrationDelta[]): number
         - If empty → return 0
         - baseScore = deltas.reduce((s, d) => s + d.confidence, 0) / deltas.length
         - Collect all parameter keys across all deltas
         - For each param: signs = deltas.filter(d => d.adjustment[param] !== undefined && d.adjustment[param] !== 0).map(d => Math.sign(d.adjustment[param]))
         - consistent = signs.length === 0 || signs.every(s => s === signs[0])
         - consistentParams = count of consistent params / total params (0 if none)
         - return Math.min(1, baseScore * (1 + consistentParams * 0.5))

    2. Extend CalibrationEngine:
       - Add private synthesizer = new ProfileSynthesizer()

         - return this.synthesizer.synthesize(deltas)
