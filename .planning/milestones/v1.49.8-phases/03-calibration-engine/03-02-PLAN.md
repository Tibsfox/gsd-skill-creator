<feature>
  <name>DeltaStore — JSON file persistence for CalibrationDelta records</name>
  <files>.college/calibration/delta-store.ts, .college/calibration/delta-store.test.ts</files>
  <behavior>
    Persistence survival test (CAL-03 core):
    - Test 1: Write a delta with store1 = new DeltaStore({ baseDir: tmpDir, userId: 'user1', domain: 'cooking' })
    - Create store2 = new DeltaStore({ baseDir: tmpDir, userId: 'user1', domain: 'cooking' })
    - store2.getHistory() returns exactly the delta written by store1 (same fields, same values)

    Date roundtrip test:
    - Test 2: Delta with timestamp = new Date('2026-03-01T12:00:00Z') → written → read back → timestamp instanceof Date AND timestamp.toISOString() === '2026-03-01T12:00:00.000Z'

    Domain isolation test:
    - Test 3: Write delta A to domain 'cooking', write delta B to domain 'mathematics', same userId
    - getHistory({ domain: 'cooking' }) returns only delta A
    - getHistory({ domain: 'mathematics' }) returns only delta B

    Ordering test:
    - Test 4: Write deltas in order [delta1, delta2, delta3] → getHistory() returns them in insertion order

    Empty history test:
    - Test 5: getHistory() on a store with no written deltas returns []

    Multi-user isolation test:
    - Test 6: Write delta for userId='alice', write delta for userId='bob' → alice's store returns only alice's delta
  </behavior>
  <implementation>
    After tests pass:

    1. Define types in delta-store.ts:
       - `DeltaStoreConfig`: { baseDir: string; userId: string; domain: string }

    2. Implement DeltaStore class:
       - Constructor(config: DeltaStoreConfig) stores config, computes storagePath = path.join(config.baseDir, config.userId, `${config.domain}.json`)
       - async save(delta: CalibrationDelta): Promise<void>
         - Read existing array from storagePath (or [] if not exists)
         - Serialize delta: { ...delta, timestamp: delta.timestamp.toISOString() }
         - Append to array and write back as formatted JSON
         - Create parent directories with fs.promises.mkdir({ recursive: true })
       - async getHistory(): Promise<CalibrationDelta[]>
         - If storagePath does not exist → return []
         - Read and JSON.parse the file
         - Deserialize: map each record, converting timestamp string back to new Date(record.timestamp)
         - Return in insertion order (array order from file)
       - async clear(): Promise<void>  (needed for test cleanup — delete the storage file if it exists)

    3. Use Node.js built-ins only: node:fs/promises, node:path, node:os
       Do NOT use any third-party persistence library.

    4. Export: DeltaStore, DeltaStoreConfig

    5. Tests use tmp directory via os.tmpdir() + crypto.randomUUID() subfolder.
       Each test gets a fresh DeltaStore instance pointing to the tmp path.
       afterEach: call store.clear() or rm -rf the temp dir via fs.promises.rm({ recursive: true }).
  </implementation>
</feature>
