## Architectural Patterns

1. **Collector pattern:** Each collector is a standalone async function returning typed data. Shared types in `collectors/types.ts`. NOT a common base class — each has unique function signatures.
2. **Graceful degradation:** Generator wraps every collector in try/catch. Pattern: "X failure never blocks dashboard generation" (7 instances).
3. **TDD discipline:** 11 test commits + 12 feat commits + 1 docs commit. Every feature has a RED test preceding the GREEN implementation.
4. **CSS custom properties:** `var(--domain-*, #fallback)` pattern — centralizes color definitions while providing hex fallbacks.
