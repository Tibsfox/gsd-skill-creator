**Setup pattern (use tmpdir isolation — same as technique-agents.test.ts):**
```typescript
beforeEach(async () => {
  brainstormDir = await mkdtemp(join(tmpdir(), 'brainstorm-e2e-'));
  sessionId = randomUUID();
  await initBrainstormSession({ brainstormDir, sessionId });
  sessionManager = new SessionManager({ brainstormDir });
  rulesEngine = new RulesEngine(DEFAULT_RULES_ENGINE_CONFIG);
  phaseController = new PhaseController(sessionManager, rulesEngine);
  bus = new SessionBus({ brainstormDir, sessionId });
});
afterEach(async () => {
  await rm(brainstormDir, { recursive: true, force: true });
});
```

**Bus integration in E2E tests:** Use `bus.publishRouted(message)` to simulate agents sending messages. Use `bus.poll('capture')` to verify Scribe-bound messages arrived. Use `bus.poll('energy')` to verify energy signals were emitted.
