**Body sections:**

2. **Deploy** - Security-first deployment: Kolla-Ansible TLS settings (`kolla_enable_tls_internal: yes`, `kolla_enable_tls_external: yes`, certificate paths), generating self-signed or CA-signed certificates, HAProxy TLS termination, internal API encryption, security group defaults, firewall rules for management network

   - TLS certificate expired (all services fail to communicate, API errors)
