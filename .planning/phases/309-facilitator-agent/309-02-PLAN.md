    --- adaptTechniqueQueue(session: SessionState, signal: TransitionSignal): TechniqueId[] ---
    Maps TransitionSignal to AdaptationSignal for PathwayRouter:
    - signal.type === 'energy_low' OR signal.type === 'saturation_detected': create AdaptationSignal { type: 'saturation', current_technique: session.active_technique ?? 'freewriting' }
    - signal.type === 'user_request': create AdaptationSignal { type: 'user_request', requested_technique: signal.recommendation as TechniqueId }
    - default: create AdaptationSignal { type: 'low_energy', completed_techniques: session.metadata.techniques_used }
    Delegates to: this.pathwayRouter.adaptTechniqueQueue(session.technique_queue, adaptSignal, session)

    Import AdaptationSignal type from ../pathways/router.js.

    --- generateGuidance(session: SessionState): FacilitatorGuidance ---
    Phase-aware guidance using encouraging language only.
    No urgency, no pressure, no "you need to", no "you should have", no "not enough".

    Guidance table by phase and energy_level:
    explore phase:
      - any energy: "Let's explore the problem space. What aspects feel most important to you?" (urgency: 'advisory')
    diverge phase:
      - energy 'high': "Great flow — keep going! All ideas are welcome here." (urgency: 'advisory')
      - energy 'medium': "What about approaching it from a completely different angle?" (urgency: 'advisory')
      - energy 'low': "Let's try a fresh angle. How about switching to a new approach?" (internal_action: 'suggest_technique_switch', urgency: 'next-pause')
      - energy 'flagging': "You've built a solid idea base. Ready to start organizing them?" (internal_action: 'suggest_phase_transition', urgency: 'next-pause')
    organize phase:
      - any: "Look for natural groupings — what themes keep appearing?" (urgency: 'advisory')
    converge phase:
      - any: "Let's find the most promising directions. What stands out to you?" (urgency: 'advisory')
    act phase:
      - any: "Time to turn ideas into actions. Which ones feel most actionable?" (urgency: 'advisory')

    If a technique is active, append technique-contextual hint:
    - 'scamper': " Remember, each lens is a different lens to look through — what could you Substitute?"
    - 'six-thinking-hats': " Stay in the current hat's mode — what does this perspective tell you?"
    - 'five-whys': " Keep asking 'why' — each answer brings us closer to the root."
    - 'brainwriting-635': " Build on the ideas above — take one and push it further."
    (Others: no additional hint appended)

    --- handleEnergySignal(level: EnergyLevel, session: SessionState): FacilitatorGuidance ---
    Maps energy level to encouraging response. NEVER pressure language.

    'high':
      message: "Great flow — keep going!"
      urgency: 'advisory'
    'medium':
      message: "What about trying a completely different angle?"
      urgency: 'advisory'
    'low':
      message: "Let's try approaching this differently. How about switching things up?"
      internal_action: 'suggest_technique_switch'
      urgency: 'next-pause'
    'flagging':
      const ideaCount = session.ideas.length
      message: `You've generated ${ideaCount} ideas — that's a solid foundation. Ready to start organizing them?`
      internal_action: 'suggest_phase_transition'
      urgency: 'next-pause'

    Validate facilitation voice: internal lint check — if message contains
    ['you need to', 'you must', 'you should have', 'not enough', 'try harder', 'more ideas']
    throw Error('Facilitation voice violation: pressure language detected in handleEnergySignal')
    This acts as a built-in safety check at development time.

    --- redirectEvaluation(agent: AgentRole, content: string): FacilitatorGuidance ---
    Gentle redirect when an agent produces evaluation during diverge.
    Always returns advisory-urgency guidance. Never judgmental.

    message: "Let's save that thought for the Converge phase — right now, the wilder the better. What other directions could we explore?"
    urgency: 'advisory'
    internal_action: undefined

    The agent and content parameters are accepted but not included in the message (privacy/non-shaming principle).
    Do NOT name the agent or quote the content back.

    --- generateSessionSummary(session: SessionState): string ---
    Pure function producing a Markdown string. Template:

    ```
    ## Session Summary

    **Problem:** {session.problem_statement}
    **Status:** {session.status} | **Phase reached:** {session.phase}
    **Duration:** {Math.round(session.timer.session_timer.elapsed_ms / 60_000)} minutes

    ### Ideas Generated
    {session.ideas.length} total ideas across {session.metadata.techniques_used.length} technique(s)

    ### Techniques Used
    {session.metadata.techniques_used.map(t => `- ${t}`).join('\n') || '- None yet'}

    ### Clusters Formed
    {session.clusters.length > 0
      ? session.clusters.map(c => `- **${c.label}** (${c.idea_ids.length} ideas)`).join('\n')
      : '- No clusters formed yet'}

    ### Top Ideas
    {session.ideas.slice(0, 5).map((idea, i) => `${i + 1}. ${idea.content}`).join('\n') || 'No ideas captured yet'}
    ```

    Use template literals. Ensure the output is valid Markdown.
