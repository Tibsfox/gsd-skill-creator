## Key Lessons from v1.21

### L1: Metaphor-to-Reality Pipeline (P-008 Strongest Instance)
The Amiga metaphor — originally a creative narrative device in educational packs — became the ACTUAL desktop paradigm. Window chrome IS Amiga chrome with close/depth/resize gadgets. The boot sequence IS chipset initialization with timing-driven state machines. Copper list IS real WebGL raster effects. This is the most dramatic instance of P-008 (Example-to-Infrastructure Pipeline) in the entire chain. The story became the architecture.

### L2: Multi-Language Test Ratio Challenge
For the first time, the test ratio dips below 1.0x (0.99x) because Rust Tauri commands require the full Tauri runtime for testing and have 0 test files. Desktop TypeScript alone is 1.18x — healthy. This reveals a structural challenge: as the project spans more languages, the test ratio becomes language-weighted. Future assessment should track per-language ratios alongside overall.

### L3: Scale Independence Final Confirmation
v1.21 (106 commits) scores 4.34. v1.19 (16 commits) scored 4.35. v1.17 (85 commits) scored 4.34. Across a 6.6x commit range (16-106), scores stay in a 0.01 band. Quality is definitively a process property, not a size property. This finding should be promoted to a system-level insight.

### L4: Dual Composition Tracks
The project now has two parallel composition pipelines:
1. **CLI-side:** Generator orchestrates ~25 modules (src/dashboard/)
2. **Desktop-side:** Shell orchestrates ~10 module groups (desktop/src/)
They connect via IPC bridge (DashboardHost renders generator HTML). This is a healthy architectural boundary — each side can evolve independently while sharing data through a narrow interface.

### L5: P-004 at Maximum Scale (Build Forward, Refine Backward)
v1.21 retroactively unifies 4 prior releases: dashboard (v1.12), terminal (v1.15), budget display (v1.19), design tokens (v1.20). The desktop shell doesn't replace these — it wraps and extends them. DashboardHost renders generator output. PaletteBridge connects user-style to existing theming. Terminal emulator wraps the terminal concept in proper PTY. This is P-004 operating at release-to-release scale, not just within a single version.
