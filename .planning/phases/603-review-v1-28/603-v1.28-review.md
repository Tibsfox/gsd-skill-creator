## Release Summary

v1.28 is the LARGEST release in chain history by lines (+116,935), spanning 19 phases (243-261) across two major subsystems:

1. **Knowledge Pack Educational Framework (phases 243-254):** Complete educational content platform — Zod type system (625 lines, 30+ schemas), content parsers (.skillmeta YAML, vision, activity, assessment, resource), pack registry with lookup/filter/tag search, module loader, dependency resolver with topological sort and cycle detection, content validator with per-file reporting, 35 domain packs (MATH-101 through LEARN-101) across 3 tiers (core_academic, applied, specialized), standards alignment docs, accessibility audits, CC-BY-SA-4.0 licensing, desktop browser panel with search/detail/skill-tree/progress-tracking/activity-suggestions, and observation layer (learning pattern detector, pathway adapter, activity scaffolder).

2. **Den Agent Orchestration (phases 255-261):** Filesystem-based agent communication framework — Zod bus types (191 lines, priority levels 0-7, 11 ISA opcodes, 12 agent IDs), ISA compact message encoder/decoder, filesystem message bus with priority routing, message pruning with count/age thresholds, bus health metrics, dispatcher agent (routing + lifecycle), coordinator (readiness + transition + escalation), relay agent, planner (trajectory management), monitor (budget tracking + alerts), configurator (topology selection + chipset readiness), executor (bus communication), verifier (4 verification gates), sentinel (recovery decision matrix), chronicler (audit trail + briefings), dashboard data model, comms-log (bus scanning + timeline), chipset YAML, and integration test suites.

## Key Findings

### 1. Test ratio crosses 0.9x target for first time (1.15x combined)
Den's 1.26x test ratio and knowledge's 1.05x combine for 1.15x — the first time the chain's test ratio exceeds the 0.9x target. This is driven by Den's disciplined pairing (24 test files for 19 impl files) and knowledge's comprehensive validator test suite (288+ cases).

### 2. P11 worsened with 9 fix commits (5.2%) — highest in recent chain
All 9 are same-session YAML fixes, but the volume indicates template validation gaps during rapid 35-pack content generation. The pack creation workflow should validate YAML schema compliance before committing.

### 3. Den's 12-layer pipeline is second-deepest in chain
The Den agent orchestration framework reaches 12 vertical layers (ISA types through chipset integration), the closest approach to v1.23's 17-layer AGC pipeline. This codifies the operational APT patterns (flight-ops, watchdog, capcom) into a formal framework.

### 4. Largest release scores near chain average
174 commits / 474 files / +116K lines scores 4.15 vs chain average 4.236. Scale independence continues to hold — release size does not predict score direction.
