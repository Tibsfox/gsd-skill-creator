**1. src/capabilities/capability-validator.ts** -- NEW file:

Create `CapabilityValidator` class:
- Constructor takes `CapabilityManifest`
- Internally builds `Set<string>` lookups: `skillNames`, `agentNames`, `teamNames` from manifest arrays for O(1) lookup
- `validateDeclarations(declarations: CapabilityRef[], source: string): ValidationResult` method:
  - For each declaration: if verb is `create`, skip (always valid)
  - For `use`, `after`, `adapt`: check if `type/name` exists in the corresponding Set
  - If not found, add a ValidationWarning with capability string, verb, source, and descriptive message
  - Return `{ valid: warnings.length === 0, warnings }`
- Export `CapabilityValidator`, `ValidationResult`, `ValidationWarning` types
