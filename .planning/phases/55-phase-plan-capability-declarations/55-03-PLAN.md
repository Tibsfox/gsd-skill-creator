- Constructor takes `CapabilityManifest` (from types.ts)

- Method: `validateDeclarations(declarations: CapabilityRef[], source: string): ValidationResult`

  - For `use`, `after`, `adapt`: check if `type/name` exists in the corresponding Set
  - If not found, add a ValidationWarning with capability string, verb, source, and descriptive message
  - Return `{ valid: warnings.length === 0, warnings }`
