Create `src/cli/commands/status-display.ts` with these exports:

**`budgetColorCode(percent: number): 'green' | 'cyan' | 'yellow' | 'red'`**
- `< 60` Ôćĺ 'green'
- `60-79.999...` Ôćĺ 'cyan'
- `80-99.999...` Ôćĺ 'yellow'
- `>= 100` Ôćĺ 'red'
Pure function, no picocolors dependency (caller applies color).

**`renderInstalledSection(result: CumulativeBudgetResult): string`**
- Header: "Installed Skills" (bold via picocolors)
- Subtitle: total count and total chars, e.g., "14 skills, 42,000 chars total"
- For each skill (sorted by size descending):
  - Percentage = `(skill.totalChars / result.installedTotal) * 100` (BC-04: denominator is total installed, NOT budget)
  - Mini bar relative to LARGEST skill: `formatProgressBar(skill.totalChars, largestSkillChars, 10)` (BC-07)
  - Format: `  [####......] skill-name  3,800 chars (24.5%)`
- If no skills: return "No skills installed." dim message
- Return multi-line string (joined with `\n`)

**`renderProjectionSection(result: CumulativeBudgetResult): string`**
- If `result.projection` is undefined: return dim message "Run with --profile to see loading projection"
- Header: "Loading Projection" (bold) with profile name
- Budget bar: `formatProgressBar(projection.loadedTotal, projection.budgetLimit)` (BC-03: progress bar uses loading budget, not installed total)
- Budget text: `"N% (loaded / limit chars)"` color-coded by `budgetColorCode` (BC-09)
- Count summary: `"N loaded, M deferred"` ÔÇö or if over budget: `"N of T skills fit"` (BC-05)
- Headroom: `projection.budgetLimit - projection.loadedTotal` ÔÇö but if negative, do NOT show negative number. Instead show: `"Budget exceeded ÔÇö N skills deferred"` (BC-06)
- If deferred skills exist, list them with `[deferred]` prefix
- If any skill has `oversized: true`, show `ÔÜá oversized` warning next to it
- Return multi-line string

**`buildStatusJson(result: CumulativeBudgetResult): object`**
- Returns a plain object with:
  ```
  {
    budget: number,
    totalInstalled: number,
    installed: { name, charCount, percentOfInstalled }[],
    projection: null | {
      profileName, budgetLimit, loadedTotal, deferredTotal,
      loaded: { name, charCount, tier, oversized }[],
      deferred: { name, charCount, tier, oversized }[]
    }
  }
  ```
- `installed` array sorted by size descending
- `percentOfInstalled` = `(charCount / totalInstalled) * 100` rounded to 1 decimal
- When `result.projection` is undefined, `projection` is `null`

Import picocolors for color application in render functions. Import `formatProgressBar` from budget-validation. Import types from loading-projection and budget-validation.
