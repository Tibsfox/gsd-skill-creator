<task type="auto">
  <name>Task 2: Create package manager abstraction library</name>
  <files>infra/scripts/lib/pkg-abstraction.sh</files>
  <action>
Create `infra/scripts/lib/pkg-abstraction.sh` -- a sourced library that provides package management operations dispatched to the correct backend.

**Detection strategy (in priority order):**
1. If `PKG_BACKEND` env var is set, use it directly (allows override)
2. If `infra/inventory/hardware-capabilities.yaml` exists, parse `package_manager:` from the distro section
3. If neither, run `discover-distro.sh` inline and capture the result
4. If all fail, attempt runtime detection: check `has_command dnf`, `has_command apt`, `has_command pacman` in order

**Public functions:**

`pkg_detect_backend()` -- Detect and set `_PKG_BACKEND` variable using the strategy above. Print the detected backend to stderr. Return 1 with informative message if no supported backend found.

`pkg_install <logical_name> [logical_name...]` -- Install one or more packages by logical name. Resolves names via pkg-names.sh, then dispatches:
- dnf: `sudo dnf install -y <names>`
- apt: `sudo apt-get install -y <names>` (runs `apt-get update` first if cache is older than 1 hour)
- pacman: `sudo pacman -S --noconfirm --needed <names>`
Return non-zero on failure with the failing package name in error message.

`pkg_remove <logical_name> [logical_name...]` -- Remove packages:
- dnf: `sudo dnf remove -y <names>`
- apt: `sudo apt-get remove -y <names>`
- pacman: `sudo pacman -R --noconfirm <names>`

`pkg_available <logical_name>` -- Check if a package is available (in repos):
- dnf: `dnf list available <name>` (exit code)
- apt: `apt-cache show <name>` (exit code)
- pacman: `pacman -Ss "^<name>$"` (exit code)
Returns 0 if available, 1 if not.

`pkg_installed <logical_name>` -- Check if a package is already installed:
- dnf: `rpm -q <name>`
- apt: `dpkg -l <name> | grep ^ii`
- pacman: `pacman -Q <name>`
Returns 0 if installed, 1 if not.

`pkg_update` -- Update package cache only (no upgrades):
- dnf: `sudo dnf makecache`
- apt: `sudo apt-get update`
- pacman: `sudo pacman -Sy`

**Error handling:**
- Unknown backend: print clear error listing supported backends (dnf, apt, pacman), return 1
- Package not found: print the logical name AND the resolved name in error message
- All functions check `_PKG_BACKEND` is set, call `pkg_detect_backend` automatically if not

**Conventions:**
- Source discovery-common.sh for has_command, warn, safe_read
- Source pkg-names.sh for resolve_pkg_name
- Use SCRIPT_DIR detection pattern from discover-distro.sh for relative sourcing
- `set -euo pipefail`
- No shebang (sourced library)
- Header comment block documenting purpose and usage example
