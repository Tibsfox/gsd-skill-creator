# gsd-skill-creator

## What This Is

An adaptive learning layer for Claude Code that creates, validates, and manages skills, agents, teams, and chipsets. Built on TypeScript + Rust + GLSL with a Tauri desktop app, mathematical foundations engine, and 158+ docs forming an educational ecosystem. Now includes the Rosetta Core translation engine (9 language panels), College Structure (knowledge as explorable code departments), and Calibration Engine (universal feedback loop) — proved with a flagship Cooking with Claude skill pack.

## Core Value

Skills are discovered from real patterns and proved against mathematical foundations — the unit circle isn't metaphor, it's architecture. The Rosetta Core IS skill-creator's identity: one concept, many expressions.

## Constraints

- **Quality:** 19,850+ tests must pass — zero regression
- **Atomicity:** Each phase leaves project buildable and testable
- **Safety:** Food safety boundaries are absolute — never overridden by calibration
- **Token budget:** Panel loading never exceeds 5% of context window
- **Calibration bounds:** No single step changes parameters by more than 20%
- **Branch:** Work on dev branch, no changes to v1.50.x branch
