\section{Command Macros}
The \gsdcmd{new-project} command starts a new GSD project. You can also
invoke it as \gsdcolon{new-project} in skill-style syntax. Common flags
include \gsdflag{workspace} and \gsdflag{dry-run}. Output files include
\gsdfile{.planning/PROJECT.md} and \gsdfile{.planning/ROADMAP.md}.
Press \gsdkbd{Enter} to confirm.
