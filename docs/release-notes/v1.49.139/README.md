## Lessons Learned
- **Time is a dimension the engine hadn't measured.** Five diurnal species would have missed the nocturnal acoustic world entirely. The owl at degree 4 forces the engine to track temporal niche alongside frequency and complexity.
- **The teacher completes the Central District graph.** Composer, performer, arranger, interpreter, educator — Standifer is the node that connects practice to transmission. Without the teacher, the tradition doesn't survive the generation.
- **Indicator species = key signature.** The Spotted Owl doesn't sing in the ecosystem's chord — it defines the KEY. Remove it and the entire harmonic structure shifts. This is the most powerful music-ecology parallel yet.
