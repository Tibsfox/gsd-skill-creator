## Lessons Learned

1. **Ceiling-raising events can occur along multiple axes simultaneously.** Degree 217 raises the callback-artist amplitude ceiling (PGMG E=5 → E=6) and the SPS acoustic-event amplitude ceiling (Grizzly roar to E=8 third-entry) simultaneously within a single degree. The engine's amplitude vocabulary should account for multi-axis ceiling-raising as a distinct pattern from single-axis ceiling-raising, because the interpretive weight of the combined event exceeds the sum of the individual events.
2. **Callback-with-amplitude-shift is a distinct structural register from floor-not-ceiling preservation.** When a callback artist returns at an amplitude different from their original appearance (PGMG E=5 → E=6), the shift is structurally meaningful rather than incidental. Future callback entries should be recorded with explicit shift-or-preservation tracking, and the engine's callback vocabulary should distinguish between floor-not-ceiling callbacks (Kinski, Helms Alee) and accumulated-context-warranted shift callbacks (PGMG) as separate first-class registers.
3. **Near-identity callback clusters are structurally distinct from strict callback-pair-identity.** Three consecutive callbacks at 103-103-102 degrees (Kinski, Helms Alee, PGMG) document near-identity rather than strict identity, and the one-degree variation is structurally meaningful. The engine's callback vocabulary should track cluster-near-identity patterns as distinct from pair-identity patterns, because the interpretive implications (callback distances are clustered rather than aligned to a specific number) differ.
