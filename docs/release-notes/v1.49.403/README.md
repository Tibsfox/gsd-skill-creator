Future acoustic-physics documentation should identify reference-event calibration profiles where they exist because the reference-event register is what distinguishes measurable documentation from impressionistic description.
4. **Geometric-constraint documentation (bounded channel, valley geometry, confinement) is a first-class structural observation rather than a secondary mechanical detail.** The Puyallup River valley's bounded-channel geometry is what allowed the Osceola-profile mass to maintain forward motion over 70 miles rather than dispersing radially, and the geometric constraint is a first-class structural observation about the acoustic event's character. Future sustained-mass-in-motion documentation should surface geometric-constraint observations explicitly because the geometry is what determines the acoustic event's sustained character rather than an incidental detail.
5. **Population-at-risk and institutional-response registers anchor physical-hazard documentation within its human-consequential context rather than treating the hazard as an abstract physical phenomenon.** The Orting population, the State Route 162 evacuation constraint, the Pierce County Emergency Management warning system, and the multi-agency coordination all anchor the lahar-hazard documentation within its human-consequential context. Future physical-hazard documentation should surface population-at-risk and institutional-response registers explicitly because the human-consequential context is a first-class structural observation about how the hazard operates within Pacific Northwest regional life.

Future cross-sync documentation of radically-different-mechanism pair-halves should operate at this level of mechanism-bridging abstraction because the cross-sync register is strengthened when the shared structural principle is pitched at the level where both pair-halves are recognizable instances.
