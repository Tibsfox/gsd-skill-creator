Future return appearances should follow the same energy-profile-matching rule rather than placing return entries at mechanically adjacent positions, because adjacency-based placement risks collapsing the return into a duplicate-adjacent repetition rather than preserving its distinct energy-profile justification.
2. **Multi-degree geophysics arcs benefit from explicit position-specific designation that distinguishes continuous-baseline, sustained-mobilized-mass, and impulsive-arrival sub-categories.** The geophysics arc 261-267 documents three structurally different acoustic sub-categories: continuous-baseline volcanic tremor (261), sustained-mobilized-mass lahar (262), and impulsive-arrival earthquake P-wave (263), with subsequent positions documenting further sub-categories. Future multi-degree acoustic-physics arcs should surface position-specific sub-category designations explicitly because generic arc-level description would collapse structurally different acoustic events into shared terms that obscure the distinctions.

Future acoustic-physics documentation should identify reference-event calibration profiles where they exist because the reference-event register is what distinguishes measurable documentation from impressionistic description.
4. **Geometric-constraint documentation (bounded channel, valley geometry, confinement) is a first-class structural observation rather than a secondary mechanical detail.** The Puyallup River valley's bounded-channel geometry is what allowed the Osceola-profile mass to maintain forward motion over 70 miles rather than dispersing radially, and the geometric constraint is a first-class structural observation about the acoustic event's character. 5. **Population-at-risk and institutional-response registers anchor physical-hazard documentation within its human-consequential context rather than treating the hazard as an abstract physical phenomenon.** The Orting population, the State Route 162 evacuation constraint, the Pierce County Emergency Management warning system, and the multi-agency coordination all anchor the lahar-hazard documentation within its human-consequential context.
