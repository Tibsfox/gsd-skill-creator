### SPS — Black-capped Chickadee: Pass 2 Refinement (4,120 words)

1. **Neural Architecture of Song Learning** — Hippocampal neurogenesis (Barnea & Nottebohm 1994), seasonal brain volume expansion, scatter-hoarding memory sharing neural substrate with vocal learning. HVC vocal production nucleus. The chickadee as model organism for avian neuroscience — degree 3 introduces the neuroscience layer to the SPS engine.

2. **The Semantic Content of D-Notes** — Deep dive into Templeton et al. 2005 (PNAS): 15 predator mounts, inverse relationship between predator size and D-note count. Information-theoretic analysis: graded alarm encodes ~2 bits vs. binary alarm's ~1 bit. Cross-species eavesdropping network: Red-breasted Nuthatch decoding chickadee alarms (Templeton & Greene 2007). First inter-degree causal link — Pacific Wren song suppression triggered by chickadee alarms.

3. **Social Hierarchy and Winter Flocking** — Linear dominance hierarchies, fee-bee as status signal, survival consequences of rank position. Vocal individual identity as "flock passport." Subordinate cache-pilfering strategy. Winter hierarchy shaping spring breeding territory allocation.

4. **Pioneer 2 Resonance** — Pioneer 2's silent failure mapped to the chickadee's specialization in detecting silent threats (stationary owls, concealed accipiters). The D-note system as biological solution to Pioneer 2's problem: converting absence of predator signal into presence of alarm signal. Four-layer communication architecture: silence (0) → complexity (1) → atmosphere (2) → meaning (3).
