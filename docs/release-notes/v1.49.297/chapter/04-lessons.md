   The Shabazz Palaces return at degree 157 is documented differently from the Shabazz Palaces debut at degree 103 — not because the project itself has changed but because the engine's cumulative structure has accumulated. The thirteen-degree Butler arc (144→103→157), the twenty-four-degree hip-hop era (134-157), the three-consecutive-E=6 plateau (155-157), and the Phasianidae quintet (153-157) all exist as structural context at the return that did not exist at the debut. Return entries therefore document both the subject and the structural weight the subject now carries, and the release-note architecture must make room for both.
