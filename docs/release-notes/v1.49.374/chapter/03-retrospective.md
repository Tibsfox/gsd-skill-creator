## Lessons Learned

1. **Repeat entries document catalog evolution rather than re-assessment.** Love Battery's return at degree 233 with an E=5→E=6 increment documents the band's catalog evolution from the 1990-1992 Sub Pop debut era to the 1993-1995 Atlas and Mercury mature era, and the increment is earned by the demonstrable heavier-and-denser character of the later material rather than by a reassessment of the debut era's reading. Future repeat entries should structure their E-increment or E-decrement readings as catalog-evolution documents rather than as re-assessments, because the evolution-document framing preserves the debut-era reading as historically accurate while accommodating the structural fact that artist catalogs develop over time.
2. **Minimum-interval heuristics for repeat entries should be established and referenced.** The twelve-degree interval between Love Battery's degree-221 debut and degree-233 return produced fresh context — long enough that the return registers as a callback rather than as an immediate repetition, long enough that the engine's documentation vocabulary had shifted in the interval. Future repeat-entry decisions should reference the Love Battery twelve-degree interval as a minimum-interval precedent, because shorter intervals risk treating the repeat entry as redundant coordinates rather than as genuine catalog-evolution documentation.
3. **Three-degree taxonomic-family core arcs operate as distinct structural blocks within broader family coverage.** The Peregrine-Prairie Falcon-Merlin core Falconidae arc (231-233) closes as a three-degree documentation block before the American Kestrel's one-degree-later degree-234 entry extends the family coverage to four species. Future taxonomic-family-arc documentation should actively distinguish between core-arc structures (three degrees) and family-arc structures (four or more degrees) when both apply, because the two structural positions carry distinct interpretive weight and the multi-scale documentation enriches the engine's family-arc vocabulary.
4. **Size-gradient-as-acoustic-descent is a first-class structural register for taxonomic-family arcs.** The three-degree core Falconidae arc's size gradient (large Peregrine → medium Prairie Falcon → small Merlin) maps onto a coherent acoustic descent (E=7 → E=6 → E=5), demonstrating that size and acoustic energy correlate within the Falconidae family in the engine's documentation. Future taxonomic-family-arc documentation should surface size-gradient-acoustic-descent correlations explicitly when they exist, because the correlation captures a specific within-family structural coherence that family-arc documentation should preserve.
5. **Subspecies-level documentation captures structural facts that species-level-only documentation obscures.** The black Merlin (*F. c. suckleyi*) year-round Pacific-coast residence is structurally distinct from the taiga Merlin (*F. c. columbarius*) winter-visitor migration pattern, and the subspecies-level differentiation captures a Pacific-Northwest-specific population-dynamics fact that species-level-only documentation would miss. Future SPS documentation should consider subspecies-level differentiation when the PNW subspecies population differs meaningfully from the nominate subspecies or the broader species range, because subspecies-level documentation enriches the SPS geographic vocabulary beyond species-level coverage.
