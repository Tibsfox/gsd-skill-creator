## Summary

Soyuz TM-7 launched 1988-11-26 15:49:34 UTC from Baikonur Cosmodrome LC-1/5 ("Gagarin's Start") on Soyuz-U2 launch vehicle — **same launch complex from which Yuri Gagarin became the first human in space on 1961-04-12**. The flight carried EO-4 launching crew (CDR Volkov + FE Krikalev) plus visiting French cosmonaut Brigadier General Jean-Loup Chrétien for the **Aragatz mission** — Chrétien's 2nd Soviet flight, 6 years 5 months after Soyuz T-6 (1982-06-24) = substrate-novel for FRENCH-COSMONAUT-RETURN-VISIT.

**Aragatz mission (CNES-funded ~200MFF / €33M):** Chrétien performed a **5h 57m EVA on 1988-12-09 with EO-3 CDR Vladimir Titov = FIRST NON-SOVIET NON-AMERICAN SPACEWALK IN HISTORY**. Experiments included ERA (Échantillon Robotique d'Assemblage deployable structure) + AVF + ALICE + biorhythm + CIRCE pendulum. Chrétien returned to Earth aboard Soyuz TM-6 descent module with returning EO-3 crew on 1988-12-21.

**EO-3 yearlong-endurance return (1988-12-21):** Vladimir Titov + Musa Manarov landed in Soyuz TM-6 descent module having spent **365 days 22 hours 39 minutes in space** — **the first humans to spend 365+ days in space (tied)**. Substrate-anchor for **YEARLONG-ENDURANCE-COMPLETED-RETURN substrate-cohort** that would substrate-cumulative with Polyakov's eventual 437.7d single-flight record (1994-1995) and Krikalev's career 803d cumulative.

**EO-4 launching crew remained until 1989-04-27:** CDR Volkov + FE Krikalev + continuing-cosmonaut Polyakov (substrate-thread from v683) landed in Soyuz TM-7 descent module = mission duration 151d 11h 8m. Polyakov substrate-thread cumulative 240d 22h 35m by v686 close.
