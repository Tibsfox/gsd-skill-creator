## Summary

Kvant-2 (77KS designator; 11F77 No. 16201) launched 1989-11-26 13:01:41 UTC from Baikonur Cosmodrome LC-200/39 on Proton-K (8K82K) heavy-lift launch vehicle. Mass ~19,565 kg; dimensions 13.73 m × 4.35 m. **Second Mir module** (substrate-cumulative MIR-MODULAR-EXPANSION cohort obs#2 from v678 Kvant 1987-03 substrate-anchor). Docked Mir aft port 1989-12-06 12:21:38 UTC via Igla rendezvous after **10-day approach + 3 docking attempts** (first 1989-12-02 aborted gyroscope failure; second 1989-12-04 aborted gyroscope; third 1989-12-06 manual cosmonaut intervention success).

**FIRST OPERATIONAL TKS-DERIVED FGB MODULE** — substrate-distinct from Kvant 1987-03 which used FGB only for transit + jettisoned at dock; Kvant-2 retains FGB as permanent service module (auxiliary attitude control + propulsion + electrical power).
