The lesson is that quadrant-opening degrees should honor the archival priority of the subject rather than defaulting to commercial-profile priority, and the precedent supports future quadrant-boundary placements (e.g., at degree 360 where the engine closes) to use the same archival-depth criterion where it applies.
2. **Cross-membership biographical documentation preserves small-scene community structure that band-by-band documentation would obscure.** The Paul Solger cross-membership fact at degrees 270 and 271 — the same musician in two bands across two engine degrees — is interpretively productive because it surfaces the small-scene community structure that reductive band-by-band documentation would lose. The lesson is that small-scene documentation should explicitly track cross-membership where the archival evidence supports it rather than reducing each band to a standalone entry. The community-structure register is a first-class element of small-scene documentation and should not be reduced to parenthetical footnote.
3. **Two-degree species portraits are available when the acoustic system genuinely warrants it.** The SRKW three-system portrait completed across degrees 270-271 validates the principle that the engine's species-entry vocabulary can extend to two-degree treatments when the acoustic system is functionally complex enough to warrant it. The lesson is that the default single-entry-per-species principle should be an empirical rather than a categorical constraint — the engine has the vocabulary to extend species portrait documentation across adjacent degrees when the subject's complexity genuinely warrants the wider treatment. Future multi-degree species treatments should follow the same warrant-by-actual-acoustic-complexity principle rather than applying the two-degree treatment uniformly to every species.
