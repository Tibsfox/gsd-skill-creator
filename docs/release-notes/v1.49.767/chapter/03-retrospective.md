When program substrate approaches a natural rotation boundary (e.g., substrate-form-exhaustion within current axis), anticipation file should list both INTRA-AXIS + ROTATION candidate sets.

State behavioral rules positively per Lesson #10406; trust the brief to carry topic-specific framing per Lesson #10407 candidate.
