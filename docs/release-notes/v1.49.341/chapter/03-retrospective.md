## Lessons Learned

Future centennial markers (degree 300 ahead, degree 0 for cyclic revisits) should consider midpoint-calibration as the default pairing strategy unless the substantive content requires amplitude extremity, and the engine's structural vocabulary should accumulate centennial-marker-calibration examples as first-class entries.

2. **Multi-entry practice for key Pacific Northwest acts provides mid-career and late-career contextualization that single-entry documentation cannot supply.** Death Cab for Cutie's return at degree 200, after the band's prior earlier-degree entry, demonstrates that multi-entry coverage supports deep engine documentation of acts whose creative arcs extend across decades. Future degrees should consider multi-entry appearance for artists whose engine-documentation weight merits mid-career or late-career contextualization, and the practice should apply especially at structurally significant markers where the returning-act's presence reinforces the marker's positional significance.

Future apex-predator documentation (Cougar, Gray Wolf, Grizzly Bear as geographic distribution permits) should default to multi-degree arcs, and the engine's long-term species-documentation strategy should anticipate arc-structure for any species whose complete register requires functionally-distinct multiple vocalizations.
