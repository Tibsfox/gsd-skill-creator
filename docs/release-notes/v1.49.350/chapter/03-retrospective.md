## Lessons Learned

1. **Formation-stage expression is a first-class register distinct from mature-stage expression across both creative and biological domains.** The Campfire OK debut and the Coyote pup yipping together establish formation-stage documentation as carrying distinct interpretive value that mature-stage documentation cannot substitute for, across radically different domains. Future engine degrees should treat formation-stage creative works (debut LPs, formation-period films, early-career works) and formation-stage biological vocalizations (juvenile calls, developmental-onset vocalizations, imprinting-period vocalizations) as first-class documentation subjects rather than as incomplete precursors to later mature forms, and the formation-stage-as-distinct-register cross-axis should be treated as a recurring documentation strategy rather than an ad-hoc observation at degree 209.

2. **Simplicity-as-functional-feature differs structurally from simplicity-as-limitation and merits explicit documentation distinction.** The Campfire OK participation-over-performance arrangement simplicity and the Coyote pup yip's staccato acoustic simplicity both function as deliberate features generating specific aesthetic and functional effects rather than as deficiencies to be overcome. Future engine documentation across musical, biological, and other domains should explicitly distinguish intentional-simplicity (deliberate functional feature) from deficient-complexity (limitation to be overcome), treating the distinction as a first-class analytical frame rather than an implicit interpretive assumption.

3. **Life-cycle acoustic arcs continue as a repeating Q3 pattern across four species sequences now.** Three adult-then-juvenile species pairs plus one species trio in Q3 (Black-tailed Deer 195-196, Black Bear 200-202, Mountain Goat 203-204, Coyote 208-209) establish the life-cycle-pattern as deeply entrenched in the Q3 documentation strategy. The pattern's continued repetition generates an emergent quadrant-level structural observation: ontogenetic acoustic variation is a primary Q3 SPS register category. Future Q4 and subsequent quadrant documentation should consider whether the life-cycle-arc strategy continues as an ongoing documentation approach or whether a different structural approach should be selected as a quadrant-specific strategy to avoid repetition-without-differentiation.

4. **Combined-amplitude mirrors across non-adjacent degrees are a second-order structural observation worth tracking explicitly.** The E=9 combined-amplitude value at degrees 204 and 209 (via different weightings of Part A and Part B amplitudes) surfaces a between-degree amplitude-composition pattern that neither degree's within-degree documentation captures alone. Future engine documentation should explicitly track combined-amplitude values across all degrees as a first-class ledger dimension, and combined-amplitude coincidences should be treated as structural observations warranting cross-sync documentation rather than as accidental numerical alignment.

5. **Multi-branch regional aesthetic clusters reward spectrum-completion documentation work.** The Pacific Northwest indie-folk revival has now been documented across seven distinct aesthetic branches (Sera Cahoone, Kimya Dawson, Mirah, Grand Archives, Brandi Carlile, Fleet Foxes, Campfire OK), and the seven-branch spectrum constitutes a complete-enough aesthetic-branch documentation that the broader tradition's internal diversity becomes legible. Future engine documentation of regional creative traditions should consider whether similar spectrum-completion work is available for other multi-branch clusters, and the spectrum-completion pattern should be treated as a deliberate documentation strategy rather than an accidental accumulation.

6. **Acoustic-environment registers capture group-dynamics that individual-source amplitude ratings cannot.** The Coyote pup litter-chorus acoustic environment at degree 209 adds a group-juvenile-vocalization register to the engine's acoustic-environment vocabulary distinct from single-source juvenile or adult-group environments. Future SPS documentation should continue distinguishing acoustic-environment registers by group-structure characteristics (single-source, multi-source-juvenile, multi-source-adult, mixed-age-group) as first-class documentation dimensions alongside absolute amplitude and habitat type.

7. **Triple-function communication is structurally richer than single-function communication and merits register-level recognition.** The Coyote pup yip's simultaneous contact-signaling, sibling-coordination, and neural-developmental-rehearsal functions demonstrate that a single vocalization can serve multiple communicative functions simultaneously, and the triple-function structure is structurally richer than single-function vocalizations. Future SPS documentation should explicitly track the function-count of documented vocalizations as a first-class ledger dimension, and vocalizations with multiple simultaneous functions should be treated as particularly rich documentation subjects.

8. **Participation-over-performance aesthetic stances are first-class music-ethics register additions to the engine's vocabulary.** Campfire OK's participation-over-performance ethic enters the engine's music-ethics vocabulary as an explicit alternative to the performance-over-participation default that most venue-circuit-dependent musical acts assume. The ethic's documentation establishes a first-class register category for evaluating how musical acts relate to their listeners (as participants, as audience, as co-creators, as observers), and future engine documentation should consider whether similar participation-over-performance ethics are documented or documentable across other musical genres and regional traditions.

9. **Independent-label release paths structurally parallel aesthetic choices in ways that deserve explicit documentation.** Campfire OK's Spark & Shine Records independent-label release of *Strange Like We Are* (2013) parallels the band's participation-over-performance aesthetic at the label-relationship level, with the independent-label choice reflecting deliberate artistic and aesthetic control over commercial-reach optimization. Future engine documentation of musical acts should consider whether label-relationship and release-path choices are treated as structural aesthetic choices warranting documentation rather than as merely commercial context, and the label-aesthetic-parallelism pattern should be treated as a deliberate documentation strategy rather than an incidental observation.

10. **Vocal-learning ontogenetic windows align with broader developmental transitions and should be documented as milestone-indicator vocalizations.** The three-week Coyote pup vocal onset aligns with broader developmental transitions (eye opening, coordinated movement onset, initial den emergence) and thus functions as a developmental-milestone-indicator vocalization beyond its immediate communicative functions. Future SPS documentation of species with documented vocal-learning windows should explicitly track the developmental-milestone-indicator function alongside the immediate communicative functions, treating vocalization-as-milestone-marker as a first-class function category in the engine's physiology-to-behavior vocabulary.
