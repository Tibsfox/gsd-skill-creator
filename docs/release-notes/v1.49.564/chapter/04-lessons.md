# Chapter 04 — Lessons Learned (v1.49.564 Degree 59)

Eight lessons from the first hard-gated forward degree.

1. **First-hard-gated-forward-degree operational discipline differs from retro-pass walker cadence.** Sequential single-mission focus beats parallel subagent fleets under hard gates. v1.60+ should follow the v1.59 single-context build template.

2. **Inline governance bump is the correct v1.4-era pattern.** The W5 wave's CHAIN-CONVENTIONS extensions landed inline at v1.59 ship rather than deferred post-hoc. v1.59 declarations reference stable governance vocabulary.

3. **Multi-dimension first-instance declarations benefit from an explicit format.** v1.59 carries five first-instance declarations (marine species, marine mammal, cetacean, critically-endangered, Anacortes/non-Seattle WA). A canonical format for packaging multiple first-instances into a single header is a v1.5 CHAIN-CONVENTIONS candidate — the MULTI-DIMENSION-FIRST-INSTANCE sub-form.

4. **CHANNEL-PARALLELISM is the right alignment for multi-entity topologies.** The three-parallel-entity topology of v1.59 (Surveyor 4 + Mount Eerie + SRKW) maps cleanly to CHANNEL-PARALLELISM. Future multi-entity missions (Mariner 6+7, Apollo CSM+LM, Viking 1+2) should plan for CHANNEL-PARALLELISM with v1.58 and v1.59 as founding exemplars.

5. **Cultural-transmission axis is now beyond single-exemplar.** Two species, two taxa, two mechanisms, one shared axis. Future candidates where social learning is the primary mode (humpback whale song culture, chimpanzee tool use, parrot vocalizations) can cite v1.58 + v1.59 as the axis founders.

6. **Sourced-fact discipline under hard gates requires primary-source verification budget.** Several `confidence:to-verify` flags remain in v1.59 documentation. Future ships should budget primary-source resolution time for each flagged claim.

7. **Forest-module-api amendment-5 SOCIAL-LEARNING-DRIVEN DSP class gained second exemplar.** v1.58 Bewick's Wren dialect-shift knob + v1.59 SRKW matriline-drift knob match the authoring pattern. Third exemplar or v1.60+ scheduled review triggers harness v1.1.0 minor bump.

8. **Surveyor-program engineering-lineage framing validated the thematic motif.** The "engineering lineage survives the loss of the individual" theme holds at three independent scales (aerospace / personal / ecological). Structural resonance is not coincidental; mission-topic selection informed by thematic fit and the fit holds under sourced-fact examination.
