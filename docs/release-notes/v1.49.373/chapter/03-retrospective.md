2. **Two-degree family-genus arcs document meaningful within-genus variation rather than mere taxonomic checklists.** The Peregrine (231) and Prairie Falcon (232) two-degree Falconidae arc documents cosmopolitan vs. endemic, vertical stoop vs. horizontal pursuit, cliff-and-city vs. canyon-and-arid-plateau variation within a single genus, and the within-genus variation register is valuable because it demonstrates that family-level documentation can carry distinctive interpretive weight through same-genus analysis. Future family-arc documentation should actively surface within-genus variation when adjacent entries share genus, because the variation often carries interpretive weight that single-species documentation obscures.
