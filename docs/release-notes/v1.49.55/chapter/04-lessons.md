2. **The mycorrhizal network IS the Rosetta Stone.**
   One network, multiple species, bidirectional nutrient transfer, mediated by fungi that translate between incompatible root systems. This is exactly what the Rosetta Stone framework describes: a translation layer that connects domains that can't directly communicate. The biological metaphor preceded the framework by millions of years. We documented what already existed.
