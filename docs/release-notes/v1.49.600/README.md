Arrived during the largest global Martian dust storm ever recorded; mission controllers implemented a **waiting protocol** — used the time productively for first close-up Phobos imagery (1971-11-30) and Deimos (1971-12-02) rather than wasting orbital science windows on dust-obscured surfaces. Mission ended 1972-10-27 due to attitude-control gas depletion; 17-month operational lifetime; 698 orbits.
