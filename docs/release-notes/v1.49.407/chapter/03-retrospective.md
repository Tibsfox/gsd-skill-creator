## Lessons Learned

1. **Geographic-register-opening documentation for a previously uncovered zone should name the zone's distinctive characteristics explicitly rather than treating the zone as equivalent to the standard-coverage area.** The Whidbey Island register at degree 266 explicitly characterizes the island as the longest in the contiguous United States, names the Island County context, documents the two fixed-link mainland connections (Clinton-Mukilteo ferry and Deception Pass Bridge), and surfaces the Naval Air Station Whidbey Island context. The explicit distinctive-characteristics naming is interpretively productive because it prevents the new register from collapsing into generic Pacific Northwest coverage and establishes the specific cultural, military, and geographic context that makes the zone distinct. Future first-entry-for-a-new-geographic-zone documentation should follow the same explicit-characteristics pattern.
2. **Parallel-formation-network genres require three-or-more-city documentation rather than single-origin-story documentation.** The crossover thrash register at degree 266 documents the genre as developed in parallel by D.R.I. (Houston), S.O.D. (New York), and The Accused (Oak Harbor) rather than as originating in any single city. The parallel-formation-network documentation is interpretively productive because it surfaces the actual structural shape of the genre's emergence rather than a retrospectively simplified single-origin narrative, and it establishes a documentation template for other parallel-formation genres elsewhere in the engine. Future multi-city-origin genres should follow the same parallel-network documentation pattern.
3. **Catalog-identity iconography (Martha Splatterhead) deserves first-class register treatment rather than footnote treatment.** The Martha Splatterhead trilogy register at degree 266 documents the recurring horror-comic-book character as a catalog anchor at first-class structural resolution rather than as album-art trivia. The first-class iconography treatment is interpretively productive because catalog-identity iconography is a genuine first-class element of how independent bands build recognition and identity across multiple records, and the engine's documentation vocabulary is stronger when iconographic-identity registers operate at the same structural level as genre and personnel registers.
