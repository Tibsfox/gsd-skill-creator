1. **Multi-degree arcs support multiple structural patterns, not a single pattern.**
   The Roosevelt Elk trio's (197-199) social-dynamic-amplitude-arc at 7-5-7 and the Black Bear trio's (200-202) escalation-plus-inversion arc demonstrate that multi-degree SPS arcs are a family of structural patterns whose internal organization depends on the subject's underlying repertoire structure rather than defaulting to any single pattern. Future multi-degree arc documentation should analyze which structural pattern the subject's repertoire actually supports and allow the arc's interpretive framing to follow from that analysis rather than applying a single default pattern.
   _⚙ Status: `investigate` · lesson #7832_

The streak-as-ecosystem-observation pattern opens future engine iterations to consider whether other labels (Sub Pop, Up Records, K Records, Kill Rock Stars, Barsuk) warrant similar extended multi-degree documentation where the roster's community-structure supports ecosystem-level observation rather than individual-band observation.
   _⚙ Status: `investigate` · lesson #7833_

3. **Contrast-sync pairings are a distinct structural type from parallel-sync pairings.**

Degree 201's E=4-plus-E=6 contrast-sync pairing introduces a distinct pairing-type where amplitudes differ significantly but both halves express the same underlying structural principle at contrast registers. Future pairing analysis should classify each pairing as parallel-sync or contrast-sync rather than treating all pairings as a single structural type.
   _⚙ Status: `investigate` · lesson #7834_
