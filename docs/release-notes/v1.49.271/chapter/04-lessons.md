1. **Broadcast-as-community-health is a structural innovation worth documenting at institutional level.**
   Music Heals** pioneered the model of a music radio station serving an explicit mental-health-awareness function before mainstream media adopted the October Mental Health Awareness programming slot. Future radio and broadcasting entries documenting institutional innovations should treat pioneering-ahead-of-mainstream as a first-class structural dimension rather than biographical detail. The specific year of pioneering (when Music Heals was first aired) versus the year of mainstream adoption is the structural metric.
   _⚙ Status: `investigate` · lesson #6775_

3. **Eelgrass obligate / indicator species documentation requires naming the ecological co-occupants.**
   The Brant documentation gains force by naming the eelgrass-bed ecological co-occupants (juvenile Chinook salmon, Pacific herring, Dungeness crab) rather than treating the bird in isolation. Future indicator-species documentation should identify the broader foundational-species ecosystem so the conservation argument extends beyond a single species. Without the salmon/herring/crab connection, the Brant-eelgrass argument would be species-level; with it, the argument is ecosystem-level.
   _⚙ Status: `investigate` · lesson #6777_

A host with weak parasocial relationship and a host with strong parasocial relationship produce different institutional value even with the same audience size.

Future genus-level documentation should track habitat-specialization axes as first-class structural dimensions alongside body size, migratory strategy, and call character. The marine-specialist role is rare in Anatidae, and its occupation by the Brant is a structural event.

8. **Padilla Bay and similar 8,000-acre-class eelgrass reserves are named-site structural features that warrant their own persistent documentation.**

Future species whose populations depend on a small number of named sites should receive site-specific documentation with reserve designations, acreages, management authorities, and public-access infrastructure (trails, viewpoints). Landscape-level documentation is incomplete without site-level grounding.

Future indicator-species documentation (whether biological or cultural) should specify the mechanism by which the indicator tracks the underlying system state, rather than asserting indicator status in general terms.
