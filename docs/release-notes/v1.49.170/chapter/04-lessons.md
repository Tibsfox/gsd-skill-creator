1. **Name collisions are a structural feature of music history, not an accident.**
   In the 1960s, there was no internet, no centralized band-name registry, no way for a Central District R&B group to know that a Boeing engineer's swing band across town had the same name. The collision survived into the archival era because one band's documentation outlasted the other's. The engine must now verify that found data matches the search target -- not just that data was found.

2. **Three documentation states form a complete taxonomy.**
   Correct documentation (the found data matches the target), absent documentation (no data found), and misdirected documentation (found data matches a different entity). The engine encountered all three within eight degrees (27, 33, 34). Every future search should classify its results into one of these three states.
   _⚙ Status: `investigate` · lesson #1180_

3. **The dialect is the engine's first acoustic element that separates rather than connects.**
   Every previous element -- language, familiarity, the hook, the congregation -- functioned to communicate, attract, or display. The dialect functions to sort: same type joins, different type passes. This is a new acoustic category. The engine's vocabulary now includes signals that build walls as well as signals that build bridges.

4. **Cone crop as resource-defined ecosystem is genuinely new.**
   Previous ecosystems were places (old-growth, riparian, mountain conifer). Cone crop is a condition (having mature cones, wherever that occurs). The crossbill's home is the food, not the forest. This adds a necessary dimension for nomadic species.
