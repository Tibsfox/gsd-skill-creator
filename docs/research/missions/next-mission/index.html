How to Use This Package

  1. Download mission.pdf and read Stage 1 (Vision) to orient yourself to the research scope and module structure.
  2. Open mission.pdf Stage 2 (Research Reference) as the primary factual foundation for all agent work.
  3. Hand Stage 3 (Mission Package) to your GSD orchestrator or Claude Code with the instruction: "Execute this mission package."
  4. Approve HITL checkpoints at post-Wave 0, post-Wave 1, and post-Wave 2 boundaries via CAPCOM.
  5. After Wave 3 completion, the verification report maps every success criterion to its test results.

LaTeX Recompilation

To modify the .tex source and rebuild the PDF, run three XeLaTeX passes:

xelatex -interaction=nonstopmode mission.tex
xelatex -interaction=nonstopmode mission.tex
xelatex -interaction=nonstopmode mission.tex

Three passes resolve the table of contents, cross-references, LastPage counter, and hyperlink destinations. Requires XeLaTeX with DejaVu fonts (pre-installed on Ubuntu 24).
