Three-Stage Structure

Stage 1: Vision Guide

Problem statement (5 concrete problems), core concept (bloom succession), 5 research modules (PNW ecology, garden design, indoor growing, global conservation, getting-started guides), chipset configuration (Squadron, 12 roles), 12 success criteria, scope boundaries, through-line connecting to Fox Companies habitat restoration vision.

Stage 2: Research Reference

Module 1: 600 native bee species in WA, 6 families profiled, seasonal bloom calendar (28 plants, Feb-Oct), threats (habitat loss, pathogen spillover, climate), convergence zone habitat (118 species in 3 urban sites). Module 2: 40+ plant profiles with Latin names, bloom times, light/pollinator data. Bloom succession month-by-month. Bee lawn alternatives. Container gardens. Module 3: CCD breakthrough (USDA-ARS 2025), $235-577B/year pollination value, 4 grassroots programs, 3 citizen science programs, EU/US policy. All from USDA, Xerces, WSU/OSU Extension, UN FAO, IPBES.

Stage 3: Mission Package

Wave execution plan (4 waves, 2 parallel tracks). 9 deliverables. 8 components with model assignments (25% Opus / 65% Sonnet / 10% Haiku). 32 tests (5 safety-critical, 14 core, 8 integration, 5 edge). Squadron crew manifest (12 roles). Estimated: 8 context windows across 2 sessions.

How to Use

Execute with GSD

This package is ready for handoff to gsd-skill-creator. The component specs in Stage 3 define the work. Each component maps to one agent invocation. The wave plan defines execution order. To execute:

1. Read the wave plan (Stage 3)
2. Create .planning/ structure
3. /gsd:plan-phase for each component
4. /gsd:execute-phase with wave-based parallelization

Recompile LaTeX

Edit mission.tex and recompile:
xelatex -interaction=nonstopmode mission.tex
xelatex -interaction=nonstopmode mission.tex
xelatex -interaction=nonstopmode mission.tex
Three passes required for TOC, cross-references, and page numbers. Requires TeX Live with XeLaTeX.
