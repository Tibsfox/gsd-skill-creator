## The Practice: At Every Completion

When you finish a phase, mission, or major workplan:

1. **Stop.** Don't rush to the next thing.
2. **Reflect.** What happened? What surprised you? What changed how you think?
3. **Document.** Create a personal journal entry (like CENTERCAMP-PERSONAL-JOURNAL.md)
4. **Teach.** Embed learning in code comments, guides, and documents

New entries will appear here automatically.
