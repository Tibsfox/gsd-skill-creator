How to Execute This Mission

  1. Feed the PDF to gsd-skill-creator via Claude Code
  2. The orchestrator reads the chipset configuration and activates Squadron profile
  3. Wave 0 produces type schemas and source index
  4. Wave 1 runs two parallel tracks (Taxonomy+Material | Stories+Pathway)
  5. Wave 2 converges: fishing TEK needs taxonomy; integration needs all modules
  6. Wave 3 synthesizes, audits safety, builds publication, and verifies
  7. CAPCOM gate at each wave boundary for human review
