Usage

  1. Download the Mission Package PDF and review all three stages.
  2. Load the PDF into a fresh Claude Code context with: gsd new-project --from mission-package
  3. The orchestrator reads the wave execution plan and deploys agents per the crew manifest.
  4. Wave 0 builds the MCP scaffold and VRAM manager; Wave 1 builds chips in parallel; Wave 2 integrates; Wave 3 verifies.
  5. Hardware requirement: NVIDIA CUDA-capable GPU (reference: RTX 4060 Ti 8GB).

Recompilation

  1. Download the LaTeX Source file.
  2. Ensure XeLaTeX and DejaVu fonts are installed: sudo apt install texlive-xetex fonts-dejavu
  3. Compile with three passes: xelatex -interaction=nonstopmode mission.tex (×3)
