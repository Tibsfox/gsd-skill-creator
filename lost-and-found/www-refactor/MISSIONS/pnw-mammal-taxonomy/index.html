Usage Instructions

  1. Load the PDF into a fresh Claude Code context or GSD skill-creator session
  2. Companion to Wings of the Pacific Northwest (Avian Taxonomy) — shared ecoregion framework
  3. Begin with Wave 0 (Foundation) — reuse/extend avian taxonomy schema where possible
  4. Wave 1 launches three parallel tracks: carnivores+ungulates, small mammals+bats, marine mammals
  5. Waves 2–4 proceed through analysis, synthesis, and verification
  6. Human approval (CAPCOM) required at each wave boundary

Recompilation

  1. Ensure XeLaTeX and DejaVu fonts are installed
  2. Run three passes: xelatex -interaction=nonstopmode pnw-mammal-taxonomy-mission.tex (×3)
