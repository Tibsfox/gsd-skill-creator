Usage Instructions

  1. Load the PDF into a fresh Claude Code context or GSD skill-creator session
  2. The mission package is self-contained — all three stages are in one document
  3. Begin with Wave 0 (Foundation) — create shared taxonomy schema, source index, and ecoregion definitions
  4. Wave 1 launches three parallel tracks: resident taxonomy, migratory species, and ecoregion communities
  5. Waves 2–4 proceed sequentially through analysis, synthesis, and verification
  6. Human approval (CAPCOM) required at each wave boundary

Recompilation

  1. Ensure XeLaTeX and DejaVu fonts are installed
  2. Run three passes: xelatex -interaction=nonstopmode pnw-avian-taxonomy-mission.tex (×3)
  3. Three passes resolve TOC, cross-references, and page count
