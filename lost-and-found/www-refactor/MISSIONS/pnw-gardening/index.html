Usage Instructions

  1. Load the PDF into a fresh Claude Code context as mission input
  2. Point gsd-skill-creator at the document — it contains all three pipeline stages
  3. The Orchestrator (FLIGHT) reads Stage 3 to decompose into waves
  4. Wave 0 produces shared schemas; Wave 1 runs two parallel research tracks
  5. Wave 2 synthesizes cross-module connections; Wave 3 publishes and verifies
  6. CAPCOM agent requests human approval at each wave boundary

Recompiling the LaTeX

To modify and recompile, ensure XeLaTeX and DejaVu fonts are installed:

xelatex -interaction=nonstopmode pnw-gardening-mission.tex (run 3 times)
